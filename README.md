## How to Use This System

### Daily
1. **Morning** (5min): Run `plan-day` from Productivity System
2. **Execution**: Work on Top 3 ACRA-aligned tasks only
3. **Evening** (5min): Run `assess-day` to log and detect patterns

### Weekly
1. **Review** (15min): Run `view-patterns` for trend analysis
2. **Adjust**: Course-correct based on detected patterns
3. **Plan**: Set next week's strategic focus

### Monthly
1. **Aggregate**: Compile weekly reviews
2. **Analyze**: ACRA performance by category
3. **Pivot**: Adjust strategy based on data

### Quarterly
1. **Strategic Review**: Are we growing in right direction?
2. **Department Audit**: Which systems need improvement?
3. **Goal Setting**: Next quarter's ACRA targets
