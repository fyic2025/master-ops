# Red Hill Fresh - Business Context

This folder contains RHF-specific knowledge that agents need.

## Files

| File | Purpose |
|------|---------|
| `overview.md` | Business overview, positioning, history |
| `products.md` | Product categories, seasonal availability |
| `delivery-zones.md` | Geographic coverage, delivery schedules |
| `customers.md` | Customer segments, personas |
| `competitors.md` | Local competition |
| `brand-voice.md` | How RHF communicates |

## Usage

RHF agents reference these files for business-specific context.
Skills (in `.claude/skills/`) provide the HOW.
These files provide the WHAT and WHO for RHF specifically.
