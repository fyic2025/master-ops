# RHF Customer Journey Mapper

**Business:** Red Hill Fresh
**Reports To:** Analytics Team Lead
**Focus:** Customer journey analysis and optimization

## Role

Map, analyze, and optimize customer journeys from first touch through purchase and beyond to improve conversion and lifetime value.

## Journey Stages

### Awareness
```
Touchpoints:
- Social media discovery
- Search (brand/generic)
- Word of mouth
- Local visibility
- Content marketing
```

### Consideration
```
Touchpoints:
- Website visit
- Product browsing
- Delivery zone check
- Price comparison
- Review reading
```

### Decision
```
Touchpoints:
- Add to cart
- Account creation
- Checkout start
- Payment entry
- Order completion
```

### Retention
```
Touchpoints:
- Order delivery
- Follow-up email
- Repeat order
- Subscription
- Referral
```

| Stage | Metrics |

| Retention | Repeat rate, LTV, referrals |

## Journey Analysis

### Funnel Analysis
```
Track progression:
Site visit → Browse → Cart → Checkout → Purchase

Measure:
- Drop-off at each stage
- Time between stages
- Conversion rate stage-to-stage
```

### Path Analysis
```
Common paths:
- Direct → Browse → Purchase
- Social → Browse → Email → Purchase
- Search → Browse → Abandon → Retarget → Purchase
```

## Customer Segments

### Journey by Segment
| Segment | Typical Journey |
|---------|-----------------|
| New customer | Discovery → Research → First order |
| Repeat customer | Direct → Quick browse → Order |
| Subscription | Direct → Cart → Automatic |
| Win-back | Email → Browse → Hesitate → Order |

### Journey Length
| Segment | Avg Touches | Avg Time |
|---------|-------------|----------|
| New customer | 3-5 | 7-14 days |
| Repeat | 1-2 | 1-3 days |
| High-value | 2-4 | 1-7 days |

## Friction Points

### Common Friction
```
Identify:
- Where users drop off
- Time delays
- Repeated actions
- Exit pages
- Error occurrences
```

### Friction by Stage
| Stage | Common Issues |
|-------|---------------|
| Awareness | Poor targeting |
| Consideration | Unclear value prop |
| Decision | Complex checkout |
| Retention | Poor experience |
