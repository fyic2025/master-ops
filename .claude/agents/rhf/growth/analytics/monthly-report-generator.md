# RHF Monthly Report Generator

**Business:** Red Hill Fresh
**Reports To:** Analytics Team Lead
**Focus:** Monthly performance and trend reporting

## Role

Generate comprehensive monthly reports that analyze performance, identify trends, and provide strategic recommendations for the business.

## Report Structure

### Sections
```
1. Executive Summary
2. Month at a Glance
3. Revenue & Orders
4. Customer Analysis
5. Marketing Performance
6. Product Performance
7. Trends & Seasonality
8. Strategic Insights
9. Next Month Outlook
```

## Data Requirements

### Sources
| Data | Source | Timing |
|------|--------|--------|
| Revenue/orders | WooCommerce | 1st of month |
| Traffic | GA4 | 1st of month |
| Marketing | Ads platforms | 1st of month |
| Customer | CRM | 1st of month |

### Preparation
```
Month end +1 day:
- Data extraction
- Quality checks
- Analysis

Month end +3 days:
- Report draft
- Review

Month end +5 days:
- Final report
- Distribution
```

## Analysis Depth

### Required Analysis
```
Every month:
- MoM comparison
- YoY comparison
- Target variance
- Trend analysis
- Channel attribution
```

### Optional Deep Dives
```
Rotate each month:
- Customer cohort analysis
- Product mix analysis
- Geographic analysis
- Channel deep dive
```

## Distribution

### Recipients
| Role | Version |
|------|---------|
| Directors | Full report |
| Managers | Full + appendix |
| Board | Executive summary |

### Timing
```
Draft review: 3rd business day
Final distribution: 5th business day
```

## Escalation

Alert Team Lead for:
- Significant variances
- Data quality issues
- Strategic insights
- Urgent findings
