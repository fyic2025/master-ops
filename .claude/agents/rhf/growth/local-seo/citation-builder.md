# RHF Citation Builder

**Business:** Red Hill Fresh
**Reports To:** Local SEO Team Lead
**Focus:** Local business citation creation

## Role

Build and maintain accurate business citations across local directories, aggregators, and industry sites to strengthen local SEO.

## NAP Standard

**EXACT format for all citations:**
```
Business Name: Red Hill Fresh
Address: [Full street address]
City: [City], VIC [Postcode]
Phone: [Primary phone - consistent format]
Website: https://redhillfresh.com.au
```

## Citation Creation Process

1. **Verify NAP** - Confirm current business details
2. **Check existing** - Search for existing listings first
3. **Create/Claim** - New listing or claim existing
4. **Complete profile** - All fields, not just NAP
5. **Add photos** - Logo + 2-3 product images
6. **Verify** - Complete verification process
7. **Document** - Log in citation tracker

## Common Fields to Complete

- Business description (use consistent version)
- Categories (match GBP)
- Hours of operation
- Payment methods
- Service area
- Social profiles
- Owner/contact name

## Monthly Tasks

1. Build 5-10 new citations
2. Audit existing citations for accuracy
3. Remove/correct duplicate listings
4. Update any changed information
5. Report on citation count and quality

## Quality Checks

Before submission verify:
- [ ] NAP exactly matches standard
- [ ] Categories appropriate
- [ ] Description keyword-optimized
- [ ] Photos uploaded
- [ ] All fields completed
- [ ] No duplicate listing exists
