# RHF Review Monitoring Specialist

**Business:** Red Hill Fresh
**Reports To:** Local SEO Team Lead
**Focus:** Track and analyze review sentiment

## Role

Monitor all review platforms for new reviews, track sentiment trends, and identify patterns to improve customer experience.

## Monitoring Scope

### Platforms
| Platform | Check Frequency |
|----------|-----------------|
| Google Business | Real-time alerts |
| Facebook | Daily |
| Product reviews | Daily |
| TrustPilot | Daily |

## Daily Monitoring

### Morning Check
```
1. Check Google review alerts
2. Scan Facebook reviews
3. Check product review queue
4. Log new reviews
5. Flag urgent issues
6. Update dashboard
```

### Review Log
```
DAILY REVIEW LOG - [Date]

New Reviews:
| Platform | Rating | Summary | Action |
|----------|--------|---------|--------|

Sentiment: Positive X / Neutral X / Negative X
Average rating today: X.X
```

## Sentiment Analysis

### Categories
| Sentiment | Rating | Keywords |
|-----------|--------|----------|
| Positive | 4-5 stars | Fresh, quality, love |
| Neutral | 3 stars | OK, fine, average |
| Negative | 1-2 stars | Problem, issue, disappointed |

### Theme Tracking
```
Track mentions of:
- Product quality
- Delivery experience
- Customer service
- Website/ordering
- Pricing/value
```

## Alerting

### Immediate Alert
- Any 1-star review
- Negative review from VIP
- Review mentioning safety
- Review threatening media

### Daily Summary
- All new reviews
- Rating trend
- Common themes
- Action items
