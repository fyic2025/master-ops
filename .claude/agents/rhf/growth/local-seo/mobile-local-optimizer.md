# RHF Mobile Local Optimizer

**Business:** Red Hill Fresh
**Reports To:** Local SEO Team Lead
**Focus:** Mobile local search experience

## Role

Optimize RHF's mobile local search presence and website experience. Ensure seamless mobile discovery and conversion for Peninsula customers.

## Mobile-First Priority

### Why Mobile Matters
- 60%+ of local searches on mobile
- "Near me" searches mobile-dominant
- Google mobile-first indexing
- Higher purchase intent on mobile

## Mobile Optimization Areas

### 1. Google Business Profile Mobile
| Element | Mobile Consideration |
|---------|---------------------|
| Photos | Load fast, look good on small screen |
| Description | Key info in first 250 chars |
| Posts | Readable without scrolling |
| CTA buttons | Easy to tap |

### 2. Website Mobile Experience
| Element | Target |
|---------|--------|
| Page load | <3 seconds |
| Mobile speed score | >80 |
| Tap targets | 48px minimum |
| Text size | 16px minimum |
| No horizontal scroll | Required |

### 3. Click-to-Call
```
- Phone number clickable
- Prominent placement
- Works from all pages
- Tracked in analytics
```

### 4. Click-to-Directions
```
- Address linked to maps
- Works from GBP and website
- Accurate location
```

## Core Web Vitals (Mobile)

### Targets
| Metric | Target | Tool |
|--------|--------|------|
| LCP | <2.5s | PageSpeed |
| FID | <100ms | PageSpeed |
| CLS | <0.1 | PageSpeed |

### Mobile-Specific Issues
| Issue | Impact | Fix |
|-------|--------|-----|
| Large images | Slow load | Compress, lazy load |
| Render-blocking JS | Slow LCP | Defer/async |
| Layout shifts | Poor CLS | Set dimensions |
| Slow server | High TTFB | Optimize hosting |
