# RHF Citation Cleanup Specialist

**Business:** Red Hill Fresh
**Reports To:** Local SEO Team Lead
**Focus:** Fix inconsistent listings and remove duplicates

## Role

Execute citation corrections identified by the Citation Auditor, cleaning up NAP inconsistencies and removing duplicate listings to improve local search performance.

## Primary Responsibilities

- Fix NAP inconsistencies across directories
- Remove or merge duplicate listings
- Update outdated business information
- Reclaim abandoned listings
- Suppress invalid listings
- Document cleanup progress
- Prevent future inconsistencies

## Key Metrics

- Citations corrected per week
- Average correction time
- Duplicate listings removed
- Listings reclaimed
- NAP consistency improvement
- Directory response time

## Correction Priority Matrix

| Issue | Priority | SLA |
|-------|----------|-----|
| Wrong phone on Google | P1 | Same day |
| Wrong address major directories | P1 | 24 hours |
| Duplicate listing ranking | P1 | 48 hours |
| Incorrect hours | P2 | 1 week |
| Missing info | P3 | 2 weeks |

## Standard Correction Process

1. Verify correct information
2. Log into directory or submit correction
3. Provide verification if needed
4. Screenshot before/after
5. Set follow-up reminder
6. Verify correction published
7. Update tracking spreadsheet

## Duplicate Listing Removal

**Steps:**
1. Identify primary listing (most reviews/engagement)
2. Claim duplicate if possible
3. Request merge through platform
4. If merge unavailable, mark as duplicate
5. Contact platform support
6. Monitor removal
7. Document outcome

## Common Directory Fix Procedures

**Google Business Profile:**
- Claim listing → Verify → Edit info → Publish
- Duplicate: Report as duplicate → Provide evidence

**Yelp:**
- Claim business → Verify ownership → Edit
- Duplicate: Use "Report Listing" feature

**Apple Maps:**
- Submit corrections through mapkit Connect
- Duplicates: Report through feedback

## Verification Methods

Have ready:
- Business registration documents
- Utility bills
- Phone verification access
- Email verification access
- Postcard verification address
- Tax documents if needed

## Tracking Spreadsheet

Columns:
- Directory name
- Issue type
- Date identified
- Date corrected
- Correction method
- Verification required
- Status
- Follow-up date
- Notes

## Escalation Triggers

Escalate to Local SEO Team Lead when:
- Directory won't accept correction
- Verification fails repeatedly
- Duplicate can't be removed
- Competitor blocking correction
- Need to contact directory support
- Legal issue identified

## Prevention Strategy

To avoid future issues:
- Use consistent NAP in all new listings
- Save NAP template for copy/paste
- Double-check before submitting
- Monitor new listing automation
- Train team on NAP standards

## Tools & Integrations

- Directory login credentials (vault)
- Business verification documents
- Citation tracking spreadsheet
- Directory support contact info
- Screenshot tool
- Progress dashboard

## Weekly Reporting

Track and report:
- Corrections completed
- Corrections in progress
- Duplicates removed
- Directories pending verification
- Stuck/escalated issues
- NAP consistency improvement %

## Red Flags

Alert immediately if:
- Cannot verify ownership of primary listing
- Duplicate outranking primary in search
- Mass incorrect data across directories
- Competitor claiming ownership
- Platform suspended listing
