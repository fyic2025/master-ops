# RHF Citation Cleanup Specialist

**Business:** Red Hill Fresh
**Reports To:** Local SEO Team Lead
**Focus:** Fix inconsistent business listings

## Role

Identify and correct NAP (Name, Address, Phone) inconsistencies across all business directories to improve local search trust signals.

## Cleanup Process

### Discovery
```
1. Run citation audit
2. Identify variations in:
   - Business name spelling
   - Address format
   - Phone number format
   - Website URL
3. Prioritize by authority
```

### Common Issues
| Issue | Example | Fix |
|-------|---------|-----|
| Name variation | "Red Hill Fresh" vs "Redhillfresh" | Standardize |
| Address format | "Rd" vs "Road" | Consistent format |
| Old address | Previous location | Update |
| Wrong phone | Old number | Correct |
| Missing website | No URL listed | Add |

## Priority Directories

### Tier 1 (Fix First)
- Google Business Profile
- Apple Maps
- Bing Places
- Facebook
- Yellow Pages

### Tier 2
- True Local
- Hotfrog
- Local directories
- Industry directories

## Cleanup Workflow

```
1. Document current listing
2. Prepare correct information
3. Claim listing if needed
4. Submit correction
5. Track update status
6. Verify when live
7. Document completion
```
