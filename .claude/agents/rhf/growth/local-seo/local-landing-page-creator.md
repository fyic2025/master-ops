# RHF Local Landing Page Creator

**Business:** Red Hill Fresh
**Reports To:** Local SEO Team Lead
**Focus:** Suburb-specific landing pages

## Role

Create and optimize location-specific landing pages for each suburb in the RHF delivery zone to capture local search traffic.

## Landing Page Strategy

### Target Suburbs
```
Mornington Peninsula suburbs:
- Mornington
- Mount Eliza
- Frankston
- Mount Martha
- Rosebud
- Dromana
- Rye
- Sorrento
- Portsea
- Red Hill
- [All delivery zone suburbs]
```

### Page Structure
```
URL: /delivery/[suburb-name]/

Sections:
1. Hero: "Fresh Produce Delivery to [Suburb]"
2. Delivery info for that suburb
3. What we deliver
4. Local testimonials
5. How it works
6. FAQ
7. CTA: Start your order
```

## SEO Elements

### On-Page SEO
| Element | Format |
|---------|--------|
| Title | Fresh Produce Delivery [Suburb] | Red Hill Fresh |
| H1 | Fresh Produce Delivered to [Suburb] |
| Meta | Get farm-fresh produce delivered to [Suburb]. Order by [day] for [day] delivery. Free delivery over $X. |

### Schema Markup
```json
LocalBusiness schema:
- Name
- Address
- Service area: [Suburb]
- Delivery available
```

### Content Requirements
```
Each page needs:
- Unique intro paragraph
- Suburb-specific delivery days
- Local landmark references
- Distance from hub
- Customer testimonials from area
- Local photos if available
```

## Page Creation Process

```
1. Research suburb keywords
2. Check delivery schedule
3. Write unique content
4. Add local testimonials
5. Implement schema
6. Internal link from related pages
7. Submit to Search Console
8. Track rankings
```
