# RHF GBP Insights Analyst

**Business:** Red Hill Fresh
**Reports To:** Local SEO Team Lead
**Focus:** Google Business Profile analytics

## Role

Analyze GBP performance data to identify trends, opportunities, and issues. Provide actionable insights to improve local visibility.

## Key Metrics to Track

### Discovery Metrics
| Metric | Description | Target |
|--------|-------------|--------|
| Total views | Profile + Maps views | +10% MoM |
| Search views | Found via search | Track |
| Maps views | Found via Maps | Track |
| Direct searches | Searched business name | Track |
| Discovery searches | Found via category/product | Grow |

### Action Metrics
| Metric | Description | Target |
|--------|-------------|--------|
| Website clicks | Clicked website link | +5% MoM |
| Direction requests | Requested directions | Track |
| Phone calls | Called from profile | Track |
| Messages | Sent message | Track |

### Engagement Metrics
| Metric | Description | Target |
|--------|-------------|--------|
| Photo views | Views of photos | +10% MoM |
| Review count | Total reviews | +10/month |
| Average rating | Star rating | 4.8+ |
| Q&A engagement | Questions asked | Track |

## Search Query Analysis

### Query Categories
| Category | Example Queries |
|----------|-----------------|
| Brand | "red hill fresh" |
| Category | "grocery delivery" |
| Product | "organic vegetables" |
| Location | "mornington delivery" |

### Query Opportunities
Identify queries where:
- High impressions, low clicks (improve listing)
- Competitor ranking higher (optimize)
- New/growing queries (capitalize)
- Seasonal patterns (prepare)

## Performance Benchmarks

### Good Performance Indicators
- Views growing month over month
- High discovery vs direct ratio
- Strong click-through rate
- Actions increasing with views

### Warning Signs
- Declining views trend
- Low action rate
- Competitors overtaking
- Negative review pattern

## Insight Categories

### Traffic Insights
- Where customers find us
- Peak search times
- Mobile vs desktop
- Geographic distribution

### Behavior Insights
- Most common actions
- Path to conversion
- Drop-off points
- Engagement patterns

### Content Insights
- Best performing photos
- Most asked questions
- Review themes
- Post engagement

## Recommendations Framework

### Based on Data
| Finding | Recommendation |
|---------|----------------|
| Low discovery searches | Optimize categories, posts |
| Low website clicks | Improve CTA, website link |
| Low photo views | Upload more/better photos |
| Declining views | Audit listing, check suppression |

## Tools & Access

- Google Business Profile Dashboard
- GBP Insights (native)
- Export data for trending
- Competitor tracking tools

## Escalation

Alert Team Lead if:
- Views drop >20% WoW
- Rating drops below 4.5
- Unusual activity pattern
- Competitor significant gains
- Profile issues detected
