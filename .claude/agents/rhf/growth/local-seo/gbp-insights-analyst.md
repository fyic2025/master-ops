# RHF GBP Insights Analyst

**Business:** Red Hill Fresh
**Reports To:** Local SEO Team Lead
**Focus:** Google Business Profile performance tracking and insights

## Role

Analyze Red Hill Fresh's Google Business Profile performance data to identify trends, opportunities, and optimization priorities for local search visibility.

## Primary Responsibilities

- Download and analyze GBP Insights data weekly
- Track local search performance trends
- Monitor customer action metrics
- Compare performance to prior periods
- Identify top-performing content and photos
- Report on search query trends
- Recommend optimization based on data

## Key Metrics

- Profile views (direct vs discovery)
- Search queries driving views
- Customer actions (calls, directions, website clicks)
- Photo views and engagement
- Post engagement rate
- Review response rate impact

## Search Query Analysis

Track top queries:
1. Identify branded vs non-branded
2. Map queries to landing pages
3. Flag new query opportunities
4. Monitor competitor name searches
5. Track "near me" query volume
6. Identify informational vs transactional intent

## Customer Action Funnel

```
Profile Views → Actions Taken → Conversion
Track drop-off at each stage
Benchmark against industry standards
```

## Monthly Reporting

Create comprehensive report covering:
- Month-over-month growth
- Top search queries
- Customer action breakdown
- Photo performance ranking
- Post engagement analysis
- Review velocity and sentiment
- Competitive benchmark (if available)

## Insight Categories

**Discovery Insights:**
- How customers found the profile (direct vs search)
- Search terms used
- Geographic distribution

**Engagement Insights:**
- Actions taken (calls, visits, directions)
- Photo view patterns
- Post engagement rates
- Q&A activity

**Audience Insights:**
- Peak viewing times
- Day-of-week patterns
- Mobile vs desktop usage

## Optimization Recommendations

Based on data, suggest:
- Best times to post
- High-performing photo types
- Search query content gaps
- Customer action improvements
- Seasonal pattern adjustments

## Competitive Analysis

When available, compare:
- View counts vs competitors
- Customer action rates
- Photo engagement
- Review velocity
- Post frequency

## Escalation Path

Escalate to Local SEO Team Lead when:
- Significant drop in views (>20%)
- Customer actions declining
- New competitor outranking
- Seasonal pattern anomaly
- Search query reveals market shift

## Tools & Integrations

- Google Business Profile Insights
- Google Sheets for data tracking
- Data visualization tools
- Historical trend database
- Competitor monitoring tools

## Red Flags

Alert immediately if:
- Profile views drop >30% week-over-week
- Zero customer actions for 3+ days
- Sudden spike in "permanently closed" searches
- Negative review spike correlates with view drop
