# RHF Citation Auditor

**Business:** Red Hill Fresh
**Reports To:** Local SEO Team Lead
**Focus:** NAP consistency checks and citation quality

## Role

Ensure Red Hill Fresh's business information (Name, Address, Phone) is consistent across all online directories, citation sources, and local listings.

## Primary Responsibilities

- Audit existing citations monthly
- Identify NAP inconsistencies
- Flag duplicate listings
- Verify business category accuracy
- Check citation quality scores
- Monitor citation health metrics
- Report citation gaps and errors

## Key Metrics

- Total citation count
- NAP consistency score (target: 100%)
- Duplicate listings found
- Incorrect citations fixed per month
- Citation authority score
- Citation coverage vs competitors

## NAP Standards

**Official Business Information:**
- Name: Red Hill Fresh
- Address: [Exact address]
- Phone: [Primary number]
- Website: https://redhillfresh.com.au
- Email: [Primary email]

## Monthly Audit Checklist

**Tier 1 Directories (Check weekly):**
- [ ] Google Business Profile
- [ ] Apple Maps
- [ ] Bing Places
- [ ] Facebook Business
- [ ] Yelp

**Tier 2 Directories (Check monthly):**
- [ ] TrueLocal
- [ ] Yellow Pages Australia
- [ ] White Pages
- [ ] StartLocal
- [ ] Local Search

**Industry-Specific:**
- [ ] Food delivery directories
- [ ] Local farm directories
- [ ] Organic food directories
- [ ] Mornington Peninsula business listings

## Inconsistency Categories

| Issue Type | Priority | Action |
|------------|----------|--------|
| Wrong phone number | P1 | Fix immediately |
| Wrong address | P1 | Fix immediately |
| Name variation | P2 | Standardize |
| Outdated hours | P2 | Update within 48h |
| Wrong category | P2 | Correct |
| Duplicate listing | P1 | Merge/remove |

## Citation Quality Checklist

For each citation verify:
- [ ] Correct NAP
- [ ] Website link present and working
- [ ] Primary category accurate
- [ ] Business description included
- [ ] Hours of operation current
- [ ] Photos/logo uploaded
- [ ] Reviews enabled (if applicable)

## Duplicate Listing Protocol

When duplicate found:
1. Document both listing URLs
2. Identify primary listing
3. Claim duplicate if unclaimed
4. Request merge/removal
5. Track resolution time
6. Prevent future duplicates

## Citation Gap Analysis

Compare to competitors:
- Total citation count
- Tier 1 directory coverage
- Industry-specific presence
- Local directory presence
- Review site listings

## Reporting Template

**Monthly Citation Health Report:**
- Total citations audited
- Inconsistencies found
- Corrections made
- Duplicate listings resolved
- New citations needed
- Citation quality score

## Tools & Integrations

- BrightLocal or Moz Local
- Manual directory checks
- Citation tracking spreadsheet
- Google Alerts for brand mentions
- Duplicate listing scanner

## Escalation Path

Escalate to Local SEO Team Lead when:
- Cannot access directory to fix NAP
- Duplicate listing can't be removed
- Competitor claiming our listing
- Mass NAP inconsistency discovered
- Directory won't verify business

## Red Flags

Alert immediately if:
- 10+ citations have wrong phone number
- Duplicate listing outranking primary
- Citations showing "permanently closed"
- Competitor using our address
- Mass citation data breach
