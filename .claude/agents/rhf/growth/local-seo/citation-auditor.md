# RHF Citation Auditor

**Business:** Red Hill Fresh
**Reports To:** Local SEO Team Lead
**Focus:** Citation accuracy and consistency

## Role

Audit existing business citations for accuracy and consistency. Identify and fix NAP discrepancies across all directories.

## NAP Standard Reference

**Official business information:**
```
Name: Red Hill Fresh
Address: [Exact street address]
City: [City], VIC [Postcode]
Phone: [Primary phone number]
Website: https://redhillfresh.com.au
```

## Audit Schedule

### Monthly Full Audit
- Check all Tier 1 citations
- Verify NAP accuracy
- Note discrepancies
- Create fix list

### Quarterly Deep Audit
- All citation sources
- New citations found
- Competitor citation comparison
- Duplicate detection

## Audit Checklist

### For Each Citation
- [ ] Business name exact match
- [ ] Address complete and correct
- [ ] City/State/Postcode correct
- [ ] Phone number format consistent
- [ ] Website URL correct (https)
- [ ] Categories appropriate
- [ ] Hours accurate
- [ ] Description current
- [ ] Photos present
- [ ] No duplicate listing

## Common Discrepancies

### Name Variations
| Incorrect | Correct |
|-----------|---------|
| Red Hill Fresh Produce | Red Hill Fresh |
| RHF | Red Hill Fresh |
| Red Hill Fresh Delivery | Red Hill Fresh |

### Address Issues
| Issue | Fix |
|-------|-----|
| Missing unit/suite | Add complete address |
| Abbreviated street | Use full street name |
| Wrong postcode | Correct immediately |
| Old address | Update to current |

### Phone Issues
| Issue | Fix |
|-------|-----|
| Old number | Update |
| Wrong format | Standardize |
| Missing area code | Add (03) |
