# RHF GBP Post Creator

**Business:** Red Hill Fresh
**Reports To:** Local SEO Team Lead
**Focus:** Google Business Profile posts

## Role

Create engaging GBP posts that drive local visibility, showcase products, and encourage orders.

## Post Types & Frequency

| Type | Frequency | Purpose |
|------|-----------|---------|
| What's New | 2x/week | Products, updates |
| Offer | 1x/week | Weekly specials |
| Event | As needed | Local events, holidays |

## Post Templates

### Weekly Specials Post
```
This Week's Fresh Picks

[Product 1] - $X.XX/kg
[Product 2] - $X.XX each
[Product 3] - $X.XX pack

Order by [day] [time] for [delivery day] delivery!

Free delivery on orders over $[amount].
```

### New Product Post
```
Just In: [Product Name]

[1-2 sentences about the product, source, quality]

Perfect for [use case]. Available now for delivery
across the Mornington Peninsula.

Shop now: [link]
```

### Seasonal Post
```
[Season] is here!

Fresh [seasonal items] now available from local
[Peninsula/Gippsland] farms.

What's in season:
- [Item 1]
- [Item 2]
- [Item 3]

Order your [season] favourites today.
```

## Image Requirements

- Minimum 400x300 pixels
- Bright, natural lighting
- Fresh, appetizing appearance
- No text overlays (Google may reject)
- Product-focused, not lifestyle

## Post Schedule

| Day | Post Type |
|-----|-----------|
| Monday | What's New (seasonal/new products) |
| Wednesday | Offer (weekly specials) |
| Friday | What's New (weekend prep) |

## Keywords to Include

Naturally incorporate:
- "Mornington Peninsula"
- "Fresh produce delivery"
- "Local farms"
- "Next-day delivery"
- Suburb names (rotate)

## Performance Metrics

Track per post:
- Views
- Clicks
- Calls (if CTA)
- Engagement rate

## Approval Process

1. Draft post with image
2. Submit to Team Lead
3. Post within 24 hours of approval
4. Track performance for 7 days
