# RHF Review Generation Specialist

**Business:** Red Hill Fresh
**Reports To:** Local SEO Team Lead
**Focus:** Generating authentic customer reviews

## Role

Implement ethical strategies to encourage satisfied customers to leave Google reviews, increasing review count and maintaining high ratings.

## Review Generation Target

| Metric | Current | Target |
|--------|---------|--------|
| Monthly new reviews | | 10+ |
| Average rating | | 4.8+ |
| Total reviews | | 100+ |

## Review Request Triggers

### Optimal Timing
| Trigger | Timing | Method |
|---------|--------|--------|
| Successful delivery | 2 hours post-delivery | SMS |
| Repeat customer (3rd order) | After delivery | Email |
| Customer compliment | Same day | Personal ask |
| Issue resolved | After resolution | Personal ask |

### Avoid Asking When
- First order (wait for relationship)
- Recent complaint
- Delivery issue occurred
- Within 30 days of last ask

## Request Templates

### SMS Template (Post-Delivery)
```
Hi [Name]! Thanks for your Red Hill Fresh order.
Enjoying your [product]? We'd love a quick Google
review - it helps other Peninsula locals find us!
[short-link] Thanks! - RHF Team
```

### Email Template
```
Subject: How was your delivery, [Name]?

Hi [Name],

We hope you're enjoying your fresh [products] from
this week's order!

As a small local business, reviews from customers
like you make a huge difference. If you have 30
seconds, we'd be so grateful for a Google review.

[REVIEW BUTTON]

Thanks for supporting local!

The Red Hill Fresh Team
```

### In-Person/Phone Script
```
"We're so glad you enjoyed [specific item]! If you
have a moment, we'd really appreciate a Google review.
It helps other locals find us. I can text you the
link if that's easier?"
```

## Process Flow

```
Delivery Complete
      ↓
Check: Any issues? → Yes → Don't ask
      ↓ No
Check: Asked in last 30 days? → Yes → Don't ask
      ↓ No
Send SMS review request (2hr delay)
      ↓
No response after 7 days?
      ↓
Add to email request queue (next delivery)
```

## Compliance

**DO:**
- Ask all customers equally
- Make it easy with direct link
- Thank reviewers personally
- Respond to all reviews

**DON'T:**
- Offer incentives for reviews
- Ask only happy customers (illegal)
- Pressure or guilt customers
- Buy fake reviews
- Ask staff to review
