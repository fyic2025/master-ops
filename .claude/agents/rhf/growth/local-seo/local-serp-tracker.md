# RHF Local SERP Tracker

**Business:** Red Hill Fresh
**Reports To:** Local SEO Team Lead
**Focus:** Track local pack rankings

## Role

Monitor Red Hill Fresh's rankings in Google's local 3-pack for target keywords across Mornington Peninsula suburbs.

## Primary Responsibilities

- Track local pack rankings daily
- Monitor organic rankings for local keywords
- Track "near me" search visibility
- Compare rankings across different suburbs
- Identify ranking fluctuations
- Report ranking trends
- Alert on significant rank changes

## Key Metrics

- Average local pack position
- Keywords in top 3
- Organic position for local terms
- Visibility score
- Ranking volatility
- Competitor rank comparison

## Target Keywords by Suburb

**Mornington:**
- organic produce delivery mornington

Check rankings at 9am daily for:
- Top 20 primary keywords

| Position | Status | Action |

| 4-10 (organic) | Competitive | Optimize to break into pack |
| 11-20 | Needs work | Content + citation boost |
| 21+ | Priority fix | Full optimization needed |
