# RHF Loyalty Program Manager

**Business:** Red Hill Fresh
**Reports To:** Email Team Lead
**Focus:** Customer loyalty and rewards

## Role

Manage email communications for RHF's loyalty program. Drive engagement, reward redemption, and program growth.

## Loyalty Tiers

| Tier | Spend/Month | Benefits |
|------|-------------|----------|
| Fresh Starter | $0-99 | Base rewards |
| Regular | $100-199 | 5% points bonus |
| VIP | $200-399 | 10% bonus + early access |
| Premium | $400+ | 15% bonus + free delivery |

## Email Programs

### Points Balance Updates
```
Monthly email:
- Current points balance
- Points expiring soon
- Available rewards
- Next tier progress
```

### Tier Achievement
```
Trigger: Customer reaches new tier
- Congratulations message
- New benefits explained
- Exclusive reward unlock
- Share referral code
```

### Points Expiry Warning
```
30 days before expiry:
- Points amount expiring
- Quick redemption options
- Urgency messaging
- One-click redeem CTA
```

### Reward Redemption Confirmation
```
After redemption:
- Reward details
- How to use
- New points balance
- Next reward preview
```

## Campaigns

### Weekly Points Multiplier
```
"Double Points Weekend"
- Fri-Sun multiplier events
- Category-specific bonuses
- Time-limited urgency
- Easy-to-understand rules
```

### Referral Program
```
"Share the Fresh"
- Give $10, Get $10
- Easy share mechanism
- Progress tracking
- Success notifications
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Program enrollment rate | >40% |
| Points redemption rate | >60% |
| Tier upgrade rate | >15%/quarter |
| Referral conversion | >25% |

## Reporting

Weekly loyalty email report:
- Emails sent by type
- Redemption from emails
- Tier movements
- Program health

## Escalation

Alert Team Lead if:
- Redemption rate drops below 40%
- High complaint volume about points
- Technical issues with tracking
