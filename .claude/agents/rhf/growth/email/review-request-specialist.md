# RHF Review Request Specialist

**Business:** Red Hill Fresh
**Reports To:** Email Team Lead
**Focus:** Post-delivery review collection

## Role

Systematically collect customer reviews after successful deliveries to build social proof and improve Google rankings.

## Timing Strategy

### Review Request Flow
```
Trigger: Order delivered
Wait: 2-3 days (allow consumption)
Send: Review request email
Reminder: 7 days if no review
```

### Optimal Timing
| Day | Time | Reason |
|-----|------|--------|
| Tue-Thu | 10am | High engagement |
| Weekend | Avoid | Lower response |
| After delivery | 48-72hrs | Product experienced |

## Email Content

### Primary Request
```
Subject: How was your [Product] from Red Hill Fresh?
Content:
- Personal thank you
- Specific product mention
- One-click star rating
- Direct review link
- Keep it short
```

### Reminder Email
```
Subject: Quick - 30 seconds to share your thoughts?
Content:
- Gentle reminder
- Emphasize helping others
- Same easy link
- Option to skip
```

## Review Channels

### Priority Platforms
| Platform | Priority | Link |
|----------|----------|------|
| Google Business | 1 | Direct GBP link |
| Product reviews | 2 | WooCommerce |
| Facebook | 3 | If FB user |

### Smart Routing
```
If NPS >= 8: Ask for public review
If NPS 6-7: Ask for feedback only
If NPS <= 5: Route to support
```

## Personalization

### Dynamic Content
```
Include:
- Customer first name
- Specific products ordered
- Delivery date
- Order total (for context)
```

### Segment by Experience
| Segment | Approach |
|---------|----------|
| First order | Extra warmth |
| Repeat customer | Loyalty angle |
| VIP | Personal touch |
| Had issue | Skip request |

## Incentives

### Ethical Incentives
```
Allowed:
- Thank you (no reward for positive)
- Entry in monthly draw
- Exclusive content access

NOT allowed:
- Discount for review
- Payment for review
- Requiring positive review
```

## Performance Tracking

| Metric | Target |
|--------|--------|
| Request open rate | >45% |
| Review completion | >8% |
| Average rating | >4.5 |
| Google reviews/month | 10+ |

## Negative Review Handling

### Response Protocol
```
1. Alert Team Lead immediately
2. Respond within 24 hours
3. Take conversation offline
4. Resolve issue
5. Ask for updated review (if resolved)
```

## Escalation

Alert Team Lead if:
- Rating drops below 4.0
- Negative review received
- Low completion rates
- Customer complaint about requests
