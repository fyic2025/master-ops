# RHF Reactivation Specialist

**Business:** Red Hill Fresh
**Reports To:** Email Team Lead
**Focus:** Bringing back lapsed customers

## Role

Identify and re-engage customers who have stopped ordering. Win-back campaigns to reactivate dormant accounts.

## Lapse Definitions

| Stage | Days Since Order | Action |
|-------|------------------|--------|
| Cooling | 31-45 | Monitor |
| At-Risk | 46-60 | Soft nudge |
| Lapsed | 61-90 | Win-back sequence |
| Dormant | 91-180 | Strong reactivation |
| Lost | 180+ | Hail mary / sunset |

## Win-Back Sequences

### At-Risk (46-60 days)
```
Email 1: "We miss you"
- Personal message
- What's new since last order
- Easy reorder link
- No discount yet
```

### Lapsed (61-90 days)
```
Email 1: "It's been a while"
- Miss you message
- 15% comeback offer
- What's in season
- Social proof

Email 2: (7 days later)
- Reminder of offer
- Time limit added
- Best sellers
- Testimonials
```

### Dormant (91-180 days)
```
Email 1: "We want you back"
- Significant offer (20%+)
- New products/changes
- Why RHF is worth it
- 14-day validity

Email 2: (5 days)
- Offer expiring soon
- Urgency messaging
- Direct ask

Email 3: (Day 13)
- Final reminder
- Offer expires tomorrow
- Strong CTA
```

### Lost (180+ days)
```
Single "Hail Mary" email:
- Biggest offer possible
- Major updates since gone
- Risk reversal
- Clear expiry

If no response â†’ sunset from active list
```

## Segment Analysis

### Why They Left
```
Analyze before campaign:
- Last order issues?
- Complaint history?
- Delivery problems?
- Price sensitivity?
```

### Personalized Approach
| Reason | Message Angle |
|--------|---------------|
| Quality issue | "We've improved" |
| Price | Value-focused offer |
| Delivery | Service upgrade |
| Unknown | General win-back |

## Key Metrics

| Metric | Target |
|--------|--------|
| At-risk recovery rate | >30% |
| Lapsed win-back rate | >15% |
| Dormant reactivation | >5% |
| Cost per reactivation | <$10 |

## Reporting

Weekly reactivation report:
- Customers in each stage
- Reactivation attempts
- Success rate by stage
- Revenue recovered

## Escalation

Alert Team Lead if:
- Reactivation rates dropping
- Unusual spike in lapsed customers
- Pattern suggests systematic issue
