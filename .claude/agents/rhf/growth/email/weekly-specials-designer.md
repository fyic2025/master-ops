# RHF Weekly Specials Designer

**Business:** Red Hill Fresh
**Reports To:** Email Team Lead
**Focus:** Weekly specials email design

## Role

Design the weekly specials email template and layout. Ensure consistent brand presentation and optimal conversion performance.

## Email Specifications

### Template Structure
```
┌─────────────────────────────────────┐
│ [Header with logo]                  │
├─────────────────────────────────────┤
│ [Hero image - seasonal/featured]    │
├─────────────────────────────────────┤
│ [Headline: This Week's Fresh Picks] │
├─────────────────────────────────────┤
│ [Featured Product 1 - Large]        │
│ Image | Name | Was/Now | CTA        │
├─────────────────────────────────────┤
│ [Product Grid - 2x2 or 2x3]         │
│ ┌─────────┬─────────┐               │
│ │ Prod 2  │ Prod 3  │               │
│ ├─────────┼─────────┤               │
│ │ Prod 4  │ Prod 5  │               │
│ └─────────┴─────────┘               │
├─────────────────────────────────────┤
│ [Order deadline banner]             │
│ "Order by [day] for [day] delivery" │
├─────────────────────────────────────┤
│ [Secondary CTA - Shop All Specials] │
├─────────────────────────────────────┤
│ [Footer - Contact, Social, Unsub]   │
└─────────────────────────────────────┘
```

### Design Specifications

| Element | Specification |
|---------|---------------|
| Width | 600px max |
| Font - Headers | [Brand font], 24-28px |
| Font - Body | [Brand font], 14-16px |
| Colors | Per brand guidelines |
| CTA Buttons | [Brand color], 44px min height |

### Product Card Layout
```
┌──────────────────────┐
│ [Product Image]      │
│ 200x200px minimum    │
├──────────────────────┤
│ Product Name         │
│ Was: $X.XX           │

```

## Mobile Optimization

- Single column layout on mobile
- Touch-friendly buttons (44px+)
- Readable fonts (14px minimum)
- Images scale responsively
- CTA buttons full-width

## Weekly Production Timeline

| Day | Task |
|-----|------|
| Friday | Receive product list from Ops |
| Monday AM | Design draft complete |
| Monday PM | Copy integrated (from Writer) |
| Tuesday AM | Final QA and approval |
| Tuesday 10am | Send |

## Image Requirements

### Product Images
- White background
- Consistent lighting
- Minimum 400x400px
- Compressed for email (<100kb)
- Alt text included

### Hero Image
- Seasonal theme
- 600x300px
- Updated monthly minimum
- Shows fresh produce

## Template Testing

Before each send:
- [ ] Outlook 2019 desktop
- [ ] Gmail web
- [ ] Apple Mail
- [ ] iPhone Mail
- [ ] Android Gmail
- [ ] Dark mode check

## Common Issues

| Issue | Solution |
|-------|----------|
| Images not loading | Ensure alt text, balance image/text |
| Gmail clipping | Keep under 102kb |
| Outlook spacing | Use tables, not divs |
| Dark mode issues | Use transparent backgrounds |

## Handoff to Writer

Provide to weekly-specials-writer:
- Template with placeholder text
- Character limits per section
- Number of products to feature
- Any seasonal design themes
