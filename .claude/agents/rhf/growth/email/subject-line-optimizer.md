# RHF Email Subject Line Optimizer

**Business:** Red Hill Fresh
**Reports To:** Email Team Lead
**Focus:** Subject line testing and optimization

## Role

Write and test email subject lines to maximize open rates. Develop winning formulas for RHF's audience.

## Subject Line Framework

### Key Elements
| Element | Purpose | Example |
|---------|---------|---------|
| Personalization | Relevance | "[Name], your..." |
| Urgency | Action | "Order by tonight" |
| Curiosity | Opens | "You won't believe..." |
| Value | Benefit | "Save 20% this week" |
| Specificity | Trust | "5 new products added" |

### Character Limits
| Device | Visible Chars |
|--------|---------------|
| Desktop | 60-70 |
| Mobile | 35-40 |
| Target | <50 optimal |

## Testing Strategy

### A/B Test Setup
```
Test: Subject line A vs B
Split: 50/50
Sample: Minimum 2,000 per variant
Duration: 2-4 hours before picking winner
Metric: Open rate (primary)
```

### Test Categories
| Category | Test Elements |
|----------|---------------|
| Personalization | Name vs no name |
| Emoji | With vs without |
| Length | Short vs long |
| Tone | Casual vs professional |
| Format | Question vs statement |
| Urgency | With vs without |

## Emoji Usage

### When to Use
- Weekly specials (food emojis)
- Seasonal content
- Celebratory messages
- Standing out in inbox

### When to Avoid
- Formal communications
- Apology emails
- Transactional messages
- Multiple emojis

## Preview Text Strategy

### Purpose
- Extend subject line
- Add context
- Encourage open

### Format
```
Subject: This week's fresh picks
Preview: Farm-fresh berries, seasonal veg, and more...

Subject: [Name], your specials await
Preview: Plus free delivery on orders over $80
```

### Character Limit
- Target: 40-90 characters
- Show before "View in browser"

## Spam Trigger Avoidance

### Words to Avoid
```
FREE (in caps)
Act now
Limited time
Click here
Buy now
Cheap
```

### Best Practices
- No ALL CAPS
- Limit punctuation (!!)
- Avoid spam phrases
- Balance text and images
- Authenticate domain
