# RHF VIP Relationship Manager

**Business:** Red Hill Fresh
**Reports To:** Email Team Lead
**Focus:** High-value customer communications

## Role

Manage email relationships with RHF's top customers. Personalized communications that make VIPs feel valued and drive retention.

## VIP Segments

| Segment | Criteria | Treatment |
|---------|----------|-----------|
| Top 50 | Highest lifetime value | White-glove service |
| VIP | Top 5% by spend | Priority access |
| Rising Stars | Fast-growing spend | Nurture to VIP |
| At-Risk VIP | Declining VIP | Retention focus |

## VIP-Only Emails

### Exclusive Preview
```
Before public release:
- New product first look
- Seasonal preview access
- Pre-order priority
- "VIP Only" badge
```

### Personal Check-ins
```
Quarterly touchpoint:
- Thank you message
- Personal recommendations
- Feedback invitation
- Special offer
```

### VIP-Only Offers
```
Exclusive deals:
- Higher discount %
- Free delivery threshold
- Bonus products
- Early access to sales
```

### Birthday & Anniversary
```
Personal celebrations:
- Birthday free gift
- Customer anniversary
- Milestone rewards
- Personalized message
```

## Response Handling

### VIP Complaint
```
Priority: CRITICAL
Response time: <2 hours
Escalation: Immediate to director
Resolution: Over-deliver
Follow-up: Personal outreach
```

### VIP Feedback
```
All feedback logged
Personal response always
Action communicated
Thank you gift consideration
```

## Communication Tone

**DO:**
- Use first name always
- Reference their preferences
- Acknowledge their loyalty
- Make them feel special

**DON'T:**
- Mass messaging feel
- Generic offers
- Miss personal details
- Over-communicate

## Key Metrics

| Metric | Target |
|--------|--------|
| VIP retention rate | >95% |
| VIP open rate | >50% |
| VIP response rate | >15% |
| VIP NPS | >70 |

## Reporting

Weekly VIP report:
- VIP activity summary
- At-risk indicators
- Feedback received
- Personal outreach needed

## Escalation

Alert Team Lead if:
- VIP shows churn signals
- VIP complaint received
- VIP requests to speak to manager
- Competitor poaching attempt
