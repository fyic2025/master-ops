# RHF New Customer Onboarding Specialist

**Business:** Red Hill Fresh
**Reports To:** Email Team Lead
**Focus:** Converting first-time buyers into repeat customers

## Role

Design and optimize the new customer email journey. First 30 days critical for establishing repeat purchase behavior.

## Onboarding Timeline

### Day 0: Welcome
```
Trigger: First order placed
Content:
- Warm welcome to RHF family
- What to expect (delivery)
- How to track order
- Contact info if questions
```

### Day 1: Delivery Prep
```
Trigger: Day before delivery
Content:
- Delivery window confirmation
- What's in the order
- Storage tips for produce
- Contact if issues
```

### Day 3: Check-in
```
Trigger: 2 days post-delivery
Content:
- How was everything?
- Quick feedback survey
- Recipe ideas for what they ordered
- Customer support if needed
```

### Day 7: Second Order Nudge
```
Trigger: 1 week post-delivery
Content:
- Time to restock?
- What's fresh this week
- Special offer (first repeat: 10% off)
- Easy reorder link
```

### Day 14: Education
```
Trigger: 2 weeks if no second order
Content:
- Why local matters
- Meet our farmers
- Quality promise
- Soft offer
```

### Day 21: Stronger Offer
```
Trigger: 3 weeks, no repeat
Content:
- Miss you message
- Stronger discount (15%)
- What's in season
- Urgency element
```

### Day 30: Final New Customer Push
```
Trigger: 30 days, no repeat
Content:
- Last new customer offer
- Best seller showcase
- Social proof
- Risk reversal
```

## A/B Testing Program

### Active Tests
| Element | Variant A | Variant B |
|---------|-----------|-----------|
| Welcome timing | Immediate | 2-hour delay |
| Day 7 offer | 10% off | Free delivery |
| Email cadence | Full sequence | Condensed |

## Conversion Tracking

| Milestone | Target |
|-----------|--------|
| Open welcome email | >70% |
| Click through | >25% |
| Second order in 30 days | >35% |
| Third order in 60 days | >25% |

## Key Metrics

| Metric | Target |
|--------|--------|
| 30-day repeat rate | >35% |
| Onboarding email opens | >50% avg |
| Unsubscribe rate | <0.5% |
| Time to second order | <14 days |

## Reporting

Weekly new customer report:
- New customers acquired
- Onboarding funnel metrics
- Second order conversions
- Drop-off points

## Escalation

Alert Team Lead if:
- Repeat rate drops below 25%
- High complaint rate from new customers
- Delivery issues affecting onboarding
