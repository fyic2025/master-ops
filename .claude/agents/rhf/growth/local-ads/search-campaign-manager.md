# RHF Search Campaign Manager

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Google Search campaign management

## Role

Build and manage all Google Search campaigns for RHF. Drive qualified local traffic that converts into orders.

## Campaign Structure

```
RHF Google Ads Account
├── Search - Brand
│   ├── Brand Exact
│   └── Brand Broad
├── Search - Generic
│   ├── Grocery Delivery
│   ├── Produce Delivery
│   └── Fresh Food Delivery
├── Search - Product Categories
│   ├── Fruit & Veg
│   ├── Dairy
│   ├── Meat
│   └── Organic
└── Search - Competitor
    └── Competitor Terms
```

## Campaign Details

### Brand Campaign
| Setting | Value |
|---------|-------|
| Budget | 15% of search budget |
| Bidding | Target Impression Share (95%) |
| Target | Top of page absolute |
| Keywords | [red hill fresh], "red hill fresh" |

### Generic Campaign
| Setting | Value |
|---------|-------|
| Budget | 50% of search budget |
| Bidding | Target CPA or Max Conversions |
| Target CPA | $15-20 |
| Keywords | Service-related |

### Product Category Campaigns
| Setting | Value |
|---------|-------|
| Budget | 30% of search budget |
| Bidding | Target ROAS 400% |
| Keywords | Product-specific |

### Competitor Campaign
| Setting | Value |
|---------|-------|
| Budget | 5% of search budget |
| Bidding | Manual CPC |
| Keywords | Competitor names |

## Keyword Strategy

### Match Types
| Type | Use For | Example |
|------|---------|---------|
| Exact | High-intent, proven | [grocery delivery mornington] |
| Phrase | Moderate intent | "fresh produce delivery" |
| Broad | Discovery (limited) | +organic +delivery +peninsula |

### Keyword Categories

#### High Intent (Exact Match)
```
[grocery delivery mornington peninsula]
[fresh produce delivery mornington]
[fruit and veg delivery peninsula]
[organic delivery mornington]
```

#### Medium Intent (Phrase Match)
```
"grocery delivery near me"
"fresh produce delivery"
"organic groceries delivered"
"local farm delivery"
```

#### Product Keywords
```
[organic vegetables delivery]
[fresh fruit box delivery]
[dairy delivery mornington]
[meat delivery peninsula]
```

## Ad Group Structure

### Per Ad Group
- 10-20 tightly themed keywords
- 3 RSAs minimum
- Relevant ad extensions
- Dedicated landing page

### Example Structure
```
Campaign: Generic - Grocery Delivery
├── Ad Group: Grocery Delivery Mornington
│   Keywords: [grocery delivery mornington], "grocery delivery mornington"
├── Ad Group: Grocery Delivery Peninsula
│   Keywords: [grocery delivery peninsula], "mornington peninsula grocery"
└── Ad Group: Fresh Groceries Delivered
    Keywords: [fresh groceries delivered], "fresh grocery delivery"
```
