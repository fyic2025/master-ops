# RHF Quality Score Optimizer

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Quality Score improvement

## Role

Optimize Quality Scores across all keywords to reduce CPCs, improve ad positions, and maximize budget efficiency.

## Quality Score Components

### Three Factors
| Factor | Weight | Our Control |
|--------|--------|-------------|
| Expected CTR | ~40% | Ad copy, targeting |
| Ad Relevance | ~20% | Keywords in ads |
| Landing Page | ~40% | Page content, speed |

### Rating Scale
| Rating | Meaning |
|--------|---------|
| Above Average | Excellent |
| Average | Acceptable |
| Below Average | Needs work |

## Quality Score Targets

### By Campaign Type
| Campaign | Target QS |
|----------|-----------|
| Brand | 8-10 |
| Generic | 7-9 |
| Product | 6-8 |

### Distribution Goals
| QS Range | Target % |
|----------|----------|
| 8-10 | 40%+ |
| 6-7 | 40% |
| 1-5 | <20% |

## Improvement Strategies

### Expected CTR (Below Average)
```
Problem: Historical CTR lower than competitors

Actions:
1. Improve ad copy (more compelling)
2. Add stronger CTA
3. Test headline variations
4. Improve ad relevance
5. Use ad extensions
6. Check targeting (exclude poor audiences)
```

### Ad Relevance (Below Average)
```
Problem: Ads don't match keyword intent

Actions:
1. Include keyword in headline
2. Include keyword in description
3. Create tighter ad groups (fewer keywords)
4. Match ad message to search intent
5. Use keyword insertion (carefully)
```

### Landing Page (Below Average)
```
Problem: Page doesn't satisfy user intent

Actions:
1. Match page to keyword intent
2. Improve page load speed
3. Make content relevant to ad
4. Ensure mobile-friendly
5. Add trust signals
6. Clear CTA on page
```

## QS Audit Process

### Weekly QS Review
```
1. Export keyword report with QS
2. Sort by impressions (high to low)
3. Flag QS <6 with significant spend
4. Check component breakdown
5. Prioritize by impact
6. Create improvement plan
```

### Priority Matrix
| QS | Impressions | Priority |
|----|-------------|----------|
| <5 | High | Critical |
| <5 | Low | Low |
| 5-6 | High | High |
| 5-6 | Low | Medium |
| 7+ | Any | Maintain |

## Keyword-Level Optimization

### Low QS Action Plan
```
Keyword: [example keyword]
Current QS: 4
Impressions: 5,000/month

Component Analysis:
- Expected CTR: Below Average
- Ad Relevance: Below Average
- Landing Page: Average

Actions:
1. Add keyword to headline (Ad Relevance)
2. Test new ad copy with stronger CTA (CTR)
3. Consider new landing page
4. If no improvement in 4 weeks, pause or restructure
```

## Landing Page Optimization

### Page Requirements
| Element | Requirement |
|---------|-------------|
| Relevance | Match keyword/ad intent |
| Speed | <3 second load |
| Mobile | Fully responsive |
| Content | Useful, original |
| Navigation | Easy to use |
| Trust | Reviews, security |

### Page Speed Actions
| Issue | Fix |
|-------|-----|
| Large images | Compress, lazy load |
| Slow server | Upgrade hosting |
| Render blocking | Optimize JS/CSS |
| Too many requests | Combine, minimize |

## Ad Copy for QS

### Best Practices
```
Headline 1: Include primary keyword
Headline 2: Benefit or CTA
Headline 3: Trust signal or offer

Description 1: Expand on headline, include keyword variant
Description 2: Features, CTA
```

### Example
```
Keyword: grocery delivery mornington peninsula

Ad:
Headline 1: Grocery Delivery Mornington Peninsula
Headline 2: Fresh to Your Door Next Day
Headline 3: Free Delivery Over $80

Description 1: Farm-fresh groceries delivered across the
Peninsula. Order by 6pm for next-day delivery.
Description 2: Local farms, quality produce. Shop now!
```

- Average QS drops below 6
- High-volume keyword QS drops 3+ points

- Unable to improve QS after 4 weeks of effort
