## Competitive Response

### When to Respond
| Situation | Response |
|-----------|----------|
| Competitor outbidding | Review bid strategy |
| New competitor | Monitor closely |
| Aggressive offer | Evaluate matching |
| Ad copy innovation | Test similar |

### Response Options
```
1. Bid adjustments
2. Ad copy improvements
3. Offer matching/beating
4. Differentiation focus
5. No action (if not impacting)
```
