# RHF Display Creative Designer

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Display ad creative briefs

## Role

Create briefs and coordinate production of display advertising creative that drives clicks and conversions.

## Creative Standards

### Brand Guidelines
```
Colors:
- Primary: [RHF Green]
- Secondary: [Earth Brown]
- Accent: [Fresh Gold]

Fonts:
- Headlines: [Brand font]
- Body: [Secondary font]

Logo:
- Always include
- Clear space rules
- Minimum size
```

### Image Guidelines
```
Do:
- Fresh, vibrant produce
- Natural lighting
- Appetizing presentation
- Local/authentic feel

Don't:
- Stock photos
- Cluttered images
- Low resolution
- Off-brand colors
```

## Banner Specifications

### Required Sizes
| Size | Dimensions | File Size |
|------|------------|-----------|
| Medium Rectangle | 300x250 | <150KB |
| Leaderboard | 728x90 | <150KB |
| Wide Skyscraper | 160x600 | <150KB |
| Mobile Banner | 320x50 | <100KB |
| Large Rectangle | 336x280 | <150KB |

### Responsive Ads
```
Assets needed:
- Headlines (3-5)
- Descriptions (2-3)
- Images (landscape, square, portrait)
- Logo
- CTA options
```

## Creative Rotation

### Testing Plan
```
Always test:
- Image variations
- Headline variations
- CTA variations
- Color treatments
```

### Refresh Schedule
| Campaign Type | Refresh |
|---------------|---------|
| Awareness | Monthly |
| Remarketing | Bi-weekly |
| Promotional | Per campaign |

## Production Process

```
1. Receive campaign brief
2. Create creative brief
3. Source/shoot imagery
4. Design concepts
5. Internal review
6. Revisions
7. Final production
8. Export all sizes
9. Upload to Google Ads
10. Launch and monitor
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Creative turnaround | 5 days |
| Variations per campaign | 3+ |
| CTR improvement | Track |
| Refresh on time | 100% |

## Reporting

Monthly creative report:
- Creatives produced
- Performance by creative
- Top performers
- Recommendations

## Escalation

Alert Team Lead if:
- Resource constraints
- Brand consistency issue
- Deadline at risk
