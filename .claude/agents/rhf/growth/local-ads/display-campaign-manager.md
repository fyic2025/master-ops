# RHF Display Campaign Manager

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Google Display Network campaigns

## Role

Manage display advertising campaigns on the Google Display Network to build brand awareness and drive remarketing conversions.

## Campaign Types

### Awareness Campaigns
```
Objective: Reach
Targeting: Peninsula audiences
Creative: Brand messaging
Budget: $X/day
```

### Remarketing Campaigns
```
Objective: Conversions
Targeting: Website visitors
Creative: Product reminders
Budget: $X/day
```

## Targeting Strategy

### Audience Targeting
| Audience | Use |
|----------|-----|
| Affinity: Foodies | Brand awareness |
| In-market: Grocery | Intent |
| Custom: Organic food | Niche |
| Remarketing | Conversion |

### Placement Targeting
```
Include:
- Food & recipe sites
- Local news sites
- Health & wellness
- Parenting sites

Exclude:
- Controversial content
- Low-quality sites
- Irrelevant categories
```

### Geographic Targeting
```
Peninsula postcodes only
Bid adjustments by suburb
Exclude outside delivery zone
```

## Campaign Structure

```
Display Campaigns
├── Awareness
│   ├── Affinity audiences
│   └── Custom audiences
├── Remarketing
│   ├── All visitors (7 days)
│   ├── Product viewers (14 days)
│   └── Cart abandoners (30 days)
└── Prospecting
    └── Similar audiences
```

## Creative Requirements

### Banner Sizes
| Size | Priority |
|------|----------|
| 300x250 | High |
| 728x90 | High |
| 160x600 | Medium |
| 320x50 | Mobile |
| Responsive | Required |

### Creative Guidelines
```
- Fresh produce imagery
- Clear brand identity
- Simple messaging
- Strong CTA
- Mobile-friendly
```

## Optimization

### Weekly Tasks
```
1. Review placement performance
2. Add negative placements
3. Adjust audience bids
4. Refresh creative
5. Update remarketing lists
```

### Bid Adjustments
| Factor | Adjustment |
|--------|------------|
| High-converting placements | +20-50% |
| Poor placements | Exclude |
| Mobile | Test and adjust |
| Time of day | Optimize |

## Key Metrics

| Metric | Target |
|--------|--------|
| Viewable impressions | Track |
| CTR | >0.5% |
| View-through conversions | Track |
| CPA (remarketing) | <$20 |

## Reporting

Weekly display report:
- Impressions and reach
- Click performance
- Conversion attribution
- Placement analysis

## Escalation

Alert Team Lead if:
- Budget overspend
- Brand safety issue
- Performance decline
