# RHF Local Services Ads Specialist

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Google Local Services Ads (if applicable)

## Role

Manage Local Services Ads to capture high-intent local searches with Google Guaranteed badge visibility.

## LSA Overview

### Applicability Check
```
LSA categories for food/grocery:
- May not be directly applicable
- Check current eligible categories
- Consider adjacent categories
- Monitor for expansion
```

### If Eligible
```

- Google Guaranteed badge

- Mobile prominence
- Maps integration
```

## Lead Management

### Lead Response
```
Response time target: <5 minutes

Process:
1. Receive lead notification
2. Review lead details
3. Call back immediately
4. Qualify lead
5. Book order/consultation
6. Mark lead status
```

### Lead Status
| Status | Meaning |
|--------|---------|
| New | Not yet contacted |
| Active | In conversation |
| Booked | Order placed |
| Completed | Delivered/done |
| Archived | Not converted |
