# RHF Bid Optimizer

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Manual bid adjustments

## Role

Fine-tune bids at keyword, ad group, and campaign level to maximize performance within budget.

## Bid Optimization Areas

### Adjustment Types
| Level | Adjustments |
|-------|-------------|
| Campaign | Budget, bid strategy |
| Ad Group | Base bids |
| Keyword | Individual bids |
| Device | Mobile/desktop |
| Time | Day/hour |
| Location | Geo bids |
| Audience | RLSA overlays |

## Optimization Framework

### Bid Decision Matrix
| Performance | Volume | Action |
|-------------|--------|--------|
| Above target CPA | High | Lower bid 10-20% |
| Above target CPA | Low | Pause or monitor |
| At target CPA | Any | Maintain |
| Below target CPA | High | Increase 10-20% |
| Below target CPA | Low | Increase cautiously |

### Bid Change Limits
```
Per adjustment:
- Maximum increase: +25%
- Maximum decrease: -30%
- Frequency: Weekly max
- Wait period: 7 days between changes
```

## Daily Optimization

### Morning Check
```
1. Review yesterday's performance
2. Check for anomalies
3. Identify urgent adjustments
4. Make time-sensitive changes
```

### Key Triggers
| Trigger | Action |
|---------|--------|
| CPA 2x target | Lower bid 20% |
| CPA 0.5x target | Increase bid 15% |
| Zero impressions | Check bid vs competition |
| Spike in CPC | Investigate |

## Weekly Optimization

### Analysis Process
```
1. Export performance data
2. Calculate metrics by level
3. Rank by opportunity
4. Prioritize changes
5. Implement adjustments
6. Document changes
```
