# RHF Audience Builder

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Custom audience creation

## Role

Build and maintain custom audiences in Google Ads for targeted advertising campaigns.

## Audience Types

### First-Party Audiences
| Audience | Source | Use |
|----------|--------|-----|
| All customers | CRM upload | Exclusion, lookalike |
| VIP customers | CRM (high LTV) | Lookalike |
| Recent buyers | CRM (30 days) | Cross-sell |
| Lapsed | CRM (90+ days) | Win-back |
| Newsletter | Email list | Targeting |

### Remarketing Audiences
| Audience | Definition | Window |
|----------|------------|--------|
| All visitors | Any page | 30 days |
| Product viewers | Product pages | 14 days |
| Category viewers | Category pages | 14 days |
| Cart visitors | Cart page | 7 days |
| Converters | Thank you page | 540 days |

### Custom Audiences
| Audience | Signals |
|----------|---------|
| Health conscious | Search: organic food, healthy eating |
| Busy families | Search: meal prep, quick dinner |
| Foodies | Search: gourmet, recipes, cooking |
| Local shoppers | Search: local produce, farmers market |

## Audience Creation

### Remarketing Setup
```
In Google Ads:
1. Go to Audience Manager
2. Create new audience
3. Define rules
4. Set membership duration
5. Name clearly
6. Add to campaigns
```

### Customer Match
```
Upload process:
1. Export clean customer list
2. Format per Google specs
3. Hash if required
4. Upload to Google Ads
5. Wait for matching
6. Check match rate
7. Create lookalikes
```

### Custom Intent
```
Build based on:
- Keywords searched
- URLs visited
- Apps used
- YouTube content
```

## Audience Naming Convention

```
Format: [Type]_[Description]_[Duration]_[Date]

Examples:
- RM_AllVisitors_30d_2024
- CM_VIPCustomers_2024Q1
- CI_OrganicFood_2024
```

### Expansion Settings
| Reach | Use |
|-------|-----|
| Narrow | Best match, lower volume |
| Balanced | Default starting point |

## Audience Maintenance

### Monthly Tasks
```
1. Review audience sizes
2. Check match rates

4. Remove old audiences
5. Create new test audiences
```

