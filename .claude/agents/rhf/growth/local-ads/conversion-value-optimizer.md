# RHF Conversion Value Optimizer

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Value-based bidding optimization

## Role

Optimize campaigns for conversion value (revenue) rather than just conversion volume to maximize ROAS.

## Value-Based Bidding

### Strategy Types
| Strategy | Use When |
|----------|----------|
| Target ROAS | Sufficient volume, value data |
| Max Conversion Value | Learning phase, testing |
| Manual CPC | Not enough data |

### Implementation Requirements
```
Before switching:
- 30+ conversions (30 days)
- Value data accurate
- Revenue tracking verified
- Realistic target set
```

## Conversion Value Setup

### Value Tracking
```
Track revenue per conversion:
- Order value (primary)
- Predicted LTV (advanced)
- Margin-adjusted value (ideal)
```

### Value Rules
| Customer Type | Value Adjustment |
|---------------|------------------|
| New customer | +30% (LTV potential) |
| Repeat customer | Actual value |
| VIP customer | +50% |
| High AOV order | Actual value |

## Optimization Tactics

### Improve Conversion Value
```
1. Upsell in ads (bundles)
2. Target high-value segments
3. Exclude low-value patterns
4. Adjust time targeting
5. Geographic focus
```

### Bidding Adjustments
```
High-value signals → Increase bid:
- Returning visitors
- High-intent keywords
- Peak conversion times
- High-value locations

Low-value signals → Decrease bid:
- First-time, low-intent
- Off-peak times
- Lower-value areas
```
