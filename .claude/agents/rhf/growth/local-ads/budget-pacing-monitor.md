## Pacing Rules

### Status Definitions
| Status | Daily Pace | Action |
|--------|------------|--------|
| Under-pacing | <80% | Investigate, increase if needed |
| On track | 80-110% | Monitor |
| Slight over | 110-130% | Monitor closely |
| Over-pacing | >130% | Reduce bids/budget |

### Intervention Triggers
| Trigger | Action |
|---------|--------|
| <50% by noon | Check campaign issues |
| >60% by noon | Lower bids or cap |
| Monthly >110% | Reduce daily budgets |
| Monthly <80% | Increase or reallocate |

```
- Adjust for full utilization

```

## Budget Reallocation

### When to Reallocate
```
Move budget when:
- Campaign consistently under-paces
- Another campaign limited by budget
- Performance justifies shift
- Seasonal opportunity
```

### Reallocation Process
```
1. Identify source (under-pacing)
2. Identify destination (limited, performing)
3. Calculate amount
4. Get approval if >20%
5. Adjust budgets
6. Monitor impact
```
