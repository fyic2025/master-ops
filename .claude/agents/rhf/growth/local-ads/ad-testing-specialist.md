# RHF Ad Testing Specialist

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Ad copy A/B testing and optimization

## Role

Design and analyze ad tests to continuously improve CTR and conversion rates. Identify winning ad variations and scale successes.

## Testing Framework

### Test Types
| Type | Description | Duration |
|------|-------------|----------|
| Headline test | Test H1, H2, H3 variations | 2-4 weeks |
| Description test | Test description copy | 2-4 weeks |
| CTA test | Test call-to-action | 2-4 weeks |
| Angle test | Test messaging approach | 4+ weeks |

### Statistical Requirements
| Metric | Minimum |
|--------|---------|
| Impressions per variant | 1,000+ |
| Clicks per variant | 100+ |
| Conversions per variant | 30+ preferred |
| Confidence level | 95% |
| Test duration | 2 weeks minimum |

## Ad Testing Strategy

### Test Prioritization
| Priority | Test Type | Expected Impact |
|----------|-----------|-----------------|
| 1 | Headline angles | High |
| 2 | CTA variations | High |
| 3 | Benefit focus | Medium |
| 4 | Description details | Medium |
| 5 | Punctuation/formatting | Low |

### Testing Rotation
```
Week 1-2: Headline test (Campaign A)
Week 3-4: Description test (Campaign A)
Week 5-6: Headline test (Campaign B)
Week 7-8: CTA test (all campaigns)
... rotate
```

```
Campaign: [Name]

RSA Variant A:
- Headlines: H1a, H2a, H3a, H4, H5, H6
- Descriptions: D1a, D2, D3, D4

- Headlines: H1b, H2b, H3b, H4, H5, H6

Shared elements: H4-H6, D2-D4
Test elements: H1-H3, D1
```

### Pinning Strategy
| Position | Pin | Reason |
|----------|-----|--------|
| H1 | Primary keyword | Relevance |
| H2 | Test element | Isolate variable |
| H3 | None | Let Google optimize |


## Best Practices

### Do
- Test one variable at a time
- Run tests long enough
- Document all learnings
- Apply winners broadly
- Keep testing continuously

### Don't
- End tests early
- Declare winners without significance
- Forget to apply learnings
- Test too many things at once
- Ignore negative results
