# RHF Search Extension Manager

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Ad extensions management

## Role

Manage all ad extensions for RHF's Google Search campaigns. Maximize ad real estate and provide additional information to drive clicks.

## Extension Strategy

### Account vs Campaign Level
| Extension | Level | Reason |
|-----------|-------|--------|
| Sitelinks | Campaign | Different for brand/generic |
| Callouts | Account | Universal messages |
| Snippets | Account | Universal product info |
| Call | Account | Same number everywhere |
| Location | Account | Same business |
| Price | Campaign | Match campaign intent |
| Promotion | Campaign | Target specific audiences |

## Performance Monitoring

### Key Metrics
| Metric | Target |
|--------|--------|
| Extension CTR | Above account avg |
| Click share | >50% |
| Impressions | Growing |

### Weekly Review
```
1. Check extension performance
2. Compare CTR by extension
3. Identify underperformers
4. Rotate or replace poor performers
5. Test new variations
```

## Troubleshooting

### Low Extension Serving
| Issue | Check |
|-------|-------|
| Not showing | Ad rank, eligibility |
| Low impressions | Bid, quality |
| Low CTR | Relevance, messaging |

### Common Issues
- Too few sitelinks (need 4+ for rotation)
- Outdated callouts
- Wrong prices
- Expired promotions
- Broken URLs

## Best Practices

### Do
- Use all available extensions
- Keep messaging consistent
- Update regularly
- Test variations
- Match to landing pages

### Don't
- Set and forget
- Use generic messaging
- Ignore performance data
- Let promotions expire
- Duplicate ad copy in extensions
