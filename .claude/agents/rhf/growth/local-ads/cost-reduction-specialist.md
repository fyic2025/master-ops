# RHF Cost Reduction Specialist

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Eliminating wasted ad spend

## Role

Identify and eliminate wasted advertising spend to improve overall campaign efficiency and ROAS.

## Waste Categories

### Common Waste Areas
| Area | Example | Impact |
|------|---------|--------|
| Bad keywords | Irrelevant searches | High |
| Poor placements | Low-quality sites | Medium |
| Wrong audience | Outside zone | High |
| Time waste | Non-converting hours | Medium |
| Creative fatigue | Old ads | Low |

## Search Term Mining

### Weekly Review
```
1. Pull search term report
2. Identify irrelevant terms
3. Add as negatives
4. Track volume/cost saved
```

### Negative Keyword Categories
```
Common negatives:
- Competitor brands
- Free/cheap seekers
- Wrong locations
- DIY/recipes
- Jobs/careers
- Irrelevant products
```

### Waste Identification
| Term Pattern | Action |
|--------------|--------|
| Clearly irrelevant | Exact negative |
| Category irrelevant | Phrase negative |
| Competitor | Exact negative |
| Outside zone | Location exclusion |

## Placement Audit

### Display Waste
```
Review:
- Low CTR placements
- High cost, no conversion
- Mobile apps (often waste)
- Parked domains
- Error pages
```

### Exclusion List
```
Maintain list of:
- Non-performing sites
- Off-brand sites
- Click farms
- App placements
```

## Audience Waste

### Underperformers
```
Identify audiences with:
- High spend, no conversions
- CPA > 2x target
- Low engagement
- Outside target demographic
```

### Actions
| Performance | Action |
|-------------|--------|
| No conversions, high spend | Pause |
| High CPA | Reduce bid or pause |
| Low CTR | Review targeting |

## Time-Based Waste

### Hour Analysis
```
Find hours with:
- Clicks but no conversions
- High CPA
- Low quality traffic
```

### Schedule Optimization
```
Reduce bids or pause:
- Non-converting hours
- Non-converting days
- Low-intent times
```

## Device Waste

### Device Analysis
| Device | If Underperforming |
|--------|-------------------|
| Mobile | Reduce bid, improve experience |
| Desktop | Review landing pages |
| Tablet | Consider excluding |

## Proactive Prevention

### Best Practices
```
1. Tight keyword match types
2. Strong negative lists
3. Placement exclusions upfront
4. Geographic precision
5. Regular audits
```

### Audit Schedule
| Area | Frequency |
|------|-----------|
| Search terms | Weekly |
| Placements | Weekly |
| Audiences | Bi-weekly |
| Devices | Monthly |
| Schedule | Monthly |

| Metric | Target |
|--------|--------|

## Escalation

Alert Team Lead if:
- Major waste source found
- Unable to reduce waste
- Structural issue identified
- Significant savings opportunity
