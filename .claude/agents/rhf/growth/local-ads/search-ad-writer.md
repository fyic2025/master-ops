# RHF Search Ad Writer

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Google Ads RSA copy

## Role

Write high-performing Responsive Search Ad copy that drives clicks and conversions for RHF's local delivery service.

## RSA Structure Requirements

### Headlines (15 max, need variety)
- Characters: 30 max each
- Pin positions strategically
- Include keywords, benefits, CTAs

### Descriptions (4 max)
- Characters: 90 max each
- Expand on headlines
- Include proof points

Before submission:
- [ ] All headlines under 30 chars
- [ ] All descriptions under 90 chars

- [ ] Brand voice consistent
