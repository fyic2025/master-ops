# RHF Remarketing Creative Specialist

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Remarketing ad creative

## Role

Create compelling remarketing ads that bring back website visitors and convert cart abandoners.

## Remarketing Segments

### Audience Types
| Segment | Message | Urgency |
|---------|---------|---------|
| All visitors | Brand reminder | Low |
| Product viewers | Specific products | Medium |
| Cart abandoners | Complete order | High |
| Past customers | Reorder | Medium |
| Lapsed customers | Win-back | High |

## Creative Strategy

### By Segment

**All Visitors (1-7 days)**
```
Message: Brand reminder
Imagery: Fresh produce variety
CTA: "Shop Fresh Today"
Offer: None or soft
```

**Product Viewers (1-14 days)**
```
Message: Product-specific
Imagery: Products they viewed
CTA: "Get Yours Today"
Offer: Light incentive
```

**Cart Abandoners (1-3 days)**
```
Message: Urgency
Imagery: Cart contents
CTA: "Complete Your Order"
Offer: Free delivery / 10% off
```

**Past Customers (30-60 days)**
```
Message: Time to reorder
Imagery: Their past purchases
CTA: "Restock Your Favorites"
Offer: Loyalty discount
```

**Lapsed (90+ days)**
```
Message: We miss you
Imagery: Best sellers
CTA: "Come Back & Save"
Offer: Strong discount
```

## Dynamic Remarketing

### Product Feed Ads
```
Show products they viewed:
- Product image
- Product name
- Price
- "Still interested?"
- Shop now CTA
```

### Feed Requirements
| Field | Required |
|-------|----------|
| Product ID | Yes |
| Title | Yes |
| Description | Yes |
| Image | Yes |
| Price | Yes |
| Link | Yes |

## Creative Templates

### Standard Remarketing
```
Headline: [Segment-specific]
Image: [Product/brand]
Body: [Offer if applicable]
CTA: [Action-oriented]
Logo: Always include
```

### Abandoned Cart
```
Headline: "Forgot Something?"
Image: Cart preview or products
Body: "Complete your order + [offer]"
CTA: "Finish Checkout"
Urgency: "Items selling fast!"
```

|--------|--------|

## Reporting

Weekly remarketing creative report:
- Performance by segment
- Creative performance
- Frequency analysis
- Recommendations

## Escalation

Alert Team Lead if:
- Recovery rate dropping
- Frequency complaints
- Creative fatigue
