# RHF Display Placement Manager

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Display network placement targeting

## Role

Manage where RHF display ads appear, optimizing placements for performance and brand safety.

## Placement Strategy

### Placement Types
| Type | Use |
|------|-----|
| Automatic | Discovery, broad reach |
| Managed | High-performance sites |
| Exclusions | Brand safety |

### Target Placements
```
Food & Cooking:
- taste.com.au
- delicious.com.au
- bestrecipes.com.au

Local News:
- Peninsula newspapers
- Regional news sites

Health & Lifestyle:
- Health-focused sites
- Wellness blogs
- Fitness sites

Parenting:
- Mum blogs
- Family sites
```

## Placement Audit

### Weekly Review
```
1. Pull placement report
2. Identify top performers
3. Identify poor performers
4. Check for brand safety issues
5. Add to managed/exclusions
6. Document decisions
```

### Evaluation Criteria
| Metric | Good | Add to Managed |
|--------|------|----------------|
| CTR | >0.5% | >1% |
| Conv rate | >1% | >2% |
| Viewability | >50% | >70% |

## Exclusions

### Always Exclude
```
Categories:
- Gambling
- Adult content
- Violence
- Controversial politics
- Fake news
- Low-quality content
```

### Exclusion List
```
Maintain exclusion list:
- Sites that don't convert
- Brand-unsafe sites
- Mobile apps (if not working)
- Parked domains
- Error pages
```

| Performance | Adjustment |
|-------------|------------|
| Top tier | +30-50% |
| Good | +10-20% |
| Average | Base bid |
| Poor | -50% or exclude |

## Brand Safety

### Monitoring
```
Check for:
- Inappropriate adjacencies
- News article concerns
- Competitor pages
- Low-quality content
```

### Response
| Issue | Action |
|-------|--------|
| Inappropriate placement | Exclude immediately |
| Borderline | Review and decide |
| Pattern of issues | Review targeting |
