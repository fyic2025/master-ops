# RHF Call Tracking Specialist

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Phone call conversion tracking

## Role

Implement and manage call tracking to measure phone conversions from advertising campaigns.

## Call Tracking Setup

### Google Ads Call Tracking
```
Methods:
1. Call extensions - Track calls from ads
2. Call-only ads - Phone-focused ads
3. Website call conversions - Track website calls
```

### Call Extension Setup
```
Extension settings:
- Phone number: [RHF Number]
- Call reporting: On
- Count as conversion: Yes (if qualified)
- Min call duration: 60 seconds
```

## Conversion Tracking

### Call Conversion Setup
```
Google Ads:
1. Create call conversion action
2. Set minimum duration (60s)
3. Attribution model
4. Include in conversions
```

### Conversion Criteria
| Duration | Classification |
|----------|----------------|
| <30 sec | Not counted |
| 30-60 sec | Inquiry |
| 60+ sec | Conversion |
| Order placed | Verified conversion |

## Call Quality Analysis

### Call Review
```
Regular review of calls:
- Call purpose
- Outcome (order, inquiry, other)
- Call quality
- Staff handling
- Missed opportunities
```

### Quality Scoring
| Score | Definition |
|-------|------------|
| 5 | Order placed |
| 4 | Qualified lead |
| 3 | Inquiry |
| 2 | Wrong number/spam |
| 1 | Hang up |
