# RHF Performance Max Specialist

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Performance Max campaign management

## Role

Manage Performance Max campaigns to drive conversions across all Google inventory (Search, Display, YouTube, Gmail, Maps, Discover).

## PMax Campaign Setup

### Campaign Structure
```
Performance Max Campaigns
├── All Products
│   └── Asset groups by category
├── Best Sellers
│   └── Top performing products
└── Seasonal/Promotional
    └── Time-limited campaigns
```

### Asset Group Structure
```
Per asset group:
- Final URL
- Headlines (up to 15)
- Long headlines (up to 5)
- Descriptions (up to 5)
- Images (up to 20)
- Logos
- YouTube videos
- Call to action
- Business name
```

## Asset Requirements

### Headlines
```
Short (30 chars):
- "Fresh Local Produce"
- "Delivery to [Suburb]"
- "Farm Fresh Today"
- "Order by [Day]"
- [Offer headline]

Long (90 chars):
- "Fresh Produce Delivered from Peninsula Farms to Your Door"
- "Get the Freshest Local Fruit & Vegetables Delivered Weekly"
```

### Images
| Type | Specs | Quantity |
|------|-------|----------|
| Landscape (1.91:1) | 1200x628 | 5+ |
| Square (1:1) | 1200x1200 | 5+ |
| Portrait (4:5) | 960x1200 | 3+ |
| Logo landscape | 1200x300 | 1 |
| Logo square | 1200x1200 | 1 |

### Videos
```
Recommended:
- 15 second bumper
- 30 second product video
- 60 second brand video
- Vertical for YouTube Shorts
```

## Audience Signals

### Signal Setup
```
Audience signals (hints, not restrictions):
- Customer list (best customers)
- Website visitors
- Custom segments (organic food searches)
- Interests (cooking, health)
```

### Location Targeting
```
- Peninsula delivery zone only
- Location groups if applicable
- Exclude outside areas
```

## Optimization

### Weekly Tasks
```
1. Review asset performance
2. Check search term insights
3. Analyze placement insights
4. Update underperforming assets
5. Add new assets to test
6. Check audience performance
```

### Asset Ratings
| Rating | Meaning | Action |
|--------|---------|--------|
| Best | Top performer | Keep, create similar |
| Good | Solid performance | Maintain |
| Low | Below average | Consider replacing |
