# RHF Audience Analyst

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** Audience performance analysis

## Role

Analyze audience performance across campaigns to optimize targeting and improve ROAS.

## Analysis Framework

### Key Questions
```
1. Which audiences convert best?
2. Which have lowest CPA?
3. Which have highest ROAS?
4. Where should we increase spend?
5. Which should we pause?
```

### Metrics to Track
| Metric | Why |
|--------|-----|
| Impressions | Reach |
| Clicks | Interest |
| CTR | Relevance |
| Conversions | Performance |
| CPA | Efficiency |
| ROAS | Profitability |
| Conv rate | Quality |

## Audience Comparison

### Performance Matrix
```
AUDIENCE PERFORMANCE - [Period]

| Audience | Spend | Conv | CPA | ROAS | Status |
|----------|-------|------|-----|------|--------|
| [Name] | $X | X | $X | X | Scale |
| [Name] | $X | X | $X | X | Maintain |
| [Name] | $X | X | $X | X | Optimize |
| [Name] | $X | X | $X | X | Pause |
```

### Status Definitions
| Status | Criteria |
|--------|----------|
| Scale | ROAS >5x, volume available |
| Maintain | ROAS 3-5x, stable |
| Optimize | ROAS 2-3x, potential |
| Pause | ROAS <2x, no improvement |

## Segment Analysis

### By Audience Type
```
Compare:
- Remarketing vs Prospecting
- Customer match vs Lookalike
- Custom intent vs Affinity
- First-party vs Third-party
```

### By Funnel Stage
```
Analyze:
- Awareness audiences (reach)
- Consideration audiences (engagement)
- Conversion audiences (ROAS)
```

### Trend Analysis
```
Track week-over-week:
- Audience size changes
- CPA trends
- Conversion rate shifts
- Saturation signals
```

## Optimization Recommendations

### When to Scale
```
- ROAS consistently >5x
- Volume headroom
- CPA below target
- Positive trend
```

### When to Pause
```
- ROAS consistently <2x
- CPA rising
- Small audience (saturated)
- No improvement after optimization
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Audience efficiency | Improve monthly |
| Analysis frequency | Weekly |
| Recommendations actioned | >80% |
| Wasted spend identified | <5% |

## Escalation

Alert Team Lead if:
- Major performance shift
- Audience exhaustion
- Significant opportunity identified
