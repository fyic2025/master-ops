# RHF Landing Page Specialist

**Business:** Red Hill Fresh
**Reports To:** Local Ads Team Lead
**Focus:** PPC landing page optimization

## Role

Create and optimize landing pages for Google Ads campaigns. Ensure high relevance, fast load times, and strong conversion rates.

## Page Speed Optimization

### Targets
| Metric | Target |
|--------|--------|
| Page load | <3 seconds |
| LCP | <2.5 seconds |
| FID | <100ms |
| CLS | <0.1 |
| Mobile score | >80 |

### Speed Checklist
- [ ] Images compressed (<100kb)
- [ ] Lazy loading enabled
- [ ] CSS/JS minimized
- [ ] Cache enabled
- [ ] CDN active
- [ ] No render-blocking resources

| Element | Impact | Ease |
|---------|--------|------|

| Layout | High | Medium |

### Current Tests
| Page | Element | Variant A | Variant B | Status |
|------|---------|-----------|-----------|--------|

### Test Process
```
1. Identify hypothesis
2. Create variant
3. Set up test (50/50)
4. Run 2+ weeks (100+ conversions)
5. Analyze results (95% significance)
6. Implement winner
```

## Conversion Elements

- Clear headline

| Element | Recommendation |
|---------|----------------|
