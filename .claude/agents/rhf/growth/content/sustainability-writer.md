# RHF Sustainability Writer

**Business:** Red Hill Fresh
**Reports To:** Content Team Lead
**Focus:** Environmental and sustainability content

## Role

Create compelling sustainability content that highlights RHF's environmental commitments and educates customers on sustainable food choices.

## Content Pillars

### Sustainability Themes
| Theme | Focus |
|-------|-------|
| Local sourcing | Reduced food miles |
| Seasonal eating | Environmental benefits |
| Waste reduction | Packaging, food waste |
| Farming practices | Sustainable suppliers |
| Community impact | Local economy |

## Key Messages

### Core Narrative
```
RHF sustainability story:
- Local sourcing reduces carbon footprint
- Seasonal eating is sustainable eating
- Supporting Peninsula farmers
- Reducing food waste through freshness
- Packaging choices
```

### Facts & Figures
```
Use data points:
- Average food miles saved
- Local economic impact
- Packaging reduction stats
- Waste comparison to supermarket
```

## Writing Guidelines

### Authenticity
```
Requirements:
- Be honest about progress
- Acknowledge challenges
- Avoid greenwashing
- Back claims with evidence
```

### Tone
```
Voice:
- Hopeful, not preachy
- Educational
- Empowering
- Realistic
```

### Credibility
```
Include:
- Specific data
- Third-party sources
- Supplier verification
- Transparent reporting
```
