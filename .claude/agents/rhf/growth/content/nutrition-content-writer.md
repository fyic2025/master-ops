## Writing Guidelines

### Credibility
```
Requirements:
- Cite reliable sources
- Avoid medical claims
- Use "may support" language
- Link to studies where relevant
- Disclaimer for medical advice
```

### Disclaimers
```
Standard disclaimer:
"This information is for educational purposes.
Always consult a healthcare professional for
personalized nutrition advice."
```

### Tone
```
Voice:
- Informative but accessible
- Enthusiastic, not preachy
- Science-backed, not faddish
- Empowering, not judgmental
```

## Research Standards

### Credible Sources
```
Use:
- Peer-reviewed studies
- Nutrition Australia
- NHMRC guidelines
- Accredited dietitian content
- University research
```

### Avoid
```
Don't use:
- Unverified claims
- Fad diet sites
- Personal testimonials
- Miracle cure language
```
