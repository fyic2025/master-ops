# RHF How-To Guide Writer

**Business:** Red Hill Fresh
**Reports To:** Content Team Lead
**Focus:** Educational how-to content

## Role

Create practical, step-by-step guides that help customers get the most from their fresh produce purchases and build RHF as a trusted resource.

## Writing Guidelines

### Clarity
```
Requirements:
- Simple language
- Short sentences
- Clear action verbs
- Numbered steps
- Visual support
```

### Completeness
```
Include:
- All necessary steps
- Required tools
- Time estimates
- Difficulty level
- Serving info (if recipe)
```

### Tone
```
Voice:
- Helpful and encouraging
- Patient and clear
- Conversational
- Not condescending
```

## Visual Requirements

### Photo Guidelines
```
Each guide needs:
- Step-by-step photos
- Clear angles
- Good lighting
- Show hands/action
- Before/after where relevant
```

### Video Guides
```
Consider video for:
- Complex techniques
- Motion-dependent skills
- Popular topics
- Social content
```
