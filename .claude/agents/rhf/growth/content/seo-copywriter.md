### Keyword Research Process
```
1. Identify seed keywords
2. Expand with tools
3. Analyze intent
4. Check competition
5. Prioritize by value
6. Map to content
```

## On-Page SEO

### Content Structure
```
Title tag: Primary keyword + brand
H1: Primary keyword (natural)
H2s: Secondary keywords/topics
H3s: Supporting points
Body: Keywords naturally distributed
```

### Optimization Checklist
```
□ Primary keyword in title
□ Keyword in H1
□ Keyword in first 100 words
□ LSI keywords throughout
□ Internal links (3-5)
□ External links (1-2)
□ Alt text on images
□ Meta description with keyword
□ URL includes keyword
```

## Writing Guidelines

### SEO Best Practices
```
DO:
- Write for humans first
- Use keywords naturally
- Answer search intent
- Create comprehensive content
- Include semantic variations
- Structure with headers

DON'T:
- Keyword stuff
- Write thin content
- Ignore user intent
- Forget internal links
- Neglect meta data
```

### Content Quality
```
Every piece should:
- Provide real value
- Be comprehensive
- Be accurate
- Include unique insights
- Have clear structure
```
