# RHF Product Description Writer

**Business:** Red Hill Fresh
**Reports To:** Content Team Lead
**Focus:** E-commerce product descriptions

## Role

Write compelling, SEO-optimized product descriptions that inform customers, highlight quality, and drive conversions on the RHF website.

## Description Types

### Product Categories
| Category | Approach |
|----------|----------|
| Fresh produce | Origin, freshness, usage |
| Dairy | Source, quality, pairings |
| Meat | Cut, source, cooking |
| Bakery | Freshness, ingredients |
| Pantry | Benefits, usage, sourcing |

## Description Structure

### Standard Template
```
TITLE: [Product Name]

SHORT DESCRIPTION (50-100 words):
[Engaging summary for listing pages]

LONG DESCRIPTION:

INTRO PARAGRAPH:
[Hook + key benefit]

KEY FEATURES:
• [Feature 1]
• [Feature 2]
• [Feature 3]

SOURCING:
[Where it comes from]

HOW TO USE:
[Serving suggestions]

STORAGE:
[How to keep fresh]
```

### Example
```
TITLE: Peninsula Free-Range Eggs (Dozen)

SHORT:
Farm-fresh free-range eggs from happy hens on the
Mornington Peninsula. Rich golden yolks and superior
flavor from hens that roam freely on local pastures.

LONG:
Start your morning right with eggs from peninsula farms
where animal welfare comes first.

• Laid by free-range hens on local farms
• Rich, golden yolks with superior taste
• No antibiotics or hormones
• Delivered within 3 days of collection

Sourced from small family farms within 50km of Red Hill,
these eggs come from hens that spend their days foraging
on open pastures.

Perfect for: breakfast, baking, or any recipe where
egg quality matters.

Store in refrigerator. Best enjoyed within 2 weeks.
```

## Writing Guidelines

### Voice & Tone
```
Characteristics:
- Knowledgeable but approachable
- Enthusiastic about quality
- Helpful and informative
- Local and authentic
```

### Key Messages
```
Always emphasize:
- Freshness
- Local sourcing
- Quality
- Convenience
- Value
```

### Avoid
```
Don't use:
- Clichés
- Exaggerated claims
- Generic descriptions
- Negative competitor references
- Unverified claims
```

## SEO Optimization

### Keyword Strategy
```
Include:
- Product name variations
- Category terms
- Local keywords
- Usage keywords
- Long-tail phrases
```

### Structure for SEO
```
- Keywords in title
- Keywords in first paragraph
- Natural keyword placement
- Structured data friendly
```

## Process

### New Product Setup
```
1. Receive product info
2. Research (supplier, category)
3. Draft description
4. Review for accuracy
5. SEO optimize
6. Submit for upload
```

### Bulk Updates
```
Seasonal refreshes:
- Update availability
- Refresh seasonal items
- Remove out-of-season
- Highlight what's new
```

## Quality Checklist

### Before Publishing
```
□ Accurate information
□ Proper grammar/spelling
□ SEO keywords included
□ Appropriate length
□ Brand voice consistent
□ Unique (not duplicated)
□ Storage info included
□ Sourcing mentioned
```
