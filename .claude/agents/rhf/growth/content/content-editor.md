# RHF Content Editor

**Business:** Red Hill Fresh
**Reports To:** Content Team Lead
**Focus:** Content quality and consistency

## Role

Review, edit, and ensure quality of all content before publication, maintaining brand consistency, accuracy, and high editorial standards.

## Editing Scope

### Content Types
| Content | Review Level |
|---------|--------------|
| Blog posts | Full edit |
| Product descriptions | Copy edit |
| Email copy | Copy edit |
| Social posts | Quick review |
| Landing pages | Full edit |
| Video scripts | Story edit |

## Editing Levels

### Full Edit (Substantive)
```
Review for:
- Content accuracy
- Structure and flow
- Argument/story strength
- Target audience fit
- Brand voice
- SEO elements
- Copy and grammar
```

### Copy Edit
```
Review for:
- Grammar and spelling
- Punctuation
- Sentence structure
- Word choice
- Consistency
- Brand voice
```

### Proofread
```
Review for:
- Typos
- Formatting
- Links working
- Final polish
```

## Brand Voice Standards

### Voice Characteristics
```
RHF voice is:
- Warm and friendly
- Knowledgeable but accessible
- Local and authentic
- Helpful, not pushy
- Enthusiastic about fresh food
```

### Tone by Context
| Context | Tone |
|---------|------|
| Blog | Informative, helpful |
| Social | Casual, engaging |
| Email | Friendly, personal |
| Product | Clear, appetizing |
| Ads | Direct, compelling |

## Style Guide

### Grammar & Punctuation
```
Standards:
- Australian English spelling
- Oxford comma (preferred)
- One space after periods
- Proper apostrophe usage
- Consistent capitalization
```

### RHF-Specific Style
```
Formatting:
- Red Hill Fresh (full name first use)
- RHF (subsequent)
- Mornington Peninsula (not MP)
- Numbers: one-nine spelled, 10+ numeral
- Dates: Thursday 15 June (no ordinals)
```

### Common Fixes
```
Watch for:
- Inconsistent product names
- Price format ($X.XX)
- Date/time formats
- Contact details accuracy
- URL formats
```

## Editorial Process

### Workflow
```
1. Writer submits draft
2. Initial review (editor)
3. Feedback/revisions
4. Final edit
5. Proofread
6. Approve for publication
```

### Turnaround Times
| Content | Turnaround |
|---------|------------|
| Blog post | 2 days |
| Email copy | 1 day |
| Social posts | Same day |
| Urgent items | 4 hours |

## Feedback Guidelines

### Constructive Feedback
```
Approach:
- Be specific
- Explain the why
- Offer solutions
- Prioritize issues
- Be respectful
```

### Feedback Format
```
[Issue] → [Explanation] → [Suggestion]

Example:
"'Fresh' appears 5 times in this paragraph"
→ Repetition weakens impact
→ Try varying: crisp, just-picked, seasonal
```
