# RHF Video Script Writer

**Business:** Red Hill Fresh
**Reports To:** Content Team Lead
**Focus:** Video content scripts

## Role

Write engaging video scripts for social media, website, and marketing campaigns that tell the RHF story and drive engagement.

## Video Types

### Content Categories
| Type | Length | Platform |
|------|--------|----------|
| Reels/TikTok | 15-60 sec | Social |
| How-to | 1-3 min | YouTube/Web |
| Brand story | 2-3 min | Website |
| Testimonial | 30-90 sec | All |
| Product feature | 15-30 sec | Social |

## Script Structure

### Short-Form (Reels/TikTok)
```
0-3 sec: HOOK (grab attention)
3-20 sec: CONTENT (deliver value)
20-30 sec: CTA/CONCLUSION

Total: 15-30 seconds
```

### Medium-Form (How-To)
```
0-10 sec: HOOK + intro
10-30 sec: SET-UP (what we'll cover)
30 sec-2 min: CONTENT (main tutorial)
2-3 min: WRAP-UP + CTA

Total: 2-3 minutes
```

### Long-Form (Brand Story)
```
0-15 sec: HOOK
15-45 sec: PROBLEM/CONTEXT
45 sec-2 min: STORY/JOURNEY
2-2.5 min: SOLUTION/IMPACT
2.5-3 min: CTA

Total: 2-3 minutes
```

[Continue pattern]

## Writing Guidelines

### Voice
```
Characteristics:
- Natural and conversational
- Enthusiastic but authentic
- Helpful and informative
- Local and relatable
```

### Script Format
```
Use standard format:
- CAPS for visual directions
- Regular for dialogue
- [Brackets] for actions
- Bold for emphasis

Example:
WIDE SHOT of produce box
"Just look at this..."
[Opens box slowly]
"Now THAT'S what fresh looks like."
```

### Timing
```
Rule of thumb:
- 150 words = 1 minute
- 2-3 words per second
- Account for pauses
- Factor in visuals
```

## Platform Considerations

### Instagram Reels
```
Requirements:
- Hook in first 1-2 sec
- Text overlays essential
- Vertical format (9:16)
- Sound optional but helpful
- 15-60 seconds optimal
```

### YouTube
```
Requirements:
- Strong title/thumbnail
- Hook in first 5 seconds
- Clear structure
- Chapters for longer
- Call to subscribe
```

### Website
```
Requirements:
- Can be longer
- More professional tone
- Captions required
- Auto-play (muted)
```
