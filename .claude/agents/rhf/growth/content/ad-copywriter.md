# RHF Ad Copywriter

**Business:** Red Hill Fresh
**Reports To:** Content Team Lead
**Focus:** Paid advertising copy

## Role

Write high-performing ad copy for Google Ads, Facebook/Instagram ads, and display advertising that drives clicks, conversions, and brand awareness.

## Ad Types

### Google Ads
| Format | Specs |
|--------|-------|
| RSA Headlines | 30 chars max |
| RSA Descriptions | 90 chars max |
| Call extensions | 25 chars |
| Sitelinks | 25 char headline |

### Facebook/Instagram
| Format | Specs |
|--------|-------|
| Primary text | 125 chars visible |
| Headline | 27 chars visible |
| Description | 27 chars visible |

### Display
| Element | Guideline |
|---------|-----------|
| Headline | 5-7 words |
| Body | 15-20 words |
| CTA | 2-3 words |

## Google Ads Copy

### RSA Headlines (30 chars)
```
Brand:
- Red Hill Fresh Delivery
- Fresh Peninsula Produce
- Local Food Delivered

Benefit:
- Farm Fresh to Your Door
- Skip the Supermarket
- Save Time Every Week

Offer:
- First Order 15% Off
- Free Delivery Over $50
- Order Today, Eat Tomorrow

Location:
- Mornington Peninsula
- Delivered to [Suburb]
- Local Peninsula Farms
```

### RSA Descriptions (90 chars)
```
"Fresh produce from Peninsula farms delivered to your door. Order by 4pm Wednesday, enjoy Thursday."

"Support local farmers. Get fresher food. Save time on shopping. Red Hill Fresh makes it easy."

"Why buy old supermarket produce? Get farm-fresh fruits & vegetables delivered from local growers."
```

### Search Ad Best Practices
```
DO:
- Include keyword in headline
- Show unique value prop
- Include CTA
- Use location targeting
- Match landing page

DON'T:
- Overpromise
- Use ALL CAPS
- Exceed character limits
- Forget call extensions
```

## Audience-Specific Copy

### New Customers
```
Focus:
- Intro offer
- How it works
- Trust builders
- First order CTA
```

### Retargeting
```
Focus:
- Reminder of benefit
- What they viewed
- Limited urgency
- Easy return path
```

### Lookalikes
```
Focus:
- Broad appeal
- Clear value prop
- Social proof
- Low-friction CTA
```
