# RHF Produce Photographer

**Business:** Red Hill Fresh
**Reports To:** Content Team Lead
**Focus:** Product and lifestyle photography

## Role

Create stunning visual content showcasing RHF produce, deliveries, and lifestyle imagery for all marketing channels.

## Photography Types

### Product Photography
| Type | Use | Requirements |
|------|-----|--------------|
| Hero shots | Website, ads | Studio quality |
| Category | Website listings | Consistent style |
| Individual | Product pages | White/clean bg |
| Group | Bundles, boxes | Styled arrangement |

### Lifestyle Photography
| Type | Use | Requirements |
|------|-----|--------------|
| In-kitchen | Social, email | Natural, authentic |
| Cooking | Recipes, blog | Action shots |
| Family | Brand story | Real moments |
| Delivery | Social proof | Doorstep context |

### Behind-the-Scenes
| Type | Use |
|------|-----|
| Farm visits | Supplier stories |
| Packing | Operations transparency |
| Team | Brand personality |
| Delivery drivers | Human connection |

## Style Guide

### Brand Aesthetic
```
Color palette: Fresh greens, natural browns, clean whites
Mood: Bright, natural, authentic
Props: Wooden boards, linen, simple ceramics
Background: Clean, uncluttered
Lighting: Natural daylight preferred
```

### Do's
- Fresh, vibrant produce
- Natural lighting
- Real imperfections (authentic)
- Peninsula context
- Seasonal relevance

### Don'ts
- Over-processed/filtered
- Artificial arrangements
- Stock photo feel
- Wilted or damaged produce
- Cluttered backgrounds

## Shot List by Category

### Produce Essentials
```
Each product needs:
- Hero shot (styled)
- Product shot (clean bg)
- Scale reference
- Detail/texture shot
- In-use lifestyle
```

### Seasonal Updates
```
Quarterly shoots:
- New seasonal items
- Box contents update
- Lifestyle refresh
- Recipe imagery
```

## Equipment & Setup

### Minimum Requirements
```
Camera: DSLR or high-end phone
Lighting: Natural + reflector
Background: White, wood, marble options
Props: Boards, linens, basic ceramics
Editing: Lightroom or equivalent
```

### Home Studio Setup
```
Location: Near large window
Time: Mid-morning for best light
Space: Clear 2m x 2m minimum
Storage: Props organized
```

## Asset Management

### File Naming
```
Format: RHF_[category]_[product]_[type]_[date]
Example: RHF_produce_strawberries_hero_20240115
```

### Storage
```
Raw files: Cloud backup
Edited: Organized folders
Quick access: Drive/Dropbox
Shared: Marketing team access
```
