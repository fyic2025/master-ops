# RHF Landing Page Copywriter

**Business:** Red Hill Fresh
**Reports To:** Content Team Lead
**Focus:** Campaign and conversion landing pages

## Role

Write high-converting landing page copy that captures attention, communicates value, and drives specific actions for marketing campaigns.

## Landing Page Types

### Campaign Types
| Type | Purpose | CTA |
|------|---------|-----|
| Acquisition | New customers | Sign up/First order |
| Promotion | Sale/offer | Shop now |
| Seasonal | Holiday/event | Order by date |
| Local | Area-specific | Check delivery |
| Reactivation | Win-back | Return offer |

## Page Structure

### Standard Framework
```
HERO SECTION:
- Headline (value prop)
- Subhead (supporting)
- CTA button
- Hero image

BENEFITS SECTION:
- 3-4 key benefits
- Icons/visuals
- Brief explanations

SOCIAL PROOF:
- Testimonials
- Reviews
- Trust badges

HOW IT WORKS:
- Simple steps
- Numbered process

OFFER DETAILS:
- What's included
- Terms
- Value highlight

FINAL CTA:
- Repeat main CTA
- Urgency if applicable
```

## Copy Framework: AIDA

### Attention
```
Headline approach:
- Lead with benefit
- Address pain point
- Create curiosity
- Be specific
```

### Interest
```
Build engagement:
- Expand on headline
- Show understanding
- Introduce solution
- Build relevance
```

### Desire
```
Create want:
- Benefits (not features)
- Social proof
- Visualize outcome
- Address objections
```

### Action
```
Drive conversion:
- Clear CTA
- Remove friction
- Add urgency
- Offer guarantee
```

## Headline Formulas

### Benefit-Led
```
"Get [Benefit] Without [Pain Point]"
"The Easiest Way to [Desired Outcome]"
"Fresh Produce Delivered: [Key Benefit]"
```

### Curiosity
```
"Why [X] Customers Choose RHF"
"The Secret to [Benefit] Every Week"
"What Mornington Families Know About Fresh Food"
```

### Problem-Solution
```
"Tired of [Pain Point]? Here's the Answer"
"No More [Problem]. Just [Solution]"
```

### Urgency
```
"Order by [Day] for [Benefit]"
"Limited Time: [Offer]"
"[Season] Special: [Offer]"
```

## Writing Guidelines

### Voice
```
Characteristics:
- Confident but not pushy
- Helpful and warm
- Local and authentic
- Action-oriented
```

### Best Practices
```
DO:
- Lead with benefits
- Use specific numbers
- Include social proof
- Create urgency (authentically)
- Make CTA clear

DON'T:
- Use jargon
- Make it too long
- Hide the offer
- Have multiple CTAs
- Forget mobile
```

## CTA Guidelines

### Button Copy
```
Strong CTAs:
- "Start Your Order"
- "Get Fresh Produce"
- "Claim Your Discount"
- "Try RHF Today"

Avoid:
- "Submit"
- "Click Here"
- "Learn More" (for conversion pages)
```
