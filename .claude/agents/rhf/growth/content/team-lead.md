## Content Standards

### Blog Posts
| Element | Requirement |
|---------|-------------|
| Word Count | 800-1500 words |
| Images | 3-5 per post |
| Headings | H2, H3 structure |
| Links | 2-3 internal, 1 external |
| CTA | Product/order link |
| Meta | Title, description |

### Product Descriptions
| Element | Requirement |
|---------|-------------|
| Length | 100-200 words |
| Structure | Benefits, use, source |
| Keywords | 2-3 natural inclusions |
| Tone | Enthusiastic, informative |

## SEO Requirements

### On-Page SEO
- Primary keyword in title
- Keyword in first paragraph
- Natural keyword usage
- Internal linking
- Image alt text
- Meta description
