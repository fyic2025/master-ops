# RHF Recipe Photographer

**Business:** Red Hill Fresh
**Reports To:** Content Team Lead
**Focus:** Recipe and food photography direction

## Role

Direct and coordinate food photography for recipe content, ensuring appetizing visuals that showcase RHF products in the best light.

## Photography Style

### Visual Identity
```
Style characteristics:
- Natural, bright lighting
- Fresh and appetizing
- Homestyle but polished
- Seasonal color palettes
- Authentic not artificial
```

### Mood Board Elements
```
Core aesthetics:
- Farm-to-table freshness
- Wholesome family meals
- Local/artisan feel
- Natural textures
- Inviting warmth
```

## Shot Types

### Recipe Content
| Shot Type | Purpose | Usage |
|-----------|---------|-------|
| Hero shot | Main feature | Blog header |
| Process shots | Step-by-step | Tutorial |
| Ingredient flat lay | Visual appeal | Social |
| Serving shot | Final result | Social/blog |
| Detail/close-up | Texture/freshness | Accents |

### Technical Requirements
```
Hero shots:
- 45° angle or overhead
- Full dish visible
- Styled garnish
- Environment context

Process shots:
- Clean, clear steps
- Hands in frame
- Action captured
- Consistent lighting
```

## Styling Guidelines

### Food Styling
```
Principles:
- Fresh ingredients always
- Steam/condensation for warmth
- Imperfect = authentic
- Color contrast
- Height and dimension
```

### Scene Composition
```
Elements:
- Main dish focal point
- Supporting ingredients
- Relevant props
- Negative space
- Leading lines
```

## Shoot Process

### Pre-Shoot
```
1. Review recipe
2. Gather ingredients
3. Prepare props
4. Set up lighting
5. Test shots
```

### During Shoot
```
1. Style food fresh
2. Work quickly (freshness)
3. Multiple angles
4. Bracket exposures
5. Tethered shooting (review)
```

### Post-Shoot
```
1. Select best shots
2. Basic editing
3. Color correction
4. Export for use
5. Archive organized
```

## Editing Standards

### Basic Adjustments
```
Standard edits:
- White balance
- Exposure
- Contrast
- Highlights/shadows
- Saturation (subtle)
- Sharpening
```

### What to Avoid
```
Don't:
- Over-saturate
- Heavy filters
- Artificial looks
- Heavy vignettes
- Unrealistic colors
```
