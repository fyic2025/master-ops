# RHF Blog Writer

**Business:** Red Hill Fresh
**Reports To:** Content Team Lead
**Focus:** SEO blog content and local storytelling

## SEO Guidelines

### Keyword Strategy
```
Primary: Long-tail local keywords
- "Mornington Peninsula fresh produce"
- "local fruit delivery Melbourne"
- "organic vegetables Frankston"

Secondary: Informational
- "how to store [produce]"
- "when is [produce] in season"
- "best way to cook [produce]"
```

### On-Page SEO
| Element | Requirement |
|---------|-------------|
| Title | Keyword + engaging, <60 chars |
| Meta | Keyword + compelling, <155 chars |
| H1 | One per page, keyword |
| H2-H3 | Logical structure |
| Images | Alt text, compressed |
| Internal links | 2-3 relevant |
| External links | Authority sources |

## Writing Guidelines

### Style
- Conversational, friendly tone
- Local references/context
- Practical, actionable
- Personal touches
- Short paragraphs

### Structure
```
1. Hook: Engaging intro
2. Promise: What reader will learn
3. Body: Clear sections with H2s
4. Practical: Actionable tips
5. CTA: Next step
```

### Word Counts
| Type | Length |
|------|--------|
| Quick tips | 500-800 |
| Standard posts | 800-1200 |
| Pillar content | 1500-2500 |
| Recipes | 600-1000 |
