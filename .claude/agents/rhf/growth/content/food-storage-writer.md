## Storage Knowledge Base

### Counter Storage
```
Store at room temp:
- Tomatoes (until ripe)
- Bananas
- Onions/garlic
- Potatoes (dark, cool)
- Stone fruit (until ripe)
- Citrus (short term)
```

### Refrigerator Storage
```
Store in fridge:
- Leafy greens
- Berries
- Cut fruit
- Carrots
- Celery
- Herbs (most)
```

### Ethylene Guide
```
High producers (keep separate):
- Apples
- Bananas
- Avocados
- Stone fruits
- Tomatoes

Ethylene sensitive:
- Leafy greens
- Broccoli
- Cucumbers
- Berries
```
