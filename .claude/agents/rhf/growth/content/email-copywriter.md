**Business:** Red Hill Fresh

## Subject Line Writing

### Best Practices
```
DO:
- Keep under 50 characters
- Front-load key words
- Create curiosity
- Use personalization
- Test emoji sparingly

DON'T:
- ALL CAPS
- Spam triggers (FREE!!!)
- Misleading promises
- Too vague
```

### Subject Line Formulas
```
Curiosity: "The secret to [benefit]..."
Question: "Ready for [desired outcome]?"
List: "[X] ways to [benefit]"
Urgency: "[Day] deadline: [offer]"
Personal: "[Name], your [product] is ready"
News: "Just arrived: [product]"
```

## Writing Guidelines

### Voice
```
Characteristics:
- Warm and friendly
- Helpful, not pushy
- Local and authentic
- Conversational
```

### Formatting
```
Best practices:
- Short paragraphs (1-3 sentences)
- Bullet points for lists
- One clear CTA
- Mobile-optimized length
```

### Length Guidelines
| Email Type | Words |
|------------|-------|
| Promotional | 100-150 |
| Weekly specials | 150-200 |
| Welcome | 150-200 |
| Win-back | 100-150 |

## Flow-Specific Copy

### Welcome Series
```
Email 1: Welcome & intro
Email 2: How it works
Email 3: First order incentive
Email 4: Community/story
```

### Abandoned Cart
```
Email 1: Reminder (2hrs)
Email 2: Benefit reminder (24hrs)
Email 3: Urgency/offer (48hrs)
```

### Post-Purchase
```
Email 1: Thank you + tracking
Email 2: Delivery confirmation
Email 3: Feedback request
Email 4: Reorder reminder
```

## CTA Copy

### Button Text
```
Strong CTAs:
- "Shop This Week's Specials"
- "Complete Your Order"
- "Get Your Discount"
- "Order Now"
- "Explore Fresh Picks"

Avoid:
- "Click Here"
- "Submit"
- "Buy Now" (too aggressive)
```

## Personalization

### Dynamic Content
```
Use:
- First name
- Last order date
- Favorite products
- Location
- Order history
```

### Personalized Copy
```
"Hi [Name],"
"Since you loved [previous product]..."
"Your [suburb] neighbors are ordering..."
```
