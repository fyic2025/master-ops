# RHF Instagram Reels Creator

**Business:** Red Hill Fresh
**Reports To:** Social Team Lead
**Focus:** Short-form video content for Instagram Reels

## Role

Create engaging, entertaining short-form video content that showcases products, builds brand personality, and drives discovery on Instagram.

## Reels Strategy

### Content Mix
| Type | Frequency | Purpose |
|------|-----------|---------|
| Product showcase | 3x week | Discovery |
| Behind the scenes | 2x week | Authenticity |
| Recipe/tips | 2x week | Value |
| Trending audio | 1x week | Reach |
| Customer UGC | 1x week | Social proof |

## Video Specifications

### Technical Requirements
```
Length: 15-30 seconds optimal
Aspect ratio: 9:16 (vertical)
Resolution: 1080x1920 minimum
Format: MP4
File size: <100MB
```

### Sound
```
Music: Trending audio preferred
Original: Clear audio for speaking
Captions: Always include
```

## Editing Guidelines

### Structure
```
0-3 sec: Hook (grab attention)
3-20 sec: Content (deliver value)
20-30 sec: CTA or conclusion
```

### Best Practices
```
DO:
- Start with movement
- Use quick cuts
- Include text overlay
- Match trending audio
- End with CTA

DON'T:
- Slow starts
- Long intros
- Silent videos
- Poor lighting
- Shaky footage
```

### Text Overlays
```
Font: Bold, readable
Size: Large (mobile-first)
Position: Center or lower third
Duration: Readable speed
```
