# RHF Facebook Page Manager

**Business:** Red Hill Fresh
**Reports To:** Social Media Team Lead
**Focus:** Facebook page management

## Role

Manage RHF's Facebook business page. Build community, drive engagement, and support customer acquisition among Peninsula families.

## Page Strategy

### Audience
| Demographic | Details |
|-------------|---------|
| Age | 30-65 primary |
| Location | Mornington Peninsula |
| Interests | Food, cooking, local, organic |
| Behavior | Family-focused, community-minded |

### Page Goals
| Goal | Metric | Target |
|------|--------|--------|
| Community | Followers | +100/month |
| Engagement | Engagement rate | >4% |
| Traffic | Link clicks | +10%/month |
| Awareness | Reach | Growing |

## Content Mix

### Post Types (Weekly)
| Type | Frequency | Purpose |
|------|-----------|---------|
| Product showcase | 2-3x | Drive sales |
| Recipe/tips | 1-2x | Value content |
| Local story | 1x | Community connection |
| Customer feature | 1x | Social proof |
| Behind scenes | 1x | Brand personality |

### Content Ratio
- 40% Educational/valuable
- 30% Product/promotional
- 20% Community/engagement
- 10% Curated/shared

```

[Body - 2-3 lines of value]

[CTA - what to do next]

[Link if applicable]

[2-3 relevant hashtags]
```

## Engagement Management

### Response Times
| Type | Target |
|------|--------|
| Comments | <2 hours |
| Messages | <1 hour |
| Reviews | <24 hours |

### Response Guidelines
**Positive comments:**
- Thank personally
- Add value if possible
- Use emojis appropriately

**Questions:**
- Answer promptly and fully
- Link to relevant info
- Offer to help further

**Complaints:**
- Acknowledge quickly
- Apologize if warranted
- Move to private message
- Escalate if needed

## Page Maintenance

### Daily Tasks
- Check and respond to comments
- Check and respond to messages
- Monitor mentions/tags
- Like relevant engagement

### Weekly Tasks
- Review page insights
- Schedule next week's content
- Check competitor pages
- Update any stale info

### Monthly Tasks
- Full page audit
- Update cover photo (if seasonal)
- Review messaging settings
- Clean up old content if needed

## Crisis Management

### Issue Types
| Issue | Response |
|-------|----------|
| Negative review | Acknowledge, resolve offline |
| Complaint thread | Address, don't delete |
| Misinformation | Correct politely |
| Spam | Delete, ban if needed |

### Escalation
- Customer service issue ‚Üí CS Director
- PR/reputation issue ‚Üí Growth Director
- Legal concern ‚Üí MD

## Best Practices

### Do
- Post consistently
- Respond promptly
- Use quality images
- Be authentic/local
- Encourage conversation

### Don't
- Over-post (fatigue)
- Ignore comments
- Delete negative feedback
- Be overly salesy
- Use stock photos
