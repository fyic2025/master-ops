## Platform Guidelines

### Instagram
| Type | Count | Size |
|------|-------|------|
| Brand | 1-2 | Any |
| Niche | 3-5 | 10K-100K |
| Medium | 3-5 | 100K-500K |
| Broad | 2-3 | 500K+ |
| **Total** | 10-15 | Mix |

### Facebook
```
Fewer hashtags:
- 3-5 maximum
- Highly relevant only
- In caption, not comment
```

## Research Process

### Finding Hashtags
```
Methods:
1. Instagram search
2. Competitor analysis
3. Influencer hashtags
4. Related suggestions
5. Hashtag tools
```

### Evaluating Hashtags
| Factor | Check |
|--------|-------|
| Size | Not too big/small |
| Relevance | Matches content |
| Engagement | Posts get likes |
| Quality | Good content |
| Recency | Active today |

### Size Sweet Spots
| Posts | Classification | Use |
|-------|----------------|-----|
| <10K | Too small | Avoid |
| 10K-100K | Niche | 30% |
| 100K-500K | Medium | 40% |
| 500K-1M | Competitive | 20% |
| >1M | Very broad | 10% |

## Banned Hashtags

### Check Before Using
```
Signs of banned:
- No recent posts showing
- "Top posts" only
- Hidden by Instagram
```

### Never Use
```
- Spammy hashtags
- Banned hashtags
- Inappropriate tags
- Irrelevant popular tags
```
