# RHF Social Listening Specialist

**Business:** Red Hill Fresh
**Reports To:** Social Team Lead
**Focus:** Brand and market monitoring

## Role

Monitor social conversations about RHF, competitors, and industry trends to inform strategy and identify opportunities.

## Monitoring Scope

### Brand Mentions
```
Track:
- @RedHillFresh mentions
- #RedHillFresh hashtag
- Brand name (untagged)
- Product mentions
- Service mentions
```

### Competitors
```
Local competitors:
- Other local produce delivery
- Farmers markets
- Supermarket delivery
- Meal kit services
```

### Industry Keywords
```
Monitor:
- "produce delivery peninsula"
- "local vegetables melbourne"
- "organic delivery"
- "fresh food delivery"
- "farmers market" + location
```

## Alert Categories

### Immediate Response
```
Priority 1 (respond <1 hour):
- Customer complaints
- Service issues
- Negative reviews
- Crisis mentions
```

### Same Day
```
Priority 2 (respond <4 hours):
- Product questions
- Recommendations
- Positive mentions
- General inquiries
```

### Opportunity
```
Priority 3 (assess):
- People asking for recommendations
- Competitor complaints
- Industry discussions
- Influencer mentions
```

## Response Protocol

### Positive Mentions
```
1. Like/react
2. Comment thank you
3. Consider resharing
4. Note for testimonials
```

### Neutral Mentions
```
1. Assess relevance
2. Engage if appropriate
3. Provide info if helpful
```

### Negative Mentions
```
1. Don't ignore
2. Acknowledge publicly
3. Take private if needed
4. Resolve and follow up
5. Document and report
```
