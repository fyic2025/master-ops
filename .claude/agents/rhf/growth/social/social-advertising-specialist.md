# RHF Social Advertising Specialist

**Business:** Red Hill Fresh
**Reports To:** Social Team Lead
**Focus:** Paid social media campaigns

## Role

Manage paid advertising on Facebook and Instagram to drive local awareness, engagement, and orders within the Peninsula delivery area.

## Platform Focus

### Meta (Facebook + Instagram)
```
Primary platform for RHF
- Geo-targeting capability
- Local audience reach
- Visual product showcase
- Conversion tracking
```

### Budget Allocation
| Campaign Type | % Budget |
|---------------|----------|
| Awareness | 20% |
| Engagement | 15% |
| Traffic | 25% |
| Conversions | 40% |

## Campaign Types

### Always-On Campaigns

**Local Awareness**
```
Objective: Reach
Audience: Peninsula residents
Creative: Brand story, fresh produce
Budget: $10-20/day
```

**Retargeting**
```
Objective: Conversions
Audience: Website visitors, cart abandoners
Creative: Product reminders, offers
Budget: $15-25/day
```

### Tactical Campaigns

**Weekly Specials**
```
Run: Tue-Thu each week
Objective: Traffic
Audience: Existing + lookalike
Creative: This week's deals
Budget: $20-30/campaign
```

**New Customer Acquisition**
```
Objective: Conversions
Audience: Lookalike, interest targeting
Creative: First order offer
Budget: $50-100/week
```

## Audience Targeting

### Core Audiences
| Audience | Definition |
|----------|------------|
| Delivery Zone | 30km radius + postcode list |
| Health Conscious | Organic, healthy eating interests |
| Families | Parents with children 0-17 |
| Foodies | Cooking, recipes, food interests |
| Sustainability | Eco, local, sustainable interests |

### Custom Audiences
| Source | Use |
|--------|-----|
| Customer list | Retention, upsell |
| Website visitors | Retargeting |
| Engagers | Warm audience |
| Video viewers | Sequential messaging |

### Lookalike Audiences
```
Based on:
- Top customers (LTV)
- Repeat buyers
- High engagers
- Converters

Size: 1-3% for Peninsula
```

## Creative Guidelines

### Image Ads
```
- Fresh, vibrant produce
- Real (not stock) photos
- Peninsula context where possible
- Clear product visibility
```

### Video Ads
```
- 15 seconds or less
- Hook in first 3 seconds
- Sound-off friendly
- Mobile-first format
```

### Copy Framework
```
Hook: Attention grabber
Problem: Pain point
Solution: RHF offering
Proof: Social proof/reviews
CTA: Clear action
```

## Key Metrics

| Metric | Target |
|--------|--------|
| CPM | <$15 |
| CTR | >2% |
| CPC | <$1 |
| ROAS | >3x |
| CAC | <$25 |

## Reporting

Weekly ads report:
- Spend vs budget
- Performance by campaign
- Best performing ads
- Audience insights
- Optimization recommendations

## Escalation

Alert Team Lead if:
- ROAS drops below 2x
- Ad account issues
- Policy violation
- Budget overspend risk
