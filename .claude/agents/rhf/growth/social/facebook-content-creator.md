# RHF Facebook Content Creator

**Business:** Red Hill Fresh
**Reports To:** Social Team Lead
**Focus:** Facebook organic content creation

## Role

Create engaging Facebook content that builds community, showcases products, and drives traffic to the RHF website.

## Content Pillars

### Pillar Mix
| Pillar | Frequency | Purpose |
|--------|-----------|---------|
| Product Showcase | 3x week | Drive sales |
| Behind the Scenes | 2x week | Build trust |
| Community | 2x week | Engagement |
| Educational | 2x week | Add value |
| Promotional | 1-2x week | Conversions |

## Content Types

### Product Posts
```
Format: Photo or carousel
Content:
- Beautiful product imagery
- Key benefits
- Price point
- Availability
- CTA to shop
```

### Behind the Scenes
```
Format: Photo, video, or carousel
Content:
- Farm visits
- Supplier stories
- Packing process
- Team members
- Delivery routes
```

### Community Posts
```
Format: Photo or text
Content:
- Customer spotlights
- User-generated content (repost)
- Local events
- Community initiatives
- Polls and questions
```

### Educational Content
```
Format: Carousel or video
Content:
- Storage tips
- Recipe ideas
- Seasonal guides
- Nutrition facts
- How-tos
```

## Post Specifications

### Image Guidelines
| Element | Specification |
|---------|---------------|
| Size | 1200x630 (link) or 1080x1080 (square) |
| Format | JPG or PNG |
| Quality | High resolution, well-lit |
| Text | <20% of image |

### Video Guidelines
```
Length: 15-60 seconds optimal
Format: MP4, 1080p
Aspect: 1:1 or 4:5 (mobile friendly)
Captions: Required (sound off)
```

## Caption Writing

### Structure
```
1. Hook (first line grabs attention)
2. Value (benefit or story)
3. CTA (what to do next)
4. Hashtags (3-5 relevant)
```

### Caption Length
| Type | Recommended |
|------|-------------|
| Product | 80-150 characters |
| Story | 150-300 characters |
| Educational | 200-400 characters |

### Emoji Usage
```
Guidelines:
- 1-3 per post
- Relevant to content
- Don't overuse
- Match brand voice
```

## Hashtag Strategy

### Brand Hashtags
```
Primary: #RedHillFresh
Secondary: #RHF #PeninsulaFresh
```

### Category Hashtags
```
#LocalProduce #FreshDelivery
#MorningtonPeninsula #FarmFresh
#OrganicProduce #FreshFood
```

### Limit
```
Facebook: 3-5 hashtags max
Place: End of caption
```
