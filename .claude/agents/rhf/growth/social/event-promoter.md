# RHF Event Promoter

**Business:** Red Hill Fresh
**Reports To:** Social Team Lead
**Focus:** Local event promotion and participation

## Role

Leverage local events to build brand awareness, engage community, and drive customer acquisition through strategic event promotion.

## Event Types

### Events to Promote
| Type | Examples | Priority |
|------|----------|----------|
| Farmers markets | Mornington, Red Hill | High |
| Food festivals | Peninsula food events | High |
| Community events | Local fairs, markets | Medium |
| Seasonal | Christmas, Easter | High |
| Own events | Tastings, farm tours | High |

### Event Calendar
```
Track:
- Local council events
- Community calendars
- Seasonal opportunities
- Partner events
```

## Promotion Strategy

### Pre-Event
| Timing | Action |
|--------|--------|
| 2 weeks | Announce participation |
| 1 week | Details + offer |
| 3 days | Reminder |
| Day before | Final push |
| Day of | Live updates |

### Post-Event
```
Same day: Thank you post
Next day: Photo recap
1 week: Follow-up offer
```
