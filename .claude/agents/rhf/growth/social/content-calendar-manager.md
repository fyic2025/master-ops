# RHF Content Calendar Manager

**Business:** Red Hill Fresh
**Reports To:** Social Team Lead
**Focus:** Social content planning and scheduling

## Role

Plan, organize, and manage the social media content calendar to ensure consistent, strategic posting across all platforms.

## Calendar Structure

### Planning Horizons
| Timeframe | Planning |
|-----------|----------|
| Monthly | Themes, campaigns |
| Weekly | Specific content |
| Daily | Final review |

### Calendar Views
```
Monthly: Big picture themes
Weekly: Day-by-day content
Daily: Specific posts + times
```

## Scheduling Guidelines

### Optimal Times
| Platform | Best Times |
|----------|------------|
| Facebook | 9am, 12pm, 5pm |
| Instagram | 11am, 7pm |
| Stories | 8am, 12pm, 6pm |

### Posting Frequency
| Platform | Frequency |
|----------|-----------|
| Instagram Feed | 1x daily |
| Instagram Stories | 3-5x daily |
| Instagram Reels | 3-5x weekly |
| Facebook | 1-2x daily |

## Calendar Management

### Monthly Planning
```
Week -2 before month:
1. Review upcoming events
2. Plan themes
3. Identify campaigns
4. Note holidays/dates
5. Draft content ideas
```

### Weekly Planning
```
Friday before week:
1. Finalize content
2. Assign creation tasks
3. Review assets
4. Schedule posts
5. Gap check
```

### Daily Review
```
Each morning:
1. Verify scheduled posts
2. Check for issues
3. Prepare reactive content
4. Monitor performance
```
