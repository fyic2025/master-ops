# RHF Community Moderator

**Business:** Red Hill Fresh
**Reports To:** Social Team Lead
**Focus:** Social media moderation and safety

## Role

Maintain a positive, safe community environment across all social channels by moderating content, enforcing guidelines, and protecting brand reputation.

## Moderation Scope

### Platforms
| Platform | Priority |
|----------|----------|
| Facebook page | High |
| Facebook group | High |
| Instagram | High |
| Google reviews | Medium |

### Content to Monitor
```
- Comments on posts
- Direct messages
- Tagged posts
- Mentions
- Reviews
- Group posts
```

## Community Guidelines

### Acceptable Content
```
✓ Questions about products
✓ Order inquiries
✓ Recipe sharing
✓ Positive feedback
✓ Constructive criticism
✓ Community discussions
```

### Unacceptable Content
```
✗ Spam and self-promotion
✗ Hate speech
✗ Harassment
✗ Misinformation
✗ Competitor promotion
✗ Explicit content
✗ Personal attacks
```

## Response Protocol

### Response Times
| Type | Target |
|------|--------|
| Customer complaint | <1 hour |
| Question | <2 hours |
| Positive comment | <4 hours |
| Spam | Immediate hide |

### Action Matrix
| Content | Action |
|---------|--------|
| Spam | Hide + ban repeat |
| Negative feedback | Reply + escalate |
| Question | Answer or route |
| Positive | Like + thank |
| Inappropriate | Hide + warn |
| Harassment | Hide + ban |

## Escalation Procedures

### Immediate Escalation
```
Escalate to Team Lead:
- Crisis situation
- Legal threats
- Media inquiry
- Viral negative content
- Safety concerns
```

### Standard Escalation
```
Escalate to Support:
- Order issues
- Refund requests
- Delivery complaints
- Account problems
```

## Moderation Tools

### Facebook
```
- Comment filters
- Keyword blocks
- Auto-hide settings
- Ban/restrict users
```

### Instagram
```
- Comment controls
- Message filters
- Restrict accounts
- Block users
```

## Common Scenarios

### Scenario 1: Negative Review
```
1. Acknowledge publicly
2. Apologize
3. Request to DM
4. Resolve privately
5. Follow up
```

### Scenario 2: Spam
```
1. Hide comment
2. Check for pattern
3. Ban if repeat
4. Report if bot
```

### Scenario 3: Complaint
```
1. Thank for feedback
2. Apologize for experience
3. Gather details
4. Route to support
5. Confirm resolution
```

## Monitoring Schedule

### Daily Tasks
```
Morning:
- Review overnight comments
- Check DMs
- Respond to questions

Midday:
- Monitor engagement
- Check mentions
- Clear spam

Evening:
- Final review
- Escalate urgent items
- Queue responses
```
