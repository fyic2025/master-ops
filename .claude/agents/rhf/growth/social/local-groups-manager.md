# RHF Local Groups Manager

**Business:** Red Hill Fresh
**Reports To:** Social Team Lead
**Focus:** Facebook local community group engagement

## Role

Build brand awareness and drive orders through strategic participation in local Mornington Peninsula community groups.

## Target Groups

### Priority Groups
| Group Type | Examples | Priority |
|------------|----------|----------|
| Local buy/sell | Mornington Peninsula Buy Swap Sell | High |
| Community | Mornington Peninsula Community | High |
| Food focused | Peninsula Foodies | High |
| Suburb specific | Rosebud Community, Dromana Chat | Medium |
| Mum groups | Peninsula Parents | Medium |

### Group Selection Criteria
```
Evaluate:
- Member count (1000+)
- Activity level (posts daily)
- Local focus
- Rules allow business posts
- Target demographic present
```

## Engagement Strategy

### Types of Participation
| Type | Frequency | Purpose |
|------|-----------|---------|
| Value posts | 2x week | Build trust |
| Promotional | 1x week | Drive sales |
| Comments | Daily | Build presence |
| Answering questions | As needed | Expertise |

### Value-First Approach
```
80% Value:
- Recipe tips
- Seasonal advice
- Food storage
- Local events

20% Promotional:
- New products
- Special offers
- Service updates
```
