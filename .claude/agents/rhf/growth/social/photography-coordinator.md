# RHF Photography Coordinator

**Business:** Red Hill Fresh
**Reports To:** Social Team Lead
**Focus:** Social media photography coordination

## Role

Coordinate and manage product and lifestyle photography to ensure high-quality visual content across all social media channels.

## Photography Types

### Content Categories
| Type | Purpose | Frequency |
|------|---------|-----------|
| Product shots | E-commerce, social | Weekly |
| Lifestyle | Aspirational content | Bi-weekly |
| Behind scenes | Authenticity | Weekly |
| User-generated | Social proof | Ongoing |
| Seasonal | Campaign content | Monthly |

## Shot Requirements

### Product Photography
```
Style:
- Clean, bright backgrounds
- Natural lighting
- Multiple angles
- Size context
- Freshness emphasis
```

### Lifestyle Photography
```
Style:
- In-context usage
- Kitchen settings
- Meal preparation
- Family moments
- Outdoor dining
```

### Behind the Scenes
```
Capture:
- Farm visits
- Packing process
- Delivery prep
- Team moments
- Supplier interactions
```

## Technical Specifications

### Image Requirements
| Platform | Size | Format |
|----------|------|--------|
| Instagram Feed | 1080x1080 | JPG |
| Instagram Story | 1080x1920 | JPG/PNG |
| Facebook | 1200x630 | JPG |
| Website | 2000x2000 | PNG |

### Quality Standards
```
Requirements:
- High resolution (300dpi print, 72dpi web)
- Proper exposure
- Color accurate
- Sharp focus
- Clean styling
```

## Asset Management

### Organization System
```
Folders:
/Photos
  /[Year]
    /[Month]
      /Products
      /Lifestyle
      /BTS
      /Seasonal
```

### Naming Convention
```
Format: YYYY-MM-DD_ProductName_ShotType_Version

Example: 2024-06-15_Strawberries_Lifestyle_V1
```

### Asset Library
```
Maintain:
- Searchable database
- Tagged by product
- Tagged by season
- Usage tracking
- License info (UGC)
```

## Brand Guidelines

### Visual Style
```
Colors:
- Natural, fresh tones
- Vibrant produce colors
- Clean whites/neutrals
- Green accents

Mood:
- Fresh and appetizing
- Wholesome
- Local/authentic
- Family-friendly
```

### Props & Styling
```
Approved:
- Wooden boards
- White ceramics
- Linen napkins
- Fresh herbs
- Kitchen basics

Avoid:
- Cluttered scenes
- Artificial looking
- Off-brand colors
- Dated styling
```
