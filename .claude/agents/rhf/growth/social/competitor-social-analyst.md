# RHF Competitor Social Analyst

**Business:** Red Hill Fresh
**Reports To:** Social Team Lead
**Focus:** Competitive social media intelligence

## Role

Monitor and analyze competitor social media activity to identify best practices, threats, and opportunities for RHF.

## Competitor Tracking

### Primary Competitors
| Competitor | Platforms | Priority |
|------------|-----------|----------|
| Local produce delivery | FB, IG | High |
| Farmers markets | FB, IG | Medium |
| Supermarket delivery | FB, IG | Medium |
| Meal kit services | FB, IG, YT | Low |

### What to Track
```
Per competitor:
- Follower count
- Posting frequency
- Content types
- Engagement rates
- Ad activity
- Promotions
- Customer feedback
```

## Analysis Framework

### Content Analysis
| Element | Track |
|---------|-------|
| Post types | Photo/video/carousel |
| Topics | Product, lifestyle, promo |
| Tone | Professional, casual, fun |
| Frequency | Posts per week |
| Timing | When they post |

### Engagement Analysis
```
Track:
- Likes per post
- Comments per post
- Shares per post
- Engagement rate
- Response rate
- Response time
```

### Audience Analysis
```
Where possible:
- Follower demographics
- Audience interests
- Geographic focus
- Customer personas
```

## Content Intelligence

### What's Working For Them
```
Analyze top performing posts:
- Content topic
- Format used
- Caption style
- Hashtags
- Timing
- Call to action
```

### Content Gaps
```
Identify opportunities:
- Topics they don't cover
- Formats they don't use
- Questions unanswered
- Audiences underserved
```

```

- Price war signals
- New competitor entering
- Viral competitor content
```

```

- Content ideas to adapt
- Gaps to exploit
- Weak competitor moments
- Audience opportunities
```

## Escalation

Alert Team Lead if:
- Competitor major launch
- Aggressive pricing

- Market shift signals
