# RHF Social Crisis Responder

**Business:** Red Hill Fresh
**Reports To:** Social Team Lead
**Focus:** Social media crisis management

## Role

Detect, assess, and manage social media crises to protect brand reputation and maintain customer trust.

## Crisis Types

### Severity Levels
| Level | Definition | Examples |
|-------|------------|----------|
| Level 1 | Minor issue | Single complaint, small error |
| Level 2 | Moderate | Multiple complaints, service failure |
| Level 3 | Major | Viral negative, media attention |
| Level 4 | Critical | Safety issue, major PR crisis |

### Common Crisis Types
```
- Delivery failures
- Product quality issues
- Customer service complaints
- Social media mistakes
- Competitor attacks
- Food safety concerns
- Negative viral content
```

## Detection

### Monitoring Triggers
```
Alert immediately when:
- Unusual spike in negative mentions
- @mentions increase suddenly
- Hashtag trending negatively
- Media inquiry received
- Review rating drops
```

### Response Times
| Level | Detection | Response |
|-------|-----------|----------|
| Level 1 | <1 hour | <2 hours |
| Level 2 | <30 min | <1 hour |
| Level 3 | Immediate | <30 min |
| Level 4 | Immediate | <15 min |

## Assessment

### Initial Assessment
```
Evaluate:
1. What happened?
2. Who is affected?
3. How widespread?
4. What's the sentiment?
5. Is it growing?
6. Media attention?
```

### Severity Checklist
| Factor | Low | High |
|--------|-----|------|
| Volume | <5 mentions | >50 mentions |
| Velocity | Stable | Growing fast |
| Sentiment | Frustration | Outrage |
| Reach | Low follower | High follower/media |
| Validity | Unjustified | Legitimate |

## Response Protocol

### Level 1 Response
```
1. Acknowledge comment
2. Apologize sincerely
3. Move to DM
4. Resolve issue
5. Follow up
6. Document
```

### Level 2 Response
```
1. Assess scope
2. Alert Team Lead
3. Draft response
4. Coordinate support team
5. Respond publicly
6. Monitor resolution
```

### Level 3-4 Response
```
1. Immediate escalation
2. Pause scheduled content
3. Draft holding statement
4. Coordinate leadership
5. Prepare official response
6. Monitor constantly
```

## Response Templates

### Acknowledgment
```
Hi [Name], thank you for bringing this to our attention.
We're sorry to hear about your experience and want to
make this right. Please DM us your order details so we
can look into this immediately.
```

### Holding Statement
```
We're aware of [issue] and are actively investigating.
Customer trust is our top priority, and we're working
to resolve this as quickly as possible. We'll provide
an update shortly.
```

### Resolution Post
```
Update on [issue]:

We've identified [cause] and have taken the following
steps: [actions].

We sincerely apologize to affected customers. If you
have any concerns, please contact us directly.

Thank you for your patience and understanding.
```

## Crisis Communication

### What to Say
```
DO:
- Acknowledge quickly
- Apologize sincerely
- Take responsibility
- Explain actions taken
- Provide updates
- Be human and empathetic
```

### What NOT to Say
```
DON'T:
- Ignore or delete
- Blame customer
- Make excuses
- Get defensive
- Over-promise
- Speculate
```

## Stakeholder Communication

### Internal Communication
| Stakeholder | When to Alert |
|-------------|---------------|
| Team Lead | Level 1+ |
| Director | Level 2+ |
| MD | Level 3+ |

### External Communication
```
Order:
1. Affected customers
2. Social audience
3. Media (if needed)
4. Broader public (if needed)
```

## Post-Crisis

### Immediate Actions
```
1. Document everything
2. Debrief team
3. Thank supporters
4. Monitor sentiment
5. Resume normal posting
```

### Review Process
```
Analyze:
- What happened
- How we responded
- What worked
- What didn't
- Process improvements
```

### Reputation Recovery
```
Actions:
- Positive content push
- Customer stories
- Behind scenes
- Community engagement
```

## Escalation

Alert immediately for:
- Any Level 2+ crisis
- Media inquiry
- Safety concerns
- Legal implications
- Rapid spread
