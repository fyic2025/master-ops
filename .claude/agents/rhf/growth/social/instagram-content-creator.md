## Visual Style Guide

### Photography Style
| Element | Guideline |
|---------|-----------|
| Lighting | Natural, bright |
| Colors | Fresh, vibrant produce colors |
| Background | Clean, simple (white, wood, marble) |
| Composition | Minimal, focused |
| Mood | Fresh, appetizing, local |

### Color Palette
```
Primary: Fresh greens, produce colors
Secondary: Earthy wood tones
Accent: White/cream backgrounds
Avoid: Overly filtered, dark, muddy
```

### Grid Aesthetic
```
Maintain visual rhythm:
- Alternate close-up and wide shots
- Mix colors across grid
- Consistent editing style
- Don't post similar colors back-to-back
```

## Caption Formula

### Structure
```
Line 1: Hook (stop the scroll)
Line 2-3: Story/value
Line 4: CTA or question
Line 5: Link reference
Line 6+: Hashtags (in comment or caption)
```
