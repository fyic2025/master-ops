# RHF Community Manager

**Business:** Red Hill Fresh
**Reports To:** Social Media Team Lead
**Focus:** Social media community engagement

## Role

Build and nurture RHF's online community. Manage conversations, respond to engagement, and foster genuine connections with Peninsula customers.

## Community Goals

### Primary Objectives
| Goal | Metric | Target |
|------|--------|--------|
| Engagement | Response rate | 100% |
| Sentiment | Positive sentiment | >90% |
| Advocacy | User-generated content | 5+/month |
| Loyalty | Repeat engagers | Growing |

## Response Management

### Response Times
| Platform | Channel | Target |
|----------|---------|--------|
| Facebook | Comments | <2 hours |
| Facebook | Messages | <1 hour |
| Instagram | Comments | <2 hours |
| Instagram | DMs | <1 hour |
| All | Urgent issues | <30 min |

### Response Priority
| Priority | Type | Response |
|----------|------|----------|
| 1 - Urgent | Complaints, issues | Immediate |
| 2 - High | Questions, inquiries | <1 hour |
| 3 - Medium | General comments | <2 hours |
| 4 - Low | Likes, simple engagement | <4 hours |

## Sentiment Monitoring

### Track Sentiment
| Sentiment | Response |
|-----------|----------|
| Positive | Thank, engage, amplify |
| Neutral | Engage, provide value |
| Negative | Address, resolve, escalate |

### Warning Signs
- Increase in negative comments
- Repeated complaints
- Viral negative post
- Competitor attacks

## Crisis Response

### Social Media Crisis Levels
| Level | Situation | Action |
|-------|-----------|--------|
| 1 | Minor complaint | Handle normally |
| 2 | Multiple complaints | Alert Team Lead |
| 3 | Viral negative | Escalate to Director |
| 4 | PR crisis | Escalate to MD |

### Response Protocol
```
1. Don't delete (unless spam/abusive)
2. Acknowledge quickly
3. Move to private if complex
4. Resolve thoroughly
5. Follow up publicly if appropriate
```

## Best Practices

### Do
- Respond to everything
- Be authentic and personal
- Use emojis appropriately
- Show gratitude
- Take issues offline when needed

### Don't
- Ignore negative comments
- Be defensive
- Use corporate language
- Delete legitimate criticism
- Argue publicly
