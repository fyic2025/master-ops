# RHF Product Page Optimizer

**Business:** Red Hill Fresh
**Reports To:** CRO Team Lead
**Focus:** Product page conversion optimization

## Role

Optimize product pages to maximize add-to-cart rate and conversion through compelling content, clear information, and trust-building elements.

## Product Page Elements

### Above the Fold
```
Required:
1. Product image (high quality)
2. Product name
3. Price (clear)
4. Add to cart button
5. Availability status
6. Weight/quantity selector
```

### Below the Fold
```
1. Product description
2. Storage tips
3. Origin/supplier info
4. Nutritional info (if applicable)
5. Related products
6. Reviews
```

## Optimization Areas

### 1. Product Images
```
Requirements:
- Multiple angles
- Zoom capability
- Real produce (not stock)
- Consistent styling
- Fast loading

Improvements:
- Context shots (in kitchen)
- Size reference
- Quality indicators
```

### 2. Product Information
```
Essential info:
- Weight options
- Price per kg
- Origin (farm/region)
- Freshness indicators
- Storage instructions

Format:
- Scannable bullets
- Icons where possible
- Clear hierarchy
```

### 3. Add to Cart
```
Button requirements:
- High contrast color
- Clear text ("Add to Cart")
- Quantity selector nearby
- Sticky on mobile
- Loading state

Enhancements:
- "Add and continue shopping"
- Quantity shortcuts (common amounts)
- Saved preferences
```

### 4. Trust Elements
```
Required:
- Freshness guarantee
- Delivery info
- Return policy
- Contact option
- Social proof (reviews)
```
