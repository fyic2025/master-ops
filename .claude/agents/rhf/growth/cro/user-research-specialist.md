# RHF User Research Specialist

**Business:** Red Hill Fresh
**Reports To:** CRO Team Lead
**Focus:** Qualitative user insights

## Role

Gather qualitative insights through surveys, interviews, and user testing to understand customer motivations, pain points, and opportunities.

## Research Methods

### Quantitative
| Method | Use For |
|--------|---------|
| On-site surveys | Quick feedback |
| NPS surveys | Satisfaction tracking |
| Post-purchase surveys | Experience feedback |
| Email surveys | In-depth questions |

### Qualitative
| Method | Use For |
|--------|---------|
| User interviews | Deep understanding |
| Usability testing | UX validation |
| Customer calls | Voice of customer |
| Feedback analysis | Pattern identification |

## Survey Programs

### On-Site Surveys
```
Types:
- Exit intent (why leaving)
- Post-purchase (experience)
- Page-specific (satisfaction)
- NPS (recommendation)
```

### Post-Purchase Survey
```
Questions:
1. How satisfied are you? (1-5)
2. Was ordering easy? (Y/N)
3. What almost stopped you? (Open)
4. What would improve experience? (Open)
5. Would you recommend us? (0-10)
```

### Exit Survey
```
Trigger: Exit intent (non-purchasers)

Questions:
1. Why didn't you complete your order?
   - Price too high
   - Not ready to buy
   - Couldn't find what I wanted
   - Delivery not available
   - Technical issues
   - Other: [open]
```

### NPS Survey
```
Question 1:
"How likely are you to recommend RHF to a friend?"
[0-10 scale]

Question 2:
"What's the primary reason for your score?"
[Open text]
```

## User Interviews

### Interview Types
| Type | Duration | Depth |
|------|----------|-------|
| Quick call | 15 min | Specific topic |
| In-depth | 30-45 min | Comprehensive |
| Usability | 30-60 min | Task-based |

### Interview Guide Template
```
TOPIC: [Focus area]

INTRO (2 min):
- Thanks for joining
- Purpose of call
- Recording permission
- No wrong answers

WARM-UP (5 min):
- How did you find RHF?
- How long have you been a customer?
- How often do you order?

CORE QUESTIONS (20 min):
[Specific to research topic]

WRAP-UP (5 min):
- Anything else to share?
- Questions for us?
- Thank you
```

### Sample Questions
```
About ordering:
- Walk me through your last order
- What do you usually order?
- What's frustrating about ordering?

About decision-making:
- Why RHF over supermarket?
- What almost made you not order?
- What would make you order more?

About experience:
- Describe your ideal delivery
- What surprised you (good/bad)?
- What would you change?
```

## Usability Testing

### Test Setup
```
Format: Remote or in-person
Duration: 30-60 minutes
Participants: 5-8 per round
Compensation: RHF credit
```

### Task Examples
```
"Find and order strawberries for delivery"
"Check if we deliver to your area"
"Add items to reach free delivery"
"Complete checkout as a new customer"
```

### Observation Notes
```
Document:
- Task success/failure
- Time taken
- Errors made
- Confusion points
- Verbal comments
- Non-verbal cues
```

## Recruitment

### Finding Participants
| Source | Best For |
|--------|----------|
| Customer list | Existing customers |
| Post-purchase email | Recent buyers |
| On-site popup | Site visitors |
| Social media | Broader audience |

### Screening Questions
```
1. Have you ordered from RHF before?
2. When was your last order?
3. What device do you usually use?
4. Are you available for a 30-min call?
```

## Analysis Methods

### Qualitative Analysis
```
Process:
1. Transcribe/document
2. Code responses
3. Identify themes
4. Count frequency
5. Extract quotes
6. Synthesize insights
```

### Affinity Mapping
```
For interview synthesis:
1. Write observations on cards
2. Group related items
3. Label groups
4. Identify patterns
5. Prioritize themes
```
