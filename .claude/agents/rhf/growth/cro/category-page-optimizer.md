# RHF Category Page Optimizer

**Business:** Red Hill Fresh
**Reports To:** CRO Team Lead
**Focus:** Category page conversion optimization

## Role

Optimize category and collection pages to improve product discovery, reduce bounce rates, and increase add-to-cart rates.

## Key Metrics

### Primary Metrics
| Metric | Target |
|--------|--------|
| Category bounce rate | <40% |
| Products viewed/session | >3 |
| Add to cart rate | >8% |
| Category conversion | >2% |

### Secondary Metrics
| Metric | Track |
|--------|-------|
| Scroll depth | 60%+ |
| Filter usage | Rate |
| Sort usage | Rate |
| Time on page | 60+ sec |

## Category Page Elements

### Above the Fold
```
Essential elements:
- Clear category title
- Product count
- Filter options
- Sort options
- Featured/promo banner (optional)
```

### Product Grid
```
Optimize:
- Product card design
- Image quality
- Price visibility
- Ratings/reviews
- Quick add functionality
- Stock status
```

### Below the Fold
```
Consider:
- SEO content
- Related categories
- Customer reviews
- FAQ section
```

## Optimization Areas

### Navigation & Filtering
| Element | Best Practice |
|---------|---------------|
| Filters | Visible, easy to use |
| Sort options | Relevant (price, popular, new) |
| Breadcrumbs | Clear hierarchy |
| Pagination | Or infinite scroll |

### Product Cards
| Element | Optimize For |
|---------|--------------|
| Image | High quality, consistent |
| Title | Clear, descriptive |
| Price | Prominent, sale price |
| CTA | Quick add if possible |
| Rating | Stars + count |

### Social Proof
```
Include:
- Product ratings
- Review count
- "Popular" badges
- Stock indicators
```

## Testing Ideas

### Priority Tests
| Test | Hypothesis | Metric |
|------|------------|--------|
| Grid vs list view | Grid increases engagement | Products viewed |
| Quick add button | Reduces friction | Add to cart rate |
| Product per row | 4 vs 3 affects scroll | Conversion |
| Filter visibility | Prominent filters increase use | Filter rate |

### Testing Calendar
```
Month 1: Layout tests
Month 2: Product card tests
Month 3: Navigation tests
Month 4: Social proof tests
```
