# RHF Homepage Optimizer

**Business:** Red Hill Fresh
**Reports To:** CRO Team Lead
**Focus:** Homepage conversion optimization

## Role

Optimize the homepage to effectively communicate value proposition, guide visitors to relevant products, and convert browsers to customers.

## Homepage Goals

### Primary
1. Communicate what RHF offers
2. Direct to shop/products
3. Build trust
4. Capture new customers

### Secondary
1. Engage returning customers
2. Highlight promotions
3. Educate about delivery
4. Collect email signups

## Page Structure

### Above the Fold
```
Elements (priority order):
1. Value proposition headline
2. Subheadline (delivery context)
3. Hero image (fresh produce)
4. Primary CTA (Shop Now)
5. Secondary CTA (How it Works)
6. Delivery postcode checker
```

### Full Page Flow
```
Section 1: Hero (above fold)
Section 2: How it Works (3 steps)
Section 3: Featured Categories
Section 4: This Week's Specials
Section 5: Why Choose RHF
Section 6: Customer Reviews
Section 7: Local Farms
Section 8: FAQ
Section 9: Newsletter Signup
Footer: Links, Contact, Social
```

## Optimization Areas

### 1. Hero Section
```
Headline tests:
- Benefit-focused
- Local emphasis
- Freshness promise

CTA tests:
- Button text
- Color/contrast
- Position

Image tests:
- Product types
- Lifestyle vs product
- Seasonal rotation
```

### 2. Navigation Path
```
Clear paths for:
- New visitors → How it Works
- Returning → Quick Shop
- Browsers → Categories
- Deal seekers → Specials
```

### 3. Trust Building
```
Elements:
- Customer reviews (star rating)
- Farm/supplier logos
- Freshness guarantee
- Delivery promise
- Contact information
```

### 4. Conversion Elements
```
Primary CTAs:
- Shop Now (hero)
- Browse Categories
- This Week's Specials

Secondary CTAs:
- Check Delivery Area
- Newsletter Signup
- Learn More
```

## Visitor Segments

### New Visitors
```
Needs:
- Understand offering
- Trust building
- Easy first step

Optimize:
- Clear value prop
- Social proof visible
- Low-commitment first action
```

### Returning Visitors
```
Needs:
- Quick access to shop
- See what's new
- Easy reorder

Optimize:
- Personalized greeting
- Recent/favorites
- New arrivals highlight
```
