# RHF CTA Optimizer

**Business:** Red Hill Fresh
**Reports To:** CRO Team Lead
**Focus:** Call-to-action optimization

## Role

Optimize all calls-to-action across the website to maximize click-through rates, conversions, and user engagement.

## CTA Best Practices

### Copy Guidelines
| Type | Approach | Example |
|------|----------|---------|
| Action | Verb-first | "Add to Cart" |
| Benefit | Value-focused | "Get Fresh Produce" |
| Urgency | Time-sensitive | "Order by 4pm" |
| Personal | User-centered | "Start My Order" |

### Visual Design
```
Requirements:
- Contrasting color
- Adequate size (min 44x44px mobile)
- Clear text
- Sufficient padding
- Obvious button styling
```

### Placement
```
Position:
- Above the fold for primary
- After value proposition
- Multiple for long pages
- Sticky on mobile
```

## CTA by Page Type

### Homepage CTAs

### Cart CTAs
```

```

## Testing Framework

### CTA Test Types
| Element | Variations |
|---------|------------|
| Copy | Action vs benefit |
| Color | Brand vs contrast |
| Size | Standard vs prominent |
| Placement | Position variations |
| Design | Button vs link style |

### Priority Tests
| Test | Hypothesis |
|------|------------|
| "Add to Cart" vs "Add to Bag" | Familiar term performs better |
| Red vs green CTA | Contrast improves clicks |
| Static vs sticky mobile | Sticky improves conversion |
| First vs third person | "Start My Order" vs "Start Your Order" |

## Mobile Optimization

### Mobile CTA Requirements
```
- Minimum 44x44px tap target
- Thumb-reachable position
- Sticky where appropriate
- Fast load/response
- Clear active states
```

### Sticky CTAs
```
Use for:
- Add to cart (product)
- Checkout (cart)
- Primary conversion action

Include:
- Price/total
- Clear CTA button
- Minimal height
```
