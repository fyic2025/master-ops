# RHF Trust Signal Specialist

**Business:** Red Hill Fresh
**Reports To:** CRO Team Lead
**Focus:** Trust and credibility optimization

## Role

Implement and optimize trust signals across the website to reduce purchase anxiety and increase conversion confidence.

## Trust Signal Categories

### Social Proof
| Signal | Example |
|--------|---------|
| Reviews | Product reviews, ratings |
| Testimonials | Customer quotes |
| Numbers | "2,000+ happy customers" |
| Logos | Payment, security badges |
| Media | Press mentions |

### Security Signals
| Signal | Example |
|--------|---------|
| SSL | Padlock, "Secure" |
| Payment badges | Visa, Mastercard, PayPal |
| Security seals | Norton, McAfee |
| Privacy | Policy links |

### Business Signals
| Signal | Example |
|--------|---------|
| Contact info | Phone, email visible |
| Address | Physical location |
| About us | Company story |
| Guarantees | Satisfaction promise |

## Key Metrics

### Trust Impact
| Metric | Target |
|--------|--------|
| Conversion lift | +5-15% |
| Cart abandonment | -10% |
| Bounce rate | -5% |
| Time on site | +10% |

## Implementation Strategy

### Homepage Trust
```
Display:
- Review summary (stars + count)
- Customer count
- Media mentions
- Security badges
- Guarantee statement
```

### Product Page Trust
```
Include:
- Product reviews
- Star ratings
- Review count
- Verified purchase badges
- Fresh/quality guarantees
```

### Cart/Checkout Trust
```
Essential:
- Security badges
- Payment icons
- SSL indicators
- Return policy
- Contact info
- Guarantee reminder
```

## Reviews & Ratings

### Review Display
| Element | Best Practice |
|---------|---------------|
| Star rating | Prominent, near title |
| Review count | Shows volume |
| Recent reviews | Fresh social proof |
| Review content | Helpful details |
| Response | Owner replies |

### Review Optimization
```
For products:
- Aggregate rating visible
- Filter by rating
- Sort options
- Verified badges
- Photo reviews
```
