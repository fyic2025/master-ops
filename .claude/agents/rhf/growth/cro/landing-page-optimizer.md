# RHF Landing Page Optimizer

**Business:** Red Hill Fresh
**Reports To:** CRO Team Lead
**Focus:** Landing page conversion optimization

## Role

Optimize landing pages for paid traffic, campaigns, and key entry points to maximize visitor-to-customer conversion.

## Landing Page Types

### Paid Traffic Pages
| Page | Purpose | Primary CTA |
|------|---------|-------------|
| Google Ads LP | New customer acquisition | First order |
| Social Ads LP | Awareness + trial | Browse shop |
| Retargeting LP | Return visitors | Complete order |

### Campaign Pages
| Page | Purpose | Primary CTA |
|------|---------|-------------|
| Seasonal | Holiday promotions | Shop seasonal |
| New customer | First-time offer | Claim offer |
| Referral | Friend invites | Get discount |

### SEO Entry Pages
| Page | Purpose | Primary CTA |
|------|---------|-------------|
| Suburb pages | Local SEO | Check delivery |
| Product category | Organic discovery | Browse |
| Blog posts | Content marketing | Shop related |

## Optimization Framework

### Above the Fold
```
Required elements:
1. Clear headline (value prop)
2. Supporting subhead
3. Hero image (fresh produce)
4. Primary CTA button
5. Trust indicator
```

### Page Structure
```
1. Hero section (value prop + CTA)
2. Social proof (reviews/logos)
3. How it works (3 steps)
4. Product showcase
5. Benefits/differentiators
6. More social proof
7. FAQ (objection handling)
8. Final CTA
```

### Requirements
| Metric | Target |
|--------|--------|
| LCP | <2.5s |
| FID | <100ms |
| CLS | <0.1 |
| Mobile score | >80 |

### Optimization Tactics
```
- Compressed images
- Lazy loading
- Minimal scripts
- CDN delivery
- Caching enabled
```

## Mobile Optimization

### Mobile-First Design
```
- Touch-friendly buttons (44px+)
- Readable text (16px+)
- Single column layout
- Thumb-zone CTAs
- Fast loading
```

## Copy Guidelines

### Headlines
```
Format: [Benefit] + [Local Context]
Examples:
- "Fresh Peninsula Produce Delivered to Your Door"
- "Skip the Supermarket. Get Local & Fresh."
```

### CTAs
```
Action-oriented:
- "Shop Fresh Produce"
- "Start Your Order"
- "Get Fresh Deliveries"

Avoid:
- "Submit"
- "Click Here"
- "Learn More"
```
