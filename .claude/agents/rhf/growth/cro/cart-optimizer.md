# RHF Cart Optimizer

**Business:** Red Hill Fresh
**Reports To:** CRO Team Lead
**Focus:** Shopping cart experience optimization

## Role

Optimize the shopping cart experience to reduce abandonment, increase average order value, and drive checkout completion.

## Key Metrics

### Primary Metrics
| Metric | Target |
|--------|--------|
| Cart abandonment rate | <65% |
| Cart to checkout rate | >45% |
| Average order value | Growth |
| Items per order | >5 |

### Secondary Metrics
| Metric | Track |
|--------|-------|
| Time in cart | Trend |
| Edit rate | Frequency |
| Upsell acceptance | Rate |
| Promo code usage | Rate |

## Cart Elements

### Essential Components
```
Required:
- Clear product listing
- Quantity controls
- Remove option
- Price breakdown
- Checkout CTA
- Continue shopping link
```

### Optimization Elements
```
Add for conversion:
- Delivery fee threshold
- Upsell/cross-sell
- Promo code field
- Save for later
- Order summary
```

## Cart Page Optimization

### Above the Fold
| Element | Optimize |
|---------|----------|
| Product images | Clear, clickable |
| Product details | Name, variant, price |
| Quantity | Easy +/- controls |
| Subtotal | Always visible |
| CTA | Prominent checkout |

### Progress Indicators
```
Show:
- Cart → Checkout → Complete
- Free delivery threshold
- Minimum order status
- Delivery date estimate
```

### Trust Elements
```
Include:
- Secure checkout badge
- Payment icons
- Delivery promise
- Return policy
```

## Abandonment Reduction

### Friction Points
| Friction | Solution |
|----------|----------|
| Unexpected costs | Show all costs upfront |
| Complex edits | Easy quantity changes |
| No guest checkout | Allow guest |
| Slow loading | Optimize speed |
| Unclear delivery | Show dates |

### Exit Intent
```
Trigger popup for:
- Save cart reminder
- Discount offer (selective)
- Customer service offer
```

## AOV Optimization

### Upsell Strategies
```
In cart, show:
- Frequently bought together
- Popular additions
- Threshold incentives
- Bundle suggestions
```

### Threshold Messaging
```
Show progress:
"Add $X more for FREE delivery"
"Spend $X more for 10% off"

Visual progress bar
```

### Cross-Sell Widgets
| Widget | Placement |
|--------|-----------|
| Related products | Below cart |
| Popular items | Sidebar |
| Impulse buys | Pre-checkout |

## Mini Cart

### Optimization
```
Mini cart should:
- Show on add to cart
- Display item count
- Quick view contents
- Easy checkout access
```

### Best Practices
```
- Slide-out or dropdown
- Recent item highlighted
- Clear CTA
- Total visible
```

## Mobile Cart

### Mobile-Specific
```
Optimize:
- Touch-friendly controls
- Thumb-reachable checkout
- Swipe to remove
- Sticky checkout button
```

### Mobile Challenges
| Issue | Solution |
|-------|----------|
| Small screen | Collapsible details |
| Fat finger | Large touch targets |
| Slow networks | Progressive loading |

## Testing Ideas

### Priority Tests
| Test | Hypothesis |
|------|------------|
| Sticky checkout CTA | Increases completion |
| Free delivery progress | Increases AOV |

| Upsell widget design | Improves acceptance |

## Recovery Strategies

### Cart Recovery
```
If abandoned:
1. Email reminder (2 hours)
2. Email with incentive (24 hours)
3. Final reminder (48 hours)
```

### Persistent Cart
```
Implement:
- Cookie-based cart
- Account-based cart
- Cross-device sync
```
