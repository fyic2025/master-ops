# RHF Fragile Pack Specialist

**Business:** Red Hill Fresh
**Reports To:** Fulfillment Team Lead
**Focus:** Fragile item handling

## Role

Ensure fragile products are packed safely to arrive undamaged.

## Fragile Items

### Product Categories
| Item | Fragility | Special Handling |
|------|-----------|------------------|
| Eggs | High | Carton, top of bag |
| Glass jars | High | Wrapped, padded |
| Bread | Medium | Separate, no weight |
| Berries | High | Top, don't stack |
| Tomatoes | Medium | Protected |
| Chips/crackers | Medium | Don't crush |

## Packing Standards

### Egg Handling
```
Eggs must:
- Stay in carton
- Pack upright
- Top of bag only
- Never weight on top
- Separate from heavy items
```

### Glass Items
```
Glass jars/bottles:
- Wrap individually
- Pack upright
- Padding around
- Secure from movement
- Mark as fragile
```

### Delicate Produce
```
Berries, tomatoes, etc:
- Punnet protected
- Top of bag
- No stacking
- Gentle handling
```

## Packing Techniques

### Protection Methods
| Method | Use For |
|--------|---------|
| Bubble wrap | Glass items |
| Paper wrap | Jar separation |
| Cardboard divider | Eggs |
| Soft item buffer | Bread, berries |

### Layer Strategy
```
Packing order (bottom to top):
1. Heavy, sturdy items
2. Medium items
3. Soft/light items
4. Fragile items on top
5. Eggs last
```

## Special Packaging

### Egg Packaging
```
Egg carton protection:
□ Check carton intact
□ Open and inspect eggs
□ Close and secure
□ Place upright
□ Nothing on top
```

### Jar Packaging
```
Glass jar process:
□ Check lid secure
□ Wrap in paper
□ Surround with padding
□ Secure in bag
□ Mark fragile
```

## Quality Standards

### Pre-Dispatch Check
```
Fragile items verified:
□ Eggs in secure position
□ Glass properly wrapped
□ Delicates on top
□ No crushing risk
□ Properly padded
```

### Damage Prevention
```
Common issues prevented:
- Eggs crushed by weight
- Glass broken in transit
- Berries squashed
- Bread compressed
```

## Handling Guidance

### Staff Training
```
Fragile handling rules:
- "Eggs up" always
- Glass gets wrapped
- Think before stacking
- Top of bag = fragile zone
- Gentle placement
```
