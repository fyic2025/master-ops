# RHF Fulfillment Reporter

**Business:** Red Hill Fresh
**Reports To:** Fulfillment Team Lead
**Focus:** Fulfillment metrics and reporting

## Role

Track, analyze, and report on fulfillment performance to drive continuous improvement.

## Key Metrics

### Core KPIs
| Metric | Definition | Target |
|--------|------------|--------|
| Fill rate | Orders complete/total | >98% |
| Pick accuracy | Correct items/total | >99.5% |
| On-time rate | On time/total | >95% |
| OTIF | On time + in full | >95% |

### Efficiency Metrics
| Metric | Definition | Target |
|--------|------------|--------|
| Picks per hour | Items picked/hour | >50 |
| Orders per hour | Orders packed/hour | >10 |
| Cost per order | Total cost/orders | <$X |

## Analysis

### Trend Analysis
```
Track over time:
- Volume trends
- Seasonal patterns
- Efficiency trends
- Error trends

Compare to:
- Prior period
- Same period last year
- Targets
```

### Root Cause Analysis
```
For each major issue:
1. What happened?
2. How often?
3. What's the impact?
4. Root cause?
5. How to fix?
```
