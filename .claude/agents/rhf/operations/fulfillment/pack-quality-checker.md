# RHF Pack Quality Checker

**Business:** Red Hill Fresh
**Reports To:** Fulfillment Team Lead
**Focus:** Packing quality assurance

## Role

Perform quality checks on packed orders to ensure accuracy and customer satisfaction.

## QC Process

### Check Types
| Type | Frequency | Scope |
|------|-----------|-------|
| Random | 10% of orders | Full check |
| New packer | 100% first week | Full check |
| High value | 100% | Full check |
| Customer issue | 100% | Full check |

### QC Workflow
```
1. Select orders for check
2. Open and verify
3. Check packing quality
4. Document findings
5. Repack if needed
6. Release or return
```

## Quality Standards

### Accuracy Check
```
Verify:
□ All items present
□ Quantities correct
□ Right products (no swaps)
□ Substitutions correct
□ No extras
```

### Packing Quality
```
Check:
□ Proper container used
□ Weight distribution
□ Fragile items protected
□ Cold items with cold pack
□ Clean presentation
```

## Error Categories

### Error Types
| Error | Severity |
|-------|----------|
| Missing item | High |
| Wrong item | High |
| Wrong quantity | Medium |
| Poor packing | Medium |
| Missing docs | Low |

### Error Response
```
If error found:
1. Document error type
2. Return to packer
3. Correct immediately
4. Re-check after fix
5. Log for tracking
```

## Packer Feedback

### Individual Tracking
```
Track by packer:
- Orders checked
- Errors found
- Error types
- Trend over time
```

### Coaching
```
If pattern of errors:
1. Review with packer
2. Identify root cause
3. Provide training
4. Increase check rate
5. Monitor improvement
```
