# RHF Order Packer

**Business:** Red Hill Fresh
**Reports To:** Fulfillment Team Lead
**Focus:** Order packing process

## Role

Pack customer orders accurately and safely, ensuring products arrive in optimal condition.

## Packing Process

### Workflow
```
1. Receive picked items
2. Verify against order
3. Pack appropriately
4. Add cold packs if needed
5. Seal and label
6. Stage for delivery
```

### Packing Station Setup
```
Station equipment:
- Packing bench
- Bag/box selection
- Cold packs
- Labels/printer
- Order documents
- Supplies
```

## Packing Standards

### Product Handling
| Product Type | Handling |
|--------------|----------|
| Fragile (eggs) | Separate, top of bag |
| Heavy (milk) | Bottom, distribute weight |
| Cold (meat) | With cold pack |
| Bread | Top, separate bag |
| Produce | Gentle, no crushing |

### Temperature Sensitive
```
Cold products:
□ Pack with cold pack
□ Separate bag for frozen
□ Last items packed
□ First to vehicle
```

## Bag/Box Selection

### Container Choice
| Order Size | Container |
|------------|-----------|
| <10 items | Paper bag |
| 10-20 items | Reusable bag |
| 20+ items | Box |
| Heavy items | Sturdy box |

### Bag Policy
```
Default: Reusable bags
Option: Paper bags
Avoid: Single-use plastic

Customer can request preference.
```

## Quality Verification

### Pre-Pack Check
```
Before packing:
□ All items present
□ Product quality good
□ Substitutions noted
□ No damage
□ Correct quantities
```

### Final Check
```
After packing:
□ Matches order list
□ Properly packed
□ Nothing forgotten
□ Label attached
□ Delivery note included
```

## Special Handling

### Gift Orders
```
If gift:
- Nice presentation
- Remove prices
- Add gift note
- Special care
```

### Special Requests
```
Handle customer notes:
- "Leave on porch"
- "Ring doorbell"
- "Call on arrival"
Document on label.
```

## Order Documentation

### Packing Slip
```
Include in order:
- Order number
- Customer name
- Items packed
- Substitutions made
- Delivery info
- Contact details
```

## Staging

### Delivery Staging
```
After packing:
1. Label clearly
2. Group by route
3. Cold items marked
4. Staged for loading
5. Time logged
```

### Route Organization
```
Stage by:
- Delivery route
- Time slot
- Load sequence
```
