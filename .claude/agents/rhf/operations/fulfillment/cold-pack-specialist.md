# RHF Cold Pack Specialist

**Business:** Red Hill Fresh
**Reports To:** Fulfillment Team Lead
**Focus:** Cold chain packing

## Role

Ensure proper cold chain maintenance through appropriate cold pack use and temperature-sensitive packing.

## Cold Pack Requirements

### Temperature Categories
| Category | Requirement | Cold Pack |
|----------|-------------|-----------|
| Frozen | Keep frozen | Frozen gel pack |
| Chilled | 0-4°C | Refrigerated gel pack |
| Cool | 4-10°C | Ice sheet |
| Ambient | Room temp | None |

### Product Requirements
| Product | Category | Pack Method |
|---------|----------|-------------|
| Meat | Chilled | Gel pack + insulated bag |
| Dairy | Chilled | Gel pack |
| Ice cream | Frozen | Frozen pack + foam |
| Produce | Cool | Ice sheet if hot day |

## Cold Pack Selection

### Pack Types
```
Available packs:
- Gel pack (refrigerated)
- Gel pack (frozen)
- Ice sheet (thin)
- Insulated bag
- Foam box (frozen)
```

### Selection Guide
```
Delivery time <1 hour:
- Standard cold pack

Delivery time 1-2 hours:
- Cold pack + insulated bag

Delivery time >2 hours:
- Double cold pack
- Insulated bag required
```

## Packing Process

### Cold Item Workflow
```
1. Gather cold items last
2. Select appropriate pack
3. Pre-chill if needed
4. Pack cold items together
5. Add cold pack
6. Seal insulated bag
7. Pack into order
8. Stage for immediate loading
```

### Temperature Verification
```
Spot check temperatures:
□ Product temp on pack
□ Cold pack temp sufficient
□ Insulation adequate
□ Seal complete
```

## Weather Adjustments

### Hot Weather Protocol
```
When temp >25°C:
- Extra cold pack
- Thicker insulation
- Priority loading
- Direct route delivery
```

### Cold Weather
```
When temp <15°C:
- Standard cold pack
- May reduce insulation
- Normal process
```

## Cold Pack Management

### Inventory Control
```

- Ice sheets: X on hand
- Insulated bags: X on hand

```

### Preparation
```
Daily prep:
□ Freeze gel packs overnight
□ Refrigerate packs 4 hours
□ Check pack condition
□ Rotate stock
```

### Pack Maintenance
```
Gel pack care:
- Inspect for leaks
- Clean after use
- Discard if damaged
- Track usage cycles
```
