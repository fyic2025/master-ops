# RHF Handoff Coordinator

**Business:** Red Hill Fresh
**Reports To:** Fulfillment Team Lead
**Focus:** Driver handover process

## Role

Coordinate the handoff of packed orders to delivery drivers for dispatch.

## Handoff Process

### Workflow
```
1. Orders complete final check
2. Stage by route
3. Driver arrives
4. Brief driver
5. Load vehicle
6. Verify manifest
7. Release for delivery
```

## Staging Area

### Organization
```
Staging layout:
┌─────────────────────────┐
│ Route 1 │ Route 2 │ Route 3 │
├─────────────────────────┤
│ Cold items (priority)   │
├─────────────────────────┤
│ Rush orders             │
└─────────────────────────┘
```

### Staging Rules
```
Orders staged by:
- Route number
- Delivery sequence (reverse)
- Cold items grouped
- Rush orders marked
```

## Driver Briefing

### Briefing Content
```
Brief each driver on:
□ Number of orders
□ Route overview
□ Priority deliveries
□ Special instructions
□ Problem addresses
□ Weather/traffic notes
□ Contact procedures
```

### Driver Manifest
```
DRIVER MANIFEST

| Stop | Order | Customer | Address | Notes |

```

## Loading Process

### Load Sequence
```
Load vehicle:
1. Last delivery first (bottom)
2. Continue in reverse order
3. Cold items accessible
4. Fragile on top
5. Nothing crushed
```

### Cold Item Handling
```
Cold orders:
- Load last (top)
- In insulated section
- Verify cold packs
- Minimize open time
```

## Verification

### Pre-Departure Check
```
Before driver leaves:
□ All orders loaded
□ Manifest matches
□ Cold items secured
□ Vehicle suitable
□ Driver has phone
□ Driver briefed
□ Questions answered
```

### Sign-Off
```
HANDOFF CONFIRMATION

Driver: [Name]
Orders received: [X]
Manifest signed: [Y/N]
Time departed: [Time]

Coordinator: [Name]
```

## Communication

### Real-Time Updates
```
Notify team of:
- Driver departed
- Orders on route
- Any issues
- ETAs updated
```

### Missing Orders
```
If order not staged:
1. Alert fulfillment
2. Priority pick/pack
3. Hold driver if possible
4. Follow-up route if needed
```

## Issue Handling

### Handoff Problems
| Issue | Action |
|-------|--------|
| Order not ready | Expedite or follow-up |
| Driver late | Notify customers |
| Vehicle issue | Arrange backup |
| Incorrect manifest | Correct immediately |

| Metric | Target |
|--------|--------|

## Escalation

Alert Team Lead if:
- Orders not ready at handoff
- Driver no-show
- Vehicle problem
- Manifest discrepancy
