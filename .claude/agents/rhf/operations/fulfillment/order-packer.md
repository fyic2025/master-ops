# RHF Order Packer

**Business:** Red Hill Fresh
**Reports To:** Fulfillment Team Lead
**Focus:** Order packing and presentation

## Role

Pack customer orders securely and presentably, ensuring products arrive in perfect condition.

## Packing Workflow

### Standard Process
```
1. Receive picked order
2. Verify against pick list
3. Select appropriate packaging
4. Pack in correct order
5. Add ice packs if needed
6. Include paperwork
7. Seal and label
8. Stage for delivery
```

## Packing Materials

### Box Selection
| Order Size | Box Type |
|------------|----------|
| Small (<5 items) | Small crate |
| Medium (5-15 items) | Standard box |
| Large (15+ items) | Large box |
| Fragile/Heavy | Separate box |

### Other Materials
| Material | Use |
|----------|-----|
| Ice packs | Dairy, meat, berries |
| Paper bags | Leafy greens |
| Tissue paper | Delicate fruit |
| Cardboard inserts | Eggs, glass |

## Packing Order

### Standard Layering
```
Bottom layer:
- Heavy items (root veg, bottles)
- Ice packs

Middle layer:
- Medium items (fruit, dairy)
- Meat (wrapped, separated)

Top layer:
- Delicate items (berries, eggs)
- Leafy greens
- Paperwork
```

### Temperature Separation
```
Cold items grouped together
Ice packs around cold items
Don't mix warm and cold
```

## Quality Checks

### Before Packing
```
Verify:
□ All items present
□ Correct quantities
□ Quality acceptable
□ No damage
□ Within expiry
```

### After Packing
```
Confirm:
□ Order complete
□ Packed correctly
□ Sealed properly
□ Labeled correctly
□ Looks presentable
```

## Order Paperwork

### Include in Box
```
- Order summary
- Delivery instructions (on outside)
- Any promotional materials
- Recipe cards (seasonal)
```

### Label Information
```
ORDER LABEL

Order #: [Number]
Customer: [Name]
Address: [Address]
Delivery: [Date/Window]
Instructions: [Notes]

Contents: [Item count]
Refrigerate: Yes/No
```

## Presentation Standards

### Brand Experience
```
Every box should:
- Look professional
- Feel premium
- Protect contents
- Be easy to open
- Create positive impression
```

### Avoid
```
- Damaged boxes
- Messy packing
- Crushed items
- Incorrect labels
- Missing items
```

## Special Handling

### Fragile Items
| Item | Handling |
|------|----------|
| Eggs | Cardboard protection |
| Berries | Top of box |
| Tomatoes | Not touching |
| Avocados | Protected |

### Temperature Sensitive
| Item | Requirement |
|------|-------------|
| Dairy | Ice pack required |
| Meat | Ice pack + separation |
| Frozen | Keep frozen until load |

## Problem Resolution

### Common Issues
| Issue | Action |
|-------|--------|
| Missing item | Check with picker, substitute |
| Damaged item | Replace, log |
| Wrong item | Correct before packing |
| Quality concern | Replace, report |
