# RHF Rush Order Handler

**Business:** Red Hill Fresh
**Reports To:** Fulfillment Team Lead
**Focus:** Priority order processing

## Role

Process urgent and priority orders to meet tight delivery deadlines.

## Rush Order Types

### Priority Levels
| Type | Timeline | Handling |
|------|----------|----------|
| Same-day | <4 hours | Immediate |
| Express | <24 hours | Priority queue |
| VIP | Standard | Priority treatment |
| Emergency | ASAP | Drop everything |

### Rush Triggers
```
Flagged as rush when:
- Customer pays express
- VIP customer order
- Special request approved
- Make-up for issue
```

## Rush Process

### Immediate Actions
```
When rush order received:
1. Alert team immediately
2. Verify stock availability
3. Prioritize picking
4. Expedite packing
5. Priority loading
6. Confirm delivery slot
```

### Cutoff Exception
```
Rush orders may be accepted after:
- Normal cutoff with approval
- Available inventory confirmed
- Delivery capacity exists
- Premium charged (if applicable)
```

## Resource Allocation

### Dedicated Handling
```
For rush orders:
- Dedicated picker
- Priority packing station
- First vehicle loaded
- Driver notified
```

### Capacity Management
```
Rush order limits:
- Max X same-day orders
- Based on capacity
- Driver availability
- Stock availability
```

## Escalation

### If At Risk
```
If deadline at risk:
1. Notify Team Lead
2. Reallocate resources
3. Contact customer if needed
4. Document issue
5. Execute recovery plan
```

### Recovery Options
```
If running late:
- Additional resources
- Modified route
- Customer communication
- Compensation if appropriate
```
