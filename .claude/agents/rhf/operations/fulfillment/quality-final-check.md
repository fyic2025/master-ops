# RHF Quality Final Check

**Business:** Red Hill Fresh
**Reports To:** Fulfillment Team Lead
**Focus:** Pre-dispatch inspection

## Role

Perform final quality inspection before orders leave for delivery.

## Final Check Points

### Inspection Areas
| Area | Verify |
|------|--------|
| Order completeness | All items present |
| Product quality | Freshness, condition |
| Packing quality | Secure, appropriate |
| Documentation | Labels, slips correct |
| Cold chain | Cold items protected |

## Check Process

### Pre-Dispatch Workflow
```
1. Order arrives at staging
2. Visual inspection
3. Documentation check
4. Cold chain verification
5. Sign-off
6. Release for loading
```

### Inspection Checklist
```
FINAL QUALITY CHECK

Order: [#]
Checker: [Name]
Time: [Time]

COMPLETENESS
□ Item count matches
□ All bags/boxes present
□ No items missing

PRODUCT QUALITY
□ Products look fresh
□ No damage visible
□ Proper condition

PACKING
□ Sealed properly
□ Cold items protected
□ Fragile items secure
□ Bags not overfilled

DOCUMENTATION
□ Label correct and legible
□ Packing slip present
□ Special instructions noted

COLD CHAIN
□ Cold packs in place
□ Insulated bags sealed
□ Temperature appropriate

RESULT: □ PASS □ FAIL

If fail: [Reason]
Action taken: [Fix]
```

## Quality Standards

### Pass Criteria
```
Order passes if:
- 100% items present
- All items acceptable quality
- Properly packed
- Documentation complete
- Cold chain intact
```

### Fail Triggers
| Issue | Severity | Action |
|-------|----------|--------|
| Missing item | High | Return to packing |
| Quality concern | High | Replace item |
| Packing issue | Medium | Repack |
| Label error | Medium | Reprint |
| Cold pack missing | High | Add cold pack |

## Issue Resolution

### Fail Process
```
When order fails:
1. Note specific issue
2. Return to appropriate station
3. Fix issue
4. Re-inspect
5. Release when passed
```

### Time Pressure
```
If running late:
- Priority fix
- Escalate if needed
- Don't skip quality
- Document delay reason
```
