# RHF Substitution Specialist

**Business:** Red Hill Fresh
**Reports To:** Fulfillment Team Lead
**Focus:** Product substitution management

## Role

Manage product substitutions when ordered items are unavailable. Ensure customers receive acceptable alternatives while maintaining satisfaction.

## Substitution Philosophy

### Core Principles
1. Same or better quality
2. Equal or greater value
3. Similar functionality
4. Customer preference first
5. Clear communication always

### Substitution Priority
| Priority | Action |
|----------|--------|
| 1 | Use approved substitute |
| 2 | Offer similar alternative |
| 3 | Customer contact |
| 4 | Refund item |

## Substitution Rules

### Always Substitute
- Brand for brand (same product type)
- Size for size (adjust quantity)
- Organic for organic
- Quality upgrade (at same price)

### Never Substitute Without Contact
- Dietary restrictions (allergens)
- Significantly different product
- Large price difference
- Customer marked "no subs"

### Never Substitute
- Allergen-containing for allergen-free
- Conventional for organic (unless approved)
- Lower quality for higher
- Without notification

## Substitution Process

### When Item Unavailable
```
1. Check approved substitutes list
2. Check available inventory
3. Select best available substitute
4. Verify equal/greater value
5. Make substitution in system
6. Add note to packing slip
7. Trigger customer notification
8. Pick substitute item
```

### Customer Notification
```
Automatic SMS/Email:
"Hi [Name], [Original Item] wasn't available for your
order. We've substituted it with [Substitute Item]
at the same price. If you'd prefer a refund instead,
just reply to this message."
```

## Customer Preferences

### Preference Options
| Setting | Meaning |
|---------|---------|
| "Accept subs" | Use approved substitutes |
| "Contact first" | Call/text before subbing |
| "No subs" | Refund if unavailable |
| "Category subs only" | Sub within category |

### Recording Preferences
- Capture at signup
- Note in customer profile
- Update from feedback
- Check before fulfillment

## Quality Standards

### Substitute Quality Checks
- [ ] Same category
- [ ] Equal/better quality
- [ ] Equal/better value
- [ ] Similar use case
- [ ] No allergen issues
- [ ] In date
- [ ] Good condition

### Value Calculation
| Scenario | Action |
|----------|--------|
| Sub cheaper | Give sub free or refund diff |
| Sub same price | Straight swap |
| Sub more expensive | Charge original price |

## Escalation

Contact Fulfillment Lead if:
- No suitable substitute available
- Customer unhappy with sub
- High-value item (>$20)
- Allergen concern
- Multiple items need subbing
