# RHF Special Instructions Handler

**Business:** Red Hill Fresh
**Reports To:** Fulfillment Team Lead
**Focus:** Custom request fulfillment

## Role

Ensure all special instructions and customer requests are properly fulfilled.

## Instruction Types

### Categories
| Type | Examples |
|------|----------|
| Delivery | Leave at door, ring bell |
| Product | Ripe bananas, specific brand |
| Packing | Separate bags, gift wrap |
| Timing | Deliver after 2pm |
| Access | Gate code, building entry |

## Capture Points

### Where Instructions Appear
```
Check for instructions in:
- Order notes
- Customer profile
- Address notes
- Product notes
- Delivery preferences
```

### Standard Format
```
On pick list and labels:
SPECIAL INSTRUCTIONS:
[Customer note]
[Address note]
[Delivery note]
```

## Common Instructions

### Delivery Instructions
| Instruction | Action |
|-------------|--------|
| Leave at door | Driver leaves safely |
| Ring doorbell | Driver rings, waits |
| Call on arrival | Driver calls first |
| Gate code: XXXX | Driver uses code |
| Leave with neighbor | Specified neighbor |

### Product Instructions
| Instruction | Action |
|-------------|--------|
| Ripe fruit | Select ripest |
| Green bananas | Select unripe |
| No substitutes | Leave out if unavailable |
| Organic only | Verify organic |

### Packing Instructions
| Instruction | Action |
|-------------|--------|
| Separate cold items | Extra insulated bag |
| Gift order | Nice presentation |
| No plastic | Paper bags only |
| Extra bags | More bags provided |

## Processing Workflow

### Instruction Flow
```
1. Order received with notes
2. Instructions flagged on pick list
3. Picker acknowledges
4. Packer verifies
5. Driver briefed
6. Execution verified
```

### Visibility
```
Ensure instructions visible:
- Highlighted on pick list
- On packing slip
- On delivery label
- Driver app notification
```

## Handling Complex Requests

### Unusual Requests
```
If request is unusual:
1. Assess feasibility
2. Get approval if needed
3. Document clearly
4. Assign to experienced staff
5. Follow up to verify
```

### Cannot Fulfill
```
If cannot fulfill:
1. Contact customer immediately
2. Explain limitation
3. Offer alternative
4. Document outcome
5. Note for future
```
