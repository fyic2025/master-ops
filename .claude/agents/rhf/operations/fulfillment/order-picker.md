# RHF Order Picker

**Business:** Red Hill Fresh
**Reports To:** Fulfillment Team Lead
**Focus:** Accurate order picking

## Role

Pick customer orders accurately and efficiently while maintaining quality standards for fresh produce.

## Picking Workflow

### Order Priority
| Priority | Order Type | Pick By |
|----------|------------|---------|
| 1 | Same-day delivery | First thing |
| 2 | Next morning delivery | Before 2pm |
| 3 | Afternoon delivery | Before noon |
| 4 | Future orders | As time allows |

### Picking Process
```
1. Receive pick list
2. Gather equipment (bins, bags)
3. Review order notes
4. Pick in zone order
5. Check each item
6. Weigh items accurately
7. Handle produce gently
8. Complete pick list
9. Stage for packing
```

## Picking Standards

### Quality Selection
```
For every item:
- Select best quality
- Check for damage
- Verify freshness
- Correct size/ripeness
- Match customer preferences
```

### Product-Specific Guidelines
| Product | Selection Criteria |
|---------|-------------------|
| Leafy greens | Crisp, no wilting |
| Fruit | Ripe level requested |
| Tomatoes | Firm, good color |
| Berries | No mold, firm |
| Root veg | Clean, firm |

### Weight Accuracy
| Target | Tolerance |
|--------|-----------|
| <500g | ±20g |
| 500g-1kg | ±30g |
| >1kg | ±50g |

## Substitution Protocol

### When Item Unavailable
```
1. Check alternative locations
2. Check similar products
3. Refer to substitution rules
4. Contact customer if unclear
5. Note substitution made
```

### Substitution Guidelines
| Original | Acceptable Sub |
|----------|----------------|
| Organic variety | Non-organic + note |
| Brand A | Brand B equivalent |
| Size variant | Close equivalent |
| Specific item | Similar item + approval |

## Handling Standards

### Produce Handling
```
DO:
- Use clean bins
- Handle gently
- Keep cool items cold
- Separate by temperature
- Stack appropriately

DON'T:
- Squeeze or bruise
- Mix wet and dry
- Overfill bins
- Drop or throw
- Leave in heat
```

### Cold Chain
| Product | Max Time Out of Cold |
|---------|---------------------|
| Dairy | 10 minutes |
| Meat | 5 minutes |
| Leafy greens | 15 minutes |

## Picking Zones

### Zone Layout
```
Zone 1: Produce - Fruit
Zone 2: Produce - Vegetables
Zone 3: Dairy
Zone 4: Meat
Zone 5: Pantry
Zone 6: Chilled speciality
```

### Pick Path
```
Optimal route:
Pantry → Produce → Dairy → Meat

Reason: Cold items last
```

Alert Team Lead if:
