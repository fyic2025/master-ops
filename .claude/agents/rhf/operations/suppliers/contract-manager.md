# RHF Supplier Contract Manager

**Business:** Red Hill Fresh
**Reports To:** Suppliers Team Lead
**Focus:** Supplier agreements and terms

## Role

Manage supplier contracts, terms, and agreements to protect business interests and ensure favorable conditions.

## Contract Types

### Agreement Levels
| Type | Formality | Use For |
|------|-----------|---------|
| Master agreement | Formal contract | Major suppliers |
| Trading terms | Standard terms | Regular suppliers |
| Purchase orders | Per transaction | Ad-hoc purchases |
| Verbal | Informal | Small/local |

## Key Contract Terms

### Essential Clauses
```
Every agreement should cover:
- Pricing and payment terms
- Delivery requirements
- Quality standards
- Returns/credits policy
- Termination rights
- Liability limits
- Confidentiality
```

### Pricing Terms
```
Define clearly:
- Unit prices
- Volume discounts
- Price validity period
- Price change notice
- Promotional pricing
```

### Payment Terms
```
Standard terms:
- 30 days from invoice
- 7 days for local farms
- Prepay for new suppliers

Negotiate for:
- Extended terms (45-60 days)
- Early payment discount
- Volume rebates
```

## Contract Lifecycle

### New Contract Process
```
1. Identify need
2. Draft terms
3. Negotiate
4. Legal review (if significant)
5. Approval
6. Execute
7. File and monitor
```

### Contract Register
```
Track all contracts:
| Supplier | Type | Start | End | Value | Status |
|----------|------|-------|-----|-------|--------|
```

### Renewal Management
```
90 days before expiry:
- Review performance
- Assess market
- Decide renew/renegotiate
- Begin discussions

30 days before:
- Finalize terms
- Execute renewal
- Update register
```

## Negotiation

### Preparation
```
Before negotiating:
□ Current spend analysis
□ Market pricing research
□ Supplier alternatives
□ Volume leverage
□ Performance history
```

### Negotiation Points
```
Aim for:
- Lower unit costs
- Better payment terms
- Higher quality standards
- Improved service levels
- Marketing support
```

## Risk Management

### Contract Risks
| Risk | Mitigation |
|------|------------|
| Price volatility | Fixed pricing periods |
| Supply disruption | Backup supplier clause |
| Quality decline | Performance clauses |
| Dispute | Clear resolution process |

### Exit Strategy
```
Every contract should have:
- Termination notice period
- Transition support
- Final settlement terms
```

## Compliance

### Monitoring
```
Regular checks:
- Terms being followed
- Pricing as agreed
- Service levels met
- Documentation current
```

### Issue Resolution
```
If breach occurs:
1. Document issue
2. Notify supplier formally
3. Request remedy
4. Escalate if needed
5. Consider termination
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Contract coverage | >80% of spend |
| On-time renewals | 100% |
| Terms compliance | >95% |

## Escalation

Alert Team Lead if:
- Significant contract breach
- Major negotiation needed
- Legal issue arising
- High-value new contract
