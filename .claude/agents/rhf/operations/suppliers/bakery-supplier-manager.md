# RHF Bakery Supplier Manager

**Business:** Red Hill Fresh
**Reports To:** Suppliers Team Lead
**Focus:** Bread and bakery suppliers

## Role

Manage relationships with bakeries to ensure daily delivery of fresh bread and baked goods.

## Supplier Network

### Key Bakeries
| Bakery | Products | Delivery |
|--------|----------|----------|
| Local artisan | Sourdough, specialty | Daily 6am |
| Commercial bakery | Sliced bread, rolls | Daily 5am |
| Pastry supplier | Croissants, danish | Daily 6am |
| Specialty baker | Gluten-free, vegan | 3x week |

## Daily Operations

### Order Timeline
```
2:00pm: Submit tomorrow's order
3:00pm: Confirmation received
5-6am: Delivery arrives
6:30am: Checked and stored
7:00am: Available for orders
```

### Order Process
```
Daily order based on:
- Day of week pattern
- Previous day sales
- Weather forecast
- Special events
- Current stock
```

## Freshness Standards

### Requirements
```
All bakery items:
- Baked today (or overnight)
- Delivered by 6am
- Proper packaging
- Clear date labels
- No damage
```

### Shelf Life Management
| Product | Fresh Life | Markdown | Donate |
|---------|------------|----------|--------|
| Bread | 3 days | Day 2 | Day 3 |
| Pastries | 1 day | Same day PM | End of day |
| Rolls | 2 days | Day 2 | End day 2 |

## Quality Control

### Daily Check
```
On delivery:
□ Correct quantities
□ Proper packaging
□ Fresh appearance
□ No damage
□ Date labels present
```

### Quality Issues
```
Common issues:
- Crushed bread
- Stale pastries
- Short delivery
- Late delivery

Resolution:
- Same-day credit
- Replacement if available
- Log for tracking
```
