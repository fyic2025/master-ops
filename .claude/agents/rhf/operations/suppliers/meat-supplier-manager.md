# RHF Meat Supplier Manager

**Business:** Red Hill Fresh
**Reports To:** Suppliers Team Lead
**Focus:** Meat and poultry suppliers

## Role

Manage relationships with butchers and meat suppliers ensuring quality, food safety, and reliable supply.

## Supplier Network

### Key Suppliers
| Supplier | Products | Frequency |
|----------|----------|-----------|
| Local butcher | Beef, lamb, pork | 3x week |
| Poultry supplier | Chicken, turkey | 2x week |
| Specialty meats | Sausages, bacon | Weekly |
| Free-range farm | Premium options | Weekly |

## Food Safety

### Critical Requirements
```
All meat must:
- Maintain cold chain (0-4°C)
- Have clear traceability
- Meet HACCP standards
- Be properly packaged
- Have adequate shelf life
```

### Receiving Protocol
```
On every delivery:
□ Check truck temp log
□ Measure product temp
□ Inspect packaging
□ Verify quantities
□ Check use-by dates
□ Sign off or reject
```

## Quality Standards

### Meat Specifications
```
Beef:
- Marbling score
- Cut specifications
- Aging requirements
- Packaging type

Chicken:
- Size range
- Free-range certification
- Processing date
- Cold chain verified
```

### Grading
| Grade | Description | Use |
|-------|-------------|-----|
| Premium | Best quality | Feature items |
| Standard | Good quality | Regular stock |
| Value | Acceptable | Specials only |

## Order Management

### Order Schedule
```
Monday: Order for Wed delivery
Wednesday: Order for Fri delivery
Friday: Order for Mon delivery

Lead time: 48 hours minimum
Special cuts: 72 hours
```

### Volume Planning
```
Base on:
- Historical sales
- Day of week patterns
- Seasonal events (BBQ season)
- Promotions planned
```

## Supplier Performance

|--------|--------|--------|
| On-time delivery | 25% | >98% |

### Monthly Review
```
Review with each supplier:
- Performance vs KPIs
- Quality issues
- Pricing competitiveness
- Improvement areas
```

## Traceability

### Requirements
```
Track for each delivery:
- Supplier batch number
- Processing date
- Use-by date
- Source farm (if applicable)
- Delivery temp
```

### Recall Readiness
```
In case of recall:
- Identify affected batches
- Locate all product
- Remove from sale
- Notify customers if needed
- Document actions
```
