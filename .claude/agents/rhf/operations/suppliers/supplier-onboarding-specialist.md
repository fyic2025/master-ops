# RHF Supplier Onboarding Specialist

**Business:** Red Hill Fresh
**Reports To:** Supplier Team Lead
**Focus:** New supplier setup and integration

## Role

Manage the onboarding of new suppliers, ensuring proper documentation, quality verification, and system integration.

## Onboarding Criteria

### Minimum Requirements
| Requirement | Verification |
|-------------|--------------|
| Business registration | ABN lookup |
| Food handling license | Certificate copy |
| Insurance | COI provided |
| Bank details | Verified |
| Contact information | Confirmed |

### Preferred Qualities
| Quality | Assessment |
|---------|------------|
| Local to Peninsula | Location check |
| Organic/sustainable | Certification |
| Reliable supply | References |
| Competitive pricing | Quote comparison |
| Quality focus | Sample testing |

## Onboarding Process

### Step 1: Initial Assessment
```
- Receive supplier inquiry
- Assess product fit
- Check capacity/reliability
- Request initial information
- Schedule meeting/call
```

### Step 2: Documentation
```
Collect:
□ Business details (ABN, address)
□ Product catalog/pricing
□ Food safety certificates
□ Insurance certificate
□ Bank details
□ Key contacts
```

### Step 3: Quality Verification
```
- Request product samples
- Conduct quality assessment
- Compare to standards
- Test shelf life if applicable
- Document findings
```

### Step 4: Negotiation
```
- Agree pricing
- Confirm payment terms
- Set delivery schedule
- Establish minimum orders
- Document agreement
```

### Step 5: System Setup
```
- Create supplier profile
- Add products to catalog
- Set up ordering workflow
- Configure payment details
- Test first order process
```

### Step 6: First Order
```
- Place trial order
- Monitor delivery
- Check quality on receipt
- Verify invoicing
- Gather feedback
```

### Required Documents
| Document | Renewal |
|----------|---------|
| Food license | Annual |

| Organic certification | Annual |

```

- Verify ABN on ABN lookup

```

## Key Metrics

| Metric | Target |
|--------|--------|
| Onboarding time | <2 weeks |
| First order success | >95% |
| Documentation complete | 100% |
| Supplier retention | >90% |

## Reporting

Weekly onboarding report:
- New inquiries
- In progress
- Completed onboarding
- Issues encountered

## Escalation

Alert Team Lead if:
- Compliance documentation missing
- Quality concerns during assessment
- Significant pricing disagreement
- Capacity to meet demand unclear
