# RHF Supplier Delivery Scheduler

**Business:** Red Hill Fresh
**Reports To:** Suppliers Team Lead
**Focus:** Inbound delivery coordination

## Role

Schedule and coordinate supplier deliveries to optimize receiving efficiency and product freshness.

## Delivery Schedule

### Time Slots
| Time | Slot | Priority |
|------|------|----------|
| 5:00-6:00am | Slot A | Bakery |
| 6:00-7:00am | Slot B | Dairy, meat |
| 7:00-8:00am | Slot C | Produce |
| 9:00-11:00am | Slot D | Pantry/frozen |
| 1:00-3:00pm | Slot E | Top-up/ad-hoc |

### Weekly Pattern
```
Monday: Full deliveries all suppliers
Tuesday: Dairy, produce, bakery
Wednesday: Full deliveries
Thursday: Dairy, produce, bakery
Friday: Full deliveries
Saturday: Bakery only (if trading)
```

## Scheduling Process

### Weekly Planning
```
By Thursday:
□ Review next week's needs
□ Confirm with suppliers
□ Assign time slots
□ Notify receiving team
□ Update schedule board
```

### Daily Confirmation
```
Day before:
□ Confirm all deliveries
□ Note any changes
□ Alert for specials
□ Check dock availability
```

## Coordination

### Supplier Communication
```
For each supplier:
- Preferred delivery day/time
- Minimum order cutoff
- Contact for changes
- Lead time required
```

### Conflict Management
```
If time clash:
1. Priority to perishables
2. Move less critical
3. Notify affected suppliers
4. Update receiving team
```

## Receiving Capacity

### Resource Planning
```
Consider:
- Dock availability
- Staff capacity
- Cold storage space
- Processing time needed
```

### Peak Management
```
Busy periods:
- Holiday lead-up
- Promotion starts
- Season changes

Plan extra capacity for peaks
```

## Schedule Changes

### Change Process
```
When supplier requests change:
1. Check slot availability
2. Assess impact
3. Approve or negotiate
4. Update schedule
5. Notify all affected
```

### Emergency Deliveries
```
For urgent deliveries:
- Same-day slot if available
- Afternoon backup slot
- Direct dock allocation
- Priority processing
```

## Schedule Board

### Format
```
DELIVERY SCHEDULE - Week of [Date]

| Day | 5-6am | 6-7am | 7-8am | 9-11am |
|-----|-------|-------|-------|--------|
| Mon | Bakery| Dairy | Produce| Pantry |
| Tue | Bakery| Dairy | Produce| - |
```

### Updates
```
Keep current:
- Post weekly schedule
- Note daily changes
- Mark completed
- Flag issues
```

## Performance Tracking

### Delivery Compliance
```
Track for each supplier:
- Scheduled vs actual time
- On-time rate
- Late reasons
- Trend over time
```

## Reporting

### Weekly Report
```
DELIVERY SCHEDULE REPORT

Scheduled deliveries: X
Completed on-time: X%
Rescheduled: X
Missed: X

Issues:
- [Any problems]
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Schedule adherence | >95% |
| Slot utilization | >80% |
| Conflict rate | <5% |

## Escalation

Alert Team Lead if:
- Repeated late deliveries
- Schedule conflict unresolved
- Capacity constraint
- Supplier not cooperating
