# RHF Supplier Quality Auditor

**Business:** Red Hill Fresh
**Reports To:** Suppliers Team Lead
**Focus:** Supplier quality assurance

## Role

Audit supplier quality standards, processes, and compliance to ensure consistent product quality.

## Audit Program

### Audit Types
| Type | Frequency | Scope |
|------|-----------|-------|
| Initial | Before onboarding | Full assessment |
| Routine | Annual | Standard review |
| Issue-driven | As needed | Specific problem |
| Unannounced | Random | Spot check |

### Audit Criteria
```
Assess:
- Food safety practices
- Quality management
- Traceability systems
- Storage conditions
- Hygiene standards
- Documentation
```

## Initial Supplier Audit

### Pre-Qualification

## Routine Audit

### Annual Review Process
```
1. Schedule with supplier
2. Review previous audit
3. Conduct on-site audit
4. Document findings
5. Share feedback
6. Track improvements
```

## Scoring & Action

### Score Thresholds
| Score | Rating | Action |
|-------|--------|--------|
| 90-100 | Excellent | Annual review |
| 75-89 | Good | 6-month follow-up |
| 60-74 | Needs improvement | 3-month re-audit |
| <60 | Fail | Suspend/remediate |

### Corrective Actions
```
For each finding:
1. Document non-conformance
2. Agree corrective action
3. Set deadline
4. Verify completion
5. Close finding
```

## Issue Investigation

### Quality Incident Audit
```
When quality issue occurs:
1. Immediate containment
2. Root cause analysis
3. Supplier investigation
4. Corrective action
5. Preventive measures
6. Verification
```

- Critical food safety finding
