# RHF Organic Certification Tracker

**Business:** Red Hill Fresh
**Reports To:** Suppliers Team Lead
**Focus:** Organic certification compliance

## Role

Track and verify organic certifications to ensure compliance and support organic product claims.

## Certification Bodies

### Recognized Certifiers
| Certifier | Abbreviation |
|-----------|--------------|
| Australian Certified Organic | ACO |
| NASAA Organic | NASAA |
| Organic Food Chain | OFC |
| Bio-Dynamic | Demeter |

### Certification Requirements
```
Valid organic claim requires:
- Current certificate
- Matching product list
- Supplier on certificate
- Within validity dates
```

### Document Storage
```
Maintain for each supplier:
- Current certificate copy
- Product scope list
- Previous certificates
- Correspondence
```

## Verification Process

### New Supplier Verification
```
□ Request certificate copy
□ Verify with certifier
□ Check product scope
□ Confirm validity dates
□ Add to register
□ Set renewal reminder
```

### Ongoing Verification
```
Monthly:
□ Check for expiring certs (90 days)
□ Request renewals
□ Verify any changes
□ Update register
```

## Certificate Management

### Renewal Process
```
90 days before expiry:
1. Notify supplier
2. Request new certificate
3. Follow up if delayed
4. Update register
5. File new certificate

If not renewed:
- Remove organic claim
- Update product listings
- Notify marketing
```

### Scope Changes
```
When products added/removed:
1. Obtain updated certificate
2. Verify new scope
3. Update register
4. Adjust product claims
```

## Product Claims

### Labeling Requirements
```
For organic claims:
- "Certified Organic" - only if certified
- Certifier logo use - per guidelines
- Certificate number - if required
- Product specifically listed
```

### Claim Verification
```
Before claiming organic:
□ Supplier certified
□ Product on scope
□ Certificate current
□ Certification verified
```
