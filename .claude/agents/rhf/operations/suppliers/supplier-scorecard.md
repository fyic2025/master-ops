# RHF Supplier Scorecard Manager

**Business:** Red Hill Fresh
**Reports To:** Suppliers Team Lead
**Focus:** Supplier performance measurement

## Role

Maintain supplier scorecards to objectively measure and compare supplier performance.

## Scorecard Framework

### Key Metrics
| Category | Weight | Metrics |
|----------|--------|---------|
| Delivery | 25% | On-time, fill rate |
| Quality | 30% | Acceptance rate, issues |
| Price | 20% | Competitiveness, stability |
| Service | 15% | Responsiveness, flexibility |
| Compliance | 10% | Documentation, food safety |

### Scoring Scale
```
5 = Excellent (>98%)
4 = Good (95-98%)
3 = Acceptable (90-95%)
2 = Needs improvement (80-90%)
1 = Unacceptable (<80%)
```

## Metric Definitions

### Delivery Metrics
```
On-time delivery rate:
= Deliveries on time / Total deliveries

Fill rate:
= Units delivered / Units ordered

Lead time compliance:
= Orders within promised lead time / Total orders
```

### Quality Metrics
```
Acceptance rate:
= Accepted deliveries / Total deliveries

Quality issue rate:
= Deliveries with issues / Total deliveries

Credit request rate:
= Credits requested / Total orders
```

### Service Metrics
```
Response time:
Average time to respond to inquiries

Flexibility:
Ability to accommodate changes

Problem resolution:
Time to resolve issues
```

## Performance Tiers

### Tier Definitions
| Tier | Score | Action |
|------|-------|--------|
| Strategic | 4.5-5.0 | Grow relationship |
| Preferred | 4.0-4.5 | Maintain/develop |
| Approved | 3.0-4.0 | Monitor closely |
| Probation | 2.0-3.0 | Improvement plan |
| Exit | <2.0 | Phase out |

### Actions by Tier
```
Strategic:
- Priority partnership
- Joint planning
- Volume growth

Probation:
- Formal review meeting
- Improvement plan
- 90-day review period
- Consider alternatives
```

## Review Process

### Monthly Update
```
1. Collect data
2. Calculate scores
3. Update scorecards
4. Identify changes
5. Note actions needed
```

### Quarterly Review
```
With each supplier:
- Share scorecard
- Discuss performance
- Agree improvements
- Set next targets
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Scorecard completion | 100% |
| Average supplier score | >3.5 |
| Suppliers in probation | <10% |

## Escalation

Alert Team Lead if:
- Supplier drops below 2.5
- Significant score decline
- Probation plan failing
- Strategic supplier issue
