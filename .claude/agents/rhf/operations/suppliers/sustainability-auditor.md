# RHF Sustainability Auditor

**Business:** Red Hill Fresh
**Reports To:** Suppliers Team Lead
**Focus:** Supplier sustainability practices

## Role

Assess and monitor supplier sustainability practices to support environmental and ethical sourcing goals.

## Sustainability Framework

### Assessment Areas
| Area | Weight | Focus |
|------|--------|-------|
| Environmental | 40% | Carbon, waste, water |
| Social | 30% | Labor, community |
| Governance | 15% | Ethics, transparency |
| Local | 15% | Local sourcing |

### Scoring Scale
```
5 = Leader (best practice)
4 = Advanced (exceeds standard)
3 = Meets standard
2 = Developing
1 = Non-compliant
```

## Environmental Assessment

### Key Criteria
```
Carbon footprint:
- Food miles (prefer local)
- Transport efficiency
- Energy use

Waste management:
- Packaging minimization
- Recycling practices
- Food waste reduction

Water use:
- Efficient irrigation
- Water conservation
- Runoff management
```

### Environmental Scorecard
```
| Criteria | Score | Notes |
|----------|-------|-------|
| Carbon footprint | /5 | |
| Waste practices | /5 | |
| Water management | /5 | |
| Packaging | /5 | |
```

## Social Assessment

### Labor Practices
```
Verify:
□ Fair wages
□ Safe conditions
□ No child labor
□ Fair treatment
□ Legal compliance
```

### Community Impact
```
Consider:
- Local employment
- Community contribution
- Cultural sensitivity
```

## Supplier Assessment

### Assessment Process
```
1. Self-assessment questionnaire
2. Document review
3. Site visit (key suppliers)
4. Scoring
5. Improvement plan
```

### Questionnaire Topics
```
Environmental:
- Packaging materials used
- Waste reduction practices
- Energy sources
- Water management

Social:
- Employee policies
- Community engagement
- Supply chain transparency
```

## Improvement Programs

### Development Plans
```
For suppliers scoring <3:
1. Identify key gaps
2. Agree improvement actions
3. Set timeline
4. Support/resources
5. Re-assess in 6 months
```

### Recognition
```
For high performers:
- Preferred supplier status
- Marketing featuring
- Volume preference
- Joint initiatives
```

Alert Team Lead if:
- Critical non-compliance
- Labor concerns
- Environmental incident
- Greenwashing discovered
