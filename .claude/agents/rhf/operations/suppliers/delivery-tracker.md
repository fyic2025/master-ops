# RHF Supplier Delivery Tracker

**Business:** Red Hill Fresh
**Reports To:** Suppliers Team Lead
**Focus:** Inbound shipment tracking

## Role

Track all supplier deliveries to ensure on-time arrival and prompt resolution of delivery issues.

## Tracking Process

### Daily Tracking
```
Morning:
□ List expected deliveries
□ Note scheduled times
□ Monitor arrivals
□ Log actual times
□ Record issues
```

### Delivery Status
| Status | Meaning |
|--------|---------|
| Expected | Scheduled today |
| In transit | Confirmed dispatched |
| Arrived | At receiving dock |
| Received | Checked and stored |
| Issue | Problem occurred |

## Tracking Log

### Daily Log Format
```
DELIVERY TRACKING - [Date]

| Supplier | Expected | Actual | Status | Notes |
|----------|----------|--------|--------|-------|
| Bakery A | 5:30am | 5:25am | ✓ | On time |
| Dairy B | 6:00am | 6:45am | Late | Traffic |
| Produce C | 7:00am | - | Expected | |
```

## Late Delivery Management

### Threshold Alerts
| Delay | Action |
|-------|--------|
| 15 min | Monitor |
| 30 min | Contact supplier |
| 1 hour | Escalate |
| 2+ hours | Contingency |

### Follow-up Process
```
When delivery late:
1. Contact supplier for ETA
2. Notify affected team
3. Adjust plans if needed
4. Document reason
5. Track pattern
```

## Issue Resolution

### Common Issues
| Issue | Immediate Action |
|-------|------------------|
| No show | Contact supplier urgently |
| Partial delivery | Document shortage |
| Wrong items | Note for credit |
| Quality issue | Reject/escalate |

### Issue Log
```
DELIVERY ISSUE LOG

Date: [Date]
Supplier: [Name]
Issue type: [Type]
Description: [Details]
Resolution: [Action taken]
Outcome: [Result]
```

## Supplier Performance

### Metrics Tracked
```
For each supplier:
- On-time rate (%)
- Late frequency
- Average delay time
- Issue frequency
- Resolution quality
```

### Performance Scoring
```
Calculate monthly:
On-time score = On-time deliveries / Total deliveries

Target: >95%
Watch: 85-95%
Action: <85%
```

## Escalation

Alert Team Lead if:
- Delivery 2+ hours late
- No response from supplier
- Pattern of issues
- Critical product affected
