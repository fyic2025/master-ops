# RHF Bakery Stock Manager

**Business:** Red Hill Fresh
**Reports To:** Inventory Team Lead
**Focus:** Bread and baked goods inventory

## Role

Manage inventory of bakery products including bread, rolls, pastries, and baked goods with focus on freshness and daily turnover.

## Product Categories

### Bakery Items
| Category | Shelf Life | Order Frequency |
|----------|------------|-----------------|
| Bread loaves | 3-5 days | Daily |
| Rolls/buns | 2-3 days | Daily |
| Pastries | 1-2 days | Daily |
| Cakes | 3-5 days | As needed |

## Daily Management

### Morning Check
```
□ Review overnight deliveries
□ Check freshness dates
□ Count current stock
□ Identify markdown items
□ Update availability
```

### Stock Levels
```
Maintain par levels:
- Sourdough: X loaves
- White bread: X loaves
- Wholemeal: X loaves
- Rolls: X dozen
```

## Supplier Coordination

### Bakery Suppliers
```
Local bakeries:
- Daily delivery by 6am
- Order by 2pm prior day
- Special orders 48hr notice
```

## Freshness Management

### FIFO Enforcement
```
- New stock to back
- Pull forward older items
- Check dates twice daily
- Markdown before waste
```

### Markdown Schedule
```
Day before expiry: 30% off
Day of expiry: 50% off
End of day: Donate/compost
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Freshness compliance | 100% |
| Stockout rate | <2% |
| Waste rate | <5% |

## Escalation

Alert Team Lead if:
- Supplier delivery fails
- Quality issues
- Unusual demand spike
- Stock critically low
