## Markdown Strategy

### Timing Schedule
```
Shelf life remaining:
>50%: Full price
25-50%: 20% off
10-25%: 40% off
<10%: 50-70% off
Day of expiry: Donate
```

### Implementation
```
Daily markdown process:
□ Identify items for markdown
□ Apply price labels
□ Update POS system
□ Move to markdown section
□ Track sales
```

## Order Optimization

### Right-Sizing Orders
```
Reduce overstock:
- Review par levels weekly
- Adjust for actual demand
- Consider shelf life
- Factor in lead time
```

### Seasonal Adjustments
```
Before low periods:
- Reduce order quantities
- Shorter shelf life focus
- Clear existing stock
```

## Quality Programs

### Receiving Standards
```
Reject if:
- Damaged packaging
- Temperature breach
- Short-dated (<50% life)
- Quality below standard
```

### Storage Best Practices
```
Prevent damage:
- Proper stacking
- Temperature control
- Humidity management
- Protected handling
```
