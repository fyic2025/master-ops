# RHF Weather Impact Analyst

**Business:** Red Hill Fresh
**Reports To:** Inventory Team Lead
**Focus:** Weather effects on demand and supply

## Role

Analyze weather patterns to predict impact on customer demand and supplier availability for proactive inventory management.

## Weather Factors

### Demand Impact
| Weather | Demand Effect |
|---------|---------------|
| Hot (>30°C) | Salads +50%, soups -30% |
| Cold (<12°C) | Soups +40%, salads -20% |
| Rain | Delivery demand +25% |
| Sunny weekend | BBQ items +60% |

### Supply Impact
| Weather | Supply Effect |
|---------|---------------|
| Frost | Leafy greens limited |
| Heavy rain | Harvest delays |
| Heatwave | Short shelf life |
| Storms | Delivery disruption |

## Monitoring Process

### Daily Check
```
Morning routine:
□ Check 7-day forecast
□ Note extreme conditions
□ Alert relevant teams
□ Adjust orders if needed
```

### Weather Sources
```
Primary: BOM
Backup: Weatherzone
Farm-specific: Local sensors
```

## Demand Adjustments

### Hot Weather Protocol
```
When forecast >30°C:
- Increase: Salads, berries, drinks
- Decrease: Root vegetables, soups
- Alert: Cold chain capacity
```

### Cold Weather Protocol
```
When forecast <12°C:
- Increase: Soups, root veg, citrus
- Decrease: Salad mixes
- Alert: Delivery scheduling
```

### Storm Protocol
```
When severe weather:
- Pre-stock essentials
- Communicate with suppliers
- Plan delivery alternatives
- Customer notifications
```

## Supply Chain Alerts

### Farm Impact Assessment
```
Monitor for:
- Frost warnings (leafy greens)
- Hail risk (stone fruit)
- Flood risk (all produce)
- Drought (water-dependent crops)
```

### Contingency Triggers
```
Activate backup suppliers when:
- Primary farm affected
- Quality compromised
- Delivery impossible
```
