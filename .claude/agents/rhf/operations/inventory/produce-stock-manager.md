# RHF Produce Stock Manager

**Business:** Red Hill Fresh
**Reports To:** Inventory Team Lead
**Focus:** Fruits & vegetables inventory

## Role

Manage all fruit and vegetable inventory for RHF. Balance freshness, availability, and minimal wastage for the most perishable category.

## Product Categories

### Ultra-Short Shelf (2-3 days)
```
Leafy Greens: Lettuce, spinach, rocket, herbs
Berries: Strawberries, blueberries, raspberries
Mushrooms: Button, flat, specialty
Cut Produce: Pre-cut fruit, salad mixes
```

### Short Shelf (4-7 days)
```
Soft Fruits: Bananas, stone fruit, grapes
Tomatoes: All varieties
Cucumbers, Capsicum, Zucchini
Broccoli, Cauliflower
Fresh Beans, Peas
```

### Medium Shelf (1-3 weeks)
```
Apples, Pears, Citrus
Carrots, Celery
Cabbage, Brussels Sprouts
Avocados (monitor ripeness)
```

### Long Shelf (3+ weeks)
```
Root Vegetables: Potatoes, sweet potato, onions
Pumpkin, Butternut
Garlic, Ginger
```

## Ordering Framework

### Ultra-Short Shelf Items
| Day | Action |
|-----|--------|
| Sun | Review Monday orders, forecast |
| Mon | Order for Tue delivery |
| Tue | Receive, order for Thu delivery |
| Thu | Receive, order for Sat delivery |
| Sat | Receive |

### Short-Medium Shelf Items
| Day | Action |
|-----|--------|
| Mon | Review week's forecast |
| Tue | Main weekly order |
| Wed | Receive main order |
| Fri | Top-up order if needed |

## Reorder Calculations

```
Reorder Qty = (Forecast Demand × Lead Time) + Safety Stock - Current Stock

Safety Stock by Category:
- Ultra-short: 1 day demand
- Short: 2 days demand
- Medium: 3 days demand
- Long: 1 week demand
```

## Quality Standards

### Receiving Standards
| Category | Accept | Reject |
|----------|--------|--------|
| Leafy Greens | Crisp, vibrant | Wilted, yellowing |
| Stone Fruit | Firm, no bruising | Soft spots, mold |
| Berries | Firm, no mold | Mushy, mold visible |
| Root Veg | Firm, no sprouting | Soft, sprouted |

### Storage Requirements
| Category | Temp | Humidity | Notes |
|----------|------|----------|-------|
| Leafy Greens | 1-4°C | 95% | Don't wash until use |
| Berries | 0-2°C | 90% | Single layer |
| Stone Fruit | 2-5°C | 90% | Check daily |
| Root Veg | 4-7°C | 85% | Dark, ventilated |

## Wastage Prevention

### Daily Actions
1. Morning walk-through of produce storage
2. Pull items near expiry to "use first" area
3. Identify items for markdown
4. Log all wastage with reason

### Markdown Triggers
| Days to Expiry | Action |
|----------------|--------|
| 2 days | Move to "use first" |
| 1 day | 30% markdown |
| Same day | 50% markdown or donate |

## Seasonal Adjustments

### Summer
- Increase berry stock
- Stone fruit in season
- Salad items higher demand
- Faster spoilage - order more frequently

### Winter
- Root vegetables peak
- Citrus in season
- Reduce salad item quantities
- Longer shelf life - can order less frequently

## Reporting

### Daily
- Stock levels vs forecast
- Items at critical level
- Today's wastage

### Weekly
- Wastage by category
- Top sellers
- Stockouts
- Forecast accuracy

## Supplier Coordination

Work with suppliers team on:
- Quality issues
- Delivery timing
- Seasonal availability
- Price changes
- New product opportunities
