# RHF Receiving Coordinator

**Business:** Red Hill Fresh
**Reports To:** Inventory Team Lead
**Focus:** Supplier delivery management

## Role

Coordinate all incoming supplier deliveries, verify quality and quantities, and ensure smooth receiving operations.

## Receiving Schedule

### Daily Deliveries
| Time | Supplier Type | Priority |
|------|---------------|----------|
| 5am-7am | Produce farms | High |
| 7am-9am | Dairy suppliers | High |
| 8am-10am | Meat suppliers | High |
| Variable | Other suppliers | Normal |

## Receiving Process

### Standard Workflow
```
1. Check scheduled deliveries
2. Prepare receiving area
3. Supplier arrives
4. Verify delivery documents
5. Unload and count items
6. Quality check
7. Temperature check (if required)
8. Sign delivery receipt
9. Enter into system
10. Move to appropriate storage
```

### Quality Checks
| Product | Checks |
|---------|--------|
| Produce | Visual, freshness, damage |
| Dairy | Temperature, expiry, packaging |
| Meat | Temperature, color, smell |

```
- [Any problems noted]

```

### Variance Handling
| Variance Type | Action |
|---------------|--------|
| Short delivery | Note on receipt, notify supplier |
| Quality issue | Reject/accept partial, document |
| Wrong item | Refuse, notify supplier |
| Temperature breach | Reject, document |

## Supplier Communication

### On-Delivery Issues
```
Call supplier immediately if:
- Delivery not on schedule
- Major quantity variance
- Quality concerns
- Documentation missing
```

### Issue Template
```
To: [Supplier]

- [Description]

- [Effect on operations]

- [What needs to happen]

Please advise.

[Your name]
RHF Receiving
```

## Cold Chain Management

### Temperature Requirements
| Product | Required Temp | Max Time Unrefrigerated |
|---------|---------------|-------------------------|
| Dairy | 2-4°C | 15 minutes |
| Meat | 0-2°C | 10 minutes |
| Leafy greens | 2-5°C | 20 minutes |

### Temperature Logging
```
Record at:
1. Delivery (supplier transport)
2. During inspection
3. Into storage
```
