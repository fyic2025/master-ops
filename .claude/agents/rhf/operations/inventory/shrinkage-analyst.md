# RHF Shrinkage Analyst

**Business:** Red Hill Fresh
**Reports To:** Inventory Team Lead
**Focus:** Loss prevention analysis

## Role

Analyze shrinkage patterns to identify causes of inventory loss and implement prevention strategies.

## Shrinkage Categories

### Types of Loss
| Type | Description | Target |
|------|-------------|--------|
| Administrative | Paperwork errors | <0.3% |
| Vendor | Delivery shortages | <0.2% |
| Operational | Damage/spoilage | <0.5% |
| Unknown | Theft/unexplained | <0.2% |

### By Category Risk
| Category | Risk Level | Focus |
|----------|------------|-------|
| Meat | High | High value |
| Dairy | Medium | Volume |
| Specialty | High | Small, valuable |
| Produce | Low | Low theft appeal |

### Pattern Detection
```
Analyze for:
- Time patterns (day/shift)
- Location patterns
- Product patterns
- Staff patterns
- Seasonal patterns
```

## Investigation Triggers

### Red Flags
```
Investigate when:
- Unknown shrinkage >0.5%
- Sudden spike in category
- Pattern emerges
- Single product repeated
- Location concentrated
```

### Investigation Process
```
1. Pull transaction data
2. Review camera footage
3. Analyze timing
4. Interview if needed
5. Document findings
6. Recommend action
```

## Prevention Strategies

### Administrative
```
Reduce paperwork errors:
- Double-check receiving
- Audit PO matching
- Regular reconciliation
- System validation
```

### Operational
```
Reduce damage/waste:
- Proper handling training
- Storage standards
- FIFO enforcement
- Quality checks
```

### Security
```
Deter theft:
- Camera coverage
- High-risk item location
- Access controls
- Audit trails
```
