# RHF Inventory Counter

**Business:** Red Hill Fresh
**Reports To:** Inventory Team Lead
**Focus:** Stock counting and accuracy

## Role

Maintain accurate inventory counts through regular cycle counts and periodic full counts, ensuring system accuracy.

## Count Schedule

### Daily Counts
| Area | Time | Priority Items |
|------|------|----------------|
| Produce | 6am | All leafy greens |
| Dairy | 7am | Short-dated items |
| Meat | 7am | All items |

### Weekly Counts
| Day | Focus |
|-----|-------|
| Monday | Fruit section |
| Tuesday | Vegetables section |

| Thursday | Meat full count |
| Friday | Pantry items |

### Monthly Full Count
```
End of month:
- Complete inventory
- All storage locations
- All product categories
- System reconciliation
```

## Counting Process

### Standard Count
```
1. Print count sheet
2. Verify storage locations
3. Count physically (no peaking at system)
4. Record actual quantities
5. Note quality/expiry issues
6. Enter into system
7. Investigate variances
8. Adjust if verified
```

### Count Best Practices
```
- Count same location at same time
- No movements during count
- Count twice if uncertain
- Report issues immediately
- Document everything
```

## Variance Investigation

### Variance Thresholds
| Variance | Action |
|----------|--------|
| <2% | Investigate, adjust if verified |
| 2-5% | Investigate, document cause |
| >5% | Escalate, full investigation |
| Any meat/dairy | Investigate all |

### Common Causes
| Cause | Solution |
|-------|----------|
| Receiving errors | Training, double-check |
| Picking errors | Process improvement |
| Wastage not recorded | Better wastage tracking |
| Theft | Security review |
| System errors | IT investigation |
