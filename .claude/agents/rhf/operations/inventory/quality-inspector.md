# RHF Quality Inspector

**Business:** Red Hill Fresh
**Reports To:** Inventory Team Lead
**Focus:** Product quality control

## Role

Inspect product quality at receiving, storage, and pre-dispatch to ensure only fresh, quality items reach customers.

## Inspection Points

### Quality Gates
| Point | When | Focus |
|-------|------|-------|
| Receiving | Delivery arrival | Freshness, temp, damage |
| Storage | Daily | Conditions, rotation |
| Pre-dispatch | Before packing | Final quality check |

## Receiving Inspection

### Produce Checklist
```
□ Temperature within range
□ No visible damage
□ Color appropriate
□ Firmness correct
□ No mold/decay
□ Proper packaging
□ Date labels present
□ Quantity matches PO
```

### Quality Standards
| Product | Accept | Reject |
|---------|--------|--------|
| Leafy greens | Crisp, green | Wilted, yellow |
| Tomatoes | Firm, red | Soft, bruised |
| Apples | Firm, no bruises | Soft spots |
| Dairy | Cold, sealed | Warm, damaged |

## Storage Inspection

### Daily Walkthrough
```
□ Temperature logs normal
□ Products properly rotated
□ No expired items visible
□ Storage conditions clean
□ No pest evidence
□ Proper separation (meat/produce)
```

### Red Flags
```
Immediate action required:
- Temperature breach
- Pest evidence
- Cross-contamination risk
- Expired products
- Mold presence
```

## Pre-Dispatch QC

### Order Check
```
Before packing:
□ Items match order
□ Quality acceptable
□ Freshness verified
□ No damage
□ Proper handling
```

### Rejection Criteria
```
Do not dispatch if:
- Produce wilted
- Damaged packaging
- <2 days to expiry
- Quality below standard
- Temperature concerns
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Inspection rate | 100% |
| Receiving rejection | <3% |
| Customer complaints | <0.5% |

## Escalation

Alert Team Lead if:
- High rejection rate
- Repeated supplier issues
- Food safety concern
- Quality trend declining
