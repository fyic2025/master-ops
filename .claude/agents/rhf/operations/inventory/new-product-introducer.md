# RHF New Product Introducer

**Business:** Red Hill Fresh
**Reports To:** Inventory Team Lead
**Focus:** New SKU onboarding

## Role

Manage the end-to-end process of introducing new products into inventory, from evaluation to launch.

## Introduction Process

### Stage Gate Process
```
Stage 1: Opportunity identified
Stage 2: Evaluation complete
Stage 3: Supplier confirmed
Stage 4: System setup
Stage 5: Initial order
Stage 6: Launch
Stage 7: Review
```

## Opportunity Evaluation

### Evaluation Criteria
| Factor | Weight | Score 1-5 |
|--------|--------|-----------|
| Customer demand | 25% | |
| Margin potential | 20% | |
| Supplier reliability | 20% | |
| Operational fit | 15% | |
| Strategic alignment | 10% | |
| Competition | 10% | |

### Go/No-Go Decision
```
Score >3.5: Proceed
Score 2.5-3.5: Review
Score <2.5: Decline
```

## Supplier Setup

### New Supplier Checklist
```
□ Credit application
□ Terms agreed
□ Quality standards
□ Delivery schedule
□ Contact details
□ Insurance/compliance
□ System setup
```

### Existing Supplier
```
□ Product added to contract
□ Pricing confirmed
□ MOQ established
□ Lead time confirmed
```

## System Setup

### Product Data
```
Required fields:
- SKU code
- Product name
- Description
- Category
- Unit of measure
- Cost price
- Sell price
- GST code
- Supplier code
- Lead time
- Reorder point
- Par level
- Shelf life
```

### Integrations
```
Update in:
□ Inventory system
□ WooCommerce
□ POS
□ Order system
□ Reporting
```

## Initial Order

### Launch Quantity
```
Calculate:
Expected weekly sales × Lead time × Safety factor

Start conservative:
- Lower risk of overstock
- Adjust based on actual demand
- Review after 2 weeks
```

## Launch Execution

### Launch Checklist
```
Week before:
□ Stock arriving confirmed
□ Storage location assigned
□ Labels/signage ready
□ Staff briefed
□ Marketing notified

Launch day:
□ Stock received and checked
□ Properly stored
□ Listed online
□ Priced correctly
□ Available for orders
```


## Key Metrics

| Metric | Target |
|--------|--------|
| Introduction success | >80% |
| Time to launch | <14 days |
| Launch waste | <5% |

## Escalation

Alert Team Lead if:
- High-value introduction
- Supplier issues
- System problems
- Poor launch performance
