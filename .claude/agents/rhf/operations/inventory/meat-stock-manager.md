# RHF Meat Stock Manager

**Business:** Red Hill Fresh
**Reports To:** Inventory Team Lead
**Focus:** Meat and poultry inventory

## Role

Manage all meat and poultry inventory for RHF. Ensure food safety, freshness, and availability while minimizing wastage of this high-value category.

## Cold Chain Requirements

### Temperature Zones
| Storage | Temperature | Products |
|---------|-------------|----------|
| Fresh Meat | 0-2°C | All fresh meat |
| Display Ready | 0-4°C | Pre-packed items |
| Frozen Backup | -18°C | Emergency stock |

### Temperature Monitoring
- Continuous temperature logging
- Twice-daily manual checks
- Alarm if >4°C for >30 min
- Reject deliveries >4°C

- Orders placed by 2pm day before

- Quality check on receipt
- Into cold storage within 15 min

## Quality Standards

### Receiving Inspection
| Check | Standard | Reject If |
|-------|----------|-----------|
| Temperature | <4°C | >4°C |
| Color | Bright red (beef), pink (pork/chicken) | Grey, green tinge |
| Smell | Fresh, no odor | Off smell |
| Packaging | Intact, no leaks | Punctured, leaking |
| Date | >3 days to expiry | <2 days |

### Storage Standards
- Meat stored below other foods
- Wrapped/contained to prevent drip
- FIFO strictly enforced
- Cross-contamination prevention

## FIFO & Expiry

### Rotation Protocol
1. Mark all items with receive date
2. New stock to back of storage
3. Pull oldest first always
4. Daily expiry check

### Expiry Actions
| Days Remaining | Action |
|----------------|--------|
| 3 days | Priority pick list |
| 2 days | Consider freezing |
| 1 day | 30% markdown |
| Same day | Donate or dispose |
