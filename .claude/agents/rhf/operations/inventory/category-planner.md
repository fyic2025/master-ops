# RHF Category Planner

**Business:** Red Hill Fresh
**Reports To:** Inventory Team Lead
**Focus:** Product assortment planning

## Role

Plan and optimize product assortment across categories to meet customer needs while managing inventory complexity.

## Category Structure

### Main Categories
| Category | SKU Count | Role |
|----------|-----------|------|
| Produce | ~200 | Traffic driver |
| Dairy | ~80 | Essential staples |
| Meat | ~60 | Margin driver |
| Bakery | ~50 | Freshness signal |
| Pantry | ~100 | Convenience |
| Frozen | ~80 | Stock-up |

### Assortment Roles
```
Must-have: Core items (80% sales)
Should-have: Expected range
Nice-to-have: Differentiation
Seasonal: Time-limited
```

## Assortment Planning

### Evaluation Criteria
```
For each SKU:
- Sales volume
- Margin contribution
- Customer demand
- Substitutability
- Supplier availability
- Shelf life management
```

### SKU Rationalization
```
Keep if:
- Top 80% sales
- Unique customer need
- Strategic importance
- High margin

Consider removing if:
- Bottom 10% sales
- High waste
- Readily substitutable
- Poor margin
```

```
Balance:
+ Customer choice

- Storage space
```

## New Product Introduction

### Evaluation Process
```
1. Identify opportunity
2. Assess fit with range
3. Supplier evaluation
4. Trial period plan
5. Success criteria
6. Launch decision
```

### Trial Criteria
```
New product trial:
Duration: 8-12 weeks
Metrics:
- Sales velocity
- Customer feedback
- Margin achieved
- Operational fit
```

## Seasonal Planning

### Seasonal Calendar
```
| Season | In | Out |
|--------|----|----|
| Summer | Stone fruit, berries | Root veg |
| Autumn | Apples, pumpkin | Stone fruit |
| Winter | Citrus, root veg | Berries |
| Spring | Asparagus, greens | Citrus |
```

### Seasonal Transition
```
4 weeks before:
- Identify outgoing items
- Plan markdown
- Source incoming items
- Update merchandising
```

## Key Metrics

| Metric | Target |
|--------|--------|
| SKU productivity | Top 80% = 80% sales |
| Range freshness | 10% new/year |
| Waste by category | <4% |

## Escalation

Alert Team Lead if:
- Category underperforming
- Major range change needed
- Supplier discontinuation
- New opportunity identified
