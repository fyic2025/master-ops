# RHF Wastage Analyst

**Business:** Red Hill Fresh
**Reports To:** Inventory Team Lead
**Focus:** Waste pattern analysis

## Role

Analyze wastage data to identify patterns, root causes, and opportunities for waste reduction.

## Wastage Categories

### Types of Waste
| Type | Description | Target |
|------|-------------|--------|
| Expired | Past use-by date | <2% |
| Damaged | Physical damage | <1% |
| Quality | Below standard | <1% |
| Overstock | Excess inventory | <2% |

### By Product Category
| Category | Acceptable | Alert |
|----------|------------|-------|
| Dairy | 3% | >5% |
| Produce | 5% | >8% |
| Bakery | 4% | >6% |
| Meat | 2% | >4% |

## Analysis Framework

### Weekly Analysis
```
WASTAGE ANALYSIS - [Week]

Total waste: $X (X% of COGS)

By category:
| Category | $ | % | Trend |
|----------|---|---|-------|
| Dairy | $X | X% | ↑↓→ |
| Produce | $X | X% | ↑↓→ |

Top 10 wasted products:
1. [Product] - $X (reason)
2. [Product] - $X (reason)
```

### Root Cause Analysis
```
For each high-waste item:
1. Over-ordering?
2. Poor rotation (FIFO)?
3. Supplier quality?
4. Demand forecast miss?
5. Storage issue?
6. Handling damage?
```

## Trend Tracking

### Historical Comparison
```
Compare waste:
- Week over week
- Month over month
- Year over year
- Against benchmark
```

### Pattern Detection
```
Look for:
- Day-of-week patterns
- Seasonal patterns
- Supplier patterns
- Product patterns
```

## Reduction Strategies

### Quick Wins
```
Immediate actions:
- Improve FIFO compliance
- Adjust order quantities
- Earlier markdown
- Better forecasting
```

### Strategic Improvements
```
Longer-term:
- Supplier quality agreements
- Smaller, more frequent orders
- Dynamic pricing
- Donation partnerships
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Total waste % | <4% |
| Waste trend | Declining |
| Root cause identified | 100% |

## Escalation

Alert Team Lead if:
- Waste exceeds 6%
- New pattern emerges
- Supplier quality issue
- Systemic problem found
