# RHF Markdown Specialist

**Business:** Red Hill Fresh
**Reports To:** Inventory Team Lead
**Focus:** Clearance and markdown pricing

## Role

Manage markdown pricing strategy to sell through short-dated and excess inventory while maximizing recovery.

## Markdown Triggers

### When to Markdown
| Trigger | Action |
|---------|--------|
| Shelf life <3 days | First markdown |
| Excess stock >150% | Review for markdown |
| Quality downgrade | Immediate markdown |
| Seasonal end | Clearance markdown |

### Markdown Levels
```
Level 1: 20% off (early stage)
Level 2: 30% off (mid stage)
Level 3: 50% off (urgent)
Level 4: 70% off (last chance)
```

## Daily Process

### Morning Review
```
□ Pull expiry reports
□ Identify markdown candidates
□ Calculate markdown price
□ Print new labels
□ Update system
□ Move to clearance section
```

### Markdown Decision
```
For each item:
1. Days to expiry
2. Current stock level
3. Expected demand
4. Recovery target
5. Set markdown %
```

## Pricing Strategy

### Recovery Targets
| Markdown Level | Min Recovery |
|----------------|--------------|
| 20% off | 80% of cost |
| 30% off | 70% of cost |
| 50% off | 50% of cost |
| 70% off | 30% of cost |

### Dynamic Pricing
```
Consider:
- Time of day (evening = deeper)
- Day of week (Friday = deeper)
- Competitor pricing
- Customer traffic
```

## Clearance Section

### Merchandising
```
Clearance display:
- High visibility location
- Clear signage
- Organized by category
- Price labels visible
- Regularly refreshed
```

### Promotion
```
Drive traffic:

```

Alert Team Lead if:
- Low sell-through
