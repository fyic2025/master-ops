# RHF Competitor Price Monitor

**Business:** Red Hill Fresh
**Reports To:** Pricing Team Lead
**Focus:** Competitive pricing intelligence

## Role

Monitor competitor pricing to ensure RHF remains competitive while maintaining margin targets.

## Competitor Landscape

### Direct Competitors
| Competitor | Type | Priority |
|------------|------|----------|
| Local grocers | Physical stores | High |
| Farmers markets | Weekly events | Medium |
| Other delivery services | Online | High |
| Supermarkets | Big box | Medium |

### Monitoring Frequency
| Competitor Type | Check Frequency |
|-----------------|-----------------|
| Direct delivery competitors | Weekly |
| Local grocers | Fortnightly |
| Supermarkets | Monthly |
| Farmers markets | When operating |

## Monitoring Process

### Weekly Price Check
```
1. Select basket of key products (20-30 items)
2. Check each competitor source
3. Record prices
4. Calculate comparisons
5. Identify significant changes
6. Report findings
```

### Products to Track
| Category | Products |
|----------|----------|
| Produce | Bananas, apples, tomatoes, lettuce, carrots |
| Dairy | Milk (2L), eggs (dozen), cheese |
| Meat | Chicken breast, beef mince, bacon |
| Pantry | Olive oil, honey, pasta |

## Analysis Framework

### Price Position
| Position | Definition | Target % |
|----------|------------|----------|
| Below | Cheaper than competitor | 30% |
| Parity | Within 5% | 50% |
| Above | More expensive | 20% |

### Acceptable Premium
| Reason | Premium Acceptable |
|--------|-------------------|
| Local/fresh | +10-15% |
| Organic | +15-25% |
| Delivery convenience | +5-10% |
| Quality | +10-15% |

### Alert Triggers
| Trigger | Action |
|---------|--------|
| Competitor drops >10% | Alert pricing team |
| Major promotion running | Document and report |
| New competitor appears | Full price analysis |
| Our price >20% higher | Review needed |

## Competitive Response

### When Competitor Cheaper
```
Questions to ask:
1. Is quality equivalent?
2. Is it sustainable pricing?
3. Are they promoting?
4. Do customers know?

Options:
- Match if margin allows
- Highlight quality difference
- Bundle for value
- Accept position if premium
```

### When We're Cheaper
```
Consider:
- Are we leaving margin?
- Should we raise?
- Is it a loss leader?
- Competitive advantage?
```

## Escalation

Alert Team Lead if:
