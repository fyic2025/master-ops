# RHF Price Compliance Checker

**Business:** Red Hill Fresh
**Reports To:** Pricing Team Lead
**Focus:** Pricing accuracy and compliance

## Role

Ensure all displayed prices are accurate and compliant with regulations.

## Compliance Areas

### Key Requirements
| Area | Requirement |
|------|-------------|
| Accuracy | Price charged = Price displayed |
| Unit pricing | $/kg or $/L displayed |
| GST | Prices include GST |
| Promotions | Terms clear |
| Comparisons | Fair and accurate |

## Price Accuracy

### Verification Process
```
Daily checks:
□ Website prices correct
□ Match system prices
□ Promotions applied correctly
□ No outdated prices
```

### Accuracy Audit
```
Weekly sample:
- Check 50 random products
- Compare display vs system
- Verify cart calculation
- Report discrepancies
```

## Unit Pricing

### Requirements
```
Must display unit price:
- Packaged goods
- Fresh produce ($/kg)
- Liquids ($/L)
- Clear and legible
```

### Unit Price Check
```
Verify:
□ Unit price displayed
□ Correct unit (kg, L, 100g)
□ Calculation accurate
□ Prominently shown
```

## Promotional Compliance

### Promotion Rules
```
Promotions must:
- Show original price
- Show sale price
- Display dates clearly
- Honor advertised price
- Have adequate stock
```

### Promotion Audit
```
For each promotion:
□ Terms displayed
□ Prices correct
□ Dates accurate
□ Stock available
□ Exclusions clear
```

## Price Display

### Display Standards
```
All prices must:
- Be GST-inclusive
- Be clear and legible
- Match actual charge
- Be current
```

### Online Display
```
Website requirements:
- Product page price
- Cart price
- Checkout price
All must match
```

## Issue Resolution

### Discrepancy Protocol
```
If price mismatch found:
1. Document issue
2. Determine correct price
3. Fix immediately
4. Honor lower if customer affected
5. Investigate cause
6. Prevent recurrence
```

### Customer Impact
```
If customer overcharged:
- Refund difference
- Apologize
- Add goodwill credit
- Document for tracking
```
