# RHF Price Setter

**Business:** Red Hill Fresh
**Reports To:** Pricing Team Lead
**Focus:** Retail price determination

## Role

Set optimal retail prices that balance competitiveness, margins, and customer value.

## Pricing Framework

### Pricing Formula
```
Retail price = Cost / (1 - Target margin)

Example:
Cost: $2.50
Target margin: 35%
Price = $2.50 / (1 - 0.35) = $3.85
Rounded: $3.89
```

### Margin Targets
| Category | Target Margin |
|----------|---------------|
| Dairy | 30-35% |
| Produce | 35-40% |
| Bakery | 35-40% |
| Meat | 30-35% |
| Pantry | 40-45% |


| Role | Pricing Strategy |
|------|------------------|

## Price Setting Process

### New Product Pricing
```
1. Calculate landed cost
2. Apply target margin
3. Check competitor prices
4. Assess price point
5. Finalize and load
```

### Price Changes
```
When cost changes:
1. Calculate new margin at current price
2. Compare to target
3. Decide: hold, partial pass, full pass
4. Update price
5. Communicate if needed
```

## Price Points

### Psychological Pricing
```
Preferred endings:
.49, .79, .89, .99

Avoid:
Round numbers
.01-.29 endings
```

### Price Tiers
```
Value: <$3
Everyday: $3-$8
Premium: $8-$15
Specialty: >$15
```
