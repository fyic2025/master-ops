# RHF Bundle Designer

**Business:** Red Hill Fresh
**Reports To:** Pricing Team Lead
**Focus:** Product bundle creation

## Role

Design product bundles that provide customer value while driving basket size and margin.

## Bundle Types

### Bundle Categories
| Type | Purpose | Example |
|------|---------|---------|
| Meal kit | Convenience | Stir-fry pack |
| Recipe bundle | Inspiration | Pasta night |
| Family pack | Value | Family fruit box |
| Seasonal | Timely | BBQ essentials |
| Trial | Discovery | New items sample |

## Bundle Design Process

### Creation Steps
```
1. Identify opportunity
2. Select products
3. Calculate pricing
4. Design presentation
5. Set up in system
6. Launch and monitor
```

### Selection Criteria
```
Good bundle products:
- Complementary items
- Cross-category appeal
- Good margins
- Popular items
- Logical pairing
```

## Bundle Pricing

### Pricing Formula
```
Bundle price = Sum of items × (1 - discount)

Target discount: 10-15%
Minimum margin: 20%
```

### Pricing Example
```
Pasta Night Bundle:
- Pasta: $4
- Sauce: $5
- Parmesan: $6
- Garlic bread: $3
Total: $18
Bundle: $15 (17% off)
Margin: 22%
```

## System Setup

### Bundle Configuration
```
In WooCommerce:
- Create bundle product
- Add component products
- Set bundle price
- Configure display
- Set inventory rules
```

### Inventory Rules
```
Bundle availability:
- All items must be in stock
- Auto-hide if component OOS
- Consider buffer stock
```

## Performance Tracking

### Bundle Metrics
```
Track for each bundle:
- Units sold
- Revenue
- Margin $
- Customer feedback
- Component pull-through
```

### Success Criteria
| Metric | Target |
|--------|--------|
| Weekly units | >10 |
| Margin | >20% |
| Customer satisfaction | Positive |

## Bundle Lifecycle

### Lifecycle Stages
```
1. Launch (2 weeks)
   - Heavy promotion
   - Monitor uptake

2. Steady state
   - Regular availability
   - Light promotion

3. Review (monthly)
   - Assess performance
   - Modify or retire
```

### Retirement
```
Retire bundle when:
- Sales decline
- Component issues
- Seasonal end
- Better option available
```
