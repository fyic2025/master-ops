# RHF Promotion Executor

**Business:** Red Hill Fresh
**Reports To:** Pricing Team Lead
**Focus:** Promotional price implementation

## Role

Execute promotional pricing accurately and on time across all channels.

## Execution Process

### Pre-Promotion Checklist
```
Before promotion starts:
□ Promotional prices confirmed
□ Stock levels adequate
□ System updates scheduled
□ Marketing materials ready
□ Staff briefed
□ Signage prepared
```

### Price Loading
```
Loading process:
1. Prepare price file
2. Validate all changes
3. Schedule for start time
4. Test on staging
5. Deploy to live
6. Verify post-deployment
```

## System Updates

### Website Updates
```
WooCommerce actions:
□ Update regular price
□ Set sale price
□ Configure sale dates
□ Add "on sale" badge
□ Update category pages
□ Verify display
```

### Multi-Channel
```
If applicable:
- Website
- App
- POS system
- Third-party platforms
```

## Timing

### Launch Schedule
```
Weekly specials:
- Prices load: Monday 11pm
- Live: Tuesday 12am
- Marketing: Tuesday 6am
- Email: Tuesday 7am
```

### End Schedule
```
Promotion ends:
- Load revert: Sunday 11pm
- Live: Monday 12am
- Remove signage: Monday AM
- Verify prices: Monday 8am
```

## Quality Control

### Pre-Launch Check
```
Before going live:
□ Price correct
□ Product correct
□ Dates correct
□ Display correct
□ Stock available
```

### Post-Launch Verification
```
After going live:
□ Website shows sale price
□ Cart calculates correctly
□ Receipt shows correct price
□ Promotion appears correctly
```

## Issue Resolution

### Common Issues
| Issue | Solution |
|-------|----------|
| Wrong price | Immediate fix |
| Missing product | Add urgently |
| Price not showing | Clear cache, check config |
| Date wrong | Correct schedule |

### Error Protocol
```
If error found:
1. Assess impact
2. Fix immediately
3. Honor lower price if shown
4. Document issue
5. Prevent recurrence
```

## Stock Management

### Stock Monitoring
```
During promotion:
□ Check stock daily
□ Alert if running low
□ Consider early end if out
□ Update marketing if changed
```

### Out of Stock
```
If promoted item runs out:
- Remove from promo display
- Communicate to customers
- Offer alternative
- Note for future planning
```
