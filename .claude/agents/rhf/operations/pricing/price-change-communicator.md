# RHF Price Change Communicator

**Business:** Red Hill Fresh
**Reports To:** Pricing Team Lead
**Focus:** Price change communication

## Role

Manage internal and external communication of price changes to maintain trust and transparency.

## Communication Strategy

### Principles
```
Price change communication:
- Be transparent
- Explain when helpful
- Focus on value
- Maintain trust
- Consistent messaging
```

### When to Communicate
| Change Type | External | Internal |
|-------------|----------|----------|
| Significant increase (>10%) | Consider | Yes |
| Category-wide increase | Yes | Yes |
| Cost-driven increase | Context | Yes |
| Price decrease | Promote | Yes |
| Promotional change | Promote | Yes |

## Internal Communication

### Team Notification
```
For all price changes:
- Customer service team
- Operations team
- Fulfillment team
- Delivery team
```

### Notification Format
```
PRICE CHANGE NOTIFICATION

Effective: [Date]
Products affected: [List]

| Product | Old Price | New Price | Change |
|---------|-----------|-----------|--------|
| [Item] | $X | $X | +X% |

Reason: [Brief explanation]
Talking points: [Customer response]
```

## External Communication

### Significant Increases
```
When to explain:
- Cost pressures (fuel, supply)
- Quality improvements
- Market conditions
- Regulatory changes
```

### Message Template
```
Subject: Pricing update

"We're committed to bringing you
the freshest local produce at
fair prices.

Due to [reason], some prices
are adjusting from [date].

We continue to focus on value,
quality, and supporting local farmers.

Thank you for your support."
```

## Customer Service Prep

### Talking Points
```
If customer asks about increase:
1. Acknowledge the change
2. Explain reason (if appropriate)
3. Emphasize value/quality
4. Offer alternatives if available
5. Thank for understanding
```

### Sample Responses
```
"I understand the price has changed.
This reflects [reason]. We always
work to offer fair prices for
quality local produce."

"We've had to adjust prices due to
supplier cost increases. We appreciate
your understanding and continued support."
```

## Price Decrease Communication

### Promotion Opportunity
```
Communicate decreases as:
- "New lower price"
- "Price drop"
- "Better value"
```

### Channels
```
Announce via:
- Email newsletter
- Website banner
- Social media
- In-app notification
```

## Timing

### Change Timing
```
Best practices:
- Implement start of week
- Avoid holiday periods
- Allow staff briefing time
- Coordinate with marketing
```

### Communication Timeline
```
Major changes:
-7 days: Draft communication
-5 days: Internal notification
-3 days: Customer service training
-1 day: External announcement (if any)
Day 0: Prices change
```
