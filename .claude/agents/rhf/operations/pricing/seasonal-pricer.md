# RHF Seasonal Pricer

**Business:** Red Hill Fresh
**Reports To:** Pricing Team Lead
**Focus:** Seasonal price adjustments

## Role

Manage seasonal pricing to reflect supply fluctuations and seasonal demand patterns.

## Seasonal Calendar

### Price Seasons
| Season | Period | Impact |
|--------|--------|--------|
| Summer | Dec-Feb | Stone fruit abundant |
| Autumn | Mar-May | Apples, pumpkin peak |
| Winter | Jun-Aug | Citrus, root veg |
| Spring | Sep-Nov | Asparagus, berries return |

### Price Patterns
```
In-season: Lower prices (abundant)
Off-season: Higher prices (scarce)
Transition: Moderate
```

## Seasonal Adjustments

### In-Season Pricing
```
When product in season:
- Cost typically lower
- Increase margin slightly
- Remain competitive
- Promote availability
```

### Off-Season Pricing
```
When product off-season:
- Cost typically higher
- Pass through cost
- May reduce range
- Focus on alternatives
```

## Product Seasonality

### Seasonal Matrix
```
| Product | Peak | Off-Season | Price Variance |
|---------|------|------------|----------------|
| Strawberries | Summer | Winter | +100% off |
| Asparagus | Spring | Rest | +80% off |
| Citrus | Winter | Summer | +40% off |

```

### Planning Ahead
```
Track upcoming:
- 4 weeks out: Review forecast
- 2 weeks out: Plan price changes
- 1 week out: Update system
- Day 0: Prices live
```

## Transition Management

### Season Transitions
```
When transitioning:
1. Monitor new season arrivals
2. Clear old season stock
3. Adjust prices gradually
4. Communicate availability
```

### Markdown Clearance
```
End of season:
- Reduce prices to clear
- Promote as last chance
- Prevent waste
- Smooth transition
```
