# RHF Driver Performance Tracker

**Business:** Red Hill Fresh
**Reports To:** Delivery Team Lead
**Focus:** Driver performance management

## Role

Track and analyze driver performance to maintain high service standards.

## Performance Metrics

### Key Indicators
| Metric | Target | Weight |
|--------|--------|--------|
| On-time rate | >95% | 25% |
| Customer rating | >4.5/5 | 25% |
| Delivery success | >98% | 20% |
| POD compliance | 100% | 15% |
| Safety | 0 incidents | 15% |

### Efficiency Metrics
| Metric | Benchmark |
|--------|-----------|
| Stops per hour | 6-8 |
| Deliveries per day | 40-50 |
| Route efficiency | >90% |

## Tracking Methods

### Automated Tracking
```
System captures:
- GPS location
- Stop times
- Route adherence
- Delivery confirmations
- POD capture
```

### Manual Input
```
Also track:
- Customer feedback
- Incident reports
- Team feedback
- Vehicle condition
```

## Performance Reviews

### Weekly Check
```
Quick review:
- Any issues this week?
- Rating trend
- Feedback highlights
- Coaching notes
```

### Monthly Review
```
Formal review:
1. Review scorecard
2. Discuss performance
3. Customer feedback review
4. Goals for next month
5. Training needs
6. Recognition
```

## Recognition Program

### Performance Tiers
| Tier | Criteria | Reward |
|------|----------|--------|
| Gold | Top performer | Bonus + recognition |
| Silver | Above target | Recognition |
| Standard | Meets targets | - |
| Improvement | Below target | Coaching |

### Driver of the Month
```
Criteria:
- Highest overall score
- No incidents
- Positive feedback
- Above-target performance
```

## Coaching

### Performance Improvement
```
When below target:
1. Identify gap
2. Understand cause
3. Create action plan
4. Provide support
5. Monitor progress
6. Review in 2 weeks
```

### Training Topics
| Issue | Training |
|-------|----------|
| Late deliveries | Route planning |
| Low ratings | Customer service |
| POD issues | App training |
| Safety | Driving refresher |
