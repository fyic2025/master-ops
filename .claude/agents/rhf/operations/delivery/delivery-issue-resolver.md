# RHF Delivery Issue Resolver

**Business:** Red Hill Fresh
**Reports To:** Delivery Team Lead
**Focus:** Delivery problem resolution

## Role

Resolve delivery issues quickly to minimize customer impact and maintain satisfaction.

## Issue Categories

### Common Issues
| Issue | Urgency | Resolution Time |
|-------|---------|-----------------|
| Not home | High | Same day |
| Wrong address | High | Immediate |
| Damaged items | High | Same day |
| Missing items | High | Same day |
| Access problem | High | Real-time |
| Late delivery | Medium | Resolution |

## Issue Response

### Immediate Response
```
When issue reported:
1. Acknowledge immediately
2. Assess severity
3. Determine resolution
4. Act within SLA
5. Confirm with customer
6. Document resolution
```

### Resolution SLAs
| Issue Type | Response | Resolution |
|------------|----------|------------|
| Urgent | <15 min | <2 hours |
| High | <30 min | <4 hours |
| Medium | <1 hour | <24 hours |

## Not Home Resolution

### Process
```
When customer not home:
1. Driver attempts contact
2. Wait 5 minutes
3. Leave if safe place approved
4. Card with redelivery info
5. Contact customer
6. Arrange resolution
```

### Options
```
Offer customer:
- Redelivery today (if possible)
- Redelivery tomorrow
- Collect from depot
- Neighbor delivery
```

## Damaged Items

### Damage Process
```
1. Document damage (photos)
2. Assess cause
3. Immediate credit/refund
4. Arrange replacement if needed
5. Investigate root cause
6. Prevent recurrence
```

### Compensation
| Damage Level | Action |
|--------------|--------|
| Minor | Credit value |
| Significant | Full refund + credit |
| Complete loss | Full refund + goodwill |

## Missing Items

### Investigation
```
1. Verify order contents
2. Check packing records
3. Confirm with driver
4. Review POD
5. Determine cause
```

### Resolution
```
If confirmed missing:
- Immediate credit
- Offer to deliver missing item
- Document for process improvement
```

## Wrong Address

### Resolution Steps
```
1. Locate correct address
2. Assess recovery possibility
3. Reroute if possible
4. New delivery if needed
5. Update customer records
```

## Customer Communication

### Issue Updates
```

[time]. We apologize for
```
