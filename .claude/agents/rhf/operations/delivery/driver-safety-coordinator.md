# RHF Driver Safety Coordinator

**Business:** Red Hill Fresh
**Reports To:** Delivery Team Lead
**Focus:** Driver and vehicle safety

## Role

Ensure safe delivery operations through driver training, vehicle maintenance, and safety compliance.

## Safety Standards

### Driver Requirements
| Requirement | Frequency |
|-------------|-----------|
| Valid license | Verified quarterly |
| Driving record | Annual check |
| Safety training | Initial + annual |
| First aid certificate | Current |

### Vehicle Requirements
| Check | Frequency |
|-------|-----------|
| Registration | Current |
| Insurance | Current |
| Roadworthy | Annual |
| Service | Per schedule |

## Daily Safety Checks

### Pre-Trip Inspection
```
Before each shift:
□ Tires (pressure, condition)
□ Lights (all working)
□ Brakes (responsive)
□ Mirrors (clean, adjusted)
□ Fluids (oil, coolant, washer)
□ Fuel (adequate)
□ Refrigeration (if applicable)
□ Load secured
□ Emergency kit (first aid, fire extinguisher)
```

## Fatigue Management

### Work Hour Limits
```
Maximum driving:
- 10 hours in 24-hour period
- Minimum 30-min break per 5 hours
- Minimum 10 hours rest between shifts
```

### Warning Signs
```
If driver reports:
- Excessive tiredness
- Difficulty concentrating
- Microsleeps

Action:
- Stop driving immediately
- Rest or relief driver
- Report to coordinator
```

## Emergency Procedures

### Vehicle Breakdown
```
1. Pull over safely
2. Hazard lights on
3. Call coordinator
4. Stay with vehicle
5. Await assistance
6. Document incident
```

### Accident Protocol
```
1. Ensure safety first
2. Check for injuries
3. Call emergency services if needed
4. Exchange details
5. Take photos
6. Call coordinator immediately
7. Do not admit liability
8. Complete incident report
```
