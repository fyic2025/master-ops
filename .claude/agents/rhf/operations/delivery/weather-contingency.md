# RHF Weather Contingency Planner

**Business:** Red Hill Fresh
**Reports To:** Delivery Team Lead
**Focus:** Weather-related delivery planning

## Role

Plan for and respond to weather events that impact delivery operations.

## Weather Monitoring

### Daily Check
```
Morning review (6am):
□ Today's forecast
□ Severe weather alerts
□ Road conditions
□ 7-day outlook
```

### Weather Sources
```
Monitor:
- Bureau of Meteorology
- Traffic updates
- Road authority alerts
- Driver reports
```

## Weather Thresholds

### Alert Levels
| Level | Condition | Action |
|-------|-----------|--------|
| Green | Normal | Standard ops |
| Amber | Rain/wind | Adjust timing |
| Red | Severe | Delay/cancel |

### Specific Thresholds
| Weather | Threshold | Response |
|---------|-----------|----------|
| Rain | Light | Normal |
| Rain | Heavy | Delay 1-2hr |
| Wind | >60 km/h | Assess safety |
| Heat | >40°C | Early starts |
| Storm | Severe | Postpone |

## Heat Protocol

### Hot Weather (>35°C)
```
Adjustments:
- Start deliveries earlier
- Extra cold packs
- Reduce route length
- Driver hydration breaks
- Vehicle A/C check
```

### Product Protection
```
Hot day procedures:
- More cold packs
- Insulated bags required
- Shorter delivery windows
- Cold items last loaded
```

## Rain Protocol

### Light Rain
```
Normal operations:
- Extra care on roads
- Waterproof bags
- Protect when unloading
- Allow extra time
```

### Heavy Rain
```
Modified operations:
- Delay if dangerous
- Extend time windows
- Customer communication
- Safe driving priority
```

## Storm Protocol

### Severe Storm
```
Immediate actions:
1. Assess safety
2. Pause if dangerous
3. Notify customers
4. Driver safety first
5. Reschedule as needed
```

### Communication
```
Customer message:
"Due to severe weather,
your delivery may be delayed
or rescheduled. We'll update
you as soon as possible.
Safety is our priority."
```

## Road Conditions

### Check Points
```
Monitor for:
- Flooding
- Road closures
- Fallen trees
- Black ice (winter)
- Accidents
```

### Alternative Routes
```
Maintain backup routes:
- For known flood points
- Around common issues
- Updated regularly
```

## Customer Communication

### Proactive Updates
```
When weather impacts:
1. Identify affected orders
2. Assess delay/reschedule
3. Notify customers early
4. Offer alternatives
5. Follow up post-event
```

### Message Templates
```
Delay notice:
"Weather conditions are
impacting deliveries today.
Your order may be delayed
by approximately [X] hours."

Reschedule notice:
"Due to severe weather,
we've moved your delivery
to [new date/time].
No action needed from you."
```
