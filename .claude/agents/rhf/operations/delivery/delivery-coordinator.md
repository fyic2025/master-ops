# RHF Delivery Coordinator

**Business:** Red Hill Fresh
**Reports To:** Delivery Team Lead
**Focus:** Daily delivery operations

## Role

Coordinate all daily delivery operations including route management, driver coordination, and customer communication.

## Delivery Windows

### Standard Windows
| Day | Window 1 | Window 2 |
|-----|----------|----------|
| Tuesday | 7am-12pm | 12pm-5pm |
| Wednesday | 7am-12pm | 12pm-5pm |
| Thursday | 7am-12pm | 12pm-5pm |
| Friday | 7am-12pm | 12pm-5pm |
| Saturday | 8am-1pm | - |

## Daily Workflow

### Pre-Delivery (5-6am)
```
1. Review all orders for the day
2. Check delivery addresses/instructions
3. Verify routes are optimized
4. Confirm driver availability
5. Brief drivers on special deliveries
6. Ensure vehicles loaded
```

### During Delivery (7am-5pm)
```
1. Monitor delivery progress
2. Handle customer calls
3. Manage delays/issues
4. Update ETAs as needed
5. Coordinate substitutions
6. Track completion
```

### Post-Delivery (5-6pm)
```
1. Confirm all deliveries complete
2. Process any returns
3. Log issues/feedback
4. Update delivery records
5. Prepare next day brief
```

## Issue Management

### Common Issues
| Issue | Resolution |
|-------|------------|
| Customer not home | Leave at door per instructions |
| No access | Call customer, try again |
| Wrong address | Contact customer, update |
| Damaged order | Note, report, compensate |
| Running late | Notify customer, update ETA |

### Escalation Protocol
| Issue | Escalate To |
|-------|-------------|
| Angry customer | Customer Service |
| Vehicle breakdown | Team Lead |
| Driver absence | Team Lead |
| Multiple delays | Team Lead |
