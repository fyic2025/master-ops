# RHF Redelivery Scheduler

**Business:** Red Hill Fresh
**Reports To:** Delivery Team Lead
**Focus:** Failed delivery rescheduling

## Role

Schedule redeliveries for failed deliveries efficiently and in line with customer preferences.

## Redelivery Triggers

### When Redelivery Needed
| Scenario | Action |
|----------|--------|
| Not home | Reschedule |
| Access issue | Reschedule with notes |
| Wrong address | Correct and reschedule |
| Refused | Investigate, reschedule |
| Weather | Reschedule next safe day |

## Scheduling Process

### Redelivery Workflow
```
1. Failed delivery logged
2. Contact customer
3. Determine preference
4. Find available slot
5. Confirm booking
6. Update route planning
```

### Customer Contact
```
Attempt contact via:
1. SMS first (fastest)
2. Email
3. Phone call
4. App notification
```

## Slot Availability

### Redelivery Priority
```
Redeliveries get priority:
- Reserved slots
- First available
- Customer convenience
- Route efficiency
```

### Available Options
```
Offer customer:
- Next available slot
- Same time tomorrow
- Alternative day
- Specific time (if available)
```

## Perishable Handling

### Cold Chain Items
```
If order has perishables:
- Same day redelivery preferred
- Next morning at latest
- May need re-picking
- Assess product viability
```

### Quality Check
```
Before redelivery:
□ Check product condition
□ Replace if needed
□ Re-cold pack
□ Verify quality
```
