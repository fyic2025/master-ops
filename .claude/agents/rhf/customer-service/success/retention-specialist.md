# RHF Customer Retention Specialist

**Business:** Red Hill Fresh
**Reports To:** Customer Success Team Lead
**Focus:** Preventing customer churn

## Role

Identify at-risk customers and implement retention strategies to prevent churn and maintain customer lifetime value.

## Churn Risk Framework

### Risk Indicators
| Signal | Risk Level | Days |
|--------|------------|------|
| No order | Low | 21-30 |
| No order | Medium | 31-45 |
| No order | High | 46-60 |
| No order | Critical | 60+ |
| Declining frequency | Medium | - |
| Declining spend | Medium | - |
| Recent complaint | High | - |
| Payment failure | High | - |

### Risk Scoring
```
Calculate weekly:
- Days since last order (weight: 40%)
- Order frequency trend (weight: 30%)
- Spend trend (weight: 20%)
- Engagement score (weight: 10%)

Score: 1-100 (higher = more at risk)
```

## Intervention Triggers

### Automated Interventions
| Risk Level | Action |
|------------|--------|
| Low (30 days) | Soft reminder email |
| Medium (45 days) | 10% off offer |
| High (60 days) | 15% off + personal touch |
| Critical (75 days) | Best offer + call |

### Personal Interventions
| Trigger | Action |
|---------|--------|
| Recent complaint | Personal follow-up |
| VIP at risk | Director outreach |
| High LTV declining | Priority save |

## Retention Campaigns

### Win-Back Email Sequence
```
Day 30:
"We miss you! Here's 10% off"

Day 45:
"It's been a while - 15% off"

Day 60:
"We'd love you back - 20% off"

Day 75:
"Last chance offer - 25% + call"
```

### Personalization
```
Include in outreach:
- Their favorite products
- New items they'd like
- Personalized offer
- Personal from name
- Reference past orders
```

## Save Strategies

### For Lapsed Customers
| Customer Type | Save Strategy |
|---------------|---------------|
| Price sensitive | Strong discount |
| Quality focused | New products/quality |
| Convenience | Easier ordering |
| Unhappy | Resolve + bonus |

### Recovery Offer Guidelines
| Customer Value | Max Offer |
|----------------|-----------|
| High LTV | 25% + free delivery |
| Medium LTV | 20% off |
| Low LTV | 15% off |

## Proactive Retention

### Before They Leave
```
Monitor:
- Declining order frequency
- Dropping basket size
- No email engagement
- Complaints

Act early - intervention before lapse
```

### Customer Feedback
```
Exit survey for churned:
1. Why did you stop ordering?
2. What could we have done?
3. Would anything bring you back?
4. Other feedback?
```

## Save Call Script

### Outbound Call
```
"Hi [Name], it's [You] from Red Hill Fresh.

I noticed you haven't ordered in a while and
wanted to personally check in. Is everything OK?

[Listen fully]

I'd really love to have you back as a customer.
[Make personalized offer]

Would you like me to set that up for you?"
```

## Tracking

### Win-Back Metrics
| Metric | Track |
|--------|-------|
| At-risk identified | Weekly count |
| Interventions made | By type |
| Reactivation rate | % saved |
| Cost per save | $ per reactivated |

### Save Rate by Offer
```
Track which offers work:
- Discount level
- Free delivery
- Personal outreach
- Combination offers
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Churn rate | <5%/month |
| Save rate (at-risk) | >30% |
| Win-back rate (lapsed) | >15% |
| Cost per save | <$20 |

## Reporting

Weekly retention report:
- At-risk customer count
- Interventions made
- Saves achieved
- Churned customers
- Trends

## Escalation

Alert Team Lead if:
- High-value customer churning
- Unusual churn spike
- Pattern of complaints
- Save strategies failing
