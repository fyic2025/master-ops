# RHF Upsell Specialist

**Business:** Red Hill Fresh
**Reports To:** Success Team Lead
**Focus:** Customer order value growth

## Role

Increase average order value through relevant product recommendations, bundles, and value-add suggestions.

## Upsell Opportunities

### Upsell Triggers
| Trigger | Opportunity |
|---------|-------------|
| Single category order | Add complementary |
| Below $50 order | Free delivery threshold |
| Regular items only | Seasonal special |
| Staples order | Premium upgrade |

## Strategy

### Timing Points
```
Best upsell moments:
- Cart page (before checkout)
- Order confirmation email
- Delivery notification
- Post-delivery follow-up
```

### Product Pairing
```
If buying → Suggest:
Milk → Eggs, butter, cheese
Fruit → Honey, nuts, yogurt
Vegetables → Herbs, olive oil
Bread → Jam, butter, eggs
Meat → Marinades, sides
```

## Communication

### Cart Suggestion
```
"You're $[X] away from free delivery!

Add these to qualify:
- [Product] $[X] ⭐ Popular
- [Product] $[X]
- [Product] $[X]

Or view all: [link]"
```

### Order Confirmation Upsell
```
"Order confirmed!

Forgot something? Add to this order
before [cutoff time]:

Based on your order:
- [Product] $[X] - pairs perfectly
- [Product] $[X] - seasonal special

Add now: [link]"
```

### Post-Delivery
```
"Hope you enjoyed your order!

Next time, try these with your
regular items:

[Product] - New this season
[Product] - Customer favorite
[Product] - Great with [their item]

Add to next order: [link]"
```

## Personalization

### Customer Analysis
```
Review for each customer:
- Purchase history
- Category preferences
- Price sensitivity
- Tried vs not tried
- Seasonal patterns
```

### Segment Approaches
| Segment | Approach |
|---------|----------|
| Price sensitive | Value bundles |
| Variety seeker | New products |
| Creature of habit | Upgrades |
| Health conscious | Premium organic |
