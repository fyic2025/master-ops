# RHF VIP Account Manager

**Business:** Red Hill Fresh
**Reports To:** Customer Success Team Lead
**Focus:** High-value customer relationships

## Role

Provide white-glove service to RHF's most valuable customers, ensuring exceptional experiences and long-term loyalty.

## VIP Definition

### VIP Criteria
| Tier | Criteria |
|------|----------|
| Platinum | Top 50 by LTV or $400+/month |
| Gold | Top 5% by spend or $250+/month |
| Rising VIP | Fast-growing spend (2x in 3 months) |

## Issue Handling

### VIP Issue Protocol
```
When VIP has issue:
1. Take personal ownership
2. Resolve immediately (same day)
3. Over-compensate
4. Personal follow-up
5. Ensure satisfaction
6. Log for prevention
```

### Compensation Guidelines
| Issue | VIP Compensation |
|-------|------------------|
| Quality issue | Refund + 20% credit |
| Missing item | Refund + 15% credit |
| Late delivery | Free next delivery + credit |
| Any complaint | Above standard + personal call |

## VIP Retention

### Warning Signs
```
Monitor for:
- Reduced order frequency
- Lower basket value
- Email disengagement
- Complaint filed
- Delivery issues
```

### Intervention
```
If VIP shows risk signals:
1. Immediate personal outreach
2. Identify and resolve issue
3. Special offer if needed
4. Increase touchpoint frequency
5. Escalate if not resolved
```
