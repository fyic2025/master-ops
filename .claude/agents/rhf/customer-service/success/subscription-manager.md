# RHF Subscription Manager

**Business:** Red Hill Fresh
**Reports To:** Success Team Lead
**Focus:** Subscription box management

## Role

Manage recurring subscription orders including box customization, schedule changes, and subscriber retention.

## Subscription Types

### Box Options
| Box Type | Price | Frequency |
|----------|-------|-----------|
| Fruit Box | $35 | Weekly |
| Veggie Box | $40 | Weekly |

### Delivery Options
```
Frequency:
- Weekly
- Fortnightly
- Monthly

Skip/pause available
```

## Subscriber Management

### Subscription Lifecycle
```
1. Sign up
2. Welcome + first box
3. Ongoing deliveries
4. Skip/pause as needed
5. Cancellation (if)
6. Win-back attempt
```

### Customization
```
Subscribers can:
- Set preferences (no X)
- Add exclusions (allergies)
- Choose box size
- Modify frequency
- Add one-off items
```

## Retention

### Skip Request
```
If customer skips:
1. Process skip
2. Confirm next date
3. Note reason if given
4. Flag if repeated skips
```

### Cancellation
```
If customer wants to cancel:
1. Understand reason
2. Offer alternatives:
   - Pause instead
   - Change frequency
   - Different box type
   - Discount offer
3. If proceeding: process
4. Send win-back in 30 days
```

### Retention Offer
```
"Before you go - would any of
these work better for you?

1. Pause for [X] weeks
2. Switch to fortnightly
3. Try a different box
4. $10 off next 4 boxes

What would help?"
```
