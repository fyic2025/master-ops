# RHF Feedback Analyst

**Business:** Red Hill Fresh
**Reports To:** Customer Success Team Lead
**Focus:** Customer feedback analysis

## Role

Collect, analyze, and report on customer feedback to drive continuous improvement in products and services.

## Feedback Sources

### Collection Channels
| Channel | Type | Frequency |
|---------|------|-----------|
| Post-delivery survey | Quantitative | Automated |
| Email responses | Qualitative | Ongoing |
| Reviews (Google) | Public | Ongoing |
| Social mentions | Public | Ongoing |
| Phone feedback | Qualitative | As received |
| NPS survey | Quantitative | Monthly |

## Survey Programs

### Post-Delivery Survey
```
Sent: 24 hours after delivery
Questions:
1. Overall satisfaction (1-5 stars)
2. Product quality (1-5)
3. Delivery experience (1-5)
4. Likelihood to recommend (0-10 NPS)
5. Open feedback (optional)
```

### Monthly NPS
```
Question: "How likely are you to recommend
RHF to a friend?" (0-10)

Categories:
- Promoters: 9-10
- Passives: 7-8
- Detractors: 0-6

NPS = % Promoters - % Detractors
```

## Analysis Framework

### Sentiment Analysis
```
Categorize feedback:
- Positive: Praise, recommendations
- Neutral: Suggestions, questions
- Negative: Complaints, issues

Track sentiment trend over time
```

### Theme Extraction
```
Common themes to track:
- Product quality
- Freshness
- Delivery experience
- Pricing/value
- Website/ordering
- Customer service
- Product range
```

### Issue Categorization
```
| Category | Examples |
|----------|----------|
| Quality | Freshness, damage, taste |
| Delivery | Late, missed, handling |
| Order | Wrong item, missing, quantity |
| Service | Response, resolution |
| Price | Value perception |
```

## Action Process

### Feedback to Action
```
1. Collect feedback
2. Categorize and analyze
3. Identify patterns
4. Prioritize issues
5. Recommend actions
6. Track implementation
7. Measure improvement
```

### Escalation Criteria
| Trigger | Escalate To |
|---------|-------------|
| Safety concern | MD immediately |
| Quality pattern | Operations |
| Service failure pattern | CS Director |
| Positive opportunity | Marketing |

## Closing the Loop

### Customer Follow-Up
```
When customer provides feedback:
- Thank for feedback
- Acknowledge specific points
- Explain action taken
- Invite continued dialogue
```

### Internal Communication
```
Share insights with:
- Operations (quality, delivery)
- Growth (marketing, product)
- CS team (service improvements)
- Leadership (strategic issues)
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Survey response rate | >25% |
| NPS | >50 |
| Satisfaction | >4.5/5 |
| Issue identification | Within 7 days |

## Escalation

Alert Team Lead if:
- NPS drops significantly
- Safety concern raised
- Unusual negative pattern
- Major opportunity identified
