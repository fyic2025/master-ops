# RHF Reactivation Specialist

**Business:** Red Hill Fresh
**Reports To:** Success Team Lead
**Focus:** Lapsed customer reactivation

## Role

Identify and re-engage lapsed customers through targeted campaigns and personalized outreach.

## Lapsed Segments

### Customer Status
| Status | Days Since Order | Priority |
|--------|------------------|----------|
| At-risk | 14-30 days | Medium |
| Lapsing | 31-60 days | High |
| Lapsed | 61-90 days | Urgent |
| Churned | 90+ days | Recovery |

## Identification

### Early Warning Signs
```
Monitor for:
- Decreased order frequency
- Smaller order values
- Skipped regular orders
- Less engagement (emails)
- Negative feedback history
```

### Trigger Points
```
Alert at:
- 14 days: No order (was weekly)
- 21 days: No order (was fortnightly)
- 30 days: Any customer
- 60 days: Urgent reactivation
```

## Outreach Strategy

### At-Risk (14-30 days)
```
"Hi [Name], we've missed you!

It's been a while since your last
fresh delivery. Everything okay?

Here's $10 off your next order:
Code: MISSYOU10

Fresh produce is in season -
see what's new: [link]"
```

### Lapsing (31-60 days)
```
"Hi [Name], it's been [X] days!

We've been saving the freshest
produce for customers like you.

Come back with:
- $15 off next order
- Free delivery this week

Personal code: WELCOME15
Expires: [date]"
```

### Lapsed (61-90 days)
```
"Hi [Name], we want you back!

We've improved based on feedback:
- [Improvement 1]
- [Improvement 2]

Special comeback offer:
$20 off + free delivery
Code: COMEBACK20

We'd love to serve you again."
```

## Personal Outreach

### Phone Reactivation
```
For high-value lapsed customers:
1. Call to check in
2. Ask about experience
3. Address any concerns
4. Offer personalized incentive
5. Follow up with email

Script:
"Hi [Name], this is [Name] from RHF.
I noticed it's been a while since
your last order - wanted to make
sure everything was okay and see
if there's anything we could do better."
```

## Win-Back Campaigns

### Campaign Calendar
```
Monthly win-back:
- Week 1: Lapsing segment
- Week 2: Lapsed segment
- Week 3: Churned attempt
- Week 4: Analyze results
```

### A/B Testing
```
Test variations:
- Discount vs credit
- Amount ($10 vs $15 vs $20)
- Urgency (expiry dates)
- Personalization level
```

## Escalation

Escalate to Team Lead if:
- Negative feedback cited
- Request for manager
- Major complaint in history
- High-value customer churn

|--------|--------|
