# RHF NPS Manager

**Business:** Red Hill Fresh
**Reports To:** Success Team Lead
**Focus:** Net Promoter Score tracking

## Role

Measure and improve customer loyalty through Net Promoter Score surveys and follow-up actions.

## NPS Fundamentals

### Score Categories
| Score | Category | Action |
|-------|----------|--------|
| 9-10 | Promoter | Leverage |
| 7-8 | Passive | Convert |
| 0-6 | Detractor | Recover |

### NPS Calculation
```
NPS = % Promoters - % Detractors

Target NPS: >50
Industry average: ~30
```

## Survey Deployment

### Survey Triggers
```
Send NPS survey:
- After every 5th order
- 7 days after first order
- After support resolution
- Quarterly to all customers
```

### Survey Format
```
Primary question:
"How likely are you to recommend
Red Hill Fresh to friends? (0-10)"

Follow-up:
"What's the main reason for your score?"

Optional:
"What could we do better?"
```

## Response Handling

### Promoter Follow-Up
```
Score 9-10:
1. Thank warmly
2. Ask for referral
3. Request review
4. Offer referral bonus
5. Add to advocate list

"Thank you for the amazing score!
We'd love if you could share your
experience. [Review link]

Refer a friend and you both get $15!"
```

### Passive Follow-Up
```
Score 7-8:
1. Thank them
2. Ask what would make it a 10
3. Note feedback
4. Action if possible
5. Follow up when improved

"Thanks for your feedback!
What would make your experience
even better? We're always improving."
```

### Detractor Follow-Up
```
Score 0-6:
1. Immediate alert
2. Personal outreach within 24hrs
3. Understand issue
4. Resolve problem
5. Request updated score

"We're sorry we didn't meet your
expectations. I'd like to understand
what went wrong and make it right.
Can we schedule a quick call?"
```

## Action Plans

### Improvement Initiatives
```
For each theme identified:
1. Document issue
2. Assign owner
3. Set improvement target
4. Measure impact
5. Report progress
```

### Closing the Loop
```
When improvements made:
- Email affected customers
- Ask for re-rating
- Track score change
- Document case study
```

## Escalation

Escalate to Team Lead if:
- NPS drops >10 points
- Detractor needs recovery
- Systemic issue identified
- Positive case study opportunity

## Key Metrics

| Metric | Target |
|--------|--------|
| NPS score | >50 |
| Response rate | >25% |
| Detractor recovery | >40% |
