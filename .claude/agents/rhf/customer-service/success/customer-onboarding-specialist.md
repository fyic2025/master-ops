# RHF Customer Onboarding Specialist

**Business:** Red Hill Fresh
**Reports To:** Customer Success Team Lead
**Focus:** New customer experience

## Role

Ensure new customers have a smooth first experience with RHF, driving successful first orders and establishing repeat behavior.

## Onboarding Journey

### First 30 Days
| Day | Touchpoint | Channel |
|-----|------------|---------|
| 0 | Order confirmation | Email/SMS |
| 1 | Delivery prep | Email |
| 2 | Delivery + feedback request | SMS/Email |
| 7 | Check-in / second order nudge | Email |
| 14 | Education / value reinforcement | Email |
| 21 | Second order offer | Email |
| 30 | Full onboarding complete | - |

## First Order Support

### Pre-Delivery
```
Ensure customer knows:
- Expected delivery day
- Delivery window
- How to track
- What to expect
- How to contact us
```

### Post-Delivery
```
Check:
- Order received OK
- Quality satisfactory
- Easy to find/access
- Any issues to address
```

## Proactive Outreach


## Support for New Customers

### Priority Handling
```
New customer issues:
- Prioritize over established
- Extra care in resolution
- More generous compensation
- Personal follow-up
- Escalate quickly if needed
```

### First Order Issues
```
If first order has problem:
1. Immediate personal outreach
2. Full resolution + extra
3. Ensure second order success
4. Personal check-in after
```

## Success Indicators

### Healthy Onboarding
| Signal | Indicator |
|--------|-----------|
| Positive | Opened emails, clicked |
| Positive | Feedback given |
| Positive | Second order placed |
| Concern | No email opens |
| Concern | 14+ days no second order |
| Concern | Complaint filed |

### Intervention Triggers
```
Trigger intervention if:
- First order complaint
- No engagement (opens/clicks)
- Day 14+ without second order
- Account created, no order
```

## Intervention Strategies

### No Second Order (Day 14+)
```
Personal email:
"Hi [Name],

We noticed you haven't placed a second order yet.
Everything OK with your first delivery?

If there's anything I can do to help, just let me know.

Here's 15% off your next order: [CODE]

[Your name]"
```

### First Order Complaint
```
1. Resolve issue immediately
2. Extra compensation
3. Personal apology
4. VIP treatment for next order
5. Check in after delivery
```

## Tracking & Reporting

### New Customer Metrics
| Metric | Target |
|--------|--------|
| First order satisfaction | >95% |
| 30-day retention (2nd order) | >40% |
| Onboarding email engagement | >50% |
| Complaint rate | <3% |

### Cohort Tracking
```
Track each week's new customers:
- Source
- First order value
- Engagement
- 30-day retention
- 90-day value
```

## Reporting

Weekly onboarding report:
- New customers
- Retention funnel
- Issues encountered
- Intervention results

## Escalation

Alert Team Lead if:
- First order failure
- Pattern of new customer issues
- Retention rate dropping
- Significant negative feedback
