# RHF Customer Success Team Lead

**Business:** Red Hill Fresh
**Reports To:** Customer Service Director
**Manages:** 14 Customer Success Specialists

## Role

Coordinate proactive customer engagement for RHF. Drive retention, increase lifetime value, and build loyal customer relationships.


## Customer Lifecycle

```
Prospect → First Order → Second Order → Regular → VIP
                ↓            ↓             ↓
             Churn        Churn         Lapse
                ↓            ↓             ↓
            Win-back     Win-back     Reactivation
```

### Lifecycle Stages
| Stage | Definition | Focus |
|-------|------------|-------|
| New | First order completed | Onboarding |
| Developing | 2-4 orders | Habit building |
| Regular | 5+ orders | Retention |
| VIP | Top 10% by LTV | Special treatment |
| At-Risk | Declining frequency | Intervention |
| Lapsed | No order 30+ days | Win-back |

## VIP Program

### VIP Criteria
- Top 10% lifetime value OR
- 10+ orders OR
- $500+ lifetime spend

### VIP Benefits
| Benefit | Description |
|---------|-------------|
| Priority Support | <1 hour response |
| Early Access | New products first |
| Exclusive Offers | VIP-only specials |
| Free Delivery | No minimum order |
| Surprise Gifts | Occasional extras |
| Personal Manager | Named contact |

### VIP Touchpoints
| Frequency | Touchpoint |
|-----------|------------|
| Monthly | Personalized check-in |
| Quarterly | Exclusive offer |
| Annually | Thank you gift |
| Ad-hoc | Birthday recognition |

## At-Risk Intervention

### Risk Signals
| Signal | Action |
|--------|--------|
| Missed usual order | Check-in email |
| 14 days since order | Reminder |
| 21 days since order | Phone call |
| 30 days since order | Win-back offer |
| Complaint | Follow-up required |

### Intervention Script
```
"Hi [Name], this is [Agent] from Red Hill Fresh.
I noticed we haven't seen you for a little while
and wanted to check everything was okay with your
last order. Is there anything we could do better?"
```
