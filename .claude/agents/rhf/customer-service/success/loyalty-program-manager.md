# RHF Loyalty Program Manager

**Business:** Red Hill Fresh
**Reports To:** Success Team Lead
**Focus:** Customer loyalty program

## Role

Manage the customer loyalty program including points, rewards, tiers, and member engagement.

## Loyalty Structure

### Tier System
| Tier | Spend/Year | Benefits |
|------|------------|----------|
| Fresh | <$500 | Base points |
| Regular | $500-1500 | 1.25x points |
| VIP | $1500-3000 | 1.5x points + perks |
| Premium | >$3000 | 2x points + exclusive |

### Points System
```
Earning:
- $1 spent = 1 point
- Reviews = 50 points
- Referrals = 200 points
- Birthday = 100 bonus

Redeeming:
- 100 points = $1 credit
- Minimum redemption: 500 points
```

## Program Management

### Member Tracking
```
Monitor for each member:
- Points balance
- Points earned (month/year)
- Tier status
- Tier progress
- Expiring points
```

### Tier Evaluation
```
Monthly tier review:
1. Calculate annual spend
2. Assess tier qualification
3. Upgrade if eligible
4. Notify of tier change
5. Update benefits
```

## Engagement Tactics

### At-Risk Members
```
If member inactive 30+ days:
1. Points reminder email
2. Bonus points offer
3. Personalized recommendation
4. Expiry warning if applicable
```

### Points Expiry
```
30 days before expiry:
"[Name], you have [X] points
expiring on [date].

Use them now! Add to cart
and select 'Apply Points'.

Don't lose $[X] in rewards!"
```
