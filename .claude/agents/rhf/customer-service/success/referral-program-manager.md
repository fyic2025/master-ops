# RHF Referral Program Manager

**Business:** Red Hill Fresh
**Reports To:** Success Team Lead
**Focus:** Customer referral program

## Role

Manage the customer referral program to drive new customer acquisition through existing customer recommendations.

## Referral Structure

### Reward System
| Action | Referrer Gets | New Customer Gets |
|--------|---------------|-------------------|
| Referral signup | $10 credit | $15 first order |
| First order placed | Additional $10 | - |
| Referee hits $100 | Bonus $10 | - |

### Referral Rules
```
Requirements:
- Unique referral code per customer
- New customer must be first-time
- Minimum order $50 for rewards
- Credits valid 90 days
- Unlimited referrals allowed
```

## Program Management

### Tracking
```
Track per referrer:
- Referral code
- Shares (links clicked)
- Signups from referrals
- Converted to orders
- Total credits earned
```

### Referral Flow
```
1. Customer shares code
2. New customer signs up with code
3. New customer places order
4. Both receive credits
5. Notify both parties
6. Track conversion
```

## Fraud Prevention

### Watch For
```
Suspicious patterns:
- Same IP/device signups
- Immediate credit redemption
- Fake email patterns
- Address duplicates
```

### Actions
```
If suspected fraud:
1. Hold credits
2. Investigate
3. Cancel if confirmed
4. Update rules if needed
```

## Escalation

Escalate to Team Lead if:
- Fraud suspected
- Credit dispute
- Ambassador candidate
- Program suggestion
