## Common Inquiries & Responses

### Order Status
```
Q: Where is my order?
A: Check WooCommerce order status:
   - Processing: Confirm delivery day
   - Out for Delivery: Provide ETA
   - Delivered: Confirm delivery proof
```

### Product Substitution
```
Q: You substituted my [item]
A: Check substitution policy:
   - Similar product, equal/greater value
   - Customer can reject and get refund
   - Log feedback for inventory team
```

### Delivery Reschedule
```
Q: Can I change my delivery date?
A: Check order status:
   - Before cutoff: Yes, change in system
   - After cutoff: Check if route can accommodate
   - Out for delivery: Not possible
```

## Quality Standards

### Response Quality
- Use customer's name
- Acknowledge their concern
- Explain clearly
- Offer solution
- Follow up if needed

### Tone Guidelines
- Friendly and warm
- Apologetic when at fault
- Solution-focused
- Never defensive
- Local, neighborly feel

## Escalation Path

```
Level 1: Support Agent
   ↓ Can't resolve in 2 interactions
Level 2: Support Team Lead
   ↓ Complex or sensitive
Level 3: CS Director
   ↓ Legal/media/VIP
Level 4: MD
```

2. Note attempted resolutions
3. Include customer sentiment
4. Recommend next action
