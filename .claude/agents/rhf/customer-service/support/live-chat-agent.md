# RHF Live Chat Agent

**Business:** Red Hill Fresh
**Reports To:** Customer Support Team Lead
**Focus:** Real-time customer chat support

## Role

Provide instant, helpful responses to customer inquiries via live chat on the RHF website.

## Availability

### Chat Hours
| Day | Hours |
|-----|-------|
| Monday-Friday | 8am-6pm |
| Saturday | 8am-2pm |
| Sunday | Closed |

### Outside Hours
```
Auto-message:
"Thanks for reaching out! We're currently offline
but will respond to your message first thing tomorrow.
For urgent delivery issues, email urgent@redhillfresh.com.au"
```

## Response Standards

### Timing
| Metric | Target |
|--------|--------|
| First response | <30 seconds |
| Average handle | <5 minutes |
| Queue wait | <2 minutes |
| Resolution rate | >80% |

### Tone
```
Be:
- Friendly and warm
- Helpful and proactive
- Clear and concise
- Professional

Avoid:
- Robotic responses
- Jargon
- Negative language
- Rushing the customer
```

## Common Inquiries

### Delivery Questions
```
"When will my order arrive?"
Response:
"Let me check that for you! [Order #]
Your order is scheduled for [Day] between [Window].
You'll get an SMS when it's on the way.
Anything else I can help with?"
```

### Order Changes
```
"Can I add to/change my order?"
Response:
"Happy to help! If it's before our [cutoff time]
on [day], I can definitely make changes.
What would you like to add/change?"
```

### Product Questions
```
"Do you have [product]?"
Response:
"Let me check our stock for you!
[If yes]: Yes, we have [product] available!
[If no]: Unfortunately we're out of [product]
right now, but try [alternative]?"
```

### Delivery Area
```
"Do you deliver to [suburb]?"
Response:
"Let me check! [Postcode check]
[If yes]: Great news - yes we do!
[If no]: Sorry, we don't reach [suburb] yet,
but we're expanding soon. Want me to notify you?"
```

## Scripts by Situation

### Order Issue
```
"I'm really sorry to hear that. Let me fix this
right away. Can I have your order number please?"

[After checking]

"I can see the issue. Here's what I'll do:
[Resolution]. Does that work for you?"
```

### Account Help
```
"I'd be happy to help with your account!
For security, can you confirm the email
address on your account?"
```

### Pricing Question
```
"Our prices include [inclusions].
[Product] is currently $X per [unit].
Was there a specific item you were looking at?"
```

## Handoff Procedures

### When to Escalate
| Situation | Action |
|-----------|--------|
| Complex complaint | Escalate to Team Lead |
| Refund >$50 | Escalate to Team Lead |
| Technical issue | Create ticket |
| Policy exception | Escalate |

### Warm Handoff
```
"I'm going to connect you with [Name/Team] who
can help with this. They'll be with you shortly.
Is there anything else while you wait?"
```

## Multitasking

### Managing Multiple Chats
```
- Max 3 concurrent chats
- Priority: oldest first
- Use templates wisely
- Keep notes in CRM
- Don't sacrifice quality
```

### Between Responses
```
While waiting for reply:
- Update CRM notes
- Prepare next steps
- Check other chats
- Review order details
```

## Documentation

### Chat Notes
```
Log for each chat:
- Customer issue summary
- Resolution provided
- Any escalation
- Follow-up needed
```
