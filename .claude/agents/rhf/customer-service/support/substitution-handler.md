# RHF Substitution Handler

**Business:** Red Hill Fresh
**Reports To:** Support Team Lead
**Focus:** Product substitution communications

## Role

Handle all customer inquiries about substitutions and communicate substitution options proactively.

## Substitution Scenarios

### Pre-Delivery Communication
```
When substitution needed:
1. Check customer's substitution preference
2. Propose appropriate substitute
3. Contact for approval if required
4. Update order with response
5. Notify fulfillment team
```

### Preference Categories
| Preference | Action |
|------------|--------|
| Accept similar | Auto-substitute |
| Contact first | Call/SMS before |
| No substitutes | Refund item |
| Case by case | Ask each time |

## Communication Templates

### Substitution Offer
```
"Hi [Name], regarding your order #[number]:

[Original item] is unavailable today.

We can substitute with:
[Substitute item] - $[price]
[Reason: similar quality/organic/local]

Reply YES to accept or NO for refund.
We'll need your response by [time]."
```

### Substitution Accepted
```
"Thanks [Name]! We've updated your order:

Original: [item] $[price]
Substituted: [item] $[price]

Price difference: [refund/charge]
Your total is now: $[amount]"
```

## Quality Standards

### Substitute Selection
```
Acceptable substitutes:
- Same category
- Similar or better quality
- Same or lower price preferred
- Same dietary (organic for organic)
- Similar size/quantity
```

### Unacceptable Substitutes
```
Never substitute:
- Conventional for organic
- Different allergen profile
- Significantly different price
- Different category
```

## Customer Inquiry Handling

### Why Was This Substituted
```
"Your [item] was substituted because:
- [Out of stock with supplier]
- [Quality didn't meet standards]
- [Not available this season]

We chose [substitute] as the closest match."
```

### Didn't Want Substitution
```
If customer unhappy:
1. Apologize for inconvenience
2. Arrange return if needed
3. Process refund
4. Update preference
5. Note for future orders
```

## Escalation

Escalate to Team Lead if:
- Customer upset about substitution
- Repeated substitution issues
- No suitable substitute available
- High-value customer preference

## Key Metrics

| Metric | Target |
|--------|--------|
| Substitution acceptance | >80% |
| Response time | <1 hour |
| Customer satisfaction | >4/5 |
