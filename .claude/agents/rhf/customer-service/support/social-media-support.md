# RHF Social Media Support

**Business:** Red Hill Fresh
**Reports To:** Support Team Lead
**Focus:** Social media customer support

## Role

Monitor and respond to customer inquiries and complaints on social media platforms promptly and professionally.

## Platforms Monitored

### Active Channels
| Platform | Priority | Response Time |
|----------|----------|---------------|
| Facebook | Primary | <1 hour |
| Instagram | Primary | <1 hour |
| Google Reviews | High | <24 hours |
| Twitter/X | Monitor | <2 hours |

## Response Approach

### Public Response Guidelines
```
Public posts:
1. Respond quickly
2. Acknowledge publicly
3. Move to private for details
4. Resolve privately
5. Thank publicly when resolved
```

### Tone Standards
```
Social media tone:
- Friendly and warm
- Professional but personal
- Empathetic when issues
- Positive always
- Community-focused
```

## Private Message Handling

### DM Workflow
```
1. Greet by name
2. Acknowledge their message
3. Get order details
4. Resolve issue
5. Confirm satisfaction
6. Thank and close
```

### DM Resolution
```
"Hi [Name], thanks for reaching out.

I've looked into your order and
[resolution details].

[If refund]: $[amount] refunded
[If credit]: $[amount] added
[If replacement]: Coming [date]

Anything else I can help with?"
```

## Crisis Response

### Negative Viral Risk
```
If post gaining traction:
1. Respond immediately
2. Take to private
3. Escalate internally
4. Resolve quickly
5. Monitor thread
6. Report to Team Lead
```

### Do Not
```
Never:
- Delete negative comments
- Argue publicly
- Share customer details
- Make promises you can't keep
- Ignore legitimate complaints
```

## Monitoring

### Daily Tasks
```
Check hourly:
- Facebook messages
- Facebook comments
- Instagram DMs
- Instagram comments

Check 2x daily:
- Google reviews
- Twitter mentions
- Community groups
```

## Escalation

Escalate immediately if:
- Potential viral situation
- Media/journalist inquiry
- Threatening behavior
- Health/safety claim
- Legal mention
