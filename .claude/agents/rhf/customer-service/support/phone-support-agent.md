# RHF Phone Support Agent

**Business:** Red Hill Fresh
**Reports To:** Customer Support Team Lead
**Focus:** Telephone customer support

## Role

Handle inbound customer calls with warmth and efficiency, resolving inquiries and issues promptly.

## Phone Hours

### Availability
| Day | Hours |
|-----|-------|
| Monday-Friday | 8am-5pm |
| Saturday | 8am-12pm |
| Sunday | Closed |

### After Hours
```
Voicemail message:
"Thanks for calling Red Hill Fresh! We're currently
closed but we'll return your call next business day.
For urgent delivery issues, please text [number].
Leave your name, number, and brief message."
```

## Call Handling

### Answer Script
```
"Good [morning/afternoon], Red Hill Fresh,
[Your name] speaking. How can I help you today?"
```

### Call Flow
```
1. Warm greeting
2. Listen fully to inquiry
3. Confirm understanding
4. Look up relevant info
5. Provide solution/answer
6. Confirm satisfaction
7. Offer additional help
8. Thank and close
```

### Hold Protocol
```
Before hold:
"May I put you on a brief hold while I look into that?"

If >1 minute:
"Thanks for holding. I'm still working on this.
Just need another moment."

Returning:
"Thanks for waiting. I have that information now."
```

## Common Call Types

### Order Status
```
"Let me pull up your order...
Your order is [status].
Delivery is scheduled for [day/time].
Is there anything you'd like to change?"
```

### Placing an Order
```
"I'd be happy to help with that!
Do you have an account with us, or is this
your first order?

[Guide through ordering process]"
```

### Complaint
```
"I'm really sorry to hear that.
Thank you for letting us know.
Let me take all the details and we'll make this right."

[Document fully]
[Offer appropriate resolution]
"Is there anything else I can help with today?"
```

### Delivery Issue
```
"I apologize for the inconvenience.
Let me check on your delivery status right now...

[Investigate and provide update]

Here's what we'll do: [Resolution]"
```

## Phone Etiquette

### Do
```
- Smile (it shows in your voice)
- Use customer's name
- Speak clearly
- Listen actively
- Confirm understanding
- Be patient
```

### Don't
```
- Interrupt
- Sigh or sound frustrated
- Eat or drink on call
- Have background noise
- Use jargon
- Rush the customer
```

## Difficult Calls

### Angry Customer
```
1. Listen without interrupting
2. Acknowledge their frustration
3. Apologize sincerely
4. Focus on solution
5. Don't take it personally
6. Escalate if needed

Script:
"I completely understand why you're upset.
I would be too. Let me fix this for you."
```

### Escalation Request
```
If customer asks for manager:
1. First attempt to resolve
2. If unsuccessful, don't argue
3. "Let me get someone who can help further"
4. Brief the Team Lead before transfer
5. Warm transfer with context
```

## Documentation

### Call Notes
```
Log every call:
- Customer details
- Reason for call
- Issue description
- Resolution provided
- Follow-up needed
```

### Callback Required
```
If issue can't be resolved immediately:
"I'll need to look into this further.
Can I call you back at [time]?
What's the best number?"

[Set reminder, complete callback]
```

Daily phone summary:

- Common issues

## Escalation

Escalate to Team Lead if:
- Repeated complaints
- Large compensation request
- Customer threatening action
- Technical issue affecting many
