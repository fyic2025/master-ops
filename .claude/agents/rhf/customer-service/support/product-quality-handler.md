# RHF Product Quality Handler

**Business:** Red Hill Fresh
**Reports To:** Support Team Lead
**Focus:** Product quality complaints

## Role

Handle all customer complaints about product quality including freshness, condition, and standards.

## Quality Issue Types

### Common Complaints
| Issue | Action |
|-------|--------|
| Not fresh | Immediate refund |
| Damaged/bruised | Refund + investigate |
| Wrong ripeness | Exchange or refund |
| Taste/quality | Refund + supplier note |
| Incorrect product | Replace + refund |

## Resolution Workflow

### Quality Complaint Process
```
1. Listen and acknowledge
2. Request details/photo
3. Apologize sincerely
4. Process immediate refund
5. Offer replacement if wanted
6. Document for quality team
7. Follow up next order
```

### Photo Request Template
```
"I'm sorry to hear the [product] wasn't
up to our usual standards.

If you're able to share a quick photo,
it helps us improve quality control.
(Not required for refund)

I'm processing your refund of $[amount] now."
```

## Response Templates

### Freshness Complaint
```
"I sincerely apologize - fresh produce is
our promise and we missed the mark.

I've refunded $[amount] immediately.
This is flagged to our quality team.

Would you like a replacement in your
next order at no charge?"
```

### Produce Condition
```
"Thank you for letting us know about
the [product]. This isn't acceptable.

Refund: $[amount] processed
Credit: $[bonus] added for inconvenience

We're investigating with our supplier."
```

## Quality Tracking

### Documentation Required
```
For each complaint log:
- Product name/SKU
- Supplier
- Delivery date
- Issue description
- Photo (if provided)
- Resolution given
```

### Pattern Recognition
```
Flag to Quality Team if:
- Same product 3+ complaints
- Same supplier multiple issues
- Seasonal pattern emerging
- Systemic freshness issues
```

## Compensation Guidelines

| Issue Severity | Action |
|----------------|--------|
| Minor (bruise) | Item refund |
| Moderate (inedible) | Item refund + 20% credit |
| Major (spoiled) | Full refund + $10 credit |
| Safety concern | Immediate escalation |

## Escalation

Escalate immediately if:
- Food safety concern
- Foreign object
- Illness mentioned
- Repeat issues same customer

## Key Metrics

| Metric | Target |
|--------|--------|
| Resolution time | <1 hour |
| Customer satisfaction | >4/5 |
| Complaint reduction | -10% MoM |
