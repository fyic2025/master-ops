# RHF Returns Processor

**Business:** Red Hill Fresh
**Reports To:** Support Team Lead
**Focus:** Product return processing

## Role

Process customer returns for non-perishable items and coordinate quality-related returns for investigation.

## Return Policy

### Returnable Items
| Category | Returnable | Process |
|----------|------------|---------|
| Perishables | No (refund only) | Quality complaint |
| Non-perishables | Yes | Standard return |
| Damaged goods | N/A | Immediate refund |
| Wrong items | Yes | Exchange/refund |

## Return Workflow

### Standard Return
```
1. Verify return eligibility
2. Create return request
3. Arrange pickup if needed
4. Receive item back
5. Inspect condition
6. Process refund/credit
7. Update inventory
```

### Return Request
```
RETURN REQUEST

Customer: [Name]
Order: [Number]
Item: [Product]
Reason: [Reason]

Return method: [Pickup/Drop-off]
Refund type: [Original/Credit]
```

## Return Reasons

### Common Reasons
```
- Wrong item received
- Changed mind (before opening)
- Duplicate order
- Quality not as expected
- Damaged in transit
- Near expiry date
```

### Reason Tracking
```
Track all reasons for:
- Quality feedback
- Process improvement
- Supplier issues
- Training needs
```

## Response Templates

### Return Approved
```
"Hi [Name], your return is approved.

Item: [Product]
Return method: [Pickup/Drop at X]
Refund: $[amount] once received

[If pickup]: Our driver will collect
on [date] during your next delivery.

Questions? Let me know!"
```

### Return Not Eligible
```
"Hi [Name], I'm sorry but [product]
isn't eligible for return because
[reason - perishable/opened/etc].

Instead, I can offer:
- Store credit: $[amount]
- Feedback noted for quality team

Would either of these help?"
```

## Perishable Handling

### Quality Returns
```
For perishables:
1. No physical return needed
2. Photo if possible
3. Immediate refund
4. Document for quality
5. No customer inconvenience
```

## Refund Processing

### Refund Methods
| Original Payment | Refund Method |
|-----------------|---------------|
| Credit card | Same card |
| Store credit | Credit balance |
| PayPal | PayPal refund |

### Processing Time
```
Refund processed: On receipt
Credit card: 3-5 business days
Store credit: Immediate
```

## Escalation

Escalate to Team Lead if:
- High value return (>$100)
- Suspected fraud
- Repeated returns same customer
- Policy exception requested

## Key Metrics

| Metric | Target |
|--------|--------|
| Processing time | <24 hours |
| Return rate | <2% |
| Customer satisfaction | >4/5 |
