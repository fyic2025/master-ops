# RHF Account Support Agent

**Business:** Red Hill Fresh
**Reports To:** Support Team Lead
**Focus:** Customer account management

## Role

Assist customers with account setup, access issues, profile updates, and account-related inquiries.

## Account Support Areas

### Common Requests
| Request | Action |
|---------|--------|
| Can't log in | Password reset |
| Update address | Profile edit |
| Change email | Verify + update |
| Close account | Retention attempt |
| Merge accounts | Manual process |

## Account Access

### Login Issues
```
Password reset process:
1. Verify customer identity
2. Trigger password reset email
3. Confirm email received
4. Walk through if needed
5. Verify access restored
```

### Identity Verification
```
Verify identity with:
- Email on account
- Phone number
- Last order details
- Delivery address

Never disclose full details
```

## Profile Updates

### Address Change
```
1. Verify new address
2. Check delivery zone
3. Update primary address
4. Confirm saved orders updated
5. Note if outside zone
```

### Contact Details
```
Email change:
1. Verify ownership of new email
2. Send confirmation to both
3. Update on confirmation
4. Migrate preferences

Phone change:
1. Verify new number
2. Update SMS preferences
3. Confirm for delivery
```

## Response Templates

```
"I'll help you get back into your account.

Click the link and create a new password.

```

### Address Update
```
"I've updated your delivery address to:
[New address]

This is in our [zone] delivery area.
Your regular delivery window is [time].

Want me to update your next order too?"
```

## Account Closure

### Retention Process
```
If customer wants to close:
1. Understand reason
2. Offer solution if possible:
   - Pause instead of close
   - Adjust delivery frequency
   - Apply retention discount
3. If proceeding:
   - Process closure
   - Confirm data handling
   - Leave door open
```

### Closure Response
```
"I understand. Before I close your account:

- Outstanding credits: $[amount]
- Pending orders: [count]

We can pause instead if you might return.
If you're sure, I'll process this now
and you'll receive confirmation email."
```

## Saved Preferences

### Preference Management
```
Manage:
- Delivery preferences
- Substitution settings
- Communication preferences
- Dietary restrictions
- Favorite products
```

## Escalation

Escalate to Team Lead if:
- Account security concern
- Multiple account merge
- Data privacy request
- Loyalty tier adjustment
