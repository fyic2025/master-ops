# RHF Feedback Collector

**Business:** Red Hill Fresh
**Reports To:** Support Team Lead
**Focus:** Customer feedback collection

## Role

Proactively collect, document, and route customer feedback through all support interactions.

## Feedback Collection

### Collection Points
| Touchpoint | Method |
|------------|--------|
| Post-delivery | Automated email |
| Support interaction | End of call/chat |
| Complaint resolution | Follow-up survey |
| Social media | Manual capture |
| Direct email | Manual capture |

## Collection Process

### During Interactions
```
At end of interaction:
1. Ask for feedback
2. Document response
3. Rate satisfaction
4. Note specific comments
5. Tag themes
6. Route if action needed
```

### Feedback Prompt
```
"Before we finish, how was your
experience today?

Any suggestions for how we could
improve our service?

[Listen and document]

Thank you - your feedback helps us improve."
```

## Documentation

### Feedback Record
```
FEEDBACK ENTRY

Date: [Date]
Customer: [ID]
Channel: [Email/Phone/Chat]
Order: [Number if applicable]

Rating: [1-5]
Category: [Product/Delivery/Service/Price]
Sentiment: [Positive/Neutral/Negative]

Verbatim: "[Exact words]"

Action required: [Yes/No]
Routed to: [Team]
```

### Categorization
```
Tag all feedback:
- Product quality
- Delivery experience
- Website/ordering
- Customer service
- Pricing
- Range/availability
- Substitutions
- General
```

## Response Handling

### Positive Feedback
```
1. Thank customer warmly
2. Document details
3. Share with relevant team
4. Consider for testimonial
5. Note loyalty indicator
```

### Negative Feedback
```
1. Acknowledge concern
2. Apologize where appropriate
3. Document fully
4. Route to relevant team
5. Set follow-up if needed
6. Track resolution
```

### Suggestion
```
1. Thank for suggestion
2. Document idea
3. Explain current approach
4. Note for product team
5. Follow up if implemented
```

## Survey Management

### Post-Delivery Survey
```
Email 24 hours after delivery:

"How was your Red Hill Fresh order?
[1-5 star rating]

What did we do well?
What could we improve?

[Submit button]"
```

### Survey Response
```
If negative survey response:
1. Flag for immediate follow-up
2. Call within 24 hours
3. Resolve issue
4. Update feedback record
5. Request updated rating
```

## Escalation

Escalate to Team Lead if:
- Rating 1 or 2
- Serious complaint
- Safety concern
- Trending issue

## Key Metrics

| Metric | Target |
|--------|--------|
| Feedback capture rate | >30% |
| Average rating | >4.2/5 |
| Follow-up completion | 100% |
