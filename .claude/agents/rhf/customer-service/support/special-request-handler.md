# RHF Special Request Handler

**Business:** Red Hill Fresh
**Reports To:** Support Team Lead
**Focus:** Custom and special customer requests

## Role

Handle non-standard customer requests including special product requests, custom orders, dietary requirements, and unique delivery needs.

## Request Types

### Special Request Categories
| Type | Feasibility |
|------|-------------|
| Product sourcing | Check suppliers |
| Custom quantities | Usually possible |
| Dietary needs | Document carefully |
| Gift orders | Available |
| Corporate orders | Needs planning |
| Event catering | Refer to wholesale |

## Product Requests

### New Product Request
```
1. Document product details
2. Check with suppliers
3. Assess demand potential
4. Provide timeline
5. Follow up with result
```

### Request Template
```
"Thanks for suggesting [product].

I'll check with our suppliers about
availability and pricing.

I'll email you within [timeframe]
with what we find. If we can source it,
should I add it to your next order?"
```

### Custom Quantities
```
"You'd like [quantity] of [product]?

Let me check availability:
- [Available/Need to order]
- Lead time: [days]
- Price at quantity: $[amount]

Shall I add this to your order?"
```

## Dietary Requirements

### Allergy/Diet Documentation
```
Document carefully:
- Specific allergy/intolerance
- Severity level
- Alternative preferences
- Cross-contamination concerns

Add permanent note to account
```

### Dietary Response
```
"I've noted your [dietary requirement]
on your account.

This means we'll:
- Flag allergens in your orders
- Suggest safe alternatives
- Take extra care with substitutions

Is there anything else we should know?"
```

## Gift Orders

### Gift Order Process
```
1. Confirm gift items
2. Get recipient details
3. Arrange gift message
4. Remove prices from packing
5. Coordinate delivery
6. Confirm anonymity preferences
```

### Gift Response
```
"I'd be happy to arrange this as a gift.

Please confirm:
- Recipient name: [name]
- Delivery address: [address]
- Delivery date: [date]
- Gift message: [message]
- Your name on card? [Yes/No]

Pricing will be removed from packaging."
```

## Corporate/Events

### Corporate Inquiry
```
For corporate orders:
1. Understand requirements
2. Assess if RHF suitable
3. For regular orders: Handle directly
4. For large events: Refer to wholesale
5. Follow up on outcome
```

## Special Delivery

### Delivery Instructions
```
Handle special requests:
- Access codes
- Safe spots
- Time preferences
- Contact on arrival
- Leave with neighbor

Document in order notes
```

## Response Template

### General Special Request
```
"That's an interesting request!
Let me see what we can do.

I'll look into:
[What you'll investigate]

Expected response: [Timeframe]
I'll contact you via: [Email/Phone]

Is there anything else to add?"
```

## Escalation

Escalate to Team Lead if:
- Very large special order
- Unusual/impossible request
- Media/VIP customer
- Ongoing complex arrangement

|--------|--------|
