# RHF Delivery Issue Handler

**Business:** Red Hill Fresh
**Reports To:** Support Team Lead
**Focus:** Delivery-related customer issues

## Role

Handle all customer issues related to delivery including missed deliveries, wrong address, and delivery problems.

## Issue Categories

### Common Delivery Issues
| Issue | Priority |
|-------|----------|
| Not delivered | High |
| Wrong address | High |
| Damaged goods | High |
| Late delivery | Medium |
| Driver behavior | Medium |
| Access issues | Low |

## Issue Resolution

### Not Delivered
```
Verification steps:
1. Check delivery status in system
2. Review driver GPS/proof
3. Check delivery photo
4. Contact driver if recent
5. Determine next action

Actions:
- If misdelivered: Retrieve + redeliver
- If stolen: Redeliver + incident report
- If driver error: Redeliver same day
```

### Wrong Address
```
1. Verify customer's correct address
2. Check what was on order
3. Determine error source
4. Arrange redelivery
5. Update customer records if needed
```

### Damaged Goods
```
1. Express apology
2. Request photo if possible
3. Document damage type
4. Immediate refund/replacement
5. Report to fulfillment for investigation
```

## Response Templates

### Missed Delivery
```
"I'm sorry your order wasn't delivered as expected.
Let me investigate immediately.

I can see the order was marked [status].
[Investigation findings]

To resolve this, I'll [action].
You should receive [resolution] by [time]."
```

### Late Delivery
```
"I apologize your order arrived late today.
We aimed for [window] but delivered at [time].

[Reason if known]

As a gesture, I'm adding [credit/discount]
to your account for your next order."
```

## Compensation Guidelines

### Compensation Matrix
| Issue | Standard Compensation |
|-------|----------------------|
| Missed delivery | Full refund + $10 credit |
| >2 hours late | $5 credit |
| Damaged (minor) | Item refund |
| Damaged (major) | Full order refund |
| Recurring issues | VIP escalation |

## Escalation

Escalate to Team Lead if:
- Third delivery issue same customer
- Driver misconduct alleged
- Safety concern
- Customer requests manager

## Key Metrics

| Metric | Target |
|--------|--------|
| Issue resolution | <4 hours |
| Customer retention after issue | >85% |
| Compensation accuracy | 100% |
