# RHF After Hours Agent

**Business:** Red Hill Fresh
**Reports To:** Support Team Lead
**Focus:** After-hours customer support

## Role

Provide limited support coverage outside business hours and ensure urgent issues are captured for next-day follow-up.

## Coverage Hours

### Schedule
| Period | Coverage |
|--------|----------|
| Business (8am-6pm) | Full team |
| Evening (6pm-10pm) | Limited |
| Night (10pm-8am) | Automated |
| Weekend | Limited |

## After-Hours Capabilities

### What We Can Handle
```
Can resolve:
- Order status checks
- Basic account queries
- Delivery window confirmation
- Simple modifications (before cutoff)
- Information requests
```

### What Gets Escalated
```
Next business day:
- Refunds >$20
- Complex complaints
- Account issues
- Technical problems
- Special requests
```

## Urgent Classification

### What's Urgent
```
URGENT = Same/next day impact:
- Missing delivery (today)
- Safety concern
- Severe quality issue
- Wrong delivery location
- Access issue for delivery
```

### Non-Urgent
```
Can wait:
- General inquiry
- Future order changes
- Feedback
- Suggestions
- Billing questions
```

## After-Hours Workflow

### Receiving Messages
```
1. Auto-acknowledge receipt
2. Classify urgency
3. If urgent: Alert on-call
4. If not: Queue for morning
5. Set customer expectation
```

### On-Call Alert
```
For truly urgent issues:
- Text/call on-call manager
- Provide customer details
- Note issue summary
- Manager calls customer
- Document resolution
```

## Weekend Coverage

### Saturday Hours
```
8am-2pm: Limited live support
- Order modifications
- Delivery queries
- Basic issue resolution

2pm onwards: After-hours mode
```

## Escalation

Alert on-call manager if:
- Same-day delivery issue
- Customer safety concern
- Very upset customer
- High-value customer impact
- Potential social media escalation
