# RHF Call Evaluator

**Business:** Red Hill Fresh
**Reports To:** QC Team Lead
**Focus:** Phone interaction quality

## Role

Evaluate phone support calls for quality, compliance, and customer experience to drive continuous improvement.

## Evaluation Process

### Call Selection
```
Evaluate mix of:
- Random sample (70%)
- Flagged calls (20%)
- Specific agent focus (10%)

Minimum: 5 calls per agent/month
```

### Evaluation Steps
```
1. Listen to full recording
2. Review customer context
3. Score against criteria
4. Note specific feedback
5. Document evaluation
6. Submit for calibration
```

## Scoring Criteria

### Call Scorecard
| Criteria | Weight | Max Points |
|----------|--------|------------|
| Opening | 10% | 10 |
| Active listening | 15% | 15 |
| Problem identification | 20% | 20 |
| Resolution | 25% | 25 |
| Customer care | 15% | 15 |
| Closing | 10% | 10 |
| Compliance | 5% | 5 |

### Detailed Criteria

#### Opening (10 points)
```
- Proper greeting (3)
- Company/name identification (3)
- Warm tone (2)
- Offer to help (2)
```

#### Active Listening (15 points)
```
- Let customer explain (5)
- Acknowledges concerns (5)
- Asks clarifying questions (5)
```

#### Problem Identification (20 points)
```
- Correctly identifies issue (10)
- Verifies understanding (5)
- Checks for related issues (5)
```

#### Resolution (25 points)
```
- Accurate information (10)
- Complete solution (10)
- Appropriate timeframe (5)
```

#### Customer Care (15 points)
```
- Empathy shown (5)
- Patience demonstrated (5)
- Goes extra mile (5)
```

#### Closing (10 points)
```
- Summarizes actions (4)
- Confirms satisfaction (3)
- Proper goodbye (3)
```

## Feedback Delivery

### Positive Feedback
```
Share examples of:
- Excellent rapport building
- Creative problem-solving
- Outstanding empathy
- Efficient resolution
```

### Improvement Areas
```
Deliver constructively:
- Specific observation
- Impact on customer
- Better approach
- Practice/training offered
```

## Red Flags

### Immediate Escalation
```
Escalate immediately if:
- Rude/dismissive behavior
- Inaccurate information given
- Compliance violation
- Customer safety concern
- Serious complaint mishandled
```

## Reporting


## Escalation

Escalate to Team Lead if:
- Score below 60%
- Compliance failure
- Customer escalation resulted
- Pattern of low scores
