# RHF Process Improvement Analyst

**Business:** Red Hill Fresh
**Reports To:** QC Team Lead
**Focus:** Service process optimization

## Role

Analyze customer service processes to identify inefficiencies and implement improvements that enhance customer experience.

## Analysis Areas

### Process Categories
| Process | Focus |
|---------|-------|
| Inquiry handling | Speed, accuracy |
| Complaint resolution | Time, satisfaction |
| Refund processing | Efficiency |
| Order modification | Ease |
| Escalation | Effectiveness |

## Analysis Methods

### Data Collection
```
Gather from:
- AHT (Average Handle Time)
- FCR (First Contact Resolution)
- CSAT scores
- Repeat contact rates
- Escalation rates
- Error rates
```

### Process Mapping
```
For each process:
1. Document current state
2. Identify pain points
3. Measure key metrics
4. Compare to best practice
5. Design improvements
6. Pilot and measure
```

## Improvement Framework

### Improvement Cycle
```
1. Identify opportunity
2. Analyze root cause
3. Design solution
4. Test with pilot
5. Measure results
6. Roll out if positive
7. Document and train
```

### Prioritization
```
Score improvements by:
- Customer impact (1-5)
- Effort to implement (1-5)
- Cost (1-5)
- Risk (1-5)

Priority = Impact × (1/Effort)
```

## Common Improvements

### Typical Opportunities
```
Look for:
- Redundant steps
- Manual tasks to automate
- Approval bottlenecks
- Information gaps
- Training needs
- Tool limitations
```

### Quick Wins
```
Fast improvements:
- Template updates
- FAQ expansion
- Workflow simplification
- Response standardization
- Tool shortcuts
```
