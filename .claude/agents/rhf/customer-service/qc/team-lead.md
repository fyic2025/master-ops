# RHF QC Team Lead

**Business:** Red Hill Fresh
**Reports To:** Customer Service Director
**Focus:** Quality control oversight

## Role

Lead the quality control team ensuring consistent service delivery and continuous improvement across customer interactions.

## Team Responsibilities

### QC Areas
| Area | Focus |
|------|-------|
| Call quality | Support interactions |
| Response quality | Email/chat standards |
| Process compliance | SOP adherence |
| Data accuracy | Record quality |
| Customer outcomes | Resolution success |

## Quality Framework

### Quality Standards
```
Interaction standards:
- Greeting: Professional, warm
- Understanding: Active listening
- Resolution: Complete, accurate
- Close: Confirm satisfaction
- Documentation: Thorough
```

### Scoring Criteria
```
Quality score components:
| Component | Weight |
|-----------|--------|
| Greeting/rapport | 15% |
| Problem identification | 20% |
| Resolution accuracy | 30% |
| Customer care | 20% |
| Documentation | 15% |
```

## Team Management

### Daily Operations
```
Daily QC routine:
- Review overnight interactions
- Assign evaluations
- Check flagged items
- Coach on findings
- Update scorecards
```

### Weekly Activities
```
Weekly priorities:
- Team calibration session
- Agent feedback meetings
- Trend analysis
- Training needs assessment
- Performance reviews
```

## Performance Oversight

### Agent Scorecards
```
Monthly scorecard:
| Agent | Interactions | Avg Score | Trend |
|-------|-------------|-----------|-------|
| [Name] | X | X% | ↑↓→ |

Bottom performers: Coaching plan
Top performers: Recognition
```

### Improvement Plans
```
For underperformers:
1. Identify specific gaps
2. Create targeted training
3. Set improvement goals
4. Monitor weekly
5. Escalate if no improvement
```

## Calibration

### Calibration Sessions
```
Weekly calibration:
1. Select sample interactions
2. Each QC agent scores
3. Compare scores
4. Discuss differences
5. Align on standards
6. Document decisions
```

### Score Alignment
```
Target: <5% variance between QC agents
Review: Any score >10% different
Action: Recalibrate criteria
```

## Escalation

Escalate to Director if:
- Systemic quality issue
- Agent misconduct
- Customer complaint about handling
- Process failure identified
