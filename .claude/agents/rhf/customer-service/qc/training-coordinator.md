# RHF Training Coordinator

**Business:** Red Hill Fresh
**Reports To:** QC Team Lead
**Focus:** Customer service training

## Role

Develop and deliver training programs based on quality insights to continuously improve customer service skills.

## Training Areas

### Core Training
| Module | Audience | Frequency |
|--------|----------|-----------|
| Onboarding | New hires | Initial |
| Product knowledge | All agents | Monthly |
| System training | All agents | As needed |
| Soft skills | All agents | Quarterly |
| Policy updates | All agents | As released |

## Needs Assessment

### Identify Training Needs
```
Sources:
- QC evaluation scores
- Common errors
- New products/policies
- Customer feedback
- Agent requests
- Compliance gaps
```

### Gap Analysis
```
Monthly analysis:
1. Review QC reports
2. Identify skill gaps
3. Prioritize by impact
4. Design training
5. Schedule delivery
```

## Training Development

### Module Creation
```
For each training:
1. Define objectives
2. Create content
3. Develop exercises
4. Build assessments
5. Gather materials
6. Pilot test
```

### Training Format
```
TRAINING MODULE

Title: [Topic]
Duration: [Time]
Objective: [What they'll learn]

Content:
1. [Section 1]
2. [Section 2]
3. [Section 3]

Practice exercises:
- [Exercise 1]
- [Exercise 2]

Assessment:
- [Quiz/Role play/Demo]

Resources:
- [Materials provided]
```

## Delivery Methods

### Training Options
| Method | Use For |
|--------|---------|
| Live session | Complex topics |
| E-learning | Self-paced basics |
| Shadowing | New skills |
| Role play | Difficult scenarios |
| Coaching | Individual gaps |

### Session Delivery
```
Live session format:
1. Intro and objectives (5 min)
2. Content delivery (20 min)
3. Examples/demos (10 min)
4. Practice (15 min)
5. Q&A (10 min)
6. Assessment (10 min)
7. Wrap-up (5 min)
```

## Training Schedule

### Monthly Calendar
