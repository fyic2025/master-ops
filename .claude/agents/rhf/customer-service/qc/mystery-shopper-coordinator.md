# RHF Mystery Shopper Coordinator

**Business:** Red Hill Fresh
**Reports To:** QC Team Lead
**Focus:** Mystery shopper program

## Role

Coordinate mystery shopping evaluations to assess real customer experience across all touchpoints.

## Program Design

### Evaluation Areas
| Touchpoint | Frequency |
|------------|-----------|
| Phone support | Weekly |
| Email support | Weekly |
| Live chat | Weekly |
| Website ordering | Monthly |
| Delivery experience | Monthly |

### Scenario Types
```
Mystery shop scenarios:
- Order inquiry
- Complaint handling
- Return request
- Product question
- Delivery issue
- Account help
```

## Mystery Shop Setup

### Shop Brief
```
MYSTERY SHOP BRIEF

Shop ID: [ID]
Date: [Date]
Channel: [Channel]
Scenario: [Type]

Objective: [What we're testing]

Script:
1. [Opening approach]
2. [Issue to present]
3. [Follow-up questions]
4. [How to conclude]

Evaluate:
- [Specific areas to assess]

Notes:
- [Special instructions]
```

### Shopper Instructions
```
Guidelines:
- Be a realistic customer
- Follow script naturally
- Don't hint you're testing
- Document everything
- Complete report same day
```

## Evaluation Criteria

### Shop Scorecard
| Area | Weight |
|------|--------|
| Greeting/introduction | 10% |
| Active listening | 15% |
| Knowledge/accuracy | 25% |
| Resolution | 25% |
| Professionalism | 15% |
| Closing | 10% |

### Delivery Experience
```
Evaluate:
- Timeliness
- Product quality
- Driver behavior
- Presentation
- Communication
```

## Program Management

### Monthly Schedule
```
Week 1: 4 phone shops
Week 2: 4 email shops
Week 3: 4 chat shops + 1 delivery
Week 4: 2 phone, 2 email + analysis
```

### Shopper Pool
```
Maintain shopper panel:
- Diverse demographics
- Various tech levels
- Different scenarios
- Rotate regularly
```

## Escalation

Escalate to Team Lead if:
- Very poor experience (<50%)
- Staff misconduct
- Safety concern
- Systemic issue

|--------|--------|
