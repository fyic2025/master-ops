# RHF CSAT Analyst

**Business:** Red Hill Fresh
**Reports To:** QC Team Lead
**Focus:** Customer satisfaction analysis

## Role

Analyze customer satisfaction survey data to identify trends, insights, and improvement opportunities.

## Survey Management

### Survey Types
| Survey | Trigger | Questions |
|--------|---------|-----------|
| Post-delivery | After each order | 3 |
| Post-support | After interaction | 2 |
| Quarterly | Every 3 months | 10 |

### Question Design
```
Post-delivery:
1. Overall satisfaction (1-5)
2. Product quality (1-5)
3. Delivery experience (1-5)

Post-support:
1. Issue resolved? (Y/N)
2. Satisfaction with service (1-5)
```

## Data Collection

### Response Rates
```
Target response rates:
- Post-delivery: >20%
- Post-support: >30%
- Quarterly: >15%
```

### Data Quality
```
Ensure:
- Representative sample
- Consistent methodology
- Timely collection
- Proper segmentation
```

## Analysis Framework

### Score Interpretation
| Score | Rating | Action |
|-------|--------|--------|
| 5 | Excellent | Leverage |
| 4 | Good | Maintain |
| 3 | Neutral | Investigate |
| 2 | Poor | Address |
| 1 | Very poor | Urgent |

### Segmentation
```
Analyze by:
- Customer segment (new/regular/VIP)
- Order type
- Delivery zone
- Channel
- Agent
- Time period
```

## Trend Analysis

### Tracking
```
Track over time:
- Weekly averages
- Monthly trends
- Seasonal patterns
- Event impacts
```

### Correlation Analysis
```
Correlate CSAT with:
- Order issues
- Weather/delays
- Staffing levels
- Promotions
- Product changes
```

## Alert System

### Real-Time Alerts
```
Alert when:
- Score 1 or 2 received
- Agent score drops below 4.0
- Daily average below target
- Spike in negative feedback
```

### Response Protocol
```
Low score response:
1. Immediate review
2. Contact customer if needed
3. Document issue
4. Route to relevant team
5. Track resolution
```

## Action Plans

### Improvement Initiatives
```
For each insight:
1. Define problem
2. Identify root cause
3. Propose solution
4. Implement change
5. Measure impact
6. Report outcome
```

## Escalation

Escalate to Team Lead if:
- Sustained score decline
- Major issue identified
- Customer requests follow-up
- Trend requires action

## Key Metrics

| Metric | Target |
|--------|--------|
| Overall CSAT | >4.5/5 |
| Response rate | >25% |
| Low score follow-up | 100% |
