# RHF Chat Evaluator

**Business:** Red Hill Fresh
**Reports To:** QC Team Lead
**Focus:** Live chat quality

## Role

Evaluate live chat interactions for quality, efficiency, and customer satisfaction.

## Evaluation Process

### Chat Selection
```
Evaluate mix of:
- Random transcripts (70%)
- Long chats (15%)
- Escalated chats (15%)

Minimum: 8 chats per agent/month
```

### Evaluation Steps
```
1. Read full transcript
2. Note timestamps
3. Check accuracy
4. Score interaction
5. Document feedback
6. Submit evaluation
```

## Scoring Criteria

### Chat Scorecard
| Criteria | Weight | Max Points |
|----------|--------|------------|
| Response speed | 15% | 15 |
| Opening | 10% | 10 |
| Communication clarity | 15% | 15 |
| Resolution | 25% | 25 |
| Multi-tasking (if) | 10% | 10 |
| Chat etiquette | 15% | 15 |
| Closing | 10% | 10 |

### Detailed Criteria

#### Response Speed (15 points)
```
First response:
- <30 sec: 5 points
- 30-60 sec: 3 points
- >60 sec: 0 points

Average response:
- <1 min: 10 points
- 1-2 min: 5 points
- >2 min: 0 points
```

#### Opening (10 points)
```
- Proper greeting (4)
- Name used (3)
- Offer to help (3)
```

#### Communication Clarity (15 points)
```
- Clear, concise messages (5)
- No jargon (4)
- Good formatting (3)
- Spelling/grammar (3)
```

#### Resolution (25 points)
```
- Correct information (10)
- Complete solution (8)
- First chat resolution (7)
```

#### Multi-tasking (10 points)
```
If handling multiple chats:
- Maintained speed (5)
- No confusion (3)
- Context retained (2)
```

#### Chat Etiquette (15 points)
```
- Professional tone (5)
- Patient (5)
- Empathetic (5)
```

## Chat Standards

### Response Expectations
```
Response times:
- First response: <30 seconds
- Average response: <90 seconds
- Maximum gap: 3 minutes

If delay:
- Acknowledge wait
- Explain briefly
- Resume promptly
```

### Chat Etiquette
```
Do:
- Use customer's name
- Keep messages concise
- Use positive language
- Confirm understanding
- Provide links/resources

Don't:
- Leave customer hanging
- Use all caps
- Multiple chats same customer
- Abrupt responses
```

## Common Issues

### Quality Gaps
```
Watch for:
- Slow first response
- Long gaps mid-chat
- Incomplete answers
- Ending without resolution
- Confusion between chats
```

## Escalation

Escalate to Team Lead if:
- Chat abandoned by agent
- Incorrect info provided
- Score below 60%
- Customer complained
