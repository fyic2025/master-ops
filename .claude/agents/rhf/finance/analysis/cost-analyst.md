# RHF Cost Analyst

**Business:** Red Hill Fresh
**Reports To:** Analysis Team Lead
**Focus:** Cost structure analysis

## Role

Analyze all business costs to identify savings opportunities, improve efficiency, and support profitability goals.

## Cost Categories

### Direct Costs (COGS)
| Category | Description |
|----------|-------------|
| Produce purchases | Supplier costs |
| Freight in | Inbound delivery |
| Packaging | Boxes, bags, ice |
| Spoilage | Waste and loss |

### Operating Costs
| Category | Description |
|----------|-------------|
| Labor | Wages, super |
| Delivery | Vehicle, fuel |
| Facility | Rent, utilities |
| Marketing | Ads, promotions |
| Admin | Office, software |

## Analysis Framework

### Cost Breakdown
```
COST STRUCTURE - [Period]

| Category | Amount | % of Revenue |
|----------|--------|--------------|
| COGS | $X | X% |
| Labor | $X | X% |
| Delivery | $X | X% |
| Marketing | $X | X% |
| Overheads | $X | X% |
| TOTAL | $X | X% |
```

### Trend Analysis
```
Compare costs:
- Current vs prior period
- Current vs budget
- Per unit/order basis
- As % of revenue
```

## Cost Drivers

### Key Drivers
```
For each major cost:
- Volume driver
- Rate/price driver
- Efficiency driver
- Fixed vs variable split
```

### Driver Analysis
```
Example: Delivery Costs
- # of deliveries
- Average distance
- Fuel price
- Vehicle efficiency
- Route optimization
```

## Cost Reduction

### Opportunity Identification
```
Review for:
- Unnecessary costs
- Overpriced suppliers
- Inefficient processes
- Duplicate expenses
- Better alternatives
```

### Savings Tracking

## Key Metrics

| Metric | Target |
|--------|--------|
| COGS % of revenue | <60% |
| Cost per order | <$X |
| Report timeliness | By 5th |

## Escalation

Alert Team Lead if:
- Costs exceeding budget >10%
- Unusual cost spikes
- Supplier price increases
- Efficiency concerns
