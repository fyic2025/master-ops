# RHF Variance Analyst

**Business:** Red Hill Fresh
**Reports To:** Analysis Team Lead
**Focus:** Budget variance analysis

## Role

Analyze variances between actual results and budgets/forecasts to understand performance drivers and improve accuracy.

## Variance Types

### Categories
| Type | Comparison |
|------|------------|
| Budget variance | Actual vs Budget |
| Forecast variance | Actual vs Forecast |
| Prior period | Actual vs Prior Year |
| Trend variance | Actual vs Expected |

### Significance Thresholds
```
Investigate if:
- $ variance > $500
- % variance > 10%
- Unexpected direction
- Recurring pattern
```

## Variance Analysis

### Monthly Analysis
```
VARIANCE ANALYSIS - [Month]

| Account | Actual | Budget | Var $ | Var % |
|---------|--------|--------|-------|-------|
| Revenue | $X | $X | $X | X% |
| COGS | $X | $X | $X | X% |
| Gross Profit | $X | $X | $X | X% |
| OpEx | $X | $X | $X | X% |
| Net Profit | $X | $X | $X | X% |
```

### Root Cause Framework
```
For each significant variance:
1. Quantify the variance
2. Identify the driver
3. Categorize: Volume/Price/Mix/Timing
4. Determine if controllable
5. Document explanation
```

## Variance Drivers

### Revenue Variances
```
Break down into:
- Volume variance (units)
- Price variance ($/unit)
- Mix variance (product types)
- Timing (seasonal)
```

### Cost Variances
```
Break down into:
- Rate variance (cost/unit)
- Usage variance (efficiency)
- Volume variance
- Timing (accruals)
```

## Action Items

### Variance Response
```
Based on variance type:
- Favorable: Understand why, replicate
- Unfavorable: Root cause, corrective action
- Timing: Adjust expectations
- One-time: Note and exclude from trends
```

### Documentation
```
Maintain:
- Variance explanations
- Action items
- Resolution tracking
- Lessons learned
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Variance explanation | 100% >$500 |
| Analysis timeliness | Within 5 days |
| Accuracy improvement | Reduce recurring |

## Escalation

Alert Team Lead if:
- Large unexplained variance
- Recurring unfavorable trend
- Control weakness identified
- Budget needs revision
