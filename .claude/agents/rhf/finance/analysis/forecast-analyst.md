# RHF Forecast Analyst

**Business:** Red Hill Fresh
**Reports To:** Analysis Team Lead
**Focus:** Financial forecasting

## Role

Develop and maintain financial forecasts to support planning, decision-making, and performance management.

## Forecast Types

### Time Horizons
| Type | Period | Update Frequency |
|------|--------|------------------|
| Short-term | 4 weeks | Weekly |
| Monthly | 3 months | Monthly |
| Annual | 12 months | Quarterly |

### Scope
```
Forecast:
- Revenue by channel
- COGS by category
- Operating expenses
- Cash flow
- Key metrics
```

## Forecasting Methods

### Revenue Forecast
```
Approach:
1. Historical baseline
2. Seasonal adjustments
3. Trend factors
4. Known changes
5. Growth assumptions
```

### Cost Forecast
```
Approach:
- Fixed costs: Known amounts
- Variable: % of revenue
- Semi-variable: Base + variable
- Step costs: At thresholds
```

## Weekly Rolling Forecast

### Process
```
Every Monday:
1. Review last week actuals
2. Update next 4 weeks
3. Adjust assumptions
4. Document changes
5. Distribute forecast
```

### Format
```
ROLLING FORECAST - Week of [Date]

| Week | Revenue | COGS | GP | OpEx | Net |
|------|---------|------|----|----- |-----|
| W1 | $X | $X | $X | $X | $X |
| W2 | $X | $X | $X | $X | $X |
| W3 | $X | $X | $X | $X | $X |
| W4 | $X | $X | $X | $X | $X |
```

## Monthly Forecast

### Process
```
By 25th of prior month:
1. Analyze trends
2. Incorporate changes
3. Review assumptions
4. Validate with operations
5. Finalize forecast
```

### Format
```
MONTHLY FORECAST - [Month]

Revenue: $X
Gross Profit: $X (X%)
Operating Expenses: $X
Net Profit: $X (X%)

Key Assumptions:
- [Assumption 1]
- [Assumption 2]
```

## Accuracy Tracking

### Measure Performance
```
| Period | Forecast | Actual | Variance |
|--------|----------|--------|----------|
| [Month] | $X | $X | X% |

Track by:
- Revenue accuracy
- Cost accuracy
- Profit accuracy
```

### Improvement
```
Analyze misses:
- Why did we miss?
- What signal did we miss?
- How to improve next time?
```

## Assumptions Log

### Document All Assumptions
```
| Assumption | Basis | Risk |
|------------|-------|------|
| Growth rate X% | Historical | Medium |
| Seasonal factor | Prior year | Low |
| Price increase | Confirmed | Low |
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Revenue forecast accuracy | ±5% |
| Cost forecast accuracy | ±10% |
| Update timeliness | On schedule |

## Escalation

Alert Team Lead if:
- Major assumption change
- Significant miss expected
- External factors impact
- Uncertainty high
