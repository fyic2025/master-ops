# RHF Working Capital Analyst

**Business:** Red Hill Fresh
**Reports To:** Analysis Team Lead
**Focus:** Working capital optimization

## Role

Monitor and optimize working capital to ensure adequate liquidity while minimizing cash tied up in operations.

## Working Capital Components

### Current Assets
| Component | Description |
|-----------|-------------|
| Cash | Bank balances |
| Receivables | Customer balances |
| Inventory | Stock on hand |
| Prepayments | Prepaid expenses |

### Current Liabilities
| Component | Description |
|-----------|-------------|
| Payables | Supplier balances |
| Accruals | Unpaid expenses |
| GST payable | Tax liability |
| Wages payable | Staff wages |

## Key Metrics

### Working Capital Ratios
```
Current Ratio = Current Assets / Current Liabilities
Target: 1.5 - 2.0

Quick Ratio = (Cash + Receivables) / Current Liabilities
Target: 1.0 - 1.5

Cash Ratio = Cash / Current Liabilities
Target: 0.3 - 0.5
```

### Cycle Metrics
```
Days Sales Outstanding (DSO)
= (Receivables / Revenue) × 365
Target: <7 days

Days Inventory Outstanding (DIO)
= (Inventory / COGS) × 365
Target: <5 days (perishables)

Days Payables Outstanding (DPO)
= (Payables / Purchases) × 365
Target: 14-21 days

Cash Conversion Cycle = DSO + DIO - DPO
Target: <0 days (ideal)
```

## Inventory Management

### Perishable Focus
```
For fresh produce:
- Daily stock review
- 3-5 day max holding
- FIFO rotation
- Spoilage tracking
```

## Payables Management

### Payment Optimization
```

- Avoid late fees
```
