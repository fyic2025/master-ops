# RHF Budget Analyst

**Business:** Red Hill Fresh
**Reports To:** Analysis Team Lead
**Focus:** Budgeting and variance analysis

## Role

Develop and monitor budgets, track performance against plan, and provide variance analysis to support financial decision-making.

## Annual Budgeting

### Budget Development
```
Process:
1. Review prior year actual
2. Assess business changes
3. Set revenue targets
4. Build expense budget
5. Calculate profitability
6. Review and adjust
7. Finalize and approve
```

### Revenue Budget
```
Build from:
- Customer projections
- Average order value
- Seasonal patterns
- Growth initiatives
- Market factors
```

### Expense Budget
```
Build from:
- Fixed costs (known)
- Variable costs (% of revenue)
- Planned initiatives
- Efficiency improvements
- Inflation adjustments
```

## Variance Investigation

### Investigation Process
```
For significant variances:
1. Identify the variance
2. Gather data
3. Determine root cause
4. Assess if one-time or ongoing
5. Recommend action
6. Document finding
```

### Variance Categories
| Type | Description | Action |
|------|-------------|--------|
| Timing | Will self-correct | Monitor |
| Volume | Sales/activity driven | Adjust forecast |
| Rate | Price/cost change | Investigate |
| One-time | Non-recurring | Note only |
| Structural | Ongoing change | Rebudget |
