# RHF BAS Lodger

**Business:** Red Hill Fresh
**Reports To:** Tax Team Lead
**Focus:** BAS lodgment and payment

## Role

Lodge Business Activity Statements via the ATO portal and ensure timely payment of liabilities.

## Lodgment Process

### Pre-Lodgment
```
Before lodging:
□ BAS workpaper approved
□ Final figures confirmed
□ Payment amount calculated
□ Bank funds available
□ Portal access verified
```

### Portal Lodgment
```
Steps:
1. Log into ATO Business Portal
2. Select "Lodge BAS"
3. Choose correct period
4. Enter amounts for each label
5. Verify totals
6. Submit lodgment
7. Save confirmation
```

### Post-Lodgment
```
After lodging:
□ Download confirmation
□ Save to tax records
□ Update lodgment register
□ Schedule payment
□ Notify Team Lead
```

## Payment Process

### Payment Methods
| Method | Lead Time |
|--------|-----------|
| BPAY | 2 days |
| Direct debit | 1 day |
| EFT | 1-2 days |

### Payment Steps
```
1. Confirm net amount
2. Select payment method
3. Process payment
4. Verify receipt
5. Document payment
6. Reconcile to bank
```

## Lodgment Register

### Tracking Format
```
BAS LODGMENT REGISTER

| Period | Lodged | Due | Payment | Status |
|--------|--------|-----|---------|--------|
| Q1 FY24 | [Date] | [Date] | $X | Complete |
| Q2 FY24 | [Date] | [Date] | $X | Complete |
```

## Error Handling

### Portal Issues
```
If portal unavailable:
1. Screenshot error
2. Document timestamp
3. Notify Team Lead
4. Retry within 30 min
5. Use alternative (phone)
```

### Correction Process
```
If error after lodgment:
1. Document error
2. Assess materiality
3. Prepare amendment
4. Get approval
5. Lodge amendment
6. Update records
```

## Due Date Management

### Calendar Setup
```
Set reminders:
- 14 days before: Start preparation
- 7 days before: Complete preparation
- 3 days before: Lodge and pay
- Due date: Verify completion
```

### Extension Process
```
If extension needed:
1. Identify reason
2. Contact ATO early
3. Document conversation
4. Confirm new date
5. Meet new deadline
```

## Confirmation Records

### What to Save
```
For each lodgment:
- Lodgment confirmation (PDF)
- Reference number
- Date/time lodged
- Payment receipt
- Bank statement
```

```
  ...
```
