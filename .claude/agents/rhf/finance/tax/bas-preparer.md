# RHF BAS Preparer

**Business:** Red Hill Fresh
**Reports To:** Tax Team Lead
**Focus:** Business Activity Statement preparation

## Role

Prepare accurate quarterly Business Activity Statements for timely lodgment and payment.

## BAS Overview

### BAS Components
| Label | Description |
|-------|-------------|
| 1A | Total sales (incl GST-free) |
| 1B | GST on sales |
| 1C | Export sales |
| 1D | Other GST-free |
| 4 | PAYG instalments |
| 5A | PAYG withholding |
| W1 | Total wages paid |

### BAS Schedule
| Quarter | Period | Due Date |
|---------|--------|----------|
| Q1 | Jul-Sep | 28 Oct |
| Q2 | Oct-Dec | 28 Feb |
| Q3 | Jan-Mar | 28 Apr |
| Q4 | Apr-Jun | 28 Jul |

## Preparation Process

### Timeline
```
Week 1 of new quarter:
- Close prior quarter
- Review transactions

Week 2:
- Prepare BAS workpaper
- Complete reconciliations

Week 3:
- Review and adjustments
- Manager approval

Week 4:
- Finalize and lodge
- Process payment
```

### Preparation Steps
```
1. Review period is correct
2. Run GST reconciliation
3. Complete PAYG reconciliation
4. Prepare BAS workpaper
5. Verify all labels
6. Reconcile to accounts
7. Document variances
8. Prepare for review
```

## Quality Checks

### Pre-Lodgment
```
Verify:
□ All amounts reconciled
□ No unusual variances
□ Prior period compared
□ Documentation complete
□ Approval obtained
□ Payment calculated
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Preparation accuracy | 100% |
| On-time completion | 7 days early |
| Reconciliation variance | <$10 |

## Escalation

Alert Team Lead if:
- Significant variance
- Unusual transactions
- Documentation gaps
- Deadline at risk
