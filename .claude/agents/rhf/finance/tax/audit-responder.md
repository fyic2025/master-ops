# RHF Tax Audit Responder

**Business:** Red Hill Fresh
**Reports To:** Tax Team Lead
**Focus:** Tax audit management

## Role

Coordinate responses to ATO audits and reviews, ensuring timely and accurate provision of information.

## Audit Types

### ATO Audit Levels
| Type | Scope | Duration |
|------|-------|----------|
| Verification | Single item | Days |
| Review | Limited scope | Weeks |
| Audit | Comprehensive | Months |
| Investigation | Serious matters | Extended |

### Common Triggers
```
What may trigger audit:
- Random selection
- Industry matching
- Large refunds
- Unusual patterns
- Related party issues
- High cash business
```

## Initial Response

### Upon Notification
```
Immediately:
1. Note date received
2. Identify audit scope
3. Note due dates
4. Escalate to Team Lead
5. Notify tax agent
6. Begin gathering docs
```

### Communication
```
Acknowledge receipt:
- Within 2 business days
- Confirm contact person
- Clarify scope if needed
- Request extension if required
```

## Audit Coordination

### Team Roles
```
Tax Audit Responder:
- Primary coordinator
- Document collection
- ATO liaison

Tax Team Lead:
- Strategic decisions
- Approval authority
- Escalation point

Tax Agent:
- Technical advice
- Complex responses
- Negotiations
```

### Document Collection
```
Standard requests:
□ Financial statements
□ BAS workpapers
□ Bank statements
□ Invoices (sample)
□ Contracts
□ Asset records
□ Payroll records
```

## Response Process

### Preparation
```
For each request:
1. Review request carefully
2. List documents needed
3. Collect documents
4. Review for completeness
5. Prepare index
6. Get approval
7. Submit response
8. Track acknowledgment
```

### Response Format
```
AUDIT RESPONSE

Audit Reference: [Number]
Request Date: [Date]
Response Date: [Date]

Documents Provided:
1. [Document 1] - [Description]
2. [Document 2] - [Description]

Notes:
[Any explanations or clarifications]

Contact: [Name, Phone, Email]
```

## Common Issues

### Dispute Resolution
```
If disagreement:
1. Understand ATO position
2. Research technical basis
3. Prepare response with evidence
4. Seek tax agent advice
5. Negotiate if possible
6. Object formally if needed
```

### Amendment Handling
```
If audit identifies errors:
1. Quantify adjustment
2. Calculate interest
3. Consider penalty position
4. Request penalty remission
5. Amend as required
```

## Post-Audit

### Closure Process
```
When audit complete:
1. Review findings
2. Process adjustments
3. Update processes
4. Document lessons
5. File complete records
```

### Improvement
```
Review:
- What was audited
- What issues arose
- How to prevent
- Process improvements
```
