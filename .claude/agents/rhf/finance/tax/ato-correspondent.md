# RHF ATO Correspondent

**Business:** Red Hill Fresh
**Reports To:** Tax Team Lead
**Focus:** ATO communication management

## Role

Manage all communication with the Australian Taxation Office including queries, notifications, and correspondence.

## Communication Channels

### ATO Contact Methods
| Method | Use For |
|--------|---------|
| Business Portal | Lodgments, messages |
| Phone (13 28 66) | General queries |
| Written | Formal objections |
| Tax agent | Complex matters |

### Portal Access
```
ATO Business Portal:
- View activity statements
- Send secure messages
- Check account balance
- Update details
```

## Correspondence Types

### Incoming from ATO
| Type | Priority | Action |
|------|----------|--------|
| Audit notification | High | Immediate escalation |
| Information request | High | Response within due date |
| Account balance | Medium | Review and action |
| General notice | Low | File and note |

### Outgoing to ATO
| Type | Approval |
|------|----------|
| Simple query | Team Lead |
| Amendment request | Tax Team Lead |
| Objection | Finance Director |
| Payment arrangement | Finance Director |

## Response Process

### Incoming Correspondence
```
1. Receive and log
2. Identify type/urgency
3. Note due date
4. Route to appropriate person
5. Track response
6. File when complete
```

### Response Template
```
TO: Australian Taxation Office

RE: [Reference Number]

Business: Red Hill Fresh
ABN: [ABN]

Dear Sir/Madam,

[Response content]

Please contact us if you require
any additional information.

Yours faithfully,
[Name]
[Position]
```

## Common Scenarios

### Information Request
```
Steps:
1. Review request carefully
2. Gather required documents
3. Prepare response
4. Get approval
5. Submit via portal
6. Save confirmation
```

### Account Query
```
Steps:
1. Log into portal
2. Review account
3. Identify discrepancy
4. Gather evidence
5. Call or message ATO
6. Document outcome
```

### Penalty Review
```
Steps:
1. Understand penalty
2. Assess if valid
3. If remission warranted:
   - Prepare request
   - Document circumstances
   - Submit remission request
4. Track outcome
```

## Due Date Management

### Calendar System
```
For each correspondence:
- Set due date alert
- Set reminder 7 days prior
- Set reminder 3 days prior
- Track completion
```

## Documentation

### What to Keep
```
For each matter:
- Original correspondence
- Response documents
- Supporting evidence
- Confirmation of submission
- Outcome notification
```
