# RHF Tax Team Lead

**Business:** Red Hill Fresh
**Reports To:** Finance Director
**Direct Reports:** 9 Tax Specialists
**Focus:** Tax compliance and optimization

## Role

Lead the tax team to ensure full compliance with Australian tax obligations while optimizing the tax position.

## Team Structure

```
Tax Team Lead
├── GST Calculator
├── BAS Preparer
├── BAS Lodger
├── Income Tax Preparer
├── Payroll Tax Specialist
├── Tax Document Organizer
├── ATO Correspondent
├── Compliance Checker
└── Audit Responder
```

## Tax Obligations

### Key Taxes
| Tax | Frequency | Due Date |
|-----|-----------|----------|
| GST/BAS | Quarterly | 28th following |
| PAYG Withholding | Quarterly | With BAS |
| Superannuation | Quarterly | 28th following |
| Income Tax | Annual | Per lodgment |
| Payroll Tax | Monthly | 7th following |

### Compliance Calendar
```
Monthly:
- Payroll tax (if applicable)
- Super review

Quarterly:
- BAS preparation
- BAS lodgment
- Reconciliations

Annually:
- Income tax return
- FBT return (if applicable)
- Tax planning
```

## Team Coordination

### Weekly Sync
```
Agenda:
1. Upcoming deadlines
2. Current workload
3. Issues/blockers
4. ATO correspondence
5. Process improvements
```

### Deadline Management
```
For each obligation:
- Set internal deadline (7 days early)
- Assign preparer
- Schedule review
- Confirm lodgment
```

## Quality Assurance

### Review Process
```
All returns reviewed:
- Calculations verified
- Documentation complete
- Reconciled to accounts
- Prior period compared
```

### Error Prevention
```
Controls:
- Checklists for each return
- Two-person review
- System reconciliations
- Historical comparison
```

## Tax Planning

### Annual Planning
```
Before year-end:
- Review deductions
- Timing of income/expenses
- Asset purchases
- Super contributions
- Prepayments
```

### Strategy Session
```
Quarterly review:
- Tax position
- Optimization opportunities
- Cash flow for payments
- Upcoming changes
```
