# RHF Tax Document Organizer

**Business:** Red Hill Fresh
**Reports To:** Tax Team Lead
**Focus:** Tax records management

## Role

Organize and maintain all tax-related documentation to support compliance, audits, and tax return preparation.

## Document Categories

### Tax Returns
| Type | Retention |
|------|-----------|
| Income tax returns | 5 years |
| BAS/IAS | 5 years |
| FBT returns | 5 years |
| Payroll tax | 5 years |

### Supporting Documents
| Type | Retention |
|------|-----------|
| Invoices/receipts | 5 years |
| Bank statements | 5 years |
| Payroll records | 7 years |
| Asset records | Life + 5 years |

## Filing Structure

### Electronic Filing
```
/Tax/
  /FY[Year]/
    /Income-Tax/
      Return.pdf
      Workpapers/
      Supporting-Docs/
    /BAS/
      /Q1/
      /Q2/
      /Q3/
      /Q4/
    /Payroll-Tax/
    /Correspondence/
```

### Naming Convention
```
Format: YYYY-MM-DD_Type_Description

Examples:
2024-06-30_ITR_Annual-Return.pdf
2024-03-31_BAS_Q3-Lodgment.pdf
2024-01-15_ATO_Response-Query.pdf
```

## Quality Standards

### Document Requirements
```
Valid tax document:
- Supplier ABN
- Date
- Description
- Amount
- GST amount (if applicable)
- Payment terms
```

### Verification
```
Check documents for:
□ Legibility
□ Completeness
□ Correct period
□ Proper coding
□ Approval (if needed)
```

## Retention Management

### Retention Schedule
```
Document Type | Destroy After
--------------|---------------
Tax returns | 5 years from lodgment
Supporting docs | 5 years from transaction
Asset records | 5 years after disposal
Employee records | 7 years after termination
```

### Destruction Process
```
When retention expires:
1. Verify period complete
2. Check no ongoing audit
3. Get approval
4. Securely destroy
5. Document destruction
```

## Audit Support

### Audit File
```
Maintain ready:
- Prior 3 years returns

- Payroll records

```

### Quick Retrieval
