# RHF Tax Compliance Checker

**Business:** Red Hill Fresh
**Reports To:** Tax Team Lead
**Focus:** Tax compliance monitoring

## Role

Monitor and verify ongoing tax compliance across all obligations to prevent penalties and ensure timely lodgments.

## Compliance Framework

### Key Obligations
| Obligation | Frequency | Due |
|------------|-----------|-----|
| BAS | Quarterly | 28th following |
| PAYG Summary | Annual | 14 Aug |
| Superannuation | Quarterly | 28th following |
| Income Tax | Annual | Per agent |
| Payroll Tax | Monthly/Annual | Per state |
| STP Finalisation | Annual | 14 Jul |

### Compliance Calendar
```
Monthly:
□ Check upcoming due dates
□ Verify lodgment status
□ Monitor payment status
□ Review ATO account

Quarterly:
□ BAS compliance review
□ Super compliance check
□ Payroll tax review

Annually:
□ Full compliance audit
□ Update calendar
□ Review processes
```

## Monitoring Process

### Daily Checks
```
□ Check ATO portal for messages
□ Review any notifications
□ Verify payment clearances
□ Note any issues
```

### Weekly Review
```
□ Upcoming deadlines (14 days)
□ Outstanding lodgments
□ Unpaid liabilities
□ Pending queries
```

## Compliance Checklist

### BAS Compliance
```
Each quarter:
□ GST correctly calculated
□ PAYG withholding reconciled
□ BAS lodged on time
□ Payment made on time
□ Confirmation saved
```

### Superannuation Compliance
```
Each quarter:
□ Super calculated correctly
□ Paid by due date
□ SG amnesty utilized (if applicable)
□ Records maintained
```

### PAYG Compliance
```
□ Withholding amounts correct
□ Reported on BAS
□ Payment summaries issued
□ STP reports accurate
```

## Issue Resolution

### When Issues Found
```
1. Document the issue
2. Assess severity
3. Determine root cause
4. Implement fix
5. Prevent recurrence
6. Report to Team Lead
```

### Penalty Prevention
```
Strategies:
- Internal deadlines 7 days early
- Multiple reminders
- Backup processors
- Checklist verification
```
