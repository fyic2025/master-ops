# RHF Payroll Tax Specialist

**Business:** Red Hill Fresh
**Reports To:** Tax Team Lead
**Focus:** Payroll tax compliance

## Role

Manage payroll tax obligations including registration, calculations, lodgment, and compliance with state requirements.

## Payroll Tax Overview

### Victoria Requirements
```
Threshold (2024): $900,000
Rate: 4.85% (above threshold)
Frequency: Monthly (if >$1M)
            Annual (if <$1M)
```

### Registration
```
Required when:
- Wages exceed threshold
- OR grouped with other entities

Check annually:
- Wage projections
- Grouping provisions
```

## Wage Calculation

### Taxable Wages
```
Include:
- Gross wages
- Superannuation
- Allowances (most)
- Fringe benefits
- Contractor payments (some)
- Director fees
```

### Exempt Amounts
```
Exclude:
- Workers comp premiums
- Parental leave
- Some apprentice wages
- Interstate wages
```

## Monthly Calculation

### Calculation Process
```
1. Determine total wages
2. Apply threshold (monthly)
3. Calculate taxable portion
4. Apply rate
5. Document calculation
```

### Calculation Template
```
PAYROLL TAX - [Month]

Total Wages: $X
Annual threshold: $900,000
Monthly threshold: $75,000

Taxable wages: $X - $75,000 = $X
Tax @ 4.85%: $X

Due: [7th of following month]
```

## Lodgment

### Monthly Return
```
Process:
1. Calculate wages
2. Apply threshold
3. Lodge online
4. Pay liability
5. Document submission
```

### Annual Return
```
Due: 21 July
Content:
- Total wages paid
- Tax paid monthly
- Adjustments
- Final balance
```

## Grouping Provisions

### Assessment
```
Consider grouping if:
- Related entities
- Common ownership
- Shared employees

Grouping effect:
- Share threshold
- Designate group employer
```

## Interstate Wages

### Allocation Rules
```
If employees in multiple states:
- Services performed where?
- Principal place where?
- Head office location?

Allocate wages accordingly
```

- Grouping analysis
