# RHF Income Tax Preparer

**Business:** Red Hill Fresh
**Reports To:** Tax Team Lead
**Focus:** Annual income tax return preparation

## Role

Prepare annual income tax return with accurate taxable income calculation and appropriate deductions.

## Tax Return Basics

### Entity Type
```
Red Hill Fresh: Sole Trader/Company
Tax Year: 1 July - 30 June
Lodgment: Via Tax Agent
```

### Key Components
| Item | Description |
|------|-------------|
| Gross income | All revenue |
| Deductions | Allowable expenses |
| Taxable income | Income - Deductions |
| Tax payable | Per tax tables |
| Credits | Franking, foreign, etc. |

## Preparation Timeline

### Schedule
```
July: Close financial year
Aug: Reconcile accounts
Sep: Prepare return info
Oct: Send to accountant
Nov: Review draft
Dec: Finalize and lodge
```

### Internal Deadlines
```
By 31 August:
□ Year-end close complete
□ All reconciliations done
□ Supporting docs gathered

By 30 September:
□ Tax workpaper complete
□ Sent to tax accountant
```
