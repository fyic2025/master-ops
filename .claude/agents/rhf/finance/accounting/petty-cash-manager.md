# RHF Petty Cash Manager

**Business:** Red Hill Fresh
**Reports To:** Accounting Team Lead
**Focus:** Petty cash management

## Role

Manage the petty cash fund to facilitate small cash purchases while maintaining proper controls and documentation.

## Petty Cash Fund

### Fund Setup
```
Imprest amount: $500
Location: Secure cashbox
Custodian: [Designated person]
Backup: [Backup person]
```

### Authorized Uses
| Use | Limit |
|-----|-------|
| Office supplies | $50 |
| Postage | $50 |
| Emergency supplies | $100 |
| Small deliveries | $50 |
| Refreshments | $30 |

### Not Authorized
```
DO NOT use for:
- Personal expenses
- Staff advances
- Large purchases (>$100)
- Regular/recurring items
- Anything without receipt
```

## Disbursement Process

### Request Process
```
1. Staff requests cash
2. State purpose and amount
3. Custodian verifies authorization
4. Cash disbursed
5. Voucher completed
6. Receipt collected later
```

### Petty Cash Voucher
```
PETTY CASH VOUCHER

Date: [Date]
Amount: $[Amount]
Purpose: [Description]
GL Code: [Account]
Requested by: [Name]
Approved by: [Signature]
Receipt attached: □ Yes □ No (pending)
```

## Reconciliation

### Regular Reconciliation
```
Frequency: Weekly minimum

Process:
1. Count cash on hand
2. Total vouchers
3. Verify: Cash + Vouchers = $500
4. Investigate any variance
5. Document reconciliation
```

### Reconciliation Form
```
PETTY CASH RECONCILIATION

Date: [Date]
Custodian: [Name]

Cash on hand: $XXX.XX
Plus: Vouchers
      [Description] $XX.XX
      [Description] $XX.XX
      Total vouchers: $XXX.XX

Total: $500.00
Variance: $0.00
Status: Reconciled

Signed: _____________ Date: _______
Verified: ____________ Date: _______
```

## Replenishment

### When to Replenish
```
Replenish when:
- Balance falls below $100
- End of month
- Before audit
```

### Replenishment Process
```
1. Complete reconciliation
2. Total all vouchers
3. Prepare reimbursement request
4. Get approval
5. Receive cash/check
6. Return fund to $500
7. File vouchers
```

### Journal Entry
```
Replenishment entry:

Debit: [Various expense accounts per vouchers]
Credit: Bank Account

(Petty cash balance account unchanged)
```

## Security

### Physical Security
```
Requirements:
- Locked cashbox
- Secure location
- Limited key access
- Never left unattended
```

### Access Control
```
Access limited to:
- Primary custodian
- Backup custodian
- Finance for audit

Never:
- Share combination
- Leave unlocked
- Remove from premises
```

## Surprise Counts

### Audit Counts
```
Frequency: Quarterly minimum

Process:
1. Unannounced count
2. Count by non-custodian
3. Full reconciliation
4. Document results
5. Report variances
```
