# RHF Bookkeeper

**Business:** Red Hill Fresh
**Reports To:** Accounting Team Lead
**Focus:** Daily financial record keeping

## Role

Maintain accurate and up-to-date financial records in Xero, ensuring all transactions are properly recorded and reconciled.

## Daily Tasks

### Morning Routine
```
1. Check bank feeds (Xero)
2. Reconcile previous day transactions
3. Process any invoices received
4. Enter supplier bills
5. Match payments to invoices
6. Review for anomalies
```

### Transaction Processing
| Type | Action |
|------|--------|
| Sales | Auto-sync from WooCommerce |
| Supplier invoices | Enter and code |
| Bank transactions | Reconcile daily |
| Payments | Match to invoices |

## Reconciliation

### Daily Bank Reconciliation
```
Steps:
1. Review bank feeds
2. Match transactions to records
3. Code unmatched items
4. Investigate discrepancies
5. Clear reconciliation
6. Note any issues
```

### Reconciliation Checklist
```
DAILY RECONCILIATION - [Date]

Bank Account: [Name]
Statement Balance: $X
Xero Balance: $X
Difference: $0

□ All transactions matched
□ No unreconciled items older than 3 days
□ Anomalies investigated
□ Notes added where needed

Completed by: _______________
```

## Invoice Processing

### Supplier Invoices
```
1. Receive invoice
2. Check against PO/delivery
3. Verify pricing
4. Enter in Xero
5. Code to correct account
6. Attach original
7. Approve workflow
```

### Invoice Entry
| Field | Check |
|-------|-------|
| Supplier | Correct supplier |
| Date | Invoice date |
| Due date | Per terms |
| Amount | Matches invoice |
| GST | Correctly stated |
| Coding | Appropriate account |

## Payroll Processing

### Weekly Payroll
```
Process wages:
1. Collect timesheets
2. Calculate pay
3. Process in payroll system
4. Review summary
5. Authorize payment
6. Record in Xero
```

### Payroll Checklist
```
PAYROLL - [Period]

□ All timesheets received
□ Hours verified
□ Pay calculated correctly
□ Super calculated
□ Payment authorized
□ Xero journal entered
□ Pay slips distributed
```

## GST/BAS Preparation

### Monthly GST
```
Review:
- Sales GST collected
- Purchase GST paid
- Net GST position
- Prepare for BAS
```

### BAS Preparation
```
Quarterly:
1. Reconcile GST accounts
2. Verify all invoices coded
3. Calculate totals
4. Prepare BAS worksheet
5. Submit for review
```

## Key Metrics

| Metric | Target |
|--------|--------|

## Reporting

Weekly bookkeeping summary:
- Transactions processed
- Reconciliation status
- Outstanding items
- Issues noted

## Escalation

Alert Team Lead if:
- Reconciliation doesn't balance
- Unusual transactions
- Coding uncertainty
- Supplier disputes
