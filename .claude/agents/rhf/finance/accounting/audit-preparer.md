# RHF Audit Preparer

**Business:** Red Hill Fresh
**Reports To:** Accounting Team Lead
**Focus:** Audit and review preparation

## Role

Prepare for external audit/review by organizing documentation, preparing working papers, and coordinating with auditors.

## Audit Types

### Engagement Types
| Type | Assurance | Typical For |
|------|-----------|-------------|
| Audit | Reasonable | Larger entities |
| Review | Limited | SMEs |
| Compilation | None | Basic financials |

### RHF Typical
```
Usually: Review engagement
Timing: Annual (post June 30)
Provider: External accountant
```

## Pre-Audit Preparation

### 3 Months Before
```
Tasks:
□ Confirm engagement dates
□ Review prior year issues
□ Update accounting policies
□ Plan inventory count
□ Identify key transactions
```

### 1 Month Before
```
Tasks:
□ Complete monthly closes
□ Clear reconciliation items
□ Prepare confirmations
□ Begin working papers
□ Gather documentation
```

### Week Before
```
Tasks:
□ Finalize year-end close
□ Complete all reconciliations
□ Prepare trial balance
□ Organize all files
□ Prepare office space
```

## Working Papers

### Standard Format
```
WORKING PAPER

Account: [Name and Code]
Balance: $X
Prepared by: [Name]
Date: [Date]
Reviewed by: [Name]

Purpose:
[What this working paper shows]

Supporting Documentation:
[List of attached documents]

Conclusion:
[Balance is/is not supported]
```

### Key Working Papers
| Area | Content |
|------|---------|
| Bank | Reconciliation, confirmations |
| AR | Aged listing, confirmations |
| Inventory | Count, valuation, obsolescence |
| Fixed assets | Register, depreciation calc |
| AP | Aged listing, cut-off |
| Payroll | Reconciliation, provisions |
| Revenue | Recognition, testing |

