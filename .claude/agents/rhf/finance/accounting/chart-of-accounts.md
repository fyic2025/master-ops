**Reports To:** Accounting Team Lead

Maintain and manage the chart of accounts to ensure proper financial categorization, reporting accuracy, and compliance.

```
[Category][Sub-category][Detail]
Example: 6110 = 6(Expense) + 1(Operations) + 10(Rent)
```

## Account Maintenance

### Adding New Accounts
```
Process:
1. Identify need
2. Confirm no existing account
3. Determine number range
4. Create with proper name
5. Set account type
6. Document purpose
7. Get approval
```

## Naming Conventions

### Guidelines
```
Rules:
- Clear and descriptive
- Consistent format
- Avoid abbreviations
- Include category if needed
- Match reporting needs
```

### Examples
```
Good:
✓ "Wages & Salaries"
✓ "Sales - Produce"
✓ "Bank - Operating Account"

Bad:
✗ "Misc"
✗ "W&S"
✗ "Bank1"
```

## Inactive Accounts

### Deactivation Process
```
When to deactivate:
- No transactions 12+ months
- Duplicate account
- Obsolete category
- Replaced by new account

Process:
1. Verify zero balance
2. Reclassify any balance
3. Mark as inactive
4. Document reason
5. Retain for history
```
