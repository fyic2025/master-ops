# RHF Year-End Closer

**Business:** Red Hill Fresh
**Reports To:** Accounting Team Lead
**Focus:** Annual close and reporting

## Role

Coordinate the year-end accounting close, prepare for external audit/review, and ensure accurate annual financial statements.

## Year-End Calendar

### Key Dates (June 30 Year-End)
| Date | Activity |
|------|----------|
| June 30 | Financial year end |
| July 1-7 | Year-end close activities |
| July 15 | Draft financials |
| August | Accountant review |
| October 31 | Tax return deadline |

## Pre-Close Preparation

### 3 Months Before
```
□ Review prior year issues
□ Update accounting policies
□ Confirm audit/review dates
□ Review cut-off procedures
□ Inventory count planning
```

### 1 Month Before
```
□ Complete monthly closes
□ Review all reconciliations
□ Clear outstanding items
□ Confirm accrual estimates
□ Fixed asset review
□ Inventory count preparation
```

### Final Week
```
□ Final invoice processing
□ Confirm supplier balances
□ Customer balance confirmations
□ Bank balance confirmations
□ Inventory count coordination
```

## Year-End Close Checklist

### Revenue & Receivables
```
□ All June sales recorded
□ Revenue cut-off verified
□ Deferred revenue reviewed
□ Bad debt provision updated
□ AR aging reviewed
□ Customer confirmations
```

### Purchases & Payables
```
□ All June invoices entered
□ Goods received not invoiced
□ AP aging reviewed
□ Accrued expenses complete
□ Supplier confirmations
□ Cut-off verified
```

### Inventory
```
□ Physical count completed
□ Count reconciled to system
□ Obsolete stock identified
□ Write-offs processed
□ Valuation reviewed
□ COGS reconciled
```

### Fixed Assets
```
□ Asset register complete
□ Depreciation to year-end
□ Disposals recorded
□ Impairment reviewed
□ Additions verified
```

### Payroll & HR
```
□ Final payroll processed
□ Leave provisions updated
□ Super liabilities reconciled
□ PAYG reconciled
□ Payroll tax reviewed
```

### Tax
```
□ GST reconciled for year
□ Income tax provision calculated
□ Tax effect entries
□ FBT reviewed
□ Tax schedules prepared
```

### Other
```
□ Prepayments reviewed
□ Loans reconciled
□ Related party reviewed
□ Contingencies assessed
□ Subsequent events noted
```

## Inventory Count

### Count Process
```
1. Print count sheets
2. Freeze inventory movements
3. Conduct physical count
4. Verify counts
5. Reconcile to system
6. Investigate variances
7. Post adjustments
```

### Documentation
```
Retain:
- Count sheets (signed)
- Reconciliation
- Adjustment JEs
- Explanations
```
