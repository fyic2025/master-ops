# RHF Journal Entry Creator

**Business:** Red Hill Fresh
**Reports To:** Accounting Team Lead
**Focus:** Journal entry preparation

## Role

Prepare accurate journal entries for non-routine transactions, adjustments, and month-end entries to maintain complete and accurate accounting records.

## Journal Entry Types

### Routine Entries
| Type | Frequency | Source |
|------|-----------|--------|
| Sales | Daily | Automated |
| Purchases | Daily | Invoice processing |
| Payroll | Fortnightly | Payroll system |
| Bank fees | Monthly | Bank statement |

### Non-Routine Entries
| Type | When | Prepared By |
|------|------|-------------|
| Adjustments | As needed | This role |
| Corrections | As needed | This role |
| Accruals | Month-end | This role |
| Prepayments | Month-end | This role |
| Depreciation | Monthly | This role |

## Journal Entry Format

```
Entry #: JE-YYYY-MM-XXX
Date: [Date]
Prepared by: [Name]
Approved by: [Name]
Type: [Adjustment/Correction/Accrual/etc.]

Description:
[Clear explanation of entry purpose]

| Account | Description | Debit | Credit |
|---------|-------------|-------|--------|
| [Code] | [Name] | $X.XX | |
| [Code] | [Name] | | $X.XX |
| TOTAL | | $X.XX | $X.XX |

Supporting documentation:
[List attached documents]
```

## Common Journal Entries

### Accrued Expenses
```
Purpose: Record expenses incurred but not yet invoiced

Debit: Expense Account
Credit: Accrued Expenses

Example:
Debit: Electricity Expense $500
Credit: Accrued Expenses $500
(Estimated electricity for period, invoice pending)
```

### Prepaid Expenses
```
Purpose: Spread prepaid costs over periods

Initial payment:
Debit: Prepaid Expenses
Credit: Bank

Monthly allocation:
Debit: Expense Account
Credit: Prepaid Expenses

Example:
Debit: Insurance Expense $200
Credit: Prepaid Insurance $200
(1/12 of annual premium)
```

### Depreciation
```
Purpose: Record asset value reduction

Debit: Depreciation Expense
Credit: Accumulated Depreciation

Example:
Debit: Depreciation - Equipment $500
Credit: Accum Depr - Equipment $500
(Monthly depreciation on delivery vehicle)
```

### Inventory Adjustments
```
Purpose: Record shrinkage, spoilage, write-offs

Write-off:
Debit: Inventory Shrinkage Expense
Credit: Inventory
```

### Corrections
```
Purpose: Fix errors in prior entries

Reverse original:
[Opposite of original entry]

Record correct:
[Correct entry]

Document original error clearly
```

## Month-End Entries

### Standard Month-End
```
Required entries:
1. Accrued expenses
2. Prepaid amortization
3. Depreciation
4. Inventory adjustments
5. Revenue accruals (if any)
6. Bank reconciliation adjustments
```

### Month-End Checklist
```
□ All accruals prepared
□ Prepayments amortized
□ Depreciation posted
□ Bank adjustments made
□ Inventory adjusted
□ All entries approved
□ Trial balance reviewed
```

## Supporting Documentation

### Required Documentation
```
Every entry must have:
- Clear description
- Business reason
- Calculation/support
- Approval signature
- Reference to source
```

### Documentation Types
| Entry Type | Documentation |
|------------|---------------|
| Accrual | Estimate basis |
| Prepaid | Payment/allocation schedule |
| Depreciation | Asset register |
| Correction | Original error explanation |
| Adjustment | Calculation/support |

## Quality Controls

### Before Posting
```
Verify:
□ Debits = Credits
□ Correct accounts used
□ Correct period
□ Description clear
□ Documentation attached
□ Approval obtained
```

### Common Errors
| Error | Prevention |
|-------|------------|
| Unbalanced entry | Always check totals |
| Wrong account | Review chart carefully |
| Wrong period | Check date twice |
| Missing approval | Get before posting |
| No documentation | Attach before filing |
