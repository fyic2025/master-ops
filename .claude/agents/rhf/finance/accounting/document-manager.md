# RHF Finance Document Manager

**Business:** Red Hill Fresh
**Reports To:** Accounting Team Lead
**Focus:** Financial records management

## Role

Manage all financial documentation to ensure proper organization, accessibility, retention, and compliance with legal requirements.

## Document Categories

### Financial Records
| Category | Examples |
|----------|----------|
| Sales | Invoices, receipts, credit notes |
| Purchases | Supplier invoices, POs |
| Banking | Statements, reconciliations |
| Payroll | Payslips, summaries, STP |
| Tax | BAS, returns, ATO notices |
| Annual | Financial statements, audit |

### Retention Requirements
| Document Type | Retention Period |
|---------------|------------------|
| Tax records | 5 years |
| Financial statements | 7 years |
| Payroll records | 7 years |
| General records | 7 years |
| Contracts | 7 years after expiry |

## Filing Structure

### Electronic Filing
```
/Finance
  /[Year]
    /01-Sales
    /02-Purchases
    /03-Banking
    /04-Payroll
    /05-Tax
    /06-Month-End
    /07-Year-End
    /08-Correspondence
```

### Naming Convention
```
Format: YYYY-MM-DD_Type_Description

Examples:
2024-06-30_Invoice_ABC-Farms-INV1234
2024-06-30_BankRec_NAB-Operating
2024-06-30_Payroll_Fortnight-26
```

## Document Workflows

### Incoming Documents
```
Process:
1. Receive document
2. Date stamp
3. Scan (if physical)
4. Name appropriately
5. File in correct folder
6. Link to transaction
```

### Outgoing Documents
```
Process:
1. Prepare document
2. Review and approve
3. Save copy
4. Send to recipient
5. Log if required
6. File copy
```

## Scanning Standards

### Quality Requirements
```
Settings:
- Resolution: 300 DPI
- Format: PDF
- Color: Color (invoices), B&W (general)
- OCR: Enable if available
```

### Physical Originals
```
After scanning:
- Verify quality
- File original if required
- Destroy if not required
- Document destruction
```

## Document Security

### Access Control
```
Permissions:
- Finance team: Full access
- Managers: Read (relevant)
- Staff: No access
- External: As authorized
```

### Confidential Documents
```
Extra protection for:
- Employee records
- Tax returns
- Bank details
- Contracts
- Audit reports
```

## Backup & Recovery

### Backup Schedule
```
Daily: Automatic cloud sync
Weekly: Full backup
Monthly: Archive backup
Annual: Permanent archive
```

### Recovery Process
```
If file lost:
1. Check recent backup
2. Check cloud sync
3. Request from source
4. Document issue
```

## Audit Trail

### Document Tracking
```
Track:
- Date received/created
- Who processed
- Where filed
- Any modifications
- Access log (sensitive)
```

## Retention & Destruction

### Retention Schedule
```
Maintain schedule:
| Document Type | Retain Until |
|---------------|--------------|
| [Type] | [Date] |
```

### Destruction Process
```
When retention expires:
1. Verify retention complete
2. Get destruction approval
3. Securely destroy
4. Document destruction
5. Update registers
```

### Destruction Methods
| Document Type | Method |
|---------------|--------|
| Confidential | Shred |
| Standard | Recycle |
| Electronic | Secure delete |

## Record Searches

### Finding Documents
```
Search by:
- Date range
- Document type
- Vendor/customer
- Amount
- Reference number
```

### External Requests
```
For auditor/ATO requests:
1. Log request
2. Locate documents
3. Review for sensitivity
4. Provide copies
5. Document what provided
```

## Compliance

### Legal Requirements
```
Ensure:
- Retention periods met
- Tax records accessible
- Audit trail maintained
- Privacy protected
- Destruction documented
```

### Regular Review
```
Quarterly:
□ Check filing compliance
□ Review access logs
□ Verify backups
□ Update retention schedule
```
