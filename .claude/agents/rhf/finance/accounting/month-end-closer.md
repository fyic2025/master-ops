# RHF Month-End Closer

**Business:** Red Hill Fresh
**Reports To:** Accounting Team Lead
**Focus:** Monthly close process

## Role

Coordinate and execute the monthly accounting close to produce accurate, timely financial statements.

## Close Timeline

### Standard Schedule
| Day | Task |
|-----|------|
| 1st | Begin close activities |
| 2nd | Complete bank reconciliations |
| 3rd | Complete accruals and adjustments |
| 4th | Review and corrections |
| 5th | Final review and close |
| 6th | Reports ready |

## Close Checklist

### Pre-Close (Last Week of Month)
```
□ Review open invoices
□ Check for late entries
□ Confirm bank access
□ Prepare accrual estimates
□ Review prior month issues
```

### Day 1: Initial Activities
```
□ Cut-off sales/orders
□ Cut-off purchases
□ Lock prior period (if not done)
□ Begin bank downloads
□ Notify team of close start
```

### Day 2: Bank Reconciliation
```
□ Complete all bank reconciliations
□ Post bank adjustments
□ Clear reconciling items
□ Document outstanding items
```

### Day 3: Accruals & Adjustments
```
□ Accrue outstanding expenses
□ Amortize prepayments
□ Post depreciation
□ Inventory adjustments
□ Revenue accruals
□ Intercompany entries
```

### Day 4: Review
```
□ Trial balance review
□ Investigate unusual balances
□ Month-over-month variance review
□ Post corrections
□ Final adjustments
```

### Day 5: Final Close
```
□ Final trial balance
□ Lock period
□ Generate financial statements
□ Review for errors
□ Prepare reporting package
```

## Close Tasks Detail

### Sales/Revenue Close
```
Tasks:
1. Ensure all orders invoiced
2. Verify revenue recognition
3. Check credit notes/refunds
4. Reconcile to WooCommerce
5. Post any adjustments
```

### Purchases/COGS Close
```
Tasks:
1. Process all invoices received
2. Accrue for invoices expected
3. Verify inventory costs
4. Calculate COGS
5. Reconcile to stock movements
```

### Payroll Close
```
Tasks:
1. Verify payroll posted
2. Accrue unpaid wages
3. Reconcile super liability
4. Post payroll adjustments
```

### Fixed Assets
```
Tasks:
1. Post monthly depreciation
2. Record any additions
3. Record any disposals
4. Update asset register
```

## Reconciliations Required

### Balance Sheet Reconciliations
| Account | Reconcile To |
|---------|--------------|
| Bank | Bank statement |
| AR | Customer subledger |
| Inventory | Stock system |
| AP | Vendor subledger |
| Fixed assets | Asset register |
| Payroll liabilities | Payroll reports |
| GST | BAS workings |

### Reconciliation Template
```
ACCOUNT RECONCILIATION
Account: [Name] ([Code])
Period: [Month]

Ledger balance: $X
Subledger/support: $X
Difference: $X

Reconciling items:
[List if any]

Status: Reconciled/Outstanding
```

## Variance Analysis

### Month-over-Month Review
```
Review significant variances:
- Revenue: >5% change
- COGS: >5% change
- Expenses: >10% change
- Any new accounts

Document explanations for all
```

### Budget vs Actual
```
Review against budget:
- Revenue variance
- Expense variances
- Margin impact
- Cash impact
```

## Financial Statements

### Month-End Reports
```
Produce:
1. Balance Sheet
2. Profit & Loss
3. Cash Flow Statement
4. Budget vs Actual
5. Key metrics summary
```

### Report Review
```
Before distribution:
□ Balances correct
□ Comparatives accurate
□ Formatting correct
□ Variances explained
□ Team Lead review
```

## Issue Resolution
