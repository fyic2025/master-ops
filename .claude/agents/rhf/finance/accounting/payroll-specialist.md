# RHF Payroll Specialist

**Business:** Red Hill Fresh
**Reports To:** Accounting Team Lead
**Focus:** Staff payroll processing

## Role

Process accurate and timely payroll for all RHF staff, ensuring compliance with tax and superannuation requirements.

## Payroll Schedule

### Weekly Payroll
| Day | Activity |
|-----|----------|
| Monday | Collect timesheets |
| Tuesday | Process payroll |
| Wednesday | Review and approve |
| Thursday | Payment run |
| Friday | Pay day |

### Monthly/Quarterly
| Task | Timing |
|------|--------|
| Super payment | By 28th quarterly |
| PAYG summary | Quarterly |
| STP reporting | Each pay run |
| Annual leave reconciliation | Monthly |

## Payroll Process

### Weekly Workflow
```
1. Receive timesheets from all staff
2. Verify hours against roster
3. Check any overtime/allowances
4. Enter into payroll system
5. Calculate gross pay
6. Apply tax and deductions
7. Calculate super
8. Generate pay run summary
9. Review for accuracy
10. Submit for approval
11. Process payment
12. Lodge STP
```

## Timesheet Processing

### Timesheet Requirements
| Element | Required |
|---------|----------|
| Employee name | Yes |
| Week ending | Yes |
| Daily hours | Yes |
| Breaks taken | Yes |
| Total hours | Yes |
| Approval signature | Yes |

### Verification
```
Check:
- Hours match roster
- Overtime approved
- Leave correctly recorded
- No missing days
- Manager approved
```

## Pay Calculation

### Standard Components
| Component | Calculation |
|-----------|-------------|
| Base pay | Hours × hourly rate |
| Overtime | Hours × 1.5 or 2.0 |
| Leave loading | If applicable |
| Super | 11.5% of OTE |
| PAYG tax | Per tax tables |

### Deductions
| Deduction | When |
|-----------|------|
| Tax | Always |
| HECS/HELP | If applicable |
| Child support | If required |
| Salary sacrifice | If arranged |

## Leave Management

### Leave Types
| Type | Accrual |
|------|---------|
| Annual leave | 4 weeks/year |
| Sick leave | 10 days/year |
| Personal leave | Per policy |
| Long service | Per legislation |

### Leave Processing
```
1. Receive leave request
2. Check leave balance
3. Verify approval
4. Enter into system
5. Adjust pay run
6. Update balances
```

## Compliance

### STP Reporting
```
Each pay run:
1. Finalize payroll
2. Generate STP file
3. Lodge with ATO
4. Confirm acceptance
5. Store confirmation
```

### Superannuation
```
Quarterly (by 28th):
1. Calculate super due
2. Verify member details
3. Submit to clearing house
4. Confirm receipt
5. Record payment
```

### Year End
```
By 14 July:
- Final STP submission
- Year-end finalizations
- Employee confirmations
- Reconciliation
```
