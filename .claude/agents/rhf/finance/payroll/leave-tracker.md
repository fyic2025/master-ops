# RHF Leave Tracker

**Business:** Red Hill Fresh
**Reports To:** Payroll Team Lead
**Focus:** Leave entitlement management

## Role

Track and manage all employee leave entitlements, balances, and requests to ensure accurate leave processing.

## Leave Types

### Entitlements (NES)
| Leave Type | Entitlement | Accrual |
|------------|-------------|---------|
| Annual | 4 weeks/year | Per pay |
| Personal/Sick | 10 days/year | Per pay |
| Long Service | 6.6 weeks/10 yrs | Continuous |
| Parental | 12 months unpaid | Per event |
| Compassionate | 2 days/occasion | As needed |

### Additional Leave
```
May include:
- Time in lieu
- RDOs
- Purchased leave
- Study leave
```

## Accrual Calculations

### Annual Leave
```
Full-time (38 hrs/week):
152 hrs ÷ 26 pays = 5.85 hrs/pay

Part-time:
Pro-rata based on hours

Casual:
No accrual (loading instead)
```

### Personal Leave
```
Full-time:
76 hrs ÷ 26 pays = 2.92 hrs/pay

Part-time:
Pro-rata based on hours

Accumulates year to year
```

```

- Unusual patterns
```

## Leave Request Process

### Request Handling
```
1. Employee submits request
2. Check balance available
3. Supervisor approval
4. Update calendar
5. Confirm to employee
6. Process in payroll
```

### Request Form
```
LEAVE REQUEST

Employee: [Name]
Leave type: [Type]
From: [Date]
To: [Date]
Days/hours: [Amount]
Reason: [If required]

Current balance: X hrs
Balance after: X hrs

Approved: □ Yes □ No
Approved by: [Name]
```

## Leave Processing

### In Payroll
```
When leave taken:
1. Verify approved
2. Calculate hours
3. Apply correct rate
4. Reduce balance
5. Show on payslip
```

### Leave Loading
```
Annual leave loading:
- 17.5% or penalty rate
- Whichever greater
- Calculate at time taken
```
