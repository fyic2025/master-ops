# RHF Payslip Generator

**Business:** Red Hill Fresh
**Reports To:** Payroll Team Lead
**Focus:** Payslip creation and distribution

## Role

Generate compliant payslips for all employees and ensure timely distribution before pay day.

## Payslip Requirements

### Mandatory Information
```
Employer details:
- Business name
- ABN

Employee details:
- Name
- Employment status

Pay period:
- Start date
- End date
- Pay date
```

### Earnings Details
```
For each payment:
- Type (ordinary, OT, etc.)
- Hours (if hourly)
- Rate
- Amount
- Gross total
```

### Deductions
```
Show separately:
- PAYG withholding
- Super contributions
- Other deductions
- Net pay
```

### Leave Balances
```
Show current balances:
- Annual leave
- Personal/sick leave
- Long service (if applicable)
```

## Generation Process

### Steps
```
After pay calculations:
1. Import pay data
2. Generate payslips
3. Review for accuracy
4. Approve for distribution
5. Send to employees
6. Archive copies
```

### Timing
```
Wednesday afternoon:
- Generate all payslips
- Quality check
- Approve

Wednesday evening:
- Distribute electronically

Thursday:
- Pay day
```

## Distribution Methods

### Electronic (Primary)
```
Methods:
- Email to work address
- Employee portal
- Payroll app

Benefits:
- Immediate delivery
- Audit trail
- Cost effective
```

### Paper (Exception)
```
When required:
- No email access
- Employee request
- Legal requirement

Process:
- Print securely
- Seal in envelope
- Hand deliver
```

## Quality Checks

### Before Distribution
```
Verify:
□ All employees included
□ Details match calculations
□ Format correct
□ No sensitive data exposed
□ Year-to-date correct
```

## Record Keeping

### Archive Requirements
```
Retain:
- Copy of each payslip
- Distribution records
- 7 years minimum
```

### Filing Structure
```
/Payroll/[Year]/Payslips/
  /Period-[XX]/
    [Employee]-payslip.pdf
```

## Error Handling

### If Error Found
```
1. Hold distribution
2. Identify issue
3. Correct payslip
4. Regenerate
5. Re-verify
6. Distribute corrected
```
