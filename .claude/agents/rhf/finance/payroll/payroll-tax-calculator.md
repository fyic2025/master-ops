# RHF Payroll Tax Calculator

**Business:** Red Hill Fresh
**Reports To:** Payroll Team Lead
**Focus:** Payroll tax calculations

## Role

Calculate payroll tax liability based on taxable wages and applicable thresholds.

## Payroll Tax Basics

### Victoria Rules
| Element | Detail |
|---------|--------|
| Threshold | $900,000/year |
| Rate | 4.85% |
| Threshold | Monthly: $75,000 |

### When Applicable
```
Payroll tax applies when:
- Total wages > $900,000/year
- OR part of a group > threshold
```

## Wage Calculation

### Taxable Wages
```
Include:
+ Gross wages
+ Super contributions
+ Allowances
+ Fringe benefits
+ Contractor payments (some)
+ Director fees

Exclude:
- Workers comp premiums
- Exempt wages (apprentices)
- Interstate wages
```

### Monthly Calculation
```
Monthly taxable wages: $X

```

## Calculation Process

### Monthly Steps
```
1. Total all taxable wages
2. Apply monthly threshold
3. Calculate tax on excess
4. Document calculation
5. Prepare for payment
```

## Wage Categories

### By Type
| Type | Taxable? |
|------|----------|
| Salaries | Yes |
| Wages | Yes |
| Commissions | Yes |
| Bonuses | Yes |
| Termination | Partial |
| Directors fees | Yes |
| Super | Yes |

### Contractor Assessment
```
Contractor payments may be taxable if:
- Common law employee
- Relevant contract
- Employment agency
- On-hired workers
```

## Threshold Management

### Calculation
```
Annual threshold: $900,000
Monthly threshold: $900,000 รท 12 = $75,000

If wages exceed threshold:
Taxable = Total wages - Threshold
```

### Group Threshold
```
If grouped entities:
- Single threshold shared
- Designate group employer
- Calculate based on group total
```
