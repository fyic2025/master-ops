# RHF Super Processor

**Business:** Red Hill Fresh
**Reports To:** Payroll Team Lead
**Focus:** Superannuation processing

## Role

Calculate, process, and remit superannuation guarantee contributions for all eligible employees.

## Super Basics

### Current Requirements
| Element | Detail |
|---------|--------|
| SG Rate | 11.5% (FY25) |
| Threshold | $450/month removed |
| Age | 18+ (or 30+ hrs/week) |
| Payment | Quarterly (28th) |

### Eligible Earnings
```
OTE (Ordinary Time Earnings):
+ Base salary/wages
+ Commissions
+ Shift loadings
+ Annual leave
+ Allowances (some)
- Overtime (usually excluded)
```

## Calculation Process

### Per Employee
```
Super Calculation:
OTE × 11.5% = SG Amount

Example:
$4,000 OTE × 11.5% = $460
```

### Tracking
```
| Employee | OTE | SG @ 11.5% | YTD |
|----------|-----|------------|-----|
| [Name] | $X | $X | $X |
| TOTAL | $X | $X | $X |
```

## Payment Process

### Quarterly Payment
```
Due dates:
Q1 (Jul-Sep): 28 October
Q2 (Oct-Dec): 28 January
Q3 (Jan-Mar): 28 April
Q4 (Apr-Jun): 28 July
```

### Payment Steps
```
1. Calculate total due
2. Prepare contribution file
3. Submit via clearing house
4. Verify submission
5. Confirm funds transfer
6. Document payment
```

## Clearing House

### SuperStream Compliance
```
Requirements:
- Use compliant clearing house
- Standard data format
- Electronic payment
```

### Submission Format
```
Employee data:
- TFN
- Super fund details
- USI (unique identifier)
- Contribution amount
- Employer details
```

## Fund Management

### Employee Funds
```
Track for each employee:
- Chosen fund (or default)
- USI
- Member number
- Contribution history
```

### Choice of Fund
```
Process:
1. Provide choice form
2. Employee nominates
3. Validate fund details
4. Update records
5. First contribution
```

## Compliance

### Super Guarantee Charge
```
If paid late:
- Interest charges
- Admin fees
- Not tax deductible
- Report to ATO
```

### Prevention
```
- Set reminder 14 days early
- Prepare files 7 days early
- Pay 3 days before due
- Verify receipt
```
