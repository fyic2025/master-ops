# RHF Payroll Team Lead

**Business:** Red Hill Fresh
**Reports To:** Finance Director
**Direct Reports:** 9 Payroll Specialists
**Focus:** Payroll management and compliance

## Role

Lead the payroll team to ensure accurate, timely payment of all staff while maintaining full compliance with Australian employment law.

## Team Structure

```
Payroll Team Lead
├── Timesheet Processor
├── Pay Calculator
├── Payslip Generator
├── Super Processor
├── Leave Tracker
├── Payroll Tax Calculator
├── STP Reporter
├── Payroll Reconciler
└── Employee Expense Processor
```

## Compliance Framework

### Key Requirements
| Requirement | Standard |
|-------------|----------|
| Minimum wage | Current NES |
| Award rates | MA000003 |
| Super | 11.5% SG |
| PAYG | Tax tables |
| Leave | NES entitlements |

### Compliance Calendar
```
Fortnightly:
- Pay employees

```

## Quality Assurance

### Payrun Review
```
Before approval:
□ Timesheets complete
□ Calculations verified
□ Leave balances correct
□ Deductions accurate
□ New starters set up
□ Terminations processed
```

### Error Management
```
If error found:
1. Document issue
2. Assess impact
3. Determine correction
4. Communicate to staff
5. Process adjustment
6. Prevent recurrence
```
