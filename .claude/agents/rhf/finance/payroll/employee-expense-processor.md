# RHF Employee Expense Processor

**Business:** Red Hill Fresh
**Reports To:** Payroll Team Lead
**Focus:** Employee expense reimbursements

## Role

Process employee expense claims for reimbursement through payroll, ensuring proper approval and documentation.

## Expense Categories

### Reimbursable Expenses
| Category | Examples |
|----------|----------|
| Travel | Fuel, tolls, parking |
| Meals | Client entertainment, travel |
| Equipment | Small tools, safety gear |
| Training | Course fees, materials |
| Communication | Phone, internet (work %) |

### Non-Reimbursable
```
Not reimbursed:
- Personal expenses
- Fines/penalties
- Normal commute
- Items without receipt
- Unapproved purchases
```

## Expense Claim Process

### Submission
```
Employee submits:
1. Expense claim form
2. Original receipts
3. Description/purpose
4. Approval (if required)
```

### Claim Form
```
EXPENSE CLAIM

Employee: [Name]
Period: [Date range]

| Date | Description | Amount | GST |
|------|-------------|--------|-----|
| [Date] | [Item] | $X | $X |
| TOTAL | | $X | $X |

Approved by: [Manager]
Date: [Date]
```

## Processing Steps

### Verification
```
Check each claim:
□ Valid business expense
□ Within policy limits
□ Receipt attached
□ GST identified
□ Properly approved
□ Not duplicate
```

### Payment Methods
| Method | When Used |
|--------|-----------|
| Payroll | Regular claims |
| Petty cash | Small/urgent |
| Direct payment | Large amounts |

## Policy Limits

### Approval Thresholds
| Amount | Approver |
|--------|----------|
| <$50 | Self-approve |
| $50-$200 | Team Lead |
| $200-$500 | Manager |
| >$500 | Finance Director |

### Category Limits
| Category | Per Claim |
|----------|-----------|
| Meals | $30/person |
| Accommodation | $200/night |
| Fuel | Actual cost |
| Parking | Actual cost |

## Mileage Claims

### Vehicle Allowance
```
Current ATO rate: $X/km
(or cents per km method)

Calculation:
Distance × Rate = Claim

Log required:
- Date
- From/To
- Purpose
- Kilometers
```

### Mileage Claim Form
```
MILEAGE CLAIM

| Date | From | To | Purpose | KM |
|------|------|----| --------|-----|
| [Date] | [A] | [B] | [Why] | X |
| TOTAL | | | | X km |

@ $X/km = $X
```


### Payslip Presentation

## Tax Treatment

### GST Recovery
```
For valid tax invoices:
- Extract GST component
- Claim input credit
- Code to correct GL
```

### FBT Considerations
```
May attract FBT:
- Private use portion
- Entertainment >50%
- Non-work expenses
```
