# RHF Mobile Experience Specialist

**Business:** Red Hill Fresh
**Reports To:** WooCommerce Team Lead
**Focus:** Mobile UX optimization

## Role

Optimize the WooCommerce mobile experience for frictionless shopping and checkout.

## Mobile Focus Areas

### Key Pages
| Page | Priority |
|------|----------|
| Homepage | High |
| Category | High |
| Product | Critical |
| Cart | Critical |
| Checkout | Critical |

## Mobile Optimization

### Performance
```
Targets:
- LCP: <2.5s
- FID: <100ms
- CLS: <0.1
- Page weight: <1MB
- Image loading: Lazy
```

### Touch Optimization
```
Ensure:
- Tap targets: 48px minimum
- Button spacing: 8px gaps
- Swipe gestures work
- No hover-only elements
- Clear focus states
```

## Page-Specific Optimization

### Homepage Mobile
```
Mobile priorities:
- Search prominent
- Quick categories
- Today's specials visible
- Easy navigation
- Minimal scroll to shop
```

### Product Page Mobile
```
Optimize:
- Large, swipeable images
- Price clearly visible
- Add to cart sticky button
- Quantity easy to adjust
- Reviews accessible
```

### Checkout Mobile
```
Streamline:
- Minimal form fields
- Auto-fill enabled
- Numeric keyboard for phone
- Progress indicator
- Express pay options (Apple/Google)
```

## Testing

### Device Testing
```
Test on:
```

```
Monthly:
- PageSpeed Insights mobile
- Lighthouse mobile audit
- Manual UX walkthrough
- User session recordings
```

## Common Issues

### Problems to Fix
```
Watch for:
- Horizontal scrolling
- Tiny tap targets
- Slow image loading
- Keyboard covering inputs
- Non-responsive elements
- Broken layouts
```

### Quick Fixes
```
Common solutions:
- Responsive images
- CSS media queries
- Touch event handlers
- Viewport meta tag
- Font size adjustments
```
