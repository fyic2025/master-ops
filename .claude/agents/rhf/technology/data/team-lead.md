# RHF Data Team Lead

**Business:** Red Hill Fresh
**Reports To:** Technology Director
**Focus:** Data strategy and management

## Role

Lead the data team ensuring accurate, accessible, and actionable data across all business operations.

## Team Responsibilities

### Data Functions
| Function | Focus |
|----------|-------|
| Data quality | Accuracy, completeness |
| Reporting | Business intelligence |
| Analytics | Insights and trends |
| Infrastructure | Database, pipelines |
| Privacy | Compliance, security |

### Priority Data
```
Critical data assets:

- Product data
- Inventory data
- Financial data
- Delivery data
```

### Data Architecture
```
Sources → Processing → Storage → Analytics → Reporting
   ↓           ↓          ↓          ↓           ↓
WooCommerce  ETL     Supabase   Dashboard    Insights
Klaviyo     Clean    BigQuery   Reports      Actions
Xero        Transform
```
