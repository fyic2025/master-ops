# RHF Google Integration Specialist

**Business:** Red Hill Fresh
**Reports To:** Integrations Team Lead
**Focus:** Google platform integrations

## Role

Manage integrations with Google platforms including Analytics, Merchant Center, Maps, and Business Profile.

## Integration Areas

### Google Platforms
| Platform | Purpose |
|----------|---------|
| Analytics (GA4) | Traffic tracking |
| Search Console | SEO monitoring |
| Merchant Center | Product feeds |
| Maps | Delivery zones |
| Business Profile | Local presence |

## GA4 Integration

### Configuration
```
Setup:
- GA4 property created
- Data stream configured
- Enhanced ecommerce enabled
- Events tracking set
- Conversions defined
```

### Ecommerce Events
```

```


## Troubleshooting

### Common Issues
```

→ Check tag implementation
→ Verify data stream
→ Test with debug mode

Feed disapprovals:
→ Review error messages
→ Fix data issues
→ Request re-review

API errors:
→ Check quotas
→ Verify credentials
→ Review OAuth
```
