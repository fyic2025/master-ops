## Hashtag Strategy

### Hashtag Tiers

**Tier 1: Small (under 50K posts)**
- Highly specific
- Less competition
- Target: 3-5 per post

**Tier 2: Medium (50K-500K posts)**
- Niche relevant
- Moderate competition
- Target: 5-8 per post

**Tier 3: Large (500K+ posts)**
- Broad reach
- High competition
- Target: 2-3 per post

### Hashtag Placement

**Instagram:**
- In caption (inline or at end)
- Or first comment (cleaner look)
- Use line breaks to separate

**LinkedIn:**
- End of post only
- 3-5 maximum
- Relevant industry tags

**Facebook:**
- Generally optional
- 1-3 if used
- Not as effective

## Best Practices

### Caption Writing

```
First line = Hook (most important)
Body = Value
End = CTA

Tips:
- Front-load important info
- Use line breaks for readability
- Include emojis sparingly
- Ask questions for engagement
- Keep paragraphs short
```
