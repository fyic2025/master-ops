## Phase 2: Writing

### Step 1: Hook Creation

**Write 3 hook options:**

```
Option A (Question):
[Question that addresses pain point or curiosity]

Option B (Statement):
[Bold statement or interesting fact]

Option C (Number):
[Numbered list or statistic]
```

**Test: Does it pass the scroll test?**
- Would this make you stop scrolling?
- Is it clear what value you'll get?
- Does it create curiosity?

## Templates

### Quick Post Formula

```
HOOK: [Attention-grabbing first line]

BODY:
[Value point 1]
[Value point 2]
[Value point 3]

CTA: [Specific action]

HASHTAGS: [Platform-appropriate tags]
```

### Carousel Formula

**Slide 1:** Bold hook/statement
**Slide 2:** Setup/context
**Slides 3-8:** Main points (one per slide)
**Slide 9:** Summary/recap
**Slide 10:** CTA + branding

### Reel Script Formula

```
0-3 sec: HOOK (stop the scroll)
3-10 sec: SETUP (what you're showing)
10-45 sec: VALUE (main content)
45-60 sec: CTA (what to do next)

Text overlay: Key points
Caption: Full context + hashtags
```
