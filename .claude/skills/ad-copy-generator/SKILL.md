## Google Ads Specifications

### Responsive Search Ads (RSA)

| Element | Limit | Quantity |
|---------|-------|----------|
| Headlines | 30 chars | 3-15 |
| Descriptions | 90 chars | 2-4 |
| Display URL Path | 15 chars each | 2 paths |

**Best Practices:**
- Minimum 8 headlines recommended
- Minimum 4 descriptions recommended
- Include keywords in at least 2 headlines
- Include one CTA headline
- Include one benefit headline
- Don't repeat messaging across headlines

## Ad Copy Frameworks

### Problem-Solution

```
Headline: [Problem] → [Solution]
Description: Experiencing [problem]? [Product/Service] helps you [solution]. [Trust signal]. [CTA].

Example:
Headline: Tired of Low Energy?
Description: Struggling with afternoon slumps? Our organic superfoods deliver sustained energy naturally. Certified organic. Shop now.
```

### Benefit-Feature

```
Headline: [Key Benefit]
Description: [Feature] that delivers [benefit]. [Supporting benefit]. [CTA].

Example:
Headline: Boost Your Immunity Naturally
Description: Dual-extracted mushroom formulas for maximum potency. Traditional wisdom meets modern wellness. Shop our range today.
```

### Social Proof

```
Headline: [Proof Point]
Description: [Expand on proof]. [Why it matters]. [CTA].

Example:
Headline: Trusted by 50,000+ Customers
Description: Join thousands choosing certified organic products. 4.8★ average rating. Free shipping over $99. Shop Australia's favorite organic store.
```

### Urgency/Scarcity

```
Headline: [Time/Quantity Limit]
Description: [Offer details]. [Why act now]. [CTA].

Example:
Headline: 20% Off - This Week Only
Description: Save on organic superfoods and supplements. Limited time offer ends Sunday. Free shipping over $99. Shop the sale now.
```

### Question Hook

```
Headline: [Engaging Question]
Description: [Answer/Solution]. [Benefit]. [CTA].

Example:
Headline: Ready to Transform Your Health?
Description: Discover medicinal mushrooms used for centuries. Premium extracts for modern wellness. Start your transformation today.
```

## A/B Testing Framework

### What to Test

| Element | Variations to Test |
|---------|-------------------|
| Headlines | Benefit vs feature, question vs statement, with/without numbers |
| Descriptions | Short vs long, benefit focus vs feature focus |
| CTAs | Shop Now vs Buy Now vs Discover |
| Offers | Percentage vs dollar off, free shipping vs discount |
| Urgency | With vs without time pressure |

### Testing Process

1. **Hypothesis:** "Question headlines will improve CTR"
2. **Create variations:** Control + Test variant
3. **Run test:** Equal budget, sufficient time (2+ weeks)
4. **Analyze:** Statistical significance
5. **Document:** Winner and learnings
6. **Iterate:** Test next element

### Headline Test Ideas

```
Test A: Feature-focused
"Organic Spirulina Powder"

Test B: Benefit-focused
"Boost Your Energy Naturally"

Test C: Social proof
"Australia's #1 Organic Store"

Test D: Question
"Looking for Real Organic?"
```

## Keyword Integration

### Headline Best Practices

- Include exact match keyword in at least 2 headlines
- Use keyword variations naturally
- Don't force keywords if unnatural
- Balance keyword headlines with benefit/CTA headlines

### Description Best Practices

- Include primary keyword once in each description
- Use semantic variations
- Keep it readable - don't stuff keywords
- Keyword should flow naturally

### Dynamic Keyword Insertion (DKI)

```
Format: {KeyWord:Default Text}

Example:
Headline: Shop {KeyWord:Organic Products} Online
When keyword is "spirulina": "Shop Spirulina Online"
When keyword triggers default: "Shop Organic Products Online"

Caution:
- Check all keyword variations fit character limit
- Ensure all variations make grammatical sense
- Don't use for competitor keywords
```

## Compliance Guidelines

### Google Ads Policies

**Prohibited:**
- Misleading claims
- ALL CAPS (except acronyms)
- Excessive punctuation!!!
- Inappropriate symbols (★ in headlines)
- Unsubstantiated superlatives
- Click-bait language

**Required:**
- Accurate pricing if shown
- Clear landing page relevance
- No competitor trademark misuse
- Appropriate claims only

### Meta Ads Policies

**Prohibited:**
- Personal attributes ("Are you overweight?")
- Before/after images (health)
- Misleading claims
- Clickbait headlines
- Discriminatory content

**Required:**
- Clear advertiser identity
- Honest representation
- Landing page matches ad

### Health/Supplement Guidelines

```
Avoid:
- Cure, treat, prevent language
- Medical condition targeting
- Before/after claims
- Guaranteed results

Safe:
- "Supports healthy..."
- "Traditionally used for..."
- "May help with..."
- General wellness claims
```
