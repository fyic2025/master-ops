# Platform Specifications

## Google Ads

### Responsive Search Ads (RSA)

**Headlines:**
- Character limit: 30 per headline
- Minimum required: 3
- Maximum allowed: 15
- Recommended: 8-15 for optimal combinations

**Descriptions:**
- Character limit: 90 per description
- Minimum required: 2
- Maximum allowed: 4
- Recommended: 4 for variety

**Display URL:**
- Final URL: Full landing page URL
- Path 1: 15 characters max
- Path 2: 15 characters max

**Asset Combinations:**
- Google automatically tests combinations
- Pin headlines/descriptions to specific positions if needed
- Avoid pinning unless absolutely necessary

**Quality Considerations:**
- Ad Strength indicator: Poor → Average → Good → Excellent
- Aim for "Good" or "Excellent" ad strength
- Include keywords in multiple headlines
- Vary messaging across headlines (don't repeat)

|---------|-------|

### Shopping Ads

**Product Feed Requirements:**
| Field | Limit | Notes |
|-------|-------|-------|
| Title | 150 chars | First 70 visible |
| Description | 5,000 chars | First 175 visible |
| Brand | 70 chars | Required |
| GTIN/MPN | Various | Required for known brands |

**Title Optimization:**
```
Format: [Brand] + [Product Type] + [Key Attribute] + [Size/Variant]

Example: Teelixir Lion's Mane Mushroom Extract Powder 100g

Best Practices:
- Put most important info first
- Include size/quantity
- Add key differentiators
- Use proper capitalization
```

**Description Optimization:**
```
- Lead with key benefits
- Include product specifications
- Natural keyword integration
- Avoid promotional language
```

### Ad Format Specifications

**Single Image:**
- Ratio: 1.91:1 to 1:1 (1:1 recommended)
- Resolution: 1080x1080 minimum
- File type: JPG, PNG
- Max file size: 30MB

**Carousel:**
- 2-10 cards
- Image ratio: 1:1
- Resolution: 1080x1080
- Each card: Headline + Description

**Video:**
- Ratio: 1:1, 4:5, 16:9
- Length: 1 sec - 240 min (15 sec recommended)
- Max file size: 4GB

**Stories/Reels:**
- Ratio: 9:16
- Resolution: 1080x1920
- Safe zones: Keep text in center 80%

---

### Dynamic Creative

**Asset Types:**
- Up to 10 images/videos
- Up to 5 primary text variations
- Up to 5 headlines
- Up to 5 descriptions
- Up to 5 CTAs

**Best Practices:**
- Varied messaging themes
- Different emotional appeals
- Mix of benefit/feature focus
- Test with/without emoji

## Character Counting Tips

### What Counts

```
Standard characters: 1 count each
- Letters, numbers, spaces
- Basic punctuation

Special characters: Varies
- Some symbols count as 2
- Emoji: Usually 2+ characters
- Foreign characters: May vary
```

### Validation

```
Always verify:
1. Count in platform preview
2. Test on mobile view
3. Check truncation points
4. Verify special characters
```

## Policy Quick Reference

### Google Ads

**Editorial:**
- No ALL CAPS (except acronyms)
- Proper spelling and grammar
- No excessive punctuation
- No gimmicky symbols
