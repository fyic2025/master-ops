# Ad Creation Workflow

## Google Ads RSA Workflow

### Step 1: Research & Planning

**Gather Information:**
```
Campaign Details:
- Campaign goal: [Traffic/Conversions/Brand awareness]
- Target keywords: [List primary keywords]
- Landing page: [URL]
- Offer/promotion: [If applicable]
- Unique selling points: [3-5 USPs]
- Competitor messaging: [Key competitor angles]
```

**Analyze Landing Page:**
- Key headlines on page
- Value propositions
- Trust signals
- CTAs used

### Step 2: Headline Creation

**Create 15 Headlines (30 chars each):**

**Category 1: Keyword Headlines (3-4)**
Include exact/close match keywords
```
1. [Exact keyword] Online
2. Shop [Keyword] Australia
3. Buy [Keyword] - Free Shipping
4. [Keyword] - [Year] Range
```

**Category 2: Benefit Headlines (3-4)**
Focus on what customer gets
```
5. Boost Your Energy Naturally
6. Feel Better Every Day
7. Transform Your Wellness
8. Natural Solutions That Work
```

**Category 3: Trust/Authority Headlines (2-3)**
Build credibility
```
9. Trusted by [X]+ Customers
10. Australian Owned & Operated
11. [X]★ Rated Online Store
12. Quality Guaranteed
```

**Category 4: CTA Headlines (2-3)**
Drive action
```
13. Shop Now & Save [X]%
14. Free Shipping Over $[X]
15. Order Today
```

### Step 3: Description Creation

**Create 4 Descriptions (90 chars each):**

**Description 1: Primary Value Prop + CTA**
```
[Primary benefit statement]. [Feature]. [CTA].

Example: Discover certified organic products for your wellness journey. Free shipping over $99. Shop now.
```

**Description 2: Features + Trust**
```
[Key features]. [Trust signal/proof].

Example: Premium quality supplements, superfoods & pantry essentials. Trusted by 50,000+ Australian families.
```

**Description 3: Offer + Urgency**
```
[Current offer]. [Why act now].

Example: Save 20% on your first order with code WELCOME20. Australia-wide delivery. Limited time offer.
```

**Description 4: Differentiation**
```
[What makes you different]. [Key benefit].

Example: Only certified organic products. No fillers, no compromises. Supporting your health naturally.
```

### Step 4: Display URL Paths

**Path Structure:**
```
Path 1: [Category/Product Type]
Path 2: [Offer/Qualifier]

Examples:
www.example.com/organic/shop-now
www.example.com/superfoods/free-shipping
www.example.com/supplements/australia
```

### Step 5: Review Checklist

```
Headlines:
[ ] All under 30 characters
[ ] Keywords in 2-3 headlines
[ ] Benefits clearly stated
[ ] CTAs included
[ ] Variety of angles
[ ] No repetition

Descriptions:
[ ] All under 90 characters
[ ] Primary keyword included
[ ] CTAs present
[ ] Value props clear
[ ] Trust signals included

Quality:
[ ] Ad Strength: Good or Excellent
[ ] Matches landing page
[ ] Policy compliant
[ ] Spelling/grammar correct
```

## Meta Ads Workflow

### Step 1: Campaign Planning

```
Campaign Details:
- Objective: [Awareness/Traffic/Conversions/Sales]
- Audience: [Target description]
- Placement: [Feed/Stories/Reels/All]
- Budget: [Daily/Lifetime]
- Creative format: [Image/Video/Carousel]
```

### Step 2: Primary Text Creation

**Version 1: Short (under 125 chars visible)**
```
[Hook] + [Value] + [CTA]

Example: Transform your mornings with medicinal mushrooms. ✨ Shop our premium extracts today.
```

**Version 2: Medium (125-300 chars)**
```
[Hook]
[Expanded value proposition]
[Social proof or trust signal]
[CTA]

Example:
Ready to feel your best?

Our certified organic superfoods are trusted by 50,000+ Australians.

Free shipping over $99. Shop now →
```

**Version 3: Long (for high-engagement audiences)**
```
[Story hook]
[Problem acknowledgment]
[Solution introduction]
[Benefits - bullet points]
[Social proof]
[CTA + urgency]
```

### Step 3: Headlines (40 chars)

```
1. [Primary benefit]
2. [Offer/promotion]
3. [Question hook]
4. [Social proof]
5. [Urgency/FOMO]
```

### Step 4: Link Descriptions (30 chars)

```
1. [Supporting benefit]
2. [Trust signal]
3. [Offer reminder]
```

### Step 5: CTA Selection

Match to objective:
- **Sales:** Shop Now
- **Traffic:** Learn More
- **Leads:** Sign Up
- **App:** Download

### Step 6: Review Checklist

```
Content:
[ ] Primary text compelling
[ ] Hook in first line
[ ] Benefits clear
[ ] CTA present
[ ] Emoji used appropriately

Technical:
[ ] Character limits met
[ ] Image specs correct
[ ] Landing page working
[ ] Pixel/tracking ready

Compliance:
[ ] No prohibited content
[ ] No personal attributes
[ ] Honest claims
[ ] No clickbait
```

## A/B Testing Workflow

### Step 1: Define Test

```
Hypothesis: [What you believe will improve performance]
Element: [Headlines/Descriptions/CTAs/Offers]
Metric: [CTR/Conversion Rate/CPA/ROAS]
Duration: [Minimum 2 weeks or X conversions]
```

### Step 2: Create Variations

**Control:** Current best performer
**Test A:** Variation 1
**Test B:** Variation 2 (optional)

**Testing Rules:**
- Change only one element at a time
- Keep budget equal across variants
- Run until statistically significant

### Step 3: Launch Test

```
Setup:
[ ] Equal budget allocation
[ ] Same targeting
[ ] Same bid strategy
[ ] Tracking verified
```

### Step 4: Analyze Results

```
Metrics to Compare:
- Impressions
- CTR
- Conversion rate
- Cost per conversion
- ROAS (if applicable)

Statistical Significance:
- Minimum 95% confidence
- Sufficient sample size
- Consider external factors
```

### Step 5: Document & Iterate

```
Test: [Description]
Winner: [A or B]
Improvement: [% change in key metric]
Learning: [What this tells us]
Next Test: [What to test next]
```
