# Ad Copy Generator - Quick Reference

## Google Ads Character Limits

| Element | Limit | Quantity |
|---------|-------|----------|
| RSA Headline | 30 chars | 3-15 |
| RSA Description | 90 chars | 2-4 |
| Display Path | 15 chars each | 2 |
| Long Headline | 90 chars | - |
| Business Name | 25 chars | 1 |

---

## Meta Ads Character Limits

| Element | Visible | Maximum |
|---------|---------|---------|
| Primary Text | 125 | 1,000 |
| Headline | 40 | 40 |
| Link Description | 30 | 30 |

## RSA Quick Template

```
Headlines (need 8-15):
1. [Keyword headline]
2. [Keyword headline 2]
3. [Benefit headline]
4. [Benefit headline 2]
5. [CTA headline]
6. [Trust signal]
7. [Offer/promotion]
8. [Unique selling point]

Descriptions (need 4):
1. [Primary benefit + keyword + CTA]
2. [Features + trust signal]
3. [Value prop + offer]
4. [Social proof + CTA]
```


## Compliance Quick Check

**Google - Avoid:**
- ALL CAPS
- !!!Punctuation!!!
- Misleading claims
- Unverified superlatives

**Meta - Avoid:**
- Personal attributes
- Before/after (health)
- Clickbait
- "Are you...?" questions

**Health - Avoid:**
- Cure, treat, prevent
- Medical claims
- Guaranteed results
