## Email Types & Frameworks

### Campaign Emails

#### Promotional Email Structure
```
Subject: [Benefit/Offer] + [Urgency/Curiosity]
Preview: [Expand on subject, add value]

Header: [Brand + Campaign hook]

Body:
- Opening hook (1-2 sentences)
- Value proposition (what's in it for them)
- Offer details (clear and specific)
- Supporting content (features/benefits)
- Social proof (if space)
- CTA (primary action)

Footer: [Links, unsubscribe, address]
```

#### Newsletter Structure
```
Subject: [This week's topic] + [Value hint]
Preview: [What they'll learn/get]

Header: [Brand + Edition/Issue]

Body:
- Personal intro (1-2 sentences)
- Main content section
- Secondary content
- Product spotlight (optional)
- CTA

Footer: [Social links, preferences, unsubscribe]
```

### Automated Flows

#### Welcome Series (3-5 emails)

**Email 1: Welcome (immediate)**
```
Subject: Welcome to [Brand]! Here's what to expect
Goal: Set expectations, deliver promised value
Content:
- Warm welcome
- What they signed up for
- Introduce brand story
- Welcome offer (if applicable)
CTA: Shop now / Complete profile
```

**Email 2: Brand Story (Day 2)**
```
Subject: Why we do what we do
Goal: Build emotional connection
Content:
- Brand origin story
- Mission and values
- What makes you different
CTA: Learn more / Shop bestsellers
```

**Email 3: Social Proof (Day 4)**
```
Subject: See why [X]+ customers love us
Goal: Build trust
Content:
- Customer reviews
- Testimonials
- Use cases
CTA: Read reviews / Shop now
```

**Email 4: Best Sellers (Day 7)**
```
Subject: Our most-loved products
Goal: Drive first purchase
Content:
- Top products by category
- Why they're popular
- Limited offer (optional)
CTA: Shop bestsellers
```

**Email 5: Last Chance (Day 10)**
```
Subject: Don't miss your welcome offer
Goal: Convert non-buyers
Content:
- Reminder of offer
- Urgency element
- Final push
CTA: Claim offer
```

#### Abandoned Cart Series

**Email 1: Reminder (1 hour)**

## Subject Line Best Practices

### Formula Frameworks

**Benefit + Curiosity:**
```
"The secret to [benefit] (hint: it's [product type])"
```

**Question:**
```
"Looking for [solution]?"
"Ready to [achieve goal]?"
```

**How-To:**
```
"How to [achieve benefit] with [product]"
```

**List:**
```
"5 ways to [benefit] this [season]"
"3 products for [goal]"
```

**Urgency:**
```
"Last chance: [offer] ends tonight"
"Only [X] left in stock"
```

**Personalization:**
```
"[Name], your [product] is waiting"
"Picked just for you, [Name]"
```

### Character Limits

| Client | Desktop | Mobile |
|--------|---------|--------|
| Gmail | 70 chars | 40 chars |
| Apple Mail | 70 chars | 35 chars |
| Outlook | 60 chars | 35 chars |

**Best Practice:** Keep subject lines under 50 characters for mobile optimization.

### Preview Text

| Purpose | Length |
|---------|--------|
| Expand on subject | 40-90 chars |
| Add value | Complete the thought |
| Create curiosity | Don't repeat subject |

**Formula:**
```
Subject: [Hook]
Preview: [Why they should open] + [Additional value]
```

## CTA Best Practices

### Button Copy

**Strong CTAs:**
- Shop Now
- Get Started
- Claim Offer
- Order Today
- View Collection

**Weak CTAs (avoid):**
- Click Here
- Submit
- Learn More (unless truly informational)

### Placement

- Primary CTA above fold
- Secondary CTA after main content
- Final CTA before footer
- Mobile: Large, tappable buttons

### Design

- Contrasting color
- Minimum 44x44px tap target
- Clear white space around
- Action-oriented text

## Email Copy Guidelines

### Opening Lines

**Hook immediately:**
```
Good: "Your wellness routine is about to get an upgrade."
Poor: "Dear valued customer, we hope this email finds you well."
```

**Personalize when possible:**
```
Good: "[Name], based on your last order..."
Good: "Since you loved [product]..."
```

### Body Copy

**Keep it scannable:**
- Short paragraphs (2-3 sentences max)
- Bullet points for features/benefits
- Bold key information
- Clear hierarchy

**Focus on benefits:**
```
Feature: "Contains 10:1 extract"
Benefit: "10x more potent than regular powders"
```

### Tone Consistency

- Match brand voice
- Conversational but professional
- No all caps for emphasis
- Minimal exclamation marks
- Australian English spelling

## A/B Testing Framework

### What to Test

| Element | Variations |
|---------|------------|
| Subject line | Length, emoji, personalization, question vs statement |
| Preview text | Different value props |
| Send time | Morning vs afternoon, weekday vs weekend |
| CTA | Text, color, placement |
| Content length | Long vs short |
| Images | Product vs lifestyle |

### Testing Process

1. **Hypothesis:** "Adding emoji will increase open rate"
2. **Test:** 50/50 split
3. **Sample size:** Minimum 1,000 per variant
4. **Duration:** 24-48 hours
5. **Winner criteria:** Statistical significance (95%)
6. **Document:** Record results and learnings

## Compliance Requirements

### CAN-SPAM / Australian Spam Act

```
Required in every email:
[ ] Sender name and address
[ ] Physical mailing address
[ ] Clear unsubscribe option
[ ] Unsubscribe honored within 10 days
[ ] Accurate subject line (not misleading)
```

### GDPR (if applicable)

```
[ ] Consent obtained before sending
[ ] Clear privacy link
[ ] Easy data access request process
```

### Best Practices

- Only email opted-in subscribers
- Honor unsubscribes immediately
- Don't use deceptive subject lines
- Include clear sender identification
