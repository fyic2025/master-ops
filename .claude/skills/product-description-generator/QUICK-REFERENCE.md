- [ ] Primary keyword in first 25 words
- [ ] Primary keyword in H1/title
- [ ] Secondary keywords (2-3) throughout
- [ ] Meta description with keyword
- [ ] Natural language (not stuffed)
- [ ] 1-2% keyword density max

## Quick Templates

### Short Description
```
[Benefit hook]. [Key feature]. [Usage hint or CTA].
```

### Long Description Structure
```
1. Hook (benefit/story)
2. Features list (bullets)
3. Usage/serving suggestions
4. Quality/sourcing story
5. CTA
```

### Meta Description
```
[Keyword] - [Benefit]. [Trust element]. Shop now at [Brand].
```

---

## Avoid List (All Brands)

### Health Claims
- "cures", "treats", "prevents"
- "clinically proven" (unless true)
- Disease names as benefits
- Medical terminology

### Weak Language
- "very", "really", "basically"
- "cheap", "affordable" (use "value")
- "best" (unless verifiable)

---

## Compliance Reminders

### Supplements
```
*Always read the label. Follow directions for use.*
```

### Health Foods
```
*This product is not intended to diagnose, treat, cure, or prevent any disease.*
```

### Organic Claims
- Only use if certified
- Specify certification body

## Platform Formatting

### BigCommerce
- HTML allowed
- Use `<p>`, `<ul>`, `<strong>`
- Max ~5000 characters

### Shopify
- Rich text editor
- Basic HTML
- Liquid variables available

### WooCommerce
- WordPress editor
- Full HTML support
- Shortcodes available

150 words = ~10-12 sentences
200 words = ~14-16 sentences
300 words = ~20-24 sentences
