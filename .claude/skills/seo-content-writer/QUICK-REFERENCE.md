# SEO Content Writer - Quick Reference

## Content Length Targets

| Content Type | Words | Purpose |
|--------------|-------|---------|
| Quick Answer | 500-800 | Featured snippets |
| Standard Blog | 1,000-1,500 | Regular posts |
| In-Depth Guide | 2,000-3,000 | Pillar content |
| Ultimate Guide | 3,000-5,000 | Authority pieces |
| Category Page | 300-500 | SEO + navigation |
| Landing Page | 500-1,000 | Conversions |

---

## Character Limits

| Element | Limit | Best Practice |
|---------|-------|---------------|
| Title Tag | 60 chars | 50-60 optimal |
| Meta Description | 160 chars | 150-160 optimal |
| URL Slug | 50-60 chars | 3-5 words |
| H1 | No limit | Match title intent |
| H2 | No limit | Include keywords |

---

## On-Page SEO Checklist

```
Title Tag:
✓ Primary keyword near start
✓ Under 60 characters
✓ Compelling and unique

Meta Description:
✓ Primary keyword included
✓ 150-160 characters
✓ Clear CTA

URL:
✓ Short and descriptive
✓ Primary keyword
✓ No special characters

Content:
✓ Keyword in first 100 words
✓ 1-2% keyword density
✓ H2s every 200-300 words
✓ 2-3 internal links/1K words
✓ 1-2 external authority links
✓ Images with alt text
```

|----------|------------|--------------|

| H1 | Yes | Variation OK |

---

## Readability Guidelines

```
Sentences: 15-20 words average
Paragraphs: 2-3 sentences max
Grade Level: 8-10 (Flesch-Kincaid)
Active Voice: 90%+ preferred
Transition Words: Every paragraph
```

---

## Featured Snippet Formats

**Paragraph (40-60 words):**
```
[What is X?]
X is [definition]. It [key characteristic]
and [another characteristic]. [Additional context
in 40-60 words total].
```

**List:**
```
## How to [Action]
1. Step one
2. Step two
3. Step three
[5-8 items ideal]
```

**Table:**
```
| Item | Detail 1 | Detail 2 |
|------|----------|----------|
| A | value | value |
| B | value | value |
```

---

## Internal Linking Rules

- 2-3 links per 1,000 words
- Descriptive anchor text
- Link to relevant content
- Vary anchor text
- Include category pages
- Link to conversion pages

## Common Mistakes to Avoid

| Mistake | Fix |
|---------|-----|
| Keyword stuffing | Natural integration |
| No headings | H2 every 200-300 words |
| Wall of text | Short paragraphs |
| No internal links | 2-3 per 1K words |
| Duplicate meta | Unique per page |
| Generic CTAs | Specific actions |
