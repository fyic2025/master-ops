## SEO Content Framework

### On-Page Optimization Checklist

```
URL Structure:
[ ] Short, descriptive URL (3-5 words)
[ ] Primary keyword in URL
[ ] No stop words or special characters

Title Tag (50-60 chars):
[ ] Primary keyword near beginning
[ ] Brand name at end (if space)
[ ] Compelling and clickable
[ ] Unique across site

Meta Description (150-160 chars):
[ ] Primary keyword included
[ ] Clear value proposition
[ ] Call to action
[ ] Unique per page

Headings:
[ ] H1 contains primary keyword
[ ] H2s cover main subtopics
[ ] H3s for subsections
[ ] Natural keyword variations

Content Body:
[ ] Primary keyword in first 100 words
[ ] Keyword density 1-2%
[ ] Related terms throughout
[ ] 2-3 internal links
[ ] 1-2 external links (authoritative)
[ ] Images with alt text
[ ] Short paragraphs (2-3 sentences)
```

### Keyword Integration Strategy

```
Primary Keyword:
- Title tag (beginning)
- H1 heading
- First paragraph
- 2-3 times in body
- Meta description
- URL slug
- Image alt text

Secondary Keywords:
- H2 headings
- Body content naturally
- Image captions
- FAQ sections

LSI/Related Terms:
- Throughout content naturally
- Subheadings where relevant
- Answer related questions
```

## Writing Best Practices

### For Readability

```
Sentence Structure:
- Average 15-20 words per sentence
- Mix short and long sentences
- Active voice preferred
- One idea per sentence

Paragraph Structure:
- 2-3 sentences max
- One main point per paragraph
- Use transitions between paragraphs
- White space for scannability

Formatting:
- Subheadings every 200-300 words
- Bullet points for lists
- Numbered lists for sequences
- Bold for key points
- Block quotes for emphasis
```

### For SEO

```
Keyword Usage:
- Natural integration only
- No keyword stuffing
- Semantic variations
- Answer user questions

Content Structure:
- Clear hierarchy (H1 > H2 > H3)
- Table of contents for long content
- Jump links for sections
- FAQ schema opportunities

User Experience:
- Answer query quickly (inverted pyramid)
- Provide comprehensive coverage
- Include visuals where helpful
- Mobile-friendly formatting
```

### For Conversions

```
Persuasion Elements:
- Benefits over features
- Social proof where relevant
- Trust signals (certifications)
- Clear calls to action

CTA Best Practices:
- One primary CTA per page
- Above the fold mention
- End of content CTA
- Contextual internal links
```

## Featured Snippet Optimization

### Snippet Types

| Type | Best For | Format |
|------|----------|--------|
| Paragraph | Definitions, explanations | 40-60 words answering "what is" |
| List | Steps, rankings, items | Numbered/bulleted lists |
| Table | Comparisons, data | Structured tables |

### Optimization Tips

```
Paragraph Snippets:
- Answer question directly in first sentence
- Use "is" definition format
- Keep to 40-60 words
- Place near top of content

List Snippets:
- Use clear H2 for the question
- Numbered list immediately after
- 5-8 items ideal
- Consistent formatting

Table Snippets:
- Use proper HTML tables
- Clear header row
- 3-5 columns max
- Relevant comparison data
```

## Internal Linking Strategy

### Link Placement

```
Ideal Distribution:
- 2-3 internal links per 1,000 words
- Contextual links within content
- Related posts section at end
- Navigation links don't count

Anchor Text:
- Descriptive, keyword-rich
- Natural within sentence
- Varied (not same anchor repeatedly)
- Avoid "click here" or "read more"
```
