## Pillar Page Brief

```markdown
# Pillar Page Brief: [Topic]

## Metadata
- **Business:** [BOO/Teelixir/Elevate/RHF]
- **Topic Cluster:** [cluster name]
- **Target Publish:** [date]

## Keyword Strategy

### Primary Keyword
- **Term:** [broad topic keyword]
- **Monthly Volume:** [number]
- **Difficulty:** [typically high]

### Cluster Keywords (to link to)
| Cluster Article | Target Keyword | Status |
|-----------------|----------------|--------|
| [Article 1] | [keyword] | [exists/planned] |
| [Article 2] | [keyword] | [exists/planned] |
| [Article 3] | [keyword] | [exists/planned] |
| [Article 4] | [keyword] | [exists/planned] |
| [Article 5] | [keyword] | [exists/planned] |

## Content Strategy

### Purpose
This pillar page will serve as the comprehensive resource for [topic], linking to all related cluster content.

### Audience Journey
- **Entry Point:** [how users find this]
- **User Goal:** [what they want to achieve]
- **Exit Path:** [where we want them to go]

## Content Specifications

### Format & Length
- **Format:** Ultimate Guide / Comprehensive Resource
- **Target Word Count:** 3,000-5,000 words
- **Table of Contents:** Yes (with jump links)

### Structure

#### H1: [Comprehensive Title]

**Introduction (200-300 words)**
- What this guide covers
- Who it's for
- How to use it (jump links)

#### H2: [Foundational Section]
- What is [topic]
- Key concepts explained
- Why it matters
[Link to: cluster article for beginners]

#### H2: [Section 2 - Main Topic Area]
- Overview of area
- Key information
- Best practices
[Link to: related cluster articles]

#### H2: [Section 3 - Main Topic Area]
- Overview of area
- Key information
- Advanced tips
[Link to: related cluster articles]

#### H2: [Section 4 - Practical Application]
- How to get started
- Step-by-step guidance
- Common mistakes
[Link to: how-to cluster articles]

#### H2: [Section 5 - Advanced Topics]
- Deep dive content
- Expert insights
[Link to: advanced cluster articles]

#### H2: FAQs
- 5-10 common questions
- Featured snippet optimized answers

#### H2: Conclusion & Next Steps
- Summary
- Recommended starting point
- Primary CTA

## Internal Linking Strategy

### Links FROM Pillar (to cluster content)
[Map each cluster article and where it fits in the pillar]

### Links TO Pillar (from cluster content)
[Ensure all cluster articles link back to pillar]

## SEO Elements

### Meta Content
- **Title Tag:** [Comprehensive/Ultimate Guide to [Topic] | Brand]
- **Meta Description:** [Comprehensive guide description]
- **URL:** /[topic]-guide or /[topic]-ultimate-guide

### Schema Markup
- Article schema
- FAQ schema (for FAQ section)
- HowTo schema (if applicable)

## Visual Requirements
- Hero image
- Section illustrations
- Infographic summarizing key concepts
- Embedded videos (if available)

## Success Metrics
- Ranking for primary keyword: Top 10
- Traffic: [target monthly sessions]
- Engagement: >3 min average time
- Internal link clicks: Track with GA4
```

## Product Category Page Brief

```markdown
# Category Page Brief: [Category Name]

## Metadata
- **Business:** [BOO/Teelixir/Elevate/RHF]
- **Category:** [category name]
- **URL:** /[category-slug]

## SEO Goals
- Rank for: [primary category keyword]
- Support: Product pages in this category

## Keyword Targeting

### Primary
- [category keyword] - [volume]

### Long-tail
- [modifier + category]
- [category + australia]
- [best + category]

## Content Requirements

### Above Products (300-500 words)
[SEO-friendly introduction to the category]

**Structure:**
- Opening paragraph (what is this category, who it's for)
- Benefits of category/why shop here
- Brief buying guide or selection tips
- Trust signals (certifications, sourcing)

### Below Products (200-300 words)
[Additional SEO content, FAQs]

**Structure:**
- FAQ section (3-5 questions)
- Additional category information
- Related categories links

## Meta Content
- **Title:** [Category] - [Benefit] | [Brand] ([X] Products)
- **Description:** Shop [category] at [Brand]. [USP]. [Trust signal]. Free shipping over $[X].

## Internal Linking
- Link to: [related categories]
- Link to: [relevant blog content]
- Link from: [homepage, navigation, related categories]
```
