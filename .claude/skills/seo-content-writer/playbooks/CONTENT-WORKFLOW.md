# SEO Content Creation Workflow

## Phase 1: Research & Planning

### 1.1 Keyword Selection

**Required Information:**
- Primary keyword with search volume
- Secondary keywords (3-5)
- Search intent classification
- Current ranking position (if any)

**Actions:**
1. Verify keyword in GSC data
2. Check competitor rankings
3. Confirm search intent
4. Validate business relevance

### 1.2 Competitor Analysis

**For each top 5 result:**
- Word count
- Content structure
- Unique angles covered
- Gaps in coverage
- Backlink profile

**Document findings:**
```
Competitor: [URL]
Word Count: [number]
Structure: [headings list]
Unique Value: [what they do well]
Gaps: [what's missing]
```

## Phase 2: Writing

### 2.1 First Draft Structure

**Opening (10% of content):**
- Hook reader immediately
- State what they'll learn
- Include primary keyword naturally
- Set expectations

**Body (80% of content):**
- Follow outline structure
- One main point per section
- Support claims with evidence
- Include examples and specifics
- Natural keyword integration

**Conclusion (10% of content):**
- Summarize key points
- Clear call to action
- Internal link to related content
- Encourage engagement

### 2.2 Section Writing Guidelines

**For Each Section:**
1. Start with clear H2 heading
2. Opening sentence states section purpose
3. 2-3 supporting paragraphs
4. Include at least one of:
   - Bullet list
   - Example
   - Data point
   - Quote or reference
5. Transition to next section

**Paragraph Guidelines:**
- 2-3 sentences maximum
- One idea per paragraph
- Active voice preferred
- Conversational but professional

### 2.3 Writing Voice by Business

**BOO - Buy Organics Online:**
```
Tone: Warm, supportive, educational
Voice: "We're here to help you..."
Style: Accessible expertise
Avoid: Medical claims, jargon
```

**Teelixir:**
```
Tone: Inspiring, authoritative, passionate
Voice: "Discover the ancient wisdom..."
Style: Elevated but accessible
Avoid: Oversimplification, generic
```

**Elevate Wholesale:**
```
Tone: Professional, helpful, commercial
Voice: "Partner with us for..."
Style: B2B focused, practical
Avoid: Consumer language, casual
```

**Red Hill Fresh:**
```
Tone: Friendly, authentic, local
Voice: "Fresh from our local farms..."
Style: Down-to-earth, community
Avoid: Corporate speak, generic
```

## Phase 3: SEO Optimization

### 3.1 Keyword Optimization Checklist

```
Primary Keyword Placement:
[ ] Title tag (near beginning)
[ ] Meta description
[ ] URL slug
[ ] H1 heading
[ ] First paragraph (first 100 words)
[ ] At least one H2
[ ] Body content (2-4 natural mentions)
[ ] Image alt text (at least one)
[ ] Conclusion

Secondary Keywords:
[ ] H2 headings (natural fit)
[ ] Body content (1-2 mentions each)
[ ] Image alt text variations

LSI/Related Terms:
[ ] Sprinkled throughout naturally
[ ] FAQ section
[ ] Subheadings where relevant
```

### 3.2 Meta Content Writing

**Title Tag Formula:**
```
[Primary Keyword] - [Benefit/Modifier] | [Brand]

Examples:
Organic Spirulina Powder - Benefits & How to Use | BOO
Best Medicinal Mushrooms for Immunity | Teelixir
Wholesale Organic Products Australia | Elevate
```

**Meta Description Formula:**
```
[Action verb] [primary keyword] [benefit].
[Trust signal]. [CTA]

Examples:
Discover the benefits of organic spirulina powder
for energy and nutrition. Australian certified organic.
Shop online with free shipping over $99.
```

### 3.3 Internal Linking

**Link Placement:**
- Natural context within sentences
- 2-3 links per 1,000 words
- Vary anchor text
- Link to:
  - Related blog posts
  - Product pages (relevant)
  - Category pages
  - Pillar content

**Anchor Text Guidelines:**
- Descriptive of destination
- Include target keywords when natural
- Avoid: "click here", "read more", "this article"
- Vary anchors linking to same page

## Phase 4: Quality Assurance

### 4.1 Content Quality Checklist

```
Accuracy:
[ ] Facts verified
[ ] Statistics sourced
[ ] Claims substantiated
[ ] Links working

Readability:
[ ] Grade level 8-10
[ ] Short paragraphs
[ ] Clear headings
[ ] Bullet points used
[ ] Active voice dominant

Grammar & Style:
[ ] Spelling checked
[ ] Grammar checked
[ ] Australian English
[ ] Brand voice consistent
[ ] No jargon (unless explained)

Formatting:
[ ] Proper heading hierarchy
[ ] Consistent formatting
[ ] Images properly placed
[ ] Tables formatted correctly
[ ] Mobile-friendly layout
```

### 4.2 SEO Quality Checklist

```
On-Page SEO:
[ ] Title tag optimized (50-60 chars)
[ ] Meta description optimized (150-160 chars)
[ ] URL is clean and includes keyword
[ ] H1 matches intent, includes keyword
[ ] Keyword density 1-2%
[ ] Internal links present (2-3 per 1K words)
[ ] External links to authority sources (1-2)
[ ] Images have descriptive alt text

Technical:
[ ] No duplicate content issues
[ ] Unique from other site pages
[ ] Schema markup considered
[ ] Page loads quickly
```

### 4.3 Compliance Check

**For Supplements/Health Products:**
```
[ ] No therapeutic claims
[ ] No "cure/treat/prevent" language
[ ] Traditional use framing
[ ] Required disclaimers included
[ ] Certifications accurate
```

**For All Content:**
```
[ ] No competitor defamation
[ ] Accurate pricing/offers
[ ] Correct certification claims
[ ] Honest claims only
```

## Phase 5: Publishing

### 5.1 Pre-Publish Checklist

```
Content:
[ ] Final proofread complete
[ ] All links tested
[ ] Images optimized and uploaded
[ ] Alt text added to all images
[ ] Author/date set

SEO:
[ ] Title tag set
[ ] Meta description set
[ ] URL finalized
[ ] Categories/tags assigned
[ ] Schema markup added (if applicable)

Technical:
[ ] Preview tested on desktop
[ ] Preview tested on mobile
[ ] No broken elements
[ ] Page speed acceptable
```

### 5.2 Post-Publish Actions

**Immediately:**
- [ ] Request indexing in GSC
- [ ] Check live page loads correctly
- [ ] Verify meta tags render properly
- [ ] Test all links work

**Within 24 Hours:**
- [ ] Share on social (if applicable)
- [ ] Internal team notification
- [ ] Add to content tracking
- [ ] Schedule performance review

**Within 7 Days:**
- [ ] Verify indexed in Google
- [ ] Check initial rankings
- [ ] Monitor for any issues
- [ ] Document baseline metrics

## Phase 6: Performance Monitoring

### 6.1 Metrics to Track

| Metric | Source | Check Frequency |
|--------|--------|-----------------|
| Indexing | GSC | Day 7 |
| Rankings | GSC | Weekly |
| Impressions | GSC | Weekly |
| Clicks | GSC | Weekly |
| CTR | GSC | Monthly |
| Sessions | GA4 | Monthly |
| Engagement Time | GA4 | Monthly |
| Conversions | GA4 | Monthly |

### 6.2 Performance Benchmarks

**By Content Type:**
| Type | Expect Ranking | Expect Traffic |
|------|---------------|----------------|
| Quick Answer | 2-4 weeks | 1 month |
| Standard Blog | 1-3 months | 2-3 months |
| In-Depth Guide | 2-4 months | 3-4 months |
| Pillar Content | 3-6 months | 4-6 months |

### 6.3 Optimization Triggers

**When to Update Content:**
- Ranking dropped 10+ positions
- Traffic declined 20%+ MoM
- High impressions, low CTR (<2%)
- Competitor published better content
- Information becomes outdated

**Update Actions:**
1. Re-analyze competing content
2. Identify gaps in current content
3. Expand/improve weak sections
4. Update outdated information
5. Refresh meta tags
6. Add new internal links
7. Republish with updated date
