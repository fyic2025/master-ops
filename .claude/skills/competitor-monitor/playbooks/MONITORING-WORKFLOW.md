# Competitor Monitoring Workflow

## Daily Monitoring

### Quick Checks (15 min)

```
Google Alerts:
[ ] Review competitor name alerts
[ ] Check industry news alerts
[ ] Note any urgent items

Social Media Scan:
[ ] Check competitor Instagram
[ ] Review Facebook posts
[ ] Note promotional activity
```

### Action Items

**If major promotion found:**
1. Document promotion details
2. Calculate their discount depth
3. Assess impact on our products
4. Escalate if immediate response needed

**If new product launched:**
1. Record product details
2. Note pricing
3. Identify if we have equivalent
4. Add to weekly report

## Weekly Monitoring

### Price Tracking (1 hour)

**Priority Products Check:**
```
1. Export our top 50 SKUs by revenue
2. Check competitor prices for each
3. Record in tracking spreadsheet:
   - Our price
   - Competitor A price
   - Competitor B price
   - Price difference %
   - Notes
4. Flag any significant changes (>5%)
```

**Promotional Scan:**
```
1. Visit each competitor homepage
2. Check sale/promotion pages
3. Review email if received
4. Document:
   - Promotion type
   - Discount level
   - Products included
   - Duration
   - Promo codes
```

## Monthly Analysis

### Deep Dive Analysis (4 hours)

**Price Position Report:**
```
1. Pull full product catalog
2. Match to competitor products
3. Calculate category averages
4. Determine position by category
5. Trend vs last month
6. Identify opportunities
```

**Traffic Estimation:**
```
Using SimilarWeb/SEMrush:
1. Estimate competitor traffic
2. Compare YoY change
3. Identify traffic sources
4. Note seasonal patterns
```

**SEO Analysis:**
```
1. Keyword ranking comparison
2. Content gap identification
3. New keywords they're targeting
4. Featured snippet ownership
5. Backlink growth
```

## Quarterly Review

- Competitors exited/changed

**Competitive Position:**
- Our relative strength
- Key threats emerged
- Opportunities identified
- Gap analysis update

**Strategy Alignment:**
- Are we differentiated?

- Product gaps to address?
- Marketing adjustments needed?

## Ad-Hoc Monitoring

### New Competitor Alert

**When identified:**
```
1. Basic profile creation
   - Company info
   - Product range
   - Pricing overview
   - Target market

2. Threat assessment
   - Direct competition level
   - Our vulnerable products
   - Their advantages
   - Their weaknesses

3. Response planning
   - Immediate actions
   - Medium-term strategy
   - Monitoring plan
```

### Major Competitor Move

**Response framework:**
```
1. Understand the move
   - What changed?
   - Why (if known)?
   - Duration (temporary/permanent)?

2. Assess impact
   - Products affected
   - Revenue at risk
   - Customer segments

3. Determine response
   - Match/ignore/differentiate
   - Timeline for decision
   - Resource requirements

4. Execute & monitor
   - Implement response
   - Track results
   - Adjust as needed
```

## Tools & Setup

### Google Alerts

**Create alerts for:**
- Each competitor company name
- Key product categories
- Industry terms
- Your own company (reputation)

**Settings:**
- Frequency: As-it-happens or daily
- Sources: News or Everything
- Language: English
- Region: Australia

### Social Monitoring

**Follow competitors on:**
- Instagram
- Facebook
- LinkedIn
- YouTube (if active)

**Use lists/collections:**
- Create private list of competitors
- Check regularly without following publicly

### Price Tracking Setup

**Spreadsheet structure:**
```
Columns:
- SKU
- Product Name
- Our Price
- Competitor A Price
- Competitor B Price
- Competitor C Price
- Our Position (low/comp/high)
- Last Checked
- Notes

Update: Weekly for priority, monthly for full
```
