## ðŸŽ¯ Lead Sourcing from Supabase

### Query to Get Fresh Beauty Leads

```sql
-- Get beauty leads not in completed campaigns
SELECT
  l.*,
  COUNT(e.id) as emails_sent
FROM smartlead_leads l
LEFT JOIN smartlead_emails e ON l.id = e.lead_id
WHERE
  l.custom_fields->>'lead_id' LIKE 'beauty%'
  AND l.status NOT IN ('COMPLETED', 'UNSUBSCRIBED', 'BLOCKED')
  AND (
    l.custom_fields->>'primary_category' IN (
      'Massage therapist',
      'Spa',
      'Wellness center',
      'Hair salon',
      'Beauty salon',
      'Cosmetic clinic',
      'Laser treatment',
      'Makeup artist',
      'Tanning salon'
    )
  )
GROUP BY l.id
HAVING COUNT(e.id) = 0  -- Never been emailed
ORDER BY
  CASE l.custom_fields->>'primary_category'
    WHEN 'Massage therapist' THEN 1
    WHEN 'Spa' THEN 2
    WHEN 'Wellness center' THEN 3
    WHEN 'Hair salon' THEN 4
    WHEN 'Beauty salon' THEN 5
    ELSE 6
  END,
  l.created_at DESC
LIMIT 3950;
```

### Export Format for Smartlead

CSV with columns:
```
email, first_name, last_name, company_name, website, location,
phone_number, custom_field_lead_id, custom_field_primary_category,
custom_field_assigned_category
```
